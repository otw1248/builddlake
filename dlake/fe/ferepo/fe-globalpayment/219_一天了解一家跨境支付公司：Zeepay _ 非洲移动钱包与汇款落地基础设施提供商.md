本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

2014年成立于加纳，法定名称为 Zeepay Ghana Limited，英国关联实体为 Zeepay JV UK Ltd。

**🌍****全球布局**

总部位于加纳阿克拉，在英国伦敦设有办公室。业务覆盖非洲、欧洲、加勒比地区共**20余个** 国家，通过与 MoneyGram 等合作伙伴可送达**150余个** 国家的汇款来源地。

**🛡️****合规资质**

• 金融牌照：持有加纳银行颁发的电子货币发行商（EMI）牌照、英国 FCA 授权支付机构牌照。

• 移动钱包牌照：在赞比亚、科特迪瓦、塞拉利昂、冈比亚、巴巴多斯等多国持有移动钱包运营牌照。

• 卡组织合作：集成 Visa Direct。

**📊****业务覆盖**

支持**20余个** 非洲国家汇款落地，覆盖加纳塞地（GHS）、赞比亚克瓦查（ZMW）、西非法郎（XOF）等非洲本地货币，以及美元、英镑、欧元等汇款来源货币。

**🎯****主要应用场景**

聚焦跨境汇款落地、移动钱包服务、账单支付与话费充值等，为 MoneyGram、Western Union、Ria、Remitly、Small World、TapTap Send 等**40余家** 国际汇款机构提供最终交付环节能力。

**📌****说明：** Zeepay 定位为 B2B 汇款落地基础设施提供商，而非直接面向汇款发送方的消费者平台。

## **二、服务定价概述**

**📋****服务项目清单**

汇款落地服务：为汇款机构、银行、支付服务商等合作伙伴提供汇款落地能力，支持移动钱包、银行账户、现金代理网点、Visa Direct 等多种收款方式。

移动钱包服务：面向加纳等市场个人用户，提供存取款、转账、账单支付、话费充值等功能。

API 集成服务：提供 Instntmny Remittance、Instntmny Local、Instntmny Bill Pay 等 API 产品，供合作伙伴技术集成。

**👤****消费者端费用说明**

• 钱包开户与维护：免费

• 接收海外汇款：免费

• MoneyGram 现金取款兑换至钱包：免费

• 现金领取：免费

**🏢****B2B 定价模式**

面向汇款机构、银行等机构客户采用定制化报价模式，具体费率取决于合作规模、汇款通道、交易量等因素。

## **三、全球化能力与合规体系**

###  （一）支付网络、区域布局与局限

**📤****汇款落地网络**

覆盖**20余个** 非洲国家的汇款落地能力，收款方式包括：移动钱包、银行账户、ATM 取款、现金代理网点、Visa Direct 借记卡/预付卡。

**📱****移动钱包覆盖**

在加纳、赞比亚、科特迪瓦、塞拉利昂、冈比亚、巴巴多斯**6国** 持有移动钱包运营牌照。支持跨运营商兼容转账。

**💳****卡组织集成**

与 Visa 合作集成 Visa Direct，支持汇款直接入账至加纳**200万余张** Visa 借记卡/预付卡持卡人账户。

**📍****区域运营重点**

• 西非：加纳（总部）、科特迪瓦、塞拉利昂、冈比亚、尼日利亚（落地）

• 东非/南非：赞比亚

• 加勒比：巴巴多斯、圭亚那

• 欧洲：英国

**💱****多货币能力**

支持多种非洲本地货币及主流汇款来源货币，汇款接收时自动转换为本地货币结算。

**📌****说明：** Zeepay 专注于非洲及加勒比市场的汇款落地，在亚太、拉美、中东等区域无牌照覆盖和本地支付能力。

### （二）安全体系

**🛡️****反洗钱能力**

与以色列ThetaRay 合作，采用 AI 驱动的 SONAR SaaS 反洗钱交易监控技术，据称可减少**99%** 误报率。

**👤****KYC 分层制度**

执行加纳银行要求的分层KYC制度，不同认证等级对应不同的交易限额。

**🔐****数据保护**

隐私政策覆盖 GDPR 主要要素，包含数据控制者身份、DPO 联系方式、数据处理目的、第三方传输说明等；数据主体权利部分覆盖访问、更正、删除、限制、反对权利。

### （三）生态合作网络

**🤝****汇款机构合作**

与**40余家** 国际汇款机构建立汇款落地合作，包括 MoneyGram、Western Union、Ria、Remitly、Small World、TapTap Send 等。

**🏦****银行合作**

与 Ecobank、Fidelity Bank、Absa、Access Bank、Zenith Bank、GCB Bank 等加纳及非洲主要银行合作。

**🔧****技术合作伙伴**

• ThetaRay：AI 反洗钱交易监控

• Intermex：美国至非洲汇款通道战略合作（2025年）

**🛒****电商集成**

提供 WooCommerce 官方支付插件，支持 USSD 和移动钱包支付接入；另有 Instntmny Business API 面向中小商户提供支付处理能力。

## **四、技术能力与结算效率**

###  （一）技术架构与开发者生态

**⚙️****API 设计**

采用 RESTful API 架构。提供四类 API 产品：Instntmny Remittance（跨境汇款）、Instntmny Local（国内转账）、Instntmny Bill Pay（账单支付）、Instntmny Business（商户支付）。API 采用 Bearer Token 认证。

**🧪****测试环境**

提供独立沙盒环境供开发测试。

**📚****文档平台**

采用 Mintlify 文档平台，完全公开访问，包含 API 介绍、Quickstart、认证说明、端点参考等板块。

### （二）结算效率

**⏱️****到账时效**

移动钱包落地支持实时到账，银行账户落地时效取决于本地清算网络，现金代理网点可即时取款。

## **五、区域合规与运营要点**

###  （一）加纳（核心市场）

**🛡️****市场进入与合规要点**

• 持有加纳银行 EMI 牌照，2024年5月获得出境汇款许可。

• 需遵循《支付系统与服务法案》（2019）消费者保护要求。

• 客户资金须以信托方式持有。

**🏢****运营注意事项**

• 电子交易税（E-Levy）已于2025年4月废除。

• KYC 分层制度影响交易限额。

### （二）英国

**🛡️****市场进入与合规要点**

• 持有 FCA 授权支付机构牌照。

• 需遵循英国反洗钱法规要求。

• 需遵守客户资金隔离要求。

### （三）其他非洲市场

**🇿🇲****赞比亚**

2021年收购本土移动货币运营商 Mangwee 进入市场；2023年获得赞比亚银行批准开展跨境出入境汇款业务。

**🇨🇮****西非法语区（科特迪瓦等）**

持有移动钱包牌照，受西非国家中央银行监管。

**🇬🇲****冈比亚**

获得冈比亚中央银行移动钱包牌照批准。

## **六、增值服务与附加产品**

###  （一）ZeeMicro 微型保险

**🛡️****产品概述**

面向低收入人群和汇款接收者的30天期微型保险产品，提供身故、重大疾病、住院补贴、意外残疾等基础保障。

### （二）账单支付服务

**📱****Instntmny Bill Pay API**

支持话费充值、流量包、电费、水费、DSTV、StarTimes、Netflix/Amazon Token 等账单支付。

### （三）其他增值服务

• International Airtime：为**100余家** 国际运营商购买话费。

• Mortgage Remit：海外侨民贡献房贷还款。

• Pension Collections：通过钱包收取养老金。

## **七、适配场景与使用考量**

**✅****适合使用 Zeepay 的场景****🌍****国际汇款机构寻求非洲落地伙伴**

如果您是汇款机构，需要在加纳、赞比亚、科特迪瓦等非洲市场建立汇款落地能力，Zeepay 提供移动钱包、银行账户、现金代理等多渠道交付。

**🏦****银行/支付服务商拓展非洲汇款通道**

如果您是银行或支付服务商，需要接入非洲移动钱包网络完成跨境转账落地，Zeepay 提供 API 集成方案。

**👨‍👩‍👧‍👦****非洲侨民汇款接收**

如果您是在英国、美国等地的非洲侨民，可通过合作汇款机构发送汇款，资金直接入账至家人持有的 Zeepay 钱包或银行账户。

**📱****加纳本地移动支付用户**

如果您是加纳本地用户，需要 P2P 转账、账单支付、话费充值等服务，Zeepay 钱包支持跨运营商兼容和 NFC 感应支付功能。

**💡****决策建议**

在选择非洲汇款落地服务商时，建议从以下维度进行综合评估：

**🎯****业务场景匹配度**

根据您的目标市场和收款方式需求，对比多家服务商的覆盖能力。如果聚焦西非市场，可同时了解其他服务商方案。

**🔧****技术集成能力**

评估您的团队是否具备 HTTP API 调用能力。Zeepay 主要提供 API 集成，对技术团队有一定要求。

**💰****定价透明度**

Zeepay 采用 B2B 定制化报价模式，在签约前务必明确费率、结算周期、争议处理等关键条款。
