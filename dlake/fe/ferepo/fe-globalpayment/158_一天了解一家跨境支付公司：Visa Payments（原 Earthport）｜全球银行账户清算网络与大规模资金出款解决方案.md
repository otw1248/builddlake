# 一天了解一家跨境支付公司：Visa Payments（原 Earthport）｜全球银行账户清算网络与大规模资金出款解决方案

本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业基本档案（2025 版）**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体：**

1997 年成立（原名 Electronic Fundraising Company PLC）；2019 年被 Visa Inc. 收购，2020 年更名为 Visa Payments Limited。

**🌍****全球布局：**

总部位于英国伦敦；依托 Visa 集团在纽约、迈阿密、迪拜、新加坡等地的区域办公室与创新中心。

**🛡️****合规资质：**

英国 FCA EMI 牌照；依托 Visa 集团全球牌照网络；PCI DSS Level 1 等安全认证。

**📊****业务覆盖：**

支持 **190+** 国家/地区、**160+** 货币；连接 **60+** 国家本地 ACH 网络；整合 Visa Direct 后可触达 **110 亿** 端点（企业披露）。

**🎯****行业渗透与应用场景：**

聚焦 B2B 跨境支付基础设施，主要服务市场平台出款（Marketplace Payouts）、大规模资金分发（Mass Payouts）、跨境贸易结算与订阅管理；为银行、金融机构、汇款服务商提供底层网络。

## **二、全球化能力图谱**

###  （一）支付网络与区域布局

**💳****国际卡收单：**

非面向消费者的卡交易处理；业务核心为账户到账户（A2A）资金流转，专注银行间支付路径构建。

🔗本地支付聚合：

核心能力为各国 ACH 网络深度连接，而非区域性电子钱包/现金票据等本地方式聚合。

**🔄****跨境清算路径：**

  * **ACH 网络** ：运营全球最大独立 ACH 网络之一，支持 60+ 国家本地清算；规避传统 SWIFT 中间行延迟与费用。

  * **实时支付网络：基于 Visa Direct 整合，触达 10+ 实时支付网络与 70+ ACH 网络；实现银行账户与卡网络双重覆盖。**


**📍****区域枢纽：**

  * 🇬🇧欧洲：伦敦承载 FCA 监管与 EMI 合规；

  * 🇺🇸北美：依托 Visa 美国实体承接 MSB 合规；

  * 🌏亚太：新加坡、香港等地通过 Visa 集团本地子公司运营。


**📞****服务与语言：**

主要服务大型企业与金融机构；定制化协商模式；依托 Visa 集团 30+ 语言客服体系。

### （二）合规与安全体系

**🛡️****牌照与认证：**

FCA EMI 为核心监管框架；其他区域通过 Visa 集团合作伙伴或本地牌照；遵循 PCI DSS、GDPR/CCPA 等标准。

**👤****KYC/AML：**

受 FCA 严格监管，履行反洗钱与客户尽调义务；整合 Visa 全球制裁筛查与监控体系。

**💰****客户资金管理：**

遵循 FCA 客户资金隔离（Safeguarding）要求；通过 segregation method 将客户资金与公司自有资金分离存放；日常对账确保资金安全（监管披露）。

**⚖️****合规运营：**

FCA 持续监督与审计；安全标准统一至 Visa 集团体系；年度合规报告与第三方极计。

## **三、技术能力清单**

###  （一）智能技术与架构

**☁️****云原生与多活：**

RESTful API 架构，AWS 云基础设施整合；通过 Visa Cloud Connect 提供安全连接；利用 Visa 集团全球云部署与灾备。

**🛡️****风控引擎：**

完全整合至 Visa 全球 AI 平台；规则＋机器学习双栈；基于 269 亿 VisaNet 交易数据实时评分，1 秒内完成风险分析；全球网络效应进行欺诈识别。

**📊****数据与洞察：**

基础 API 数据拉取，支持 JSON 格式；定位为管道型基础设施提供商，非精细化数据分析平台。

**🔧****开发者生态：**

提供 API 文档、沙盒环境、Webhook 机制；专注企业级集成；Visa Acceptance Solutions 提供 WooCommerce 插件；服务具备自主开发能力的大型企业。

### （二）支付与结算效率

**⏱️****资金结算时效：**

核心优势为 Visa Direct 分钟级出款（90% 交易 1 分钟内完成，基于 2024 年测试数据）；传统 ACH 网络维持 T+1 至 T+3 标准。

**💵****资金管理技术：**

多层级账户结构（Contracting Merchant/Managed Merchant/User/Beneficiary 四层级）；支持多币种余额与多业务线资金隔离。

## **四、计费结构与服务规则**

**🌐****跨境收款：**

定制化定价模式；基于交易量、业务模式、目标国家等因素商务谈判确定；为大型客户提供议价空间。

**📤****提现与对极结算：**

费用包含在整体协议中；价值主张为通过优化清算路径（ACH/实时网络）降低成本与提升效率，非单笔提现费用竞争。

**✨****增值服务：**

  * **订阅管理** ：Visa Subscription Manager（2024 年 4 月推出，特定地区试点）；

  * **争议解决** ：Rapid Dispute Resolution（RDR），自动化处理符合条件的退款申请；

  * **其他服务** ：定价与条款可能独立于核心支付服务。


**📝****费用项目与条款补充：**

B2B 定制化服务性质，账户管理、合规支持、技术对接费用打包在整体方案；根据 Visa 定价框架，包含一次性设置费用与基于使用量的月度最低费用；最终以《商户协议》为准。

**📌****小结：**

本章聚焦定制化企业服务计费，不含具体数值，以商务谈判结果为准。

## **五、汇率管理与资金流转**

**📊****汇率基准与数据源：**

2020 年剥离独立外汇业务后，不再作为核心利润中心；汇率机制融入 Visa 全球体系，参考多家市场数据源、外汇做市商定价与央行指标；Visa 提供固定汇率并维持约 24 小时。

**🔄****换汇与对冲工具：**

客户需自行或通过第三方管理汇率风险；公司专注资金流转执行通道。

**📤****资金账户与调度：**

  * 多层级账户结构（四层级），满足多实体、多项目资金隔离；

  * 核心调度能力为路径选择，可选 ACH 或 Visa Direct 实时网络出款；

  * 商户侧进行路径选择决策，非智能路由自动化。


**⏱️到账与清算时效：**

通过 Visa Direct 实现分钟级出款时效（90% 交易 1 分钟内完成）；对市场平台、零工经济高频实时付款场景至关重要。

**📌****小结：**

强化中立、高效资金流转基础设施定位，将可预测、快速到账作为核心交付价值。

## **六、生态对接与服务能力**

###  （一）核心业务平台集成

**📦****平台回款与大规模出款：**

最适配场景；通过 Visa Direct 为电商 Marketplace、内容平台等提供向全球供应商、卖家、创作者大规模高频资金分发；Visa Direct Connect 支持大规模出款，专注执行层面。

**🛒****独立站插件：**

Visa Acceptance Solutions 提供 WooCommerce 插件支持。

### （二）企业系统（ERP/财税）集成

**🔌****ERP/财务：** <极>NetSuite 通过 Card Payout极 SuiteApp 实现 Visa Direct 集成；客户依赖 RESTful API 定制开发，实现交易数据与后台系统同步。

  


**📋****对账与凭证：**

通过 API 拉取 JSON 格式交易数据。

### （三）服务交付与支持

**🔄****实施流程：**

大型企业服务商；专属解决方案架构师团队项目制支持。

**💬****支持体系：**

  * Visa 平台系统状态页透明运行状态与维护通知；

  * Visa 集团 30+ 语言客服；

  * 企业客户支持质量与响应速度通过商务协议约定。




## **七、增值工具与行业方案**

###  （一）风控套件（Visa Risk Manager）

**📡****数据信号：**

完全嵌入 Visa 全球风险管理体系；商户获得 Visa 级别欺诈防护网络；基于 269 亿全球交易数据。

**🧠****策略模型：**

Visa Risk Manager 极供规则定制能力，包括交易参数设置、阻止/排除规则、账户管理；配置在 Visa 框架内进行。

**🎯****适用场景：**

对标准化业务具有优势。

### （二）财资与资金管理（FX Manager）

**💧****账户与结构：**

全球化多币种账户体系与高效出款网络；构成企业全球财资管理基础设施；解决资金全球流转问题。

### （三）数据与对账

**📊****事件模型与 Webhook：**

基础的、事件驱动 Webhook 与 API 数据接口；允许开发者获取交易极态更新。

### （四）行业方案（可执行要点）

**🎮****游戏/数字娱乐：**

  * **适配组合** ：Visa Direct Gaming Solutions 提供实时出款到借记卡、30 分钟内到账、160+ 货币支持；

  * **能力亮点** ：核心出款能力的场景化应用，非深度定制化行业方案；

  * **上手清单** ：依赖 Visa 集团现有能力整合。


**📦****大规模出款平台：**

  * **适配组合** ：市场平台、零工经济平台、内容平台向全球进行高频资金结算；

  * **能力亮点** ：分钟级出款时效，触达 110 亿端点；

  * **上手清单** ：API 开发能力、Visa Direct 集成测试。




**📌****小结：**

增值工具体现为 Visa 集团能力整合，如 Visa Risk Manager、Visa Direct Gaming Solutions，非独立研发产品矩阵；核心价值主张为稳定、高效底层支付基础设施。

## **八、适配场景与注意事项**

###  （一）适配场景

  * **📦****大规模出款平台** ：市场平台、零工经济平台、内容平台需向全球卖家/创作者/服务商高频快速资金结算。

  * **🌐****跨国 B2B 贸易企业** ：与全球供应商频繁资金往来，通过 Global ACH 网络优化成本与时效的大中型进出口企业；Global ACH 相比 SWIFT 电汇节省 5-7 倍成本（平均费用 5 美元以下 vs 电汇 15-50 美元）。

  * **🏦****金融机构与汇款公司** ：银行、货币服务企业（MSB）等需要可靠底层网络处理跨境汇款和支付业务。

  * **🔄****订阅管理企业** ：运营全球订阅业务，利用 Visa Account Updater（VAU）与 Visa Subscription Manager 提升续订成功率；VAU 自动更新过期/变更卡片信息。




### （二）注意事项

  * ⚖️**合规** ：核心监管合规在英国 FCA 框架；Visa Direct 全球严格合规要求包括 KYC/AML/制裁筛查；其他区域合规路径需确认依赖 Visa 集团牌照还是本地合作；合同主体极清算主体法律关系需明确。

  * 🔧**技术** ：接入需具备 API 开发与集成能力。

  * 💰**资金** ：出款速度快，商户自行管理汇率风险；Visa 提供固定汇率维持约 24 小时；协议需明确不同清算路径的费用、时效及节假日影响。

  * 📊**运营** ：为商户提供专业支付基础设施；Visa Risk Manager 提供配置能力；运营效率提升依赖商户自身系统建设。




## **九、区域合规与适配方案**

###  （一）欧洲

**🛡️****准入与合规要点：**

FCA EMI 监管要求，包括客户极金隔离、年度审计、资本充足率；PSD2/SCA 与 GDPR 依赖 Visa 集团解决方案；强身份验证与数据保护机制。

**⚙️****运营/风控重点：**

商户通过 Visa 合规工具包配置。

**📋****上线清单：**

EMI 合规资料、PSD2/SCA 策略配置、GDPR 数据处理协议。

### （二）北美及其他地区

**🛡️****准入与合规要点：**

通过 Visa 集团美国注册实体与合作伙伴网络；依赖 Visa 对州级 MSB、FinCEN 注册、OFAC 制裁筛查、CCPA 数据隐私的整体框架。

**⚙️****运营/风控重点：**

商务谈判中就合规责任、牌照依托、当地税务（销售税/GST）、数据隐私法（CCPA/LGPD）详细尽调。

**📋****上线清单极**

主体资料、合规框架确认、税务与隐私法适配。

### （三）亚太市场

**🛡️****准入与合规要点：**

通过 Visa 集团本地子公司与牌照持有实体；遵循当地 PSP 或 E-Money 法规；新加坡 MAS、澳大利亚 APRA、香港 SFC 等监管。

**⚙️****运营/风控重点：**

依托 Visa 集团当地实体与牌照。

**📋****上线清单：**

当地监管要求确认、Visa 集团本地实体合规路径。

**📌****小结：**

合规优势集中在英国 FCA 框架，全球其他区域更多依托 Visa 集团当地实体与牌照；具体落地细节需商务谈判与法律尽调阶段确认。
