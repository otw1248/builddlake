本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

2011 年成立，公司法定全称为宝付网络科技（上海）有限公司。

**🌍****全球布局**

总部位于中国上海，并在北京、深圳、杭州等多个城市设有分支机构，业务网络覆盖全国。

**🛡️****合规资质**

支付牌照： 中国人民银行颁发的《支付业务许可证》（互联网支付-全国）。

外汇牌照： 获得国家外汇管理局上海分局颁发的跨境外汇支付业务资质，支持货物贸易、航旅、留学教育等多种业务类型。

卡组织资质： 获批具备 VISA 卡境内收单 QSP 资质。

安全认证： 通过国家权威机构的个人金融信息安全认证与信息系统安全等级保护测评；已获得ISO 27001信息安全管理体系认证；官网声明已通过PCI-DSS支付卡行业数据安全标准认证。

**📊****业务覆盖**

业务核心在中国市场，为企业级商户提供收单、结算、跨境支付与行业化解决方案。跨境业务主要服务于 Amazon、eBay、Etsy 等平台的中国卖家及独立站卖家，支持近 **10 个** 主流币种的收付款。根据易观等第三方机构报告，宝付支付在中国第三方互联网支付市场份额稳居行业前五，年交易规模达到**万亿级别** 。

**⚠️****说明：**

因场景与合作方的定义、统计范围存在差异，具体覆盖范围以签约时点及合同清单为准。

**🎯****主要应用场景**

聚焦消费金融、保险、物流、B2B/B2C 电商、航旅、共享出行等行业，提供网银支付、协议支付、聚合支付、实时分账、代收代付、跨境收付款等综合支付与金融科技服务。

## **二、计费结构与服务规则**

**📋****服务项目清单****💳****收单服务**

按支付方式划分，覆盖个人网银支付（借记卡/贷记卡）、企业网银支付、聚合支付（微信支付/支付宝）、协议支付/认证支付等。

**🏦****结算服务**

代付： 按笔收费，用于商户向对公或对私银行账户的付款，如理赔、佣金发放等。

提现： 将商户在宝付账户的资金结算至其指定的银行账户。

**🌐****跨境服务**

包含跨境收款、货币转换、提现等环节的综合计费。

**✨****增值服务**

实时分账： 平台型商户的分账处理服务。

行业解决方案： 针对特定行业的定制化支付方案，可能包含账户体系、资金存管等模块。

合规支持： KYC/KYB 审核、交易监控等基础合规服务。

**📝****费用项目补充说明**

定价模式为非公开的定制化模式，具体费率取决于商户的行业类型、交易规模、支付方式组合、风险等级等多种因素。可能包含交易手续费、代付服务费等项目。

**📌****小结：**

本章聚焦收费项目与适用规则，不含数值，最终以《商户协议》为准。

## **三、全球化能力图谱**

###  （一）支付网络与区域布局

**💳****卡组织收单**

国际卡： 具备 VISA QSP 资质，可处理 VISA 国际卡交易。

本地卡： 深度整合中国银联网络，支持银联云闪付、银联在线支付及各类银联卡交易。

**🔗****本地支付聚合**

支付网络主要聚焦中国市场，深度整合主流支付方式：

数字钱包： 支付宝、微信支付。

银行转账： 全面覆盖中国工商银行、中国建设银行、招商银行等国内主要商业银行及大部分全国性股份制银行的个人及对公网银支付。

**🔄****跨境清算路径**

依托其跨境外汇支付资质，为中国跨境电商卖家提供从海外平台到境内的收款服务。支持美元（USD）、欧元（EUR）、英镑（GBP）、日元（JPY）等近 **10 个** 币种的接收，并通过合作银行渠道完成结汇与人民币入境。

**📍****区域运营中心**

运营中心与服务团队均位于中国境内，包括上海总部以及北京、深圳、杭州等分支机构，为全国范围内的商户提供本地化服务。

### （二）合规与安全体系

**🏛️****牌照与认证**

通过国家金融科技认证中心的个人金融信息安全认证、上海市信息安全测评认证中心的信息系统安全等级保护测评。

官网声明通过 ISO 27001 信息安全管理体系认证及 PCI-DSS 支付卡行业数据安全标准。

**👤****反洗钱能力**

遵循中国人民银行《支付机构反洗钱和反恐怖融资管理办法》等监管要求，建立客户身份识别（KYC）、交易监控与可疑交易报告机制，并按规定保存客户身份资料与交易记录至少 **5年** 。

**💰****客户资金管理**

作为持牌支付机构，严格遵守中国人民银行关于客户备付金的集中存管规定，确保商户资金与公司自有资金的隔离，保障资金安全。

**⚖️****合规运营**

商户审核： 建立了严格的商户入网审核标准与资质定期复查机制。

数据合规： 遵循中国《网络安全法》、《数据安全法》、《个人信息保护法》等法律法规，将境内收集的用户个人信息存储于中国境内。

## **四、技术能力与结算效率**

###  （一）智能技术与架构

**⚙️****技术架构**

提供 RESTful 风格的 API，支持商户通过接口进行系统对接。

**🛡️****风控引擎**

内置智能风控系统，覆盖交易监控、反欺诈、黑名单管理等功能。风控体系能够基于商户行业特点、交易行为特征进行风险识别与处置，如延迟结算、关停交易等。

**🔧****开发者生态**

API 与文档： 提供公开的 API 参考文档与集成指南，覆盖支付、退款、代付、分账、查询等核心功能。

SDK 与示例： 提供 JAVA、PHP、.NET 等语言的 DEMO 示例代码包，帮助开发者快速集成。

Webhook： 支持支付成功、退款、分账、投诉等关键业务事件的异步通知，并提供渐进式重试机制，确保通知的可靠性。

测试环境： 提供独立的沙盒测试环境及 API 端点，便于商户在上线前进行开发与调试。

### （二）支付与结算效率

**⏱️****到账时效分级**

跨境收款： 针对跨境电商平台回款，最快可实现 **T+0** 到账，加速卖家资金周转。

境内结算： 提供 **7x24小时** 、全年无休的结算与代付服务，支持商户在节假日进行资金划拨。

**💵****资金管理**

商户后台： 提供功能完善的商户管理后台，支持交易管理、订单查询、财务对账、报表下载等功能。

多币种账户： 方便商户在全球范围内开展业务。

## **五、增值工具与行业方案**

###  （一）风控套件

**📡****数据信号与策略**

风控体系整合了身份认证、交易监控、行为分析等多维度数据。商户入网时需经过严格的资质审核，交易过程中系统会进行实时监控，并根据预设的行业化风控规则进行风险识别与处置。

**⚙️****商户可配置项**

商户可在其业务流程中，通过 API 调用身份认证（如银行卡要素验证）等工具，加强前端风险防范。

### （二）行业方案

宝付的核心竞争力体现在为不同垂直行业提供的深度定制化解决方案。

**🏦****消费金融**

适配组合： 协议支付/代收（用于自动还款）、大额网银支付（用于放款）、实时代付（用于结算）。

能力亮点： 提供覆盖开户、放款、还款、分账的全流程支付方案，提供大额支付通道支持，可满足消费金融行业的高额资金收付需求，并提供"同卡进出"方案以符合监管要求。

**💡****应用场景：**

一家汽车金融公司，可使用宝付的协议支付功能与用户签订自动还款协议，每月自动从用户绑定的银行卡扣除车贷月供。当需要向合作的汽车经销商支付购车款时，可通过大额代付接口完成。

**🛡️****保险行业**

适配组合： 代收（保费收取）、代付（理赔款支付）、实时分账（佣金分配）。

能力亮点： 渠道覆盖国内主流银行，支持信用卡扣款；灵活的分账和对账体系，能自动处理复杂的佣金结构；理赔款可快速代付至被保人账户。

**🚚****物流行业**

适配组合： 账户体系、代收代付、账务清分。

能力亮点： 提供统一的账户系统管理货款、运费等资金，支持货款的智能代收付，并能按需进行实时账务清分，提升物流平台与司机、货主之间的结算效率。

**🛍️****平台电商/共享经济**

适配组合： 聚合支付（用户付款）、实时分账。

能力亮点： 提供完整的实时分账解决方案，支持每笔订单实时或延时分账，可将资金按规则分配给平台及多个入驻商户或个人，有效解决平台型业务的"二清"（即二次清算，指未获得支付牌照的平台进行的资金结算行为）合规问题。

合规要点： 实时分账功能是平台型业务满足中国支付清算领域合规要求的关键工具。

**💡****应用场景：**

一个本地生活服务平台，用户通过微信支付购买了一张价值 100元的餐饮券。交易成功后，宝付的实时分账系统可根据平台预设的 90/10 规则，立即将 90元划拨至餐饮商户的子账户，10元划拨至平台的佣金账户。

## **六、适配场景分析**

**✅****适合使用宝付支付的场景****🏭****深耕特定行业的企业**

如果您是消费金融、保险、物流、平台电商等行业的企业，业务流程对支付的合规性、资金处理效率和定制化有特殊要求，宝付的行业解决方案能提供比标准化支付网关更贴合的服务。

**🏪****以中国市场为主的平台型企业**

如果您运营的是多商户入驻的电商平台、共享出行或本地生活服务平台，需要处理复杂的资金分账以满足合规要求，宝付的实时分账产品是核心适用工具。

**💰****有大额收付需求的企业**

如果您的业务涉及单笔金额较高（如数十万人民币）的收付款场景，宝付提供的高额度支付通道能够满足需求。

**🌐****服务中国的跨境电商卖家**

如果您是亚马逊、eBay 等平台的中国卖家，需要将海外销售款项便捷、快速地结算为人民币并提现至国内，宝付的跨境收款服务提供了 T+0 的到账时效。

**🤔****决策建议**

在选择支付服务商时，建议从以下维度进行综合评估：

**🌍****业务核心市场**

如果您的核心目标市场在中国，且业务模式复杂，宝付的行业方案值得深入了解。如果您的业务主要面向全球消费者，需要广泛覆盖海外本地支付方式，则应同时评估 Stripe、Adyen 等全球化支付平台。

**🎯****行业匹配度**

评估服务商是否对您所在的行业有深刻的理解和成熟的解决方案。特定行业的支付需求往往超越了基础的收单，涉及账户体系、合规、资金管理等多个层面。

**🔧****技术与服务模式**

评估服务商是提供标准化的产品，还是提供顾问式的定制化服务。宝付的模式更偏向后者，通过客户经理提供一对一服务，适合需要深度合作的企业。

**⚖️****合规与牌照**

确保服务商在您的核心市场持有必要的监管牌照。对于在中国运营的业务，持有人民银行颁发的支付牌照是基础门槛。
