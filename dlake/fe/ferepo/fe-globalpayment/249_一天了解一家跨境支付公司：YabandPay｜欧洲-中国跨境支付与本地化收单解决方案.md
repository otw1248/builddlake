本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

2022年注册成立，总部位于荷兰海牙里斯韦克（Rijswijk），法定名称为 Yabandpay B.V.。

**🌍****全球布局**

荷兰为全球运营中心，在德国杜塞尔多夫、中国广州、立陶宛维尔纽斯设有办公室。北欧市场通过与 YabandPay Nordic 的战略合作覆盖冰岛。

**🛡️****合规资质**

  * 金融牌照：荷兰 DNB 支付机构许可；爱沙尼亚 FSA 跨境服务注册

  * 卡组织资质：Visa、Mastercard 商户收单服务资质（PF）；iDEAL 官方收单支付服务商（CPSP）；银联国际欧洲区域收单机构；微信支付与支付宝+ 欧洲区域收单机构

  * 安全认证：PCI DSS Level 1

  * 合规框架：EU PSD2、Dutch Wwft（荷兰反洗钱法）、Dutch Wft（荷兰金融监管法）、GDPR


**💰****客户资金保护**

设立独立基金会 Stichting Derdengelden YabandPay，受 DNB 直接监管，商户资金完全隔离，不得用于 YabandPay 运营，符合 PSD2 客户资金隔离保护要求。

**📊****业务覆盖**

支持多种支付方式，涵盖国际卡组织、中国支付网络、欧洲本地支付方式等。交易币种覆盖多种主要货币，结算币种提供 11 种选择。

**🎯****主要应用场景**

主要服务欧洲商户，使其能够接收来自中国游客和消费者的付款。覆盖行业包括旅游酒店、零售、交通出行、餐饮等。服务的客户包括 Schiphol 机场、Brussels Airport、LVMH、Sixt、Rail Europe 等企业。

## **二、服务项目与计费规则**

###  服务项目清单

**💳****在线支付**

覆盖消费卡、商务卡、银联、微信支付、支付宝+、欧洲本地支付、SEPA、Klarna、PayPal 等。

**🏪****POS（线下）支付**

覆盖借记卡、信用卡、Visa、Mastercard、商务卡、微信支付、支付宝+、银联 QuickPass 等。

**🇪🇺****本地支付**

荷兰、比利时、卢森堡、波兰、西班牙、奥地利、北欧等区域提供专属计费规则。

**🏦****结算服务**

  * 标准结算（T+1）：达到门槛金额免费，低于门槛收取手续费（具体以协议约定为准）

  * 结算周期：每日、每周、每月可选

  * 结算货币：EUR、GBP、CHF、USD 等 11 种


**📱****POS 设备**

  * 租赁方案：按月计费

  * 购买方案：一次性付费

  * 附加费用：月度连接费、4G 数据费（如适用）


**✨****增值服务**

  * 退款处理：原交易费不退还

  * 拒付管理：按协议约定收取管理费


**📋****计费规则**

按交易类型、卡类型、支付方式、地区等因素收费，企业客户可获定制费率与专属经理。

**📝****费用项目补充说明**

可能包含延迟付款管理费、违约罚款、货币转换费等项目，具体以协议约定为准。

**📌****小结：** 本章聚焦收费项目与适用规则，不含数值，最终以《商户协议》为准。

## **三、支付网络与合规体系**

###  （一）支付网络与区域布局

**💳****卡组织收单**

持有 Visa、Mastercard 商户收单服务资质（PF），为商户提供卡支付收单服务；银联国际欧洲区域收单机构，支持闪付、二维码、应用内支付、H5 等模式。

**🇨🇳****中国支付网络**

整合微信支付、支付宝+、银联三大中国主流支付网络，支持二维码、应用内支付、H5、小程序等多种接入模式。支付宝+（Alipay+）整合多个亚太及全球区域合作钱包，覆盖中国大陆、香港、韩国、东南亚等主要市场。

**🇪🇺****欧洲本地支付**

覆盖欧洲主要本地支付方式：iDEAL（荷兰）、Bancontact（比利时）、Payconiq（比利时、卢森堡）、BLIK（波兰）、Bizum（西班牙）、Trustly（北欧）、EPS（奥地利）、SEPA 直接借记、贷记转账（欧盟）。

**🔄****先买后付（Buy Now Pay Later）**

Klarna（立即付款、稍后付款）

**📱****移动支付**

Apple Pay、Google Pay

**💱****多货币能力**

支持欧洲、北美、亚太、大洋洲等主要币种交易，API 提供实时汇率查询与转换。

### （二）合规与安全体系

**🛡️****反洗钱能力**

遵循 Dutch Wwft 反洗钱法规，提供制裁名单筛查、负面舆情检查、欺诈检测、黑白名单管理、风险评分、可疑交易报告等能力。

**🔒****数据保护**

遵循 GDPR，历史交易保留 7 年（法律要求）。

**⚖️****区域合规要点**

欧洲市场为 YabandPay 核心覆盖区域，商户接入时需关注以下合规要点：

  * PSD2 强客户认证（SCA）：在线卡支付需配置 3DS 验证策略

  * GDPR 数据保护：跨境数据传输与处理协议需签署完整

  * 本地支付规则：iDEAL、SEPA 等支付方式有各自的对账与结算规则




## **四、技术能力与结算效率**

###  （一）API 与开发者工具

**⚙️****API 架构**

RESTful 风格接口（v3.5 版本），采用 HTTPS 加密传输、JSON 数据格式、HMAC-SHA256 签名认证。

**🔧****支持的 API 功能**

  * 创建支付（v3.CreatePayments，支持多种支付方式）

  * 快速支付、扫码支付

  * 支付查询、支付捕获

  * 对账报告、汇率查询

  * Webhook 回调通知


**📦****SDK 与插件**

提供 PHP SDK 及主流电商平台插件（WooCommerce、Magento、PrestaShop 等）。

**📚****开发者文档**

包含 API 接口说明、Webhook 事件、测试账号等基本要素。

**🔔****Webhook 回调通知**

支持 paid、authorized、pending、processing、verify、Cancelled、Expired、Declined、Failed 等状态通知，采用 HMAC-SHA256 签名验证。

### （二）支付与结算效率

**⏱️****到账时效**

清算周期为每日荷兰时间凌晨 4 点，结算周期为 T+1。

**🔐****卡支付高级功能**

  * 3D Secure 2.0 三域安全验证（用于在线支付身份核验）：支持完整的状态码

  * 卡片令牌化存储：PCI DSS Level 1 合规

  * 预授权与扣款分离：支持预授权后再捕获扣款




## **五、产品矩阵与服务支持**

###  （一）收款产品

**🛒****在线收单**

通过 API 或电商插件接入，支持网站、移动应用、微信小程序等在线支付场景。

**🏪****POS 终端**

提供移动 POS、台式 POS、无人值守终端等多种设备，支持非接触、插卡、扫码三种方式，可便捷切换，适用于移动收款、收银台、自助终端等场景。

**🔗****支付链接（Pay by Link）**

生成唯一支付链接，通过社交媒体、即时通讯、邮件等渠道发送给客户，客户点击链接即可完成支付，无需商户具备技术开发能力。

### （二）电商平台分账服务

**💼****功能特点**

  * 子商户入驻与 KYC（客户身份识别）、KYB（企业身份识别）

  * 分账（Split Payment）：单 API 支持最多 10 个分账订单，支持多种分账角色配置

  * PSD2 合规的子商户管理

  * 交易监控：制裁筛查、欺诈检测、黑白名单、风险评分




### （三）微信小程序合作

**🤝****合作伙伴**

有赞（Youzan）、AllValue、Cakra、iMenu、微盟（Weimicro）

**✨****功能特点**

可建立在线商店，集成所有 YabandPay 支付方式，无需额外开发成本。

### （四）营销推广支持

**📢****营销推广支持**

YabandPay 与微信支付、支付宝官方建立合作关系，为商户提供营销推广支持，包括：

  * 红包活动：配合微信支付、支付宝官方营销活动

  * 平台曝光：支付宝 App 曝光、微信小程序曝光

  * 物料支持：提供收款码、宣传物料等




### （五）服务交付与支持

**🚀****入驻流程**

  * 审核时间：根据商户类型和资料完整程度而定，子商户通常可在 1 个工作日内完成

  * KYC 要求：营业执照、身份证明、银行账户、业务描述

  * 入驻部署支持：提供技术工程师部署支持


**📞****客户支持**

  * 支持渠道：在线聊天、邮件、电话、微信

  * 企业客户：专属经理、定制费率、优先入驻


**📊****商户后台**

支持交易管理、财务报告、支付设置、POS 管理；提供 iOS、Android 移动端 App。

## **六、增值服务与行业配置**

**🏦****企业融资服务**

YabandPay 提供企业贷款服务（Business Loans），面向有资金周转需求的中小企业（需符合相关资质要求）。

### （二）风控与拒付管理

**🛡️****风控机制**

参数监控、持续监控、3D Secure 验证。

**⚖️****拒付管理**

拒付发生时，商户需承担相应费用，平台将按协议约定流程处理。如业务场景拒付风险较高，建议在签约前与客户经理确认风控能力边界。

## **七、适配场景分析**

###  适合使用 YabandPay 的场景

**🇪🇺****欧洲本地商户接受中国消费者支付**

如果您在欧洲经营零售、餐饮、酒店、景点等业务，客群包含中国游客或留学生，YabandPay 同时持有微信支付、支付宝+、银联欧洲区域收单资质，可统一接入中国消费者常用支付方式。

**🏪****线下零售与旅游场景**

如果您需要在机场、免税店、百货商场、餐厅等线下场景部署收款设备，POS 终端支持多种支付方式，一台设备覆盖欧洲本地卡与中国移动支付。

**📱****微信小程序商城**

如果您希望通过微信小程序吸引中国消费者，可借助与有赞、微盟等 SaaS 服务商的合作快速搭建小程序商城。

**💬****社交电商与轻量收款**

如果您通过社交媒体、即时通讯与客户沟通，支付链接可快速生成收款链接，无需技术开发能力。

**🇳🇱****荷兰、比利时本地支付需求**

如果您的业务主要面向荷兰、比利时消费者，iDEAL、Bancontact 等本地支付方式费率具有竞争力（具体以协议为准），适合本地化运营。

**💼****平台分账需求**

如果您运营电商平台或外卖平台，需要将订单款项自动分配给多个商户或服务商，平台分账功能支持单笔订单自动拆分结算。

### 决策建议

**📋****决策建议**

在选择跨境支付服务商时，建议从以下维度进行综合评估：

**🎯****业务场景匹配程度**

根据您的主要市场、客户支付习惯、交易规模等因素，对比多家服务商的匹配程度。如果您的业务聚焦欧洲-中国跨境场景，YabandPay 持有微信支付、支付宝+、银联欧洲区域收单资质；如果您的业务需要覆盖北美、拉美、中东、非洲等市场，可同时了解 Adyen、Stripe、Checkout.com 等全球性支付服务商。

**🔧****技术实施能力**

评估您的团队是否具备 API 对接能力。如使用 PHP 以外的开发语言或 Shopify 等平台，需基于 API 文档自行开发或寻求第三方集成方案。如技术资源有限，可优先考虑支付链接、电商插件等无需开发的接入方式。

**💰****费用与合同条款**

在签约前，务必与客户经理沟通，明确计费规则、结算周期、拒付处理、数据安全、争议处理等关键条款，选择最符合您业务需求的方案。
