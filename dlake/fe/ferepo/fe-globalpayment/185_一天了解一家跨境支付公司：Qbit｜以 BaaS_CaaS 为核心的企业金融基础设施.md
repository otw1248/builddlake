## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

2019 年成立，同年在美国和中国香港设立公司主体，并在杭州设立运营中心。

**🌍****全球布局**

采用多总部运营模式，在亚洲（香港、杭州、深圳）、北美（加州圣何塞）等地设有核心办公室，并在全球 7 个城市建立本地化服务中心。

**🛡️****合规资质**

金融牌照： 美国 FinCEN MSB、中国香港 MSO、中国香港 TCSP

卡组织资质： Visa 主会员（Principal Member）

行业认证： PCI-DSS 认证（企业声明）

**📊****业务覆盖**

服务覆盖 **180+** 国家和地区，支持在全球范围内进行 **40+** 种货币的收付款。截至 2024 年，已服务超过 **40,000** 家企业客户，累计处理交易金额数十亿美元。

**⚠️****说明：**

因场景与合作方的定义、统计范围存在差异，具体覆盖范围以签约时点及合同清单为准。

**🎯****主要应用场景**

聚焦跨境电商、B2B 贸易、数字营销（MCN）、Fintech 平台等领域，提供以银行即服务（BaaS）和卡即服务（CaaS）为核心的金融基础设施，核心产品包括全球账户、Qbit 量子卡（虚拟/实体卡）、全球收单及 Qbit InstaPay 即时支付。

**📋****基本服务能力概览**

  * 🛒**平台支持** ： Shopify、PayPal 等

  * ⚡**到账速度** ： Qbit InstaPay 支持秒级到账

  * 🌎**主要覆盖市场** ： 北美（MSB 合规）、亚太（香港 MSO/TCSP 合规及本地化团队）

  * 💡**主要服务特点** ： BaaS/CaaS API 解决方案、强大的虚拟卡发行与管理能力、服务商生态 0 费用收款、多币种全球账户

针对生态内账户间的资金划转，提供 0 手续费的即时收款服务。

**🔌****BaaS/CaaS API 服务**

为企业客户提供模块化的金融能力输出，定价模式通常为基于用量和解决方案复杂度的定制化方案。

**✨****增值服务**

**📝****费用项目补充说明**

服务条款中明确，除 Qbit 收取的费用外，部分交易可能产生由收款人银行、平台或支付网络收取的第三方费用。

通过与全球领先银行和金融机构合作，构建支持 40+ 种货币的全球清算网络，支持多币种直接交易，旨在降低传统跨境支付中的汇兑成本。

亚太： 以香港、杭州、深圳为核心，覆盖 MSO 及 TCSP 监管，并提供本地化客户支持。

**🏛️****牌照与认证**

企业声明通过 PCI-DSS 认证。

**👤****反洗钱（AML）能力**

**💰****客户资金管理**

**📋****合规运营**

## **七、区域合规与避坑指南**

###  （一）北美（美国）

**🛡️****准入与合规要点**

MSB 牌照要求企业建立完善的反洗钱（AML）和了解你的客户（KYC）体系。

需遵循《银行保密法》（BSA）和美国爱国者法案，向 FinCEN 报告可疑交易。

严格遵守 OFAC 的制裁合规要求，对交易对手进行筛查。

**🏢****运营与风控重点**

自动化 KYC 和 AML 流程是满足合规要求和提升效率的关键。

交易监控需覆盖身份验证、制裁筛查和风险行为标记等多个环节。

**📋****上线清单**

  * ✅企业注册文件与受益所有人（UBO）信息

  * ✅详尽的业务模式说明

  * ✅AML/KYC 政策文件

  * ✅数据安全与隐私保护评估




### （二）亚太（中国香港）

**🛡️****准入与合规要点**

MSO 牌照受香港海关监管，需严格遵守香港《打击洗钱及恐怖分子资金筹集条例》。

TCSP 牌照要求在提供信托或公司服务时，履行客户尽职调查和记录保存义务。

客户资金需与公司运营资金严格隔离。

**🏢****运营与风控重点**

对所有交易进行尽职调查，并保留完整的交易记录。

建立可疑交易报告机制，及时向监管机构报送。

**📋****上线清单**

  * ✅香港公司注册文件及董事名册

  * ✅KYC/KYB 资料

  * ✅业务合规承诺书

  * ✅数据处理与隐私条款




## **八、增值工具与行业方案**

###  （一）银行即服务（BaaS）与卡即服务（CaaS）

**📦****服务内容**

通过模块化的 API，向企业客户输出底层的金融能力，包括但不限于：

全球账户 API： 集成开户、多币种支付和换汇功能。

批量付款 API： 适用于大额、高频的国际 B2B 支付场景。

卡片管理 API： 提供卡片发行、激活、冻结、消费控制等全生命周期管理。

**💡****应用场景：**

一家 Fintech 公司希望为其企业客户提供费用管理解决方案，但自身没有发卡资质；通过集成 Qbit 的 CaaS API，该公司可以在几周内快速上线自己的品牌虚拟卡，并为其客户提供消费限额、商户类别限制等精细化管理功能，而无需自行申请金融牌照和对接银行系统。

### （二）Qbit 量子卡（企业费用管理）

**📦****服务内容**

提供企业级虚拟卡和实体卡服务，是其 CaaS 能力的直接产品体现。

卡片类型： 支持 Visa 和 Mastercard 网络的虚拟卡与实体卡。

核心功能： 批量开卡、实时交易监控、自定义支出控制（如每日限额、商户类别限制、地理位置围栏）、多用户权限管理。

移动应用： 提供 Qbit Card App，方便员工管理卡片和提交费用凭证。

**💡****应用场景：**

一家跨境电商公司需要在 Google 和 Facebook 等多个海外平台投放广告；通过使用 Qbit 量子卡，公司可以为每个广告平台、每个广告活动创建独立的虚拟卡，并设置精准的预算上限；这不仅能有效控制广告超支风险，还能在某张卡信息泄露时立即冻结，保护主账户资金安全。

## **九、适配场景分析**

**✅****适合使用 Qbit 的场景****🔌****有嵌入式金融需求的企业**

如果您的企业（如 Fintech 平台、SaaS 服务商）希望在自身产品中无缝集成账户、支付或发卡功能，Qbit 的 BaaS/CaaS 解决方案提供了标准化的 API 接口，可以大幅降低开发成本和时间。

**💳****有精细化费用管理需求的企业**

如果您是跨境电商、数字营销或有海外团队的企业，需要对广告费、采购款、员工差旅费等进行严格的预算控制，Qbit 量子卡提供了强大的实时监控和自定义支出策略功能。

**🏢****服务商或平台生态企业**

如果您运营的是 MCN、在线教育平台或任何需要向大量下游服务商、创作者或教师进行高频付款的平台，Qbit InstaPay 提供的 0 费用、即时支付解决方案具有显著的成本和效率优势。

**🤝****B2B 跨境贸易企业**

如果您需要与全球多个供应商进行交易，涉及多币种的批量付款，Qbit 的全球账户和批量付款功能可以简化您的财务操作流程。

**💡****决策建议**

在选择跨境支付服务商时，建议从以下维度进行综合评估：

业务核心需求： 明确您的核心痛点是简单的收款，还是复杂的费用管理与金融功能集成。如果您的需求是后者，Qbit 的 BaaS/CaaS 模型可能更具优势。对于需要广泛本地支付方式以触达 C 端用户的场景，可同时了解 Adyen、Stripe 等方案。

技术实施能力： 评估您的团队是否具备 API 对接能力。如果技术资源有限，可优先使用其开箱即用的全球账户和量子卡管理后台；如果技术能力强，则可以最大化利用其 API 的灵活性。

目标市场合规： Qbit 在美国和香港的合规资质较为完善。如果您的核心业务在欧洲，则需要额外评估其在该区域的牌照覆盖和 GDPR 合规完整性。
