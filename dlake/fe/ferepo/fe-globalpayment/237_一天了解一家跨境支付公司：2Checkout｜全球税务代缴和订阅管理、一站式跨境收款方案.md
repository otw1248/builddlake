## **二、计费结构与服务规则**

**📋****服务项目清单****💳****交易手续费（三档套餐）**

  * **2Sell：** 适用于标准一次性交易，面向需要基础收款能力的商户

  * **2Subscribe：** 适用于订阅/周期性付款业务，包含订阅管理、失败重试、催款管理（Dunning）等能力

  * **2Monetize：** 适用于企业级数字商品销售，包含税务合规自动化、发票管理、MoR 全托管服务


**🌐****跨境交易费**

  * 适用于商户注册国与客户卡片发卡国不同的交易

  * 高风险国家适用更高费率

  * 部分国家可豁免跨境费（具体清单以官方文档为准）


**💱****货币转换费**

  * 当交易货币与结算货币不同时适用

  * 在市场汇率（如欧洲央行汇率）基础上收取价差

  * 价差幅度根据交易情况有所不同


**⚖️****拒付处理费**

  * 根据商户拒付率分级收取

  * 拒付率较低时费用较低，拒付率较高时费用较高

  * 拒付胜诉后费用处理方式以协议为准


**🏦****结算/提现费**

  * 电汇（Wire Transfer）方式收取固定费用

  * 设有最低结算金额要求


**💵****支付方式附加费**

  * PayPal 支付在标准费率基础上收取额外费用

  * 本地化支付方式（如 Boleto、Konbini 等）有单独定价

  * Apple Pay、Google Pay 等数字钱包费率以实际为准


**📊****计费规则**

采用组合费率模式（百分比 + 固定费用），按套餐类型、交易是否跨境、支付方式等因素实行差异化收费。无月费、无设置费，按交易付费。定价策略遵循行业内服务中小型商户的惯例，为公开透明的标准化模式。

**📝****费用项目补充说明**

可能包含滚动保证金（Rolling Reserve）、高风险国家附加费、专业服务费（可选）、账户激活相关审核等项目。企业客户可联系销售团队获取定制化报价。

**⏱️****结算周期**

  * **2Sell/2Subscribe：** 每周结算（Weekly Payout）

  * **2Monetize：** 默认每月结算（Monthly Payout），可协商调整

  * **结算方式：** 电汇（Wire Transfer）、PayPal、Payoneer

  * **结算币种：** USD、EUR、GBP（单一币种，可在后台切换）


**💰****滚动保证金（Rolling Reserve）**

采用滚动保证金机制，按一定比例保留资金，保留期满后释放。账户终止后可能延长保留期以覆盖潜在拒付，具体比例和周期以协议为准。滚动保证金会影响现金流，建议在签约前确认具体规则，并将其纳入资金规划。

**📌****小结：** 本章聚焦收费项目与适用规则，不含数值，最终以《商户协议》为准。

**🤔****决策建议**

在选择跨境支付服务商时，建议从以下维度进行综合评估：

**🎯****业务场景匹配度**

根据您的主要市场、客户支付习惯、是否需要 MoR 模式等因素，对比多家服务商的适配程度。如果您主要需要税务合规托管，可重点了解 2Checkout、Paddle、FastSpring 等 MoR 模式方案；如果您仅需支付网关功能，可同时了解 Stripe、Adyen、Braintree 等方案。

**💰****费率结构评估**

2Checkout 采用组合费率（百分比 + 固定费用），跨境交易有额外费用，货币转换有汇率价差。建议根据您的交易量、跨境比例、主要货币计算实际成本，与其他方案对比。

**💵****结算周期与现金流**

2Sell/2Subscribe 每周结算，2Monetize 默认每月结算。同时需关注滚动保证金对现金流的影响，尤其是业务起步阶段或季节性波动较大的业务。

**🔧****技术实施能力**

评估您的团队是否具备 API 对接能力。如技术资源有限，可优先考虑支付链接（Buy Links）和插件集成等无需开发的功能；如需深度定制，可使用 REST API 和 SDK 进行集成。

**🚫****行业准入限制**

部分行业（如代发货模式、高价值实物商品）需额外审核，建议在申请前确认业务类型是否符合平台要求。
