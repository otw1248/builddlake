## 一、费率体系全景解析

### 🔍 定制化费率框架

Thunes 无公开标准费率，所有报价需根据具体业务场景与官方团队单独协商费率方案会根据商户的业务场景、交易规模和目标市场等因素进行定制。

### 主要费用构成

### 📌 费率协商关键因素

商户在获取报价时，需重点提供以下信息以获得精准定价：

  * 业务场景与行业类型（高风险行业如汇款、电信等费率通常较高）

  * 预期月度 / 年度交易规模（交易量越大越可能获得阶梯式优惠）

  * 主要使用的支付方式（信用卡、电子钱包等成本差异显著）

  * 目标市场分布（新兴市场与成熟市场处理成本不同）

  * 技术集成复杂度（标准 API 与定制开发费用差异较大）




### 特殊及补充费用说明

**风险准备金 (Reserve)：** 高风险行业（如电信、虚拟商品）或大额业务可能要求预先缴纳保证金；此资金用于应对清算争议或拒付风险，通常在合作终止且经过一个争议观察期（如90天）后，若无未决争议则予以返还

**拒付/争议处理费 (Chargeback Fee)：** 当交易产生拒付或争议时，平台可能根据协议独立收取处理费用，此项常见于信用卡等高风险支付方式

**固定服务费：** 通用方案通常免收月费/年费；但针对企业级或VIP客户的定制化服务包，可能约定按月或按年收取固定的技术服务或账户管理费

**合规审查费：** 在部分对监管要求严格的国家或地区，为满足本地深度的KYC/AML要求（如协助资料补正、第三方认证），可能会产生额外的合规服务费用

## 二、汇率管理与资金流转

### 🔄 动态汇率机制

Thunes 以银行间市场实时汇率为基准，构建了灵活的汇率管理体系，核心特点体现在以下三方面：

#### 基准汇率体系

  * 数据源：对接银行间市场实时报价，确保汇率的市场公允性

  * 定价模式：交易发生时实时锁定汇率，支持 T+0 锁定功能，避免异地结算时的汇率波动风险

  * 透明性：所有货币兑换的点差均公开明示，商户可通过后台实时查询，符合全程透明 FX的官方承诺




#### 智能汇率风险管理

  * 实时锁汇功能：允许商户在交易创建时确定汇率，特别适用于极端市场波动或大额转账场景

  * 批量汇兑工具：支持分笔自动兑换与一键全额转换，便于商户根据资金归集需求分批锁定汇率，降低波动敞口

  * 多币种资金池：整合 80 + 结算币种，支持自动化汇兑与清算，适配定期、实时和一次性批量转换等场景




### ⏱️ 到账时效分级

Thunes 通过本地化清算网络与智能路由技术，实现了不同场景下的高效资金流转：

#### 时效影响因素解析

  * 区域差异：东南亚、非洲等新兴市场的热门电子钱包支持 T+0 到账，部分小型银行可能延迟至 T+2；欧美成熟市场通过 SEPA 等系统实现多数交易实时到账

  * 交易特性：大额、高频交易可能因合规审查加强导致延迟；法定节假日会顺延至下一个工作日

  * 技术优化：通过 SmartX Treasury System 智能路由引擎，动态选择最优路径，90% 以上交易可在一小时内完成到账




## 三、合规架构与安全体系

### 🏛️ 全球合规资质矩阵

Thunes 构建了覆盖主要市场的合规网络，其核心牌照在当地监管框架下具有重要意义：

### 🛡️ 安全保障体系

**数据安全：** 采用 AES-256 加密算法保护传输与存储数据，通过 TLS/SSL 协议确保传输安全，密钥统一管理且物理隔离

**网络防护：** 配备企业级防火墙、IDS/IPS 及 DDoS 防护机制，定期与外部安全机构合作进行渗透测试

**反欺诈技术：** 融合 AI 与机器学习的实时监控系统，集成 Tookitaki 和 ComplyAdvantage 平台，通过设备指纹、异常检测等识别欺诈模式

**资金安全：** 严格执行客户资金与自有资金隔离政策，合作银行均为当地受监管的顶级机构，确保资金安全

### 合规运营实践

**动态 KYC 机制：** 实施多级商户准入审核，结合持续动态监控，确保客户资质合规

**反洗钱措施：** 系统自动识别高风险交易，实时生成可疑交易报告（STR/SAR），符合 goAML 等报送标准

**跨境资金合规：** 通过本地持牌机构处理跨境资金流，严格遵守各国外汇管制与资本管制要求

## 四、生态对接与服务能力

### 🧩 全球支付网络集成

Thunes 以 API 为核心构建生态对接能力，虽未提供官方电商平台专用插件，但通过标准化接口实现广泛兼容：

#### 集成方式特点

  * 标准化 API/SDK：基于 REST 架构，提供详尽文档、sandbox 测试环境与交互式 API 探索器，支持多环境配置与 IP 白名单管理

  * 第三方生态集成：被 Velmie、BR-DGE 等支付科技公司集成，间接支持部分电商平台插件生态

  * 定制化模块：针对平台类业务提供 Collection API 和收单模块，支持多商户分账、退款等定制流程




#### 官方宣称的集成优势

  * 一站式对接：通过一套 API 连接 130 + 国家的 300 + 支付方式，减少多国本地支付方式单独对接的复杂度

  * 实时管理能力：提供交易追踪、资金监控与自动对账功能，提升跨境运营效率

  * 本地化适配：自动匹配目标市场的主流支付方式，优化用户支付体验




### 🌍 本地化服务网络

Thunes 通过其遍布全球关键市场的本地办公室，构建了响应迅速的全球化服务网络：

#### 区域服务能力

  * 亚太枢纽：新加坡总部负责战略决策，上海、北京、香港办公室支持大中华区业务，马尼拉、曼谷团队解决东南亚本地化问题

  * 新兴市场布局：内罗毕、约翰内斯堡等非洲办公室，迪拜、利雅得中东团队，圣保罗南美团队，深入服务高增长新兴市场




#### 客户支持体系

  * 多语言服务：以英语为基础，配备中文、法语、西班牙语、葡萄牙语等本地语言支持

  * 响应机制：通过 Zendesk 系统提供 24/7 服务，包括电话、邮件、在线聊天等渠道，多数咨询可一次响应解决

  * 技术支持：各区域设有技术团队，提供现场或远程集成协助，重点客户可获得专属对接工程师
