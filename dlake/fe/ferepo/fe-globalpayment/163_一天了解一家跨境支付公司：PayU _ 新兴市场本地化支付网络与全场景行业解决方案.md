## **一、企业基本档案**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体：**

2002年（前身 Pagosonline 在哥伦比亚成立），2014 年品牌统一；总部法律实体为 MIH PayU B.V.（荷兰），隶属于 Prosus N.V. 及南非 Naspers Limited 集团。

**🌍****全球布局：**

总部位于荷兰，并在印度、波兰、哥伦比亚、南非、土耳其等地设有核心区域枢纽，业务运营覆盖 **50 +** 新兴市场。

**🛡️****合规资质：**

印度储备银行（RBI）支付聚合商（PA）授权；PCI-DSS Level 1 认证；ISO 27001 认证（印度）；SOC 2 Type II 合规报告（通过子公司 Wibmo）。

**📊****业务覆盖：**

服务全球超过 **45 万家** 商户，支持 **400 余种** 支付方式；在印度、拉丁美洲、中东欧及非洲等新兴市场具备深度的本地化支付能力。

**🎯****行业渗透与应用场景：**

深度服务电子商务、金融服务、SaaS、旅游、游戏及市场平台等行业；提供本地化收单、智能路由、网络令牌化、多方分账、订阅管理、商户信贷及优先结算等全链路服务能力。

## **二、全球化能力图谱**

###  （一）支付网络与区域布局

**💳****国际卡收单：**

卡组织直连：与 Visa、Mastercard 建立直接合作关系，并通过与 Silverflow 的技术合作，实现通过单一 API 直接对接卡组织网络。

安全认证：支持 3D Secure 2.0，在印度市场率先为所有国内及国际交易提供 3DS 2.0 SDK 相关认证支持。

网络令牌化：支持 Visa Token Service (VTS) 与 Mastercard Digital Enablement Service (MDES)，在拉美等市场提供直通模式（PassThrough）以提升交易成功率 / 授权通过率。

智能路由：可根据持卡人与商户位置、货币及卡信息，自动选择最优的收单行或处理商进行路由分配。

高级功能：支持 Apple Pay、Google Pay 的集成对接，并明确区分商户发起交易（MIT）与持卡人发起交易（CIT）。

**🔗****本地支付聚合：**

拉丁美洲：深度覆盖市场，支持巴西的 Pix、Boleto Bancário；墨西哥的 OXXO、SPEI；哥伦比亚的 PSE；阿根廷的 RAPIPAGO 等现金与银行转账方式。

印度：获得 RBI 支付聚合商授权，支持 UPI、RuPay 及超过 **150 种** 本地支付方式，包含网银、钱包及 EMI（分期付款）及 BNPL（先买后付）。

非洲：支持 M-Pesa 等移动支付网络。

其他区域：业务网络覆盖中东欧、土耳其及部分东南亚市场。

**🔄****跨境清算路径：**

出款网络：支持通过 Visa Direct 进行主动付款，实现向 Visa 卡绑定的银行账户的即时转账。

路由优化：拥有智能路由技术，可根据成本、速度、成功率等因素动态选择最优清算路径。

**📍****区域枢纽：**

  * 🇳🇱荷兰总部：全球战略、公司治理与产品开发监督中心

  * 🇮🇳印度古尔冈：亚太地区核心运营中心

  * 🇵🇱波兰波兹南：欧洲业务运营基地

  * 🇨🇴哥伦比亚波哥大：拉丁美洲区域总部

  * 🇿🇦南非开普敦：非洲区域中心。


**📋****服务与语言：**

具体服务协议条款，以双方最终签订的正式协议文件为准。

### （二）合规与安全体系

**🏛️****核心监管资质：**

印度：获得印度储备银行（RBI）颁发的支付聚合商（Payment Aggregator）最终授权。

其他市场：通过在波兰、哥伦比亚、南非等地的本地实体，遵守当地金融监管规定。

**📜****行业认证：**

PCI-DSS：持有 PCI-DSS Level 1 认证。

ISO 27001：印度公司获得信息安全管理体系认证。

SOC 2 Type II：通过其子公司 Wibmo 获得该 SOC 2 Type II 合规报告，覆盖安全性、可用性与隐私三大标准。

**👤****KYC/AML：**

技术集成：与 AU10TIX 合作，采用其基于 Microsoft Azure 的身份验证解决方案，将平均身份核验时长缩短至 **6 秒** 。

流程能力：作为受多地监管的金融机构，执行严格的客户验证流程，以遵守反洗钱（AML）法规。

**💰****客户资金管理：**

关于客户资金存管（Safeguarding）的具体条款、合作托管银行等信息，均在与商户签署的协议中予以明确约定，并严格遵守各运营地的监管法规。

## **三、技术能力清单**

###  （一）智能技术与架构

**☁️****平台架构：**

采用云原生架构，部署于 Amazon Web Services (AWS) 之上，借助 AWS Network Firewall 与 Global Accelerator 实现多区域部署与自动化扩缩容。

**📈****系统可靠性：**

服务协议：关于系统可用性服务等级协议（Uptime SLA）的具体数值，参照服务大型企业的行业惯例，在商业合同中明确约定。

实时监控：为 PaymentsOS、拉美等多个平台提供公开的系统运行状态页面，拉美（Latam）平台近 30 天可用率达 **99.97%** 。

**🛡️****风控引擎（PayU Anti-Fraud Module）：**

技术构成：采用规则引擎与机器学习驱动的评分系统相结合的双栈架构模式，并集成 Feedzai 的先进机器学习技术。

数据信号：功能支持：支持生物识别认证（如 Flash Pay），并通过实时交易监控与人工核验结合的方式降低欺诈率。

商户配置权限：支持商户自定义 3DS 触发流程与反欺诈功能。

**🔧****开发者生态：**

API 与 SDK：提供 RESTful API 及支持 Java、PHP、Android、iOS 等开发语言的官方 SDK。

开发工具：为欧洲、拉美等区域提供独立的开发者文档中心、沙盒环境以及支持丰富事件列表的稳定可靠的 Webhook 机制。

预置对接方案：提供官方 WooCommerce 及 Magento 插件，并为拉美市场提供专属的 Magento 支付网关。

### （二）支付与结算效率

**⏱️****交易处理性能：**

响应时间：系统响应时间已优化至 **400 毫秒** （基于 AWS 架构升级后的性能提升）。

吞吐量：平台每日处理超过 **1000 万笔** 交易，拥有高并发处理能力。

**💵****资金结算时效（Priority Settlement）：**

分级结算服务：提供从 T+15 分钟到 T+1 天的灵活结算周期选择。

全天候支持：支持周末及银行节假日的 7×24 小时结算。

**🔄****资金管理技术：**

多币种账户：支持多币种支付与结算，并提供转账模块支持商户在虚拟账户与银行账户之间进行资金调度。

可编程的自动化付款能力：提供 Payouts API，支持商户通过 API 指令实现自动化付款。

## **四、计费结构与服务规则**

**📋****整体定价原则：**

采用透明的混合费率（Blended Rate）模型，为不同市场提供本地化的定价。例如在印度市场，对不同支付方式（国际卡、本地卡、UPI 等）采用明确的阶梯费率。

**💳****收款服务：**

标准平台服务费（印度市场参考）：国际交易费率为 **3%** （另加 18% GST 税费）；UPI 交易费率为 **0.9%** （另加 18% GST 税费）。

开通费用：不收取账户开通费（印度市场；欧洲等其他市场可能有注册费用）。

**🏦****结算服务：**

提供 Priority Settlement（优先结算）增值服务，允许商户根据现金流需求调整结算周期，相关费用结构将在服务开通时明确约定。

**💱****货币兑换（FX）：**

提供动态货币转换（DCC）与国际货币支付（ICP）服务，汇率每日更新，汇率价差规则在服务协议中予以明确。

**✨****增值服务：**

订阅管理、高级风控、商户融资等增值服务的计费模式，作为定制化解决方案的一部分，通过商务洽谈确定。

**📌****小结：**

本章聚焦收费模式及项目明细，公开费率仅为市场参考，不构成最终报价，具体以商务谈判及《商户协议》为准。

## **五、汇率管理与资金流转**

**📊****汇率基准与透明度：**

数据源：汇率以欧洲央行（ECB）等官方参考汇率为基准，并通过 API 从卡组织或第三方实时获取更新。

价差结构：采用 "一价全包汇率" 模式，并提供价差计算公式，确保透明度。

**🔄****换汇与对冲工具：**

即期外汇兑换：支持国际货币支付（ICP）服务，协助商户完成跨币种结算。

汇率锁定：提供汇率锁定功能，商户可选择固定汇率直至交易结算，从而对冲汇率风险。

**💵****多币种账户体系：**

账户结构：支持交易币种与结算币种的独立配置，可在多种主要货币间实现灵活结算（具体支持货币数量因市场而异）。

资金调度：提供转账模块，支持商户将 PayU 虚拟账户资金转至已注册的银行账户，且支持配置自动转账功能。

**⏱️****到账与清算时效：**

清算时效说明：通过 Priority Settlement 服务，在印度等市场可实现 **T+15 分钟** 的准实时结算。

全天候支持：结算服务覆盖周末与银行节假日，确保资金流转的连续性。

**📌****小结：**

通过汇率锁定与 T+15 分钟的优先结算服务，为新兴市场商户提供了兼具汇率风险规避与高时效优势的资金流转解决方案。

提供可覆盖支付、转账、争议处理、账户低余额预警等多种事件类型的 Webhook 通知。

## **八、适配场景与注意事项**

###  （一）适配场景

  * **🌍****新兴市场深耕者** ：业务重心在拉丁美洲、印度、中东欧、非洲，需要极高覆盖率的本地支付方式支持的企业。

  * **🏪****平台型企业** ：运营电商平台、交易市场、SaaS 平台等，需处理复杂的多方分账、子商户进件与订阅管理。

  * **🚀****高增长行业企业** ：处于旅游、游戏、金融服务等行业，需要高度定制化、包含金融增值服务的支付解决方案。

  * **💸****复杂资金流企业** ：对结算时效性（快至 T+15 分钟）和汇率风险管理（汇率锁定）有较高要求的企业。




### （二）注意事项

  * ⚖️**合规** ：PayU 在全球多地通过不同法律实体运营，签约前需审慎核实对应区域的服务主体与牌照资质。

  * 🔧**技术** ：PayU 功能强大但体系复杂，充分借助其开发者文档与沙盒环境进行集成测试尤为重要。

  * 💰**资金** ：Priority Settlement 等增值服务会涉及额外成本，需在协议中明确约定；争议处理周期通常为 30-120 天，具体时长因国家 / 地区及争议类型不同而有所差异。

  * 🌐**运营** ：PayU 在不同区域的品牌（如 Iyzico（土耳其地区））和服务细节可能存在差异，需与当地团队核实。




## **九、区域合规与适配方案**

###  （一）拉丁美洲（核心优势区域）

**🛡️****准入与合规要点：**

API 与结账页面默认支持巴西 CPF（个人税号）、CNPJ（企业税号）等本地身份信息及税号字段的采集与验证。

**🏢****运营重点：**

为 Boleto、OXXO 等本地现金支付凭证提供从生成、状态跟踪到核销对账的全流程服务解决方案；支持本地信用卡分期付款。

**📋****上线清单：**

准备 CPF/CNPJ 税务登记证明，配置票据生成与核销的 Webhook 数据回传。

### （二）印度（核心优势区域）

**🛡️****准入与合规要点：**

持有 RBI 支付聚合商最终授权，并严格遵守 RBI（印度储备银行）的数据本地化存储强制性要求。

**🏢****运营重点：**

深度整合 UPI 网络，支持 UPI 授权、确认、退款等全流程操作。

**📋****上线清单：**

完成 RBI 监管框架下的 KYB 流程，完成 UPI 支付的集成与测试。

### （三）欧洲（成熟市场）

**🛡️****准入与合规要点：**
