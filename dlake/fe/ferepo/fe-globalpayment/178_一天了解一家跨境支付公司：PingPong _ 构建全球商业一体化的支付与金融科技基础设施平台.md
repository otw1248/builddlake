## **一、企业基本档案**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体：**

2015 年 6 月（公司注册成立）；法律实体为杭州乒乓智能技术有限公司，是一家专注于服务全球跨境企业的金融科技公司。

**🌍****全球布局：**

全球运营中心位于中国杭州；在卢森堡（欧洲总部）、新加坡（亚太枢纽）、迪拜（中东枢纽）、纽约、伦敦、东京等地设有核心办公室。

**🛡️****合规资质：**

全球牌照矩阵：在全球主要市场持有超过 **60 项** 支付牌照或许可。

卡组织与行业认证：作为 Visa、Mastercard、UnionPay 等卡组织的欧洲地区主会员，并已取得行业相关安全认证。

**📊****业务覆盖：**

服务网络覆盖全球 **200 +** 国家 / 地区，支持 **16 +** 主流结算货币及 **266 种** 本地支付方式；支付网络深度整合全球主流卡组织与各区域本地支付及实时清算网络。

**🎯****行业渗透与应用场景：**

深度聚焦跨境电商、B2B 贸易、SaaS 及平台经济等领域；提供集全球收单、智能风控、高级财资管理（Currents-E, HedgingX）、订阅管理、批量出款及金融增值服务于一体的金融科技基础设施。

### (一) 支付网络与区域布局

卡组织直连：作为 Visa、Mastercard、银联国际在欧洲及香港地区的主会员，具备直接收单能力。

网络令牌化：对接 Visa VTS 与 Mastercard MDES，支持卡片信息令牌化存储。

智能优化：采用智能支付路由动态选择最优收单行，并通过账户更新器（Visa/Mastercard 账户更新服务，ABU）自动更新卡片信息，提升授权成功率。

高级功能：支持 Apple Pay、Google Pay、Klarna/Afterpay 等先买后付（BNPL），并具备延迟 / 部分扣款、交易软描述及 MIT/CIT 区分能力。

### (二) 合规与安全体系

**🏛️****核心监管牌照**

中国：通过收购浙江航天电子信息产业有限公司持有中国人民银行颁发的《支付业务许可证》。

欧洲：持有英国 FCA 颁发的电子货币机构（EMI）牌照与卢森堡 CSSF 颁发的 EMI 牌照。

新加坡：持有新加坡金融管理局（MAS）颁发的大型支付机构（MPI）牌照。

美国：在 FinCEN 注册为货币服务企业（MSB）并持有各州货币传输牌照。

**👤****KYC/AML**

技术集成：对接主流 KYC 数据库，采用 OCR 与生物识别技术进行自动化 KYC，审核时效≤24 小时。

交易监控：机器学习模型实时监测异常交易，自动提交可疑活动报告（SAR）。

**💰****客户资金管理**

资金隔离：严格遵循 FCA CASS 等监管规定，将客户资金独立存放于花旗、摩根大通、汇丰等国际一级清算银行的备付金账户。

## **四、计费结构与服务规则**

**📋****整体定价策略：**

采用 "混合费率 + 阶梯折扣" 模型，并为大客户或高风险行业提供定制化报价。

**💳****核心服务定价**

定价维度：费率与月交易规模、行业风险、目标市场及支付方式挂钩。

账户管理费：全球收款账户免费开立和持有，部分区域本地账户（如欧洲虚拟国际银行账户号码，vIBAN）可能收取少量账户管理费。

**💱****货币兑换（外汇，FX）**

汇率基准：基于国际市场实时汇率，采用透明的定价结构。

**✨****增值服务定价**

锁汇服务：提供远期锁汇和汇率风险对冲工具，提供汇率锁定服务。

高级风控：提供基于机器学习的实时风控服务，支持月度订阅或按交易量计费模式。

订阅管理：为 SaaS 和订阅制企业提供生命周期管理服务，支持按活跃用户数或交易额计费。

虚拟卡（PingPong 福卡）：提供美元结算的虚拟信用卡服务，支持多种充值方式和全球支付。

争议处理：提供专业的拒付和争议管理服务。

**🏦****企业金融服务**

供应链金融：为优质商户提供订单融资服务，基于历史交易流水进行信用评估。

外汇理财：提供汇率波动收益分享产品，支持定期锁汇与收益分成模式。

## **五、汇率管理与资金流转**

**📊****汇率基准与透明度**

汇率数据源：主流币种参考路透社等权威实时中间价，新兴市场币种采用本地主流银行牌价，价格刷新频率为秒级。

定价透明度：采用 "基准汇率 + 预设价差" 的公开结构，整体汇率信息透明可查。

**🔄****换汇与对冲工具（HedgingX）**

支持即期结售汇，可通过平台后台或 API 按需发起实时换汇操作。

提供灵活的远期锁汇服务（HedgingX），常规可支持最长 90 天的汇率锁定，并可根据客户资质与需求提供长达 6 个月甚至更长期的定制化对冲方案。

提供自动化策略：可配置汇率预警、挂单提现和条件自动换汇，灵活应对汇率波动风险。

**💵****多币种账户体系（Currents-E）**

提供主账户 + 虚拟子账户结构，子账户可按项目、店铺等维度独立设立，实现资金独立隔离与独立核算。

跨境本地虚拟账户（虚拟国际银行账户号码，vIBAN / 本地账户号码，LAN）覆盖欧盟、英国、美国、新加坡、香港等主要市场，支持通过本地清算网络收款。

**🏢****企业财资管理（Currents-E）**

资金归集模式支持手动与自动（按周期或金额阈值），可全面适配集团和多实体的资金统一管理。

内部调拨支持同币种和跨币种资金的实时调拨，提升资金使用效率。

智能清算路径：系统内置动态引擎，根据成本、时效、节假日等多维度自动匹配最优出款清算路径，保障企业全球资金流转效率最优。

**📌****小结：**

以 Currents-E 财资管理系统和 HedgingX 汇率风险管理工具为核心，PingPong 为全球化企业提供多币种账户、汇率对冲、跨币种资金归集、实时调拨和自动化清算等端到端资金流转与风险管理解决方案，服务能力处于行业领先水平。

## **八、适配场景与注意事项**

### (一) 适配场景

  * **🌍****面向欧洲、东南亚、拉美开展本地化运营的出海企业** ：重点需求本地支付方式（钱包、银行转账、票据）以及区域合规（如强客户认证，SCA、电子发票支持）。

  * **🛍️****多渠道、混合模式零售商** ：同时运营电商平台与独立站，需要统一资金归集和自动化对账，支持 MT940/CAMT.053 银行报文格式。

  * **🌐****平台型 / 创作者经济企业** ：需处理复杂的多方分账、全球批量实时出款（如 Visa Direct、SEPA Instant）、自动化子商户入驻审核流程。

  * **🚀****高增长 SaaS 与订阅制企业** ：需依托循环计费和订阅管理，关注订阅生命周期管理及全球数字服务税合规。

  * **🏢****集团化、复杂财资跨国企业** ：拥有多个法律实体，对多币种资金池、远期锁汇和内部审批流有强烈需求。




### (二) 注意事项

  * 💰**费用考量** ：部分高级功能（如锁汇、高级风控、加急结算）需额外订阅费或按笔计费，企业需综合评估成本收益。

  * 🔧**技术能力** ：平台具备高度可定制性，企业应配备足够技术开发和二次集成能力以实现深度对接。

  * ⚖️**合规要求** ：合规工具完善，终端商户需确保业务模式符合合规使用政策，主动配合深度尽职调查（增强型尽职调查，EDD）。

  * ⏱️**结算时效** ：各渠道和币种结算 SLA 存在差异，上线前需要结合 PingPong 官方的《结算时效清单》及客户经理建议合理规划现金流。




## **九、区域合规与适配方案**

### (一) 欧洲（核心市场）

**🛡️****准入与合规：**

持有英国 FCA 和卢森堡 CSSF 双 EMI 牌照，全面遵循《支付服务指令 2》（PSD2）及强客户认证（SCA）和《通用数据保护条例》（GDPR）数据合规要求。支持企业签署 DPA 协议，并为欧盟客户提供本地数据中心选项。

**🏢****本地化运营：**

深度整合 SEPA Instant 等实时清算网络（欧洲总部承载合规与清算职能），支持强客户认证（SCA）豁免策略配置，提供增值税（VAT）及电子发票一站式服务。

**📋****上线要点：**

完成 DPA 签署，配置强客户认证（SCA）合规和增值税（VAT）发票字段映射。

### (二) 北美（核心市场）

**🛡️****准入与合规：**

持有美国 MSB 牌照，按《银行保密法》（BSA）、反洗钱（AML）等金融监管规定执行运营，支持自动化税务工具（如 TaxJar）解决多州销售税问题。

**🏢****本地化运营：**

支持直连 ACH、FedNow 等本地支付系统，并通过风控模型和地址验证服务（AVS）、卡验证码（CVV）等验证机制应对友好欺诈等风险。

**📋****上线要点：**

提供最终受益所有人（UBO）相关信息，配置地址验证服务（AVS）、卡验证码（CVV）规则和税务数据自动集成。

### (三) 拉丁美洲（深度适配市场）

**🛡️****准入与合规：**

API 支持采集巴西纳税人识别号（CPF / 个人税号、CNPJ / 企业税号）等关键本地税号，实现税务合规。
