本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

成立时间与主体

2013 年成立，总部位于中国香港，主要运营实体包括 Payssion Limited 和 Payssion International Limited。

全球布局

香港为总部，在上海、深圳、南京、新加坡等地设有分支机构，服务团队覆盖亚太地区。

合规资质

  * 金融牌照：香港金钱服务营运者牌照（MSO）

  * 安全措施：通过 HTTPS（TLS）协议加密传输数据


业务覆盖

支持 **200+** 本地及国际支付方式，涵盖亚太、欧洲、拉美、中东、非洲五大区域。

主要应用场景

聚焦跨境电商、独立站收单、SaaS 订阅服务、虚拟商品销售等，提供统一 API 集成、托管收银台、支付链接、订阅管理等能力。

  


## **二、计费模式与费用说明**

交易手续费

按支付方式、地区、交易规模等因素定价，仅在成功支付时收取。

支持的支付类型

  * 国际卡收单：支持 Visa、Mastercard 等主流卡组织

  * 本地支付：电子钱包、银行转账、现金票据、先买后付（BNPL）等

  * 订阅计费：定期自动扣款、试用期、优惠券


提现与结算

支持通过国际电汇（SWIFT）等方式提现，可随时申请，无最低提现额。

费用说明

无设置费、无月费、无失败交易费。具体交易费率按支付方式定制，需联系销售获取报价。

**📌****注意：** 本章所述费用规则最终以《服务协议》为准。

  


## **三、全球支付网络与合规体系**

###  （一）支付网络与区域布局

**国际卡收单**

支持 Visa、Mastercard 等主流国际卡组织及部分本地信用卡（如巴西 Elo 等），具体支持的卡组织以后台列表和合同为准。

**本地支付聚合**

整合多种支付形态，覆盖全球五大区域：

**银行转账与清算路径**

结算方面，通过 SWIFT 进行跨境电汇结算；收单方面，Payssion 提供包括 SEPA（欧洲）、PIX（巴西）、UPI（印度）、SPEI（墨西哥）等本地清算系统对应的支付方式。

**多货币能力**

支持多种主要国际及本地货币，包括 USD、EUR、GBP、CNY、JPY、INR、BRL、MXN、THB、IDR 等。支持实时汇率转换，客户可用本地货币支付。

### （二）合规体系与监管记录

**数据保护**

已发布隐私政策和 AML/KYC 政策。建议由专业法律顾问评估其 GDPR 合规程度。

**反洗钱能力**

提供 AML/KYC 政策声明，设有合规官，负责制定和执行反洗钱、反恐融资政策，对客户进行尽职调查。具体 KYC/AML 深度与流程以最新政策为准。

## **四、技术能力与平台集成**

###  （一）API 与开发者工具

**API 架构**

采用标准的 RESTful 接口设计，通过网址路径区分接口版本，提供支付创建、查询、管理及交易通知等接口。

**开发者文档**

使用 ReadMe.io 平台托管，支持中英文双语，包含快速入门、API 参考、代码示例、错误码说明、Webhook 文档、测试指南，无需注册即可访问。

**Webhook 支持**

支持支付状态变更实时通知，提供签名验证机制，支持 Dashboard 和 API 两种配置方式。

**测试环境**

提供独立的 test/live 模式，测试与正式环境使用独立密钥，提供测试卡号和测试场景。

### （二）电商平台集成

**独立站插件**

提供 WooCommerce、Magento、OpenCart、ZenCart、PrestaShop 等官方插件，支持一键安装、多支付方式展示、订阅与分期能力开关。WooCommerce 插件保持持续更新，活跃安装超过 **300 个** 。

**💡****应用场景：** 如果您使用 WooCommerce 搭建独立站，可通过官方插件快速接入本地支付方式，插件支持保存支付方式、订阅计费等功能，安装后即可在后台配置启用。

**SaaS 建站平台**

与 Shoplazza（店匠）建立官方合作，作为推荐支付服务商，提供集成指南和技术支持。

**其他集成**

  * ISPsystem（BILLmanager）：官方集成模块

  * Corefy：支付路由平台合作伙伴




### （三）运营支持

**商户后台**

提供统一管理后台，支持交易管理、退款操作、结算管理、API Key 管理等功能。

**客户支持**

提供邮件支持（通用咨询、商务合作、技术支持）和后台工单系统，配套在线文档和开发者指南。

  


## **五、订阅管理与收款工具**

###  （一）订阅管理服务

**服务内容**

Payssion 支持周期性扣款（定期付款），商户可结合自身系统和第三方 SaaS 平台（如 Shoplazza 等）实现多种计费周期与优惠策略。如需更复杂的订阅管理功能（如套餐升级、降级、试用期自动转正等），可结合业务系统或专业订阅管理平台实现。

### （二）无代码收款工具

**支付链接**

提供支付链接（收款单页）功能，无需网站即可收款，可通过邮件、短信、社交媒体分享，支持一次性支付和自定义外观。

**托管收银台**

提供完全托管的支付页面，页面自动适配手机和电脑，支持多语言、多币种，自动展示本地支付方式。

  


## **六、适用场景与选型参考**

适合使用 Payssion 的场景**🛒****跨境电商独立站**

如果您运营独立站并面向全球销售，需要接入大量本地支付方式（如支付宝、GrabPay、Boleto、iDeal 等），Payssion 提供统一接入方案，通过 WooCommerce/Magento 插件可快速集成。

**🌍****新兴市场拓展**

如果您的业务聚焦东南亚、拉美、中东等新兴市场，当地消费者更习惯使用本地钱包和银行转账而非信用卡，Payssion 在这些区域广泛支持本地支付方式。

**🔄****SaaS 订阅服务**

如果您提供订阅制产品，需要定期自动扣款，Payssion 提供周期性扣款功能，可结合业务系统实现订阅管理。

**🚀****初创企业/小微商户**

如果您处于业务起步阶段，预算有限且交易量较小，Payssion 无设置费、无月费的定价模式可降低启动成本。

**🔧****无代码收款需求**

如果您没有技术团队或网站，需要快速收款，Payssion 提供支付链接功能，可通过邮件或社交媒体分享收款。

选型参考

在选择跨境支付服务商时，建议从以下维度进行综合评估：

**🎯****业务场景匹配度**

根据您的主要市场、客户支付习惯、交易规模等因素，对比多家服务商的适配程度。如果您的业务主要集中在欧美成熟市场且以信用卡为主，可结合自身需求对比 Stripe、Adyen 等主流服务商；如果聚焦新兴市场本地支付，可重点评估本地支付覆盖能力。

**🔧****技术实施能力**

评估您的团队是否具备 API 对接能力。如技术资源有限，可优先考虑电商平台插件实现快速集成。

**💰****费用与合同条款**

在签约前，务必获取书面报价单，明确各支付方式费率、结算周期、退款/拒付处理规则等关键条款，选择最符合您业务需求的方案。
