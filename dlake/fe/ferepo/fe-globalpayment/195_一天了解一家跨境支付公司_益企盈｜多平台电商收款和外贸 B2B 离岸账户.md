本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理,仅供参考。

**📅****成立时间与主体**

2017 年 11 月成立,主体为汇创网融科技有限公司,总部位于福建泉州。

**🌍****全球布局**

泉州为运营中心,在香港、美国设有子公司。

**🛡️****合规资质**

金融牌照:香港海关 MSO 牌照(持牌主体:Vcan Fintech Limited)

行业认证:PCI-DSS 认证

平台合作:于 2021 年 6 月官宣加入亚马逊支付服务商计划(Payment Service Provider Program, PSPP)

**📊****业务覆盖**

服务中国电商卖家及外贸企业,延伸至香港、美国、英国、法国等地个人与企业用户。

**🎯****主要应用场景**

聚焦跨境电商平台收款、独立站信用卡收单、外贸 B2B 离岸账户收款,提供 VAT 代缴、锁汇、物流对接等增值服务。

**💪****基本服务能力概览**

平台支持:Amazon、eBay、Walmart、Wish、Shopee、Lazada、TikTok Shop、Airbnb 等

币种支持:支持 **18 种** 货币(平台收款)、**50+** 种货币(独立站收款),包括美元、欧元、英镑、日元、港币等

主要覆盖市场:跨境电商平台(欧美、东南亚)、外贸 B2B(全球电汇)

## **二、计费结构与服务规则**

**📋****服务项目清单****💳****跨境收款**

按交易规模、币种、服务层级等因素实行分档与区域差异化的收费规则。

**💳****国际卡收单**

覆盖 Visa、Mastercard 等主流卡组织,具体费率与增值服务条款以协议为准。

**🏦****本地支付**

目前主要支持国际信用卡和 SWIFT 电汇。

**💵****提现与结算**

人民币、外币分通道到账,支持标准到账与加急到账,兼容平台回款与独立站收单路径。

**✨****增值服务**

VAT 相关:注册、申报、代缴等

汇率管理:锁汇服务、暂存款项等待理想汇率

账户服务:免费离岸账户开立与管理

**📝****费用项目补充说明**

可能包含货币转换加价、退款处理费、拒付处理费、加急到账费等项目。货币转换加价幅度需签约时确认。

## **三、全球化能力图谱**

### (一) 支付网络与区域布局

**💳****国际卡收单**

支持 Visa、Mastercard 等卡组织,提供基础的信用卡收单能力,主要用于独立站收款业务。

**🔗****本地支付覆盖**

支付方式主要覆盖国际信用卡和 SWIFT 电汇。对于欧洲(如 SEPA、iDEAL)、东南亚(如 GrabPay、GCash)或拉美(如 Boleto)等区域的本地钱包、本地银行转账或 BNPL 支付方式,如有需求需提前与商务团队确认覆盖范围。

**🔄****跨境清算路径**

主要通过 SWIFT 支持外贸 B2B 业务的 T/T 电汇收款。

**📍****区域运营中心**

  * 🇨🇳中国:泉州总部负责全球运营与国内客户服务,持有中国增值电信业务经营许可证

  * 🇭🇰香港:持有 MSO 牌照,覆盖港澳及亚太部分区域业务

  * 🇺🇸美国:设有子公司,承接北美市场业务




### (二) 合规与安全体系

**📜****牌照与认证**

持有香港 MSO 牌照。

**💰****客户资金管理**

外贸 B2B 业务资金由合作银行(如稠州银行)监管,客户资金在合作银行通道内运作。

## **四、技术能力、结算效率与资金管理**

### (一) 技术基础

**🔧****开发者生态**

官网介绍提供完整的 Web 端和移动端对接的 API 文档以及30 多个主流建站工具的插件(如 WooCommerce、Magento)。

### (二) 结算效率

**⏱️****到账时效**

认证审核:1 个工作日内完成(需以实际办理为准)

外贸 B2B 收款:SWIFT 电汇,一般 1-7 个工作日

平台回款与独立站收单:到账时效需与商务团队确认

### (三) 汇率管理与资金流转

**📊****汇率管理机制**

汇率基准:提现汇率参考中国银行的现汇买入价

换汇与对冲工具:

锁汇服务:为客户提供锁汇服务,用于规避汇率波动风险

择时结汇:支持客户将款项暂存在平台,等待合适的汇率时点再执行结汇或提款操作

**💵****资金账户与调度**

资金管理:提供 Web 端后台及微信小程序益企盈,支持用户查询账户资产、收汇明细、执行转账汇款(结汇)、管理收款账户等。

已加入亚马逊支付服务商计划(PSPP)。

## **六、区域合规与风险提示**

### (一) 主要市场合规要点

**🇭🇰****香港**

准入与合规:持有 MSO 牌照,需遵循香港《打击洗钱及恐怖分子资金筹集条例》

运营重点:客户尽职审查、可疑交易报告、记录保存至少 7 年

**🇺🇸****美国**

准入与合规:需遵守美国反洗钱与金融合规要求

运营重点:AML 合规官任命、独立审计、可疑活动报告(SAR)

**🇨🇳****中国大陆**

准入与合规:通过与持牌银行合作提供境内跨境支付服务

运营重点:遵守《数据安全法》、《个人信息保护法》

### (二) 通用风险提示

**⚠️****区域注意事项**

全球支付方式主要覆盖国际信用卡和 SWIFT 电汇,如业务需要大量本地支付方式,需提前与商务团队确认覆盖范围。

## **七、增值工具与行业方案**

### (一) 外贸 B2B 离岸账户

**🏦****为外贸 B2B 企业提供离岸账户开立服务。**

合作银行:与浙江稠州银行等机构合作。

账户功能:用于接收境外 B2B 贸易款项、PayPal 提款等。

服务特点:支持企业(包括无进出口权企业)、个体工商户及个人申请。

### (二) 行业方案

**🛒****跨境电商**

适配组合:

平台回款 + 独立站收款

VAT 代缴与税务服务

多币种账户与锁汇管理

**💪****能力亮点:**

提供离岸账户(稠州银行)服务

**📋****上手清单**

开通前准备:

营业执照、法人身份证(用于 KYB 认证)

店铺后台截图或链接(证明正在运营相关平台)

测试验证:小额测试订单,确认对账数据准确

上线后运营:

每日核对:检查订单与资金流水是否匹配

定期提现:根据资金需求设置提现计划

定期对账:导出对账报表,与财务数据核对

## **八、适配场景分析与决策建议**

### (一) 适合使用益企盈的场景

  * **🛒****多平台电商卖家** :如果您同时运营 Amazon、Shopee、Lazada、TikTok Shop 等多个电商平台,需要一个统一的收款工具来管理多个店铺的资金回款。

  * **🏢****外贸 B2B 中小企业** :如果您是从事 B2B 实物出口的中小企业(含个体工商户或个人),需要一个离岸账户(如稠州银行)来接收 SWIFT 电汇或处理 PayPal 提款。

  * **🛍️****基础独立站卖家** :如果您的独立站(如 WooCommerce)仅需基础的 Visa/Mastercard 信用卡收单,且对本地支付方式(如电子钱包、BNPL)没有要求。

  * **💱****有锁汇需求的商户** :如果您收取多种外币,担心汇率波动,益企盈提供锁汇服务,支持将款项暂存等待理想汇率。




### (二) 决策建议

在选择跨境支付服务商时,建议从以下维度进行综合评估:

**🎯****业务场景匹配度**

根据您的主要市场(平台/B2B/独立站)和核心需求(收款/结汇/锁汇)进行匹配。

益企盈的优势场景在于平台收款和 B2B 离岸账户。

如果您的业务需要大量本地支付方式(如东南亚钱包、欧洲 SEPA),需重点考察服务商的本地支付覆盖。

如果您的业务模式涉及订阅制,需选择支持订阅管理的服务商。

**🔧****技术实施能力**

如您需要 API 对接独立站,鉴于完整技术文档需注册后获取,您需要先完成注册和商务沟通,在获取完整 API 文档后再评估技术可行性,这会增加决策的时间成本。

评估您的团队是否具备 API 集成能力或依赖插件的技术能力。如技术资源有限,可优先考虑平台回款等无需复杂开发的功能。

**🛡️****合规与安全验证**

建议您在签约前主动要求服务商提供:

有效的 PCI-DSS 认证证书(含级别与有效期)

所有金融牌照的完整证明文件

完整的《服务条款》和《隐私政策》文本

详细的合规与安全措施说明

**💰****费用与合同条款**

在签约前,务必与多家服务商沟通,并要求益企盈提供完整的费率表,明确包含:

所有收款渠道(平台/独立站/B2B)的交易费率

货币转换的明确加价幅度(FX Markup)

拒付、退款、提现等所有附加费用明细
