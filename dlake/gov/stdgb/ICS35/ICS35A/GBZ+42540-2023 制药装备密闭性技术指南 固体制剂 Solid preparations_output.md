——OEB1级～2级的产品线，建议设置局部引风设施，如在暴露点设置有效的局部通风。

c）气闸；

d) 快速传递端口；

e） 分离式蝶阀；

f) 取样、取样器；

g) 袋进袋出；

h) 推换式过滤器；

i) 气动传输系统；

i) 一次性使用技术；

k) 顶锥阀；

1) 安全柜；

m）薄膜连接系统；

n）层流系统（低端流气流系统）。

## 6 密闭技术应用

### 6.1 进料、出料

在选用不同密闭技术满足进料、出料 OEB 需求时，宜考虑与设备需求功能的匹配性和技术本身适用性风险。

进料、出料过程中应用到的常见密闭技术如下：

a）隔离器；

b) 快速传递端口；

c) 分离式蝶阀；

d) 顶锥阀；

e) 气动传输系统；

f) 一次性使用技术；

g）安全柜。

### 6.2 流转过程

流转过程需要考虑排除密闭技术存在的风险。如气动传输系统在经过物料输送之后，管道内部残留大量API，需考虑设置足够的清洗点以满足清洁需求。也可使用套袋技术形成局部密闭效果再转移到集中清洗工位进行清洗。流转过程中应用到的常见密闭技术如下：

a）快速传递端口；

b) 带分离式蝶阀的密闭容器；

c) 气动传输系统；

d）一次性使用技术中的连续套袋技术和薄膜封闭技术。

### 6.3 取样分析

#### 6.3.1 离线取样分析

在离线取样分析过程中，在样品脱离密闭系统时需考虑样品及外包装等表面附着的 API 粉末的清理。常见的清理方法包括气闸抽风除尘、擦拭、喷淋、连续套袋等。离线取样分析可以应用如下技术：

a）隔离器；

b) 快速传递端口；

c）一次性使用技术中的连续套袋技术；

d) 安全柜；

e）气闸。

#### 6.3.2 在位取样分析

在位取样分析过程中，由于样品并没有离开密闭系统，所以在避免泄漏方面是比较安全的。同时，由于密闭系统需要进行必要的清洁，所以密闭系统内部用于取样分析的仪器的电气防护等级达到IP65能满足清洗要求。在位取样分析可以应用如下技术：

a）隔离器；

b) 安全柜。

#### 6.3.3 在线分析

在线分析过程中被分析对象未脱离生产流程，故该技术在密闭系统中能够有效避免 API 泄漏的可能，有条件时优先使用。

### 6.4 清洁

#### 6.4.1 润湿

在密闭系统内，由于设备的某些零部件结构特殊，表面容易吸附粉末，先对其进行润湿可以防止扬尘。润湿之后打开密闭系统，拆卸、移出零部件进行彻底清洗。

润湿的方法有擦拭、喷淋、灌注等。

采用润湿方法的零部件通常有流化床捕集袋、金属网状物、物料转移管路等。

#### 6.4.2 API 处理

宜对密闭系统内设备表面及使用过的清洗介质，进行 API 活性成分的灭活。灭活剂可采用氢氧化钠(NaOH)。选用其他类型灭活剂时，宜考虑灭活之后废液的处理风险，不产生二次污染。

#### 6.4.3 清洗

6.4.3.1 密闭系统清洗参照 GB/T 36030 相关条款执行，首先在不打开密闭系统的状况下进行清洗。

6.4.3.2 特殊零部件可采取人工或拆卸清洗，拆离密闭系统时需对其进行润湿或 API 处理。清洗过程常用清洗球、手动清洗、喷枪等进行。

6.4.3.3 为了便于清洗，建议在清洗用水中加入适量碳酸氢钠或其他清洗剂。

### 6.5 密闭系统废弃物处理

#### 6.5.1 密闭系统的废弃物分类

密闭生产过程产生的污染残留物，大多以物料残留、包装废弃物、过滤废弃物和清洁废弃物的形式产生，这些废弃物有固体、液体和气体三种形式。下面分类概述了可预期但不一定完整的废弃物存在种类：

a）物料残留：在密闭系统的各个环节均有可能存在；

b）包装废弃物：在隔离器、限制进入障碍系统、快速传递端口、分离式蝶阀、密闭连接系统中有可能存在；

c) 过滤废弃物: 在除了快速传递端口、分离式蝶阀、顶锥阀以外的各个场合都有可能存在, 主要存在于拦截 API 的各类过滤器中;

d）清洁废弃物：在密闭系统的各个环节均有可能存在；

e）气体废弃物：在密闭系统的各个环节的排气口过滤器外侧存在。

#### 6.5.2 密闭系统的废弃物移除

##### 6.5.2.1 物料残留

所有残余物料向外转移通过气闸或快速传递端口实现，以确保不产生二次污染。可以使用真空吸尘器对表面的粘连和粉尘沉积物进行预处理，以减少清洗液中残留物质的数量。最终的清洁是用清洗介质进行，水和加清洗剂的水溶液是最常用的清洗介质。

注：在生产过程中，系统中存在残余物料是常见的，通常是内表面的粘连物和粉尘沉积物，也可能是配套设备、配件和工具上沾附的物质。

##### 6.5.2.2 包装废弃物

API 进入隔离器通常采用气闸或快速传递端口进行。无论部分或全部取出 API，包装材料都不可避免地受到 API 的污染。对于除去污染，可以采取风淋、清洗、湿润后套袋等方式，但应在气闸中进行。

##### 6.5.2.3 过滤废弃物

密闭系统进、排风管道的过滤器表面会残留有 API，需要通过安全的方法从系统中更换、移除。采用袋进袋出和推换式过滤器方法可实现安全包装废弃过滤器，然后直接进入受控的废弃物处理过程中。

##### 6.5.2.4 清洁废弃物

清洁废弃物通常以液体的形式出现，如稀释的碱、酸、清洗剂的水溶液。宜从密闭系统通过密闭管道排放到废水箱中。虽然化学物质和清洗剂的水混合物对人的风险要低得多，但在密闭的管道中排出会更安全。清洁用辅助工具如刷子、海绵、清洁布等宜通过气闸以无污染的方式从密闭系统中移除。

#### 6.5.3 废弃物处理与排放

废弃物离开密闭系统后，进入药品生产企业的废弃物分类处理系统，药品生产企业宜充分考虑废弃物排放风险。

固体废弃物的输送、贮存、处理过程宜有防止污染环境的措施。有毒有害废渣、易扬尘废渣的装卸和运输，采取密闭和增湿等措施，防止发生污染。管道输送要考虑清洗和维修的需要。

液体废弃物根据废水中污染物种类分类收集，分别进行预处理。拟定废水处理工艺时，优先选用无毒、低毒、高效或污染较轻的水处理药剂。污水处理所产生的油泥、浮渣和剩余活性污泥等宜按照要求处理或处置。处理过的废水排入城市排水系统时，其水质符合排入城市下水道的水质标准要求或下游城镇污水处理厂、园区污水处理厂的接纳协议要求。排入表面水体时，其水质符合国家、行业和地方污水污染物排放标准。

根据 GB 37823 的废气排放要求，密闭容器的废气排放需要经过废气处理并且排放达标。

## 7 评估

### 7.1 密闭系统泄漏率

#### 7.1.1 场合

密闭系统安装、调试完成之后以及重新生产之前，宜经过泄漏率检测，满足条件之后才可进入生产环节。在正常生产过程中，为了确保设备的密闭性满足要求，宜不定期作评估。

该评估只适用于泄漏率小于  $ 10^{-1} $  h $ ^{-1} $  的场合，不适用于未形成密闭的场合，比如层流系统。

#### 7.1.2 操作

封闭密闭系统的所有进出口，将密闭系统内部压力升高或降低到表压  $ 200 \, Pa \sim 250 \, Pa $ （绝对值）且达到稳定状态。

分别记录初始大气压值、初始密闭系统内压力值、初始密闭系统内温度、开始时间和结束大气压力值、结束密闭系统内压力值、结束密闭系统内温度和结束时间。

由于测试过程在恒温、恒大气压力条件下进行，故测试时间不能过长，建议 CPT 范围为 OEB5 级，时间采用 6 min；CPT 范围为 OEB4 级，时间采用 1.5 min。测试过程中密闭系统内温度发生变化或密闭系统外大气压发生变化，记录后代入公式计算，不影响计算结果的正确性。

#### 7.1.3 计算

泄漏率计算方法参照 EJ/T 1096—1999 中 5.2.6 附录 B 进行，用压力变化法计算泄漏率的示例见附录 B。

#### 7.1.4 泄漏率与 OEB 等级的关系

泄漏率不构成与 OEB 等级的绝对对应关系，被测设备能达到的 OEB 等级，与设备的结构和 API 的性质有关。但总的趋势是，绝对值较小的泄漏率能适应较高的 OEB 等级。例如，对于隔离器来

说，泄漏率绝对值小于  $ 2.5 \times 10^{-3} \, h^{-1} $  时能适应 OEB5 级，小于  $ 1 \times 10^{-2} \, h^{-1} $  时能适应 OEB4 级。

### 7.2 颗粒密封性能

设备安装到现场之后，为了验证密闭系统是否符合 CPT 需求，建议进行颗粒密闭评估测试，测试可参照附录 C 进行。

试验结果提供了模拟 API 在预定工艺条件下的颗粒密封性能信息，不视为现场性能的保证。

注：中国医药企业管理协会 EHS 专业技术委员会发布的《中国制药工业 EHS 指南》(2020 版) 中 4.3.1 也给出了相应的验证途径。

### 7.3 清洗质量标准及检测方法

参照 GB/T 36030 执行。

## 密闭技术

### A.1 隔离器

A.1.1 隔离器一般由密封的外壳和带有操作手套的检查窗组成，用于防止物料处理和转移过程中API的暴露。隔离器常与5.2中的若干项技术组合使用来达到密闭目的。只有药物生产或密闭工艺所需的仪器、设备可位于密闭隔离器舱内，其他部件尽量设置在密闭隔离器舱外部。隔离器内设置满足清洁要求的自动清洗点，避免清洗死角，无法避免的部位要考虑满足密闭下手动清洗的条件。隔离器在每次生产前进行泄漏率检测，以防止易损部位（如手套、密封圈等）磨损而引起污染物泄漏。

A.1.2 隔离器由以下主要部件组成：

a）外壳：外壳用于建立物理屏障，其内部有工艺设备与检测设备和被处理的物质。外壳通常是为特定需求定制的。外壳可以使用柔性材料或硬质材料；

b）带有操作手套的检查窗：检查窗可以布置在隔离器的所需位置，首先根据工艺相关要求布置初定位，再根据与操作者的相匹配的人体工程定位；

c) 通风系统：通风系统的目的在于控制隔离器内部压力，以阻止 API 逃逸。隔离器可以按工艺需要由数个腔体组成，且有各自的压力和风量。进、排气过滤器确保进入和排出隔离器的空气洁净度达到要求；

d）传递系统：用于输入、内部转移、移出物料、工具、包装物等，且不影响系统密封性能。包括快速传递端口、分离式蝶阀、气闸、气动传输系统、薄膜连接系统等；

e）清洗系统：可选 CIP 和手动方式。

A.1.3 根据具体情况，隔离系统也可能用到温湿度控制，过氧化氢消毒和在有防爆要求时使用氮气吹扫隔离器内部。

### A.2 局部抽气排风

局部抽气排风通过过滤器的物理阻拦作用使得空气中的颗粒被过滤，从而获得洁净的空气，保障环境的洁净度要求。收集环节在处理颗粒过滤中非常重要，如果不能充分捕获颗粒，那么过滤器无法正常起作用。采用压差检测方式可确定过滤器是否需要更换。

封闭系统中进气时，隔离器腔体将颗粒源与工作环境隔离。通过排气口吸排出确定流量的气流，从而在腔体中产生负压，防止隔离器内的颗粒物质逃逸到工作环境中。流入隔离器的空气通过经选择适当的开口进入，所以需要控制隔离器的泄漏量。排气口的流量和进气口的选择要根据所需换气量的大小来设置。

### A.3 气闸

气闸是辅助隔离设备，主要用于洁净区与洁净区之间、洁净区与非洁净区之间小件物品的传递。双门互为连锁，有效阻止交叉污染，设有电子或机械连锁装置。配置净化过滤送风，除去物品上的浮尘。有的气闸的外侧壁上安装有密闭的操作手套，便于对经过气闸的物品进行处理。

### A.4 快速传递端口

快速传递端口是既保持系统密闭，同时又实现物料、工具、包装等进出密闭系统的技术。快速传递端口普遍应用于API及其中间体的生产过程，在OEL小于 $ 1\mu g/m^{3} $ 的环境下应用。

快速传递端口工作原理：由两个独立的单元组成，被固定安装在隔离器等外壳上的带锁扣单元为“ $ \alpha $ 

端口”，在可移动单元上的为“ $ \beta $  端口”。当两个单元合并或对接时形成一个封闭的系统。此时两端口的盖子之间形成一个密闭空间，使其虽然在隔离器内部，但该密闭空间不受污染。

两个单元可以再次彼此分离，使其具有在密闭的情况下从原密闭舱转移物料到另一个密闭舱的可能。

使用过程中分离  $ \alpha $  端口和  $ \beta $  端口时，接触点之间的密封件外表面可能受到物质污染。从经验来看，只有非常轻微的污染，一般情况下快速传递端口可以满足 OEB5 级。

快速传递端口在传递 API 过程的前后，都不建议在非隔离区打开移动单元上的  $ \beta $  端口，以防止污染环境。在打开  $ \beta $  端口时，建议在气闸中打开并彻底做内外表面的清洗或进行润湿。

### A.5 分离式蝶阀

分离式蝶阀及其辅助设施所组成的系统，可实现两个密闭系统的对接与物料的无尘转移。

分离式蝶阀的基本原理是蝶阀阀板从平行于阀板中间方向分成两个具有单独可关闭密封结构的部分，分别形成主动阀与被动阀。分离式蝶阀只有在主动阀与被动阀结合之后才能被打开，彼此分开之后未被污染的阀板面朝外从而避免泄露风险。如图 A.3 所示。

分离分离式蝶阀时，密封件的外表面和阀板外部可能受到物质的污染。这类污染一般非常轻微，一般能够满足 OEL 小于  $ 10 \mu g/m^{3} $  的需求。在阀板完全脱离之前设置专门的除尘或清洗装置，分离式蝶阀的密闭性能够满足 OEL 小于  $ 1 \mu g/m^{3} $  的需求。但是，由于系统制作的差异，应用之前建议经过评估。

分离式蝶阀系统有易损件，建议定期进行维护和目视检查。

### A.7 袋进袋出

袋进袋出操作的特点是安装、更换过滤器时，操作都在袋中进行，被更换的过滤器完全与外界不接触。在保证人员与环境的安全的同时，操作比较方便。袋进袋出安装及更换过滤器流程如下。

a）初次安装时，新过滤器直接从检修口安放到位。该检修口有特殊的法兰边可供定制的袋子套上，并用弹性件保持密封，袋子折叠好后关上检修门。

b）更换过滤器时，打开检修口，隔着袋子操作取出过滤器，使得过滤器滑入袋中，如图A.6a)所示。

c) 距离相隔不远处二次扎紧袋子。在两个扎紧处之间剪断袋子，并用胶带密封剪口，使用过的过滤器与设备分离，如图 A.6 b) 所示。此刻原来的袋子还有被扎紧的一部分套在检修口法兰上，起着阻隔作用。

d）在一个新袋子里放入新过滤器，套在检修口的法兰边上，用弹性件保持密封，如图 A.6 c) 所示。

e）隔着袋子把上一个袋子的余留部分拽下法兰口，使之滑落到袋子底部，然后放入新过滤器，如图 A.6 d) 所示。

f) 隔着袋子把新过滤器推入设备安装到位，再二次扎紧袋子底部的上一个袋子的余留部分的上口，沿二次扎紧的中间剪断，并用胶带密封剪口，上次袋子的余留部分就脱离设备，如图 A.6 e) 所示。

g）新口袋剩下的部分叠好，盖上检查口即可，如图 A.6 f) 所示。

### A.8 推换式过滤器

推换式过滤器的基本结构如图 A.7 所示。由一个圆形外壳、上下两端的活门、两个与圆筒密封配合的圆形过滤器芯和一些配气管路组成。圆筒密封安装在隔离器上，隔离器外的圆筒部分上有受控的排气口。

为了保持隔离器经常处在负压状态，需要由风机驱动排气，此时隔离器内含有API物质的粉尘就被两个过滤器芯过滤，这两个推换式过滤器芯的工作状态由两个压差计监控。靠近隔离器里面的过滤器芯截留了绝大部分粉尘，因此首先被堵塞。当压差计值达到预设值时，就必须更换过滤器芯。

更换过滤器时，打开过滤器外面的圆形外壳的活门，用工具推入一个新过滤器芯到原来外侧的一个过滤器芯的位置，原来的里侧的被污染的过滤器芯就会自动推开里侧的活门离开圆形外壳。完成过滤器芯的更换。被推入隔离器的被污染器芯经由密闭手套的操作，装入废料袋封闭，再经气闸扫除废料袋上的粉尘后取出按规定途径处置。

### A.9 气动传输系统

气动传输系统工作原理是利用真空产生的负压将物料从进料口被吸入。然后在分离容器的过滤仓内，空气与物料进行分离。物料因重力作用而自动落下，物料被收集在料仓中。

待料仓充满物料时，真空自动关闭。放料阀开启，物料通过出料口落入对接容器。出料时，压缩空气通过脉冲反吹阀自动清洁过滤器芯。如物料较为粘稠，可以安装特制的振动器辅助下料。下料结束出料舱自动关闭，整个输送流程再次启动。工作原理如图A.8所示。

特别要提醒的是:这里使用的过滤器取出更换时,建议采用袋进袋出技术,以防止污染环境。

除了已经提到的气动真空输送系统（负压输送）外，气动压力输送系统（正压输送）也在工业中使用。气动压力输送系统由于输送过程采取正压，需要特别注意容器框架内的系统密封性，所以很少在有较高密闭要求的区域使用。

### A.10 一次性使用技术

#### A.10.1 一次性隔离器

一次性隔离器适用于称重、产品取样、工艺系统的连接、更换过滤器的保护系统或维护和清洁工作。使用前需要测试是否泄漏。隔离器能达到的密闭级别取决于连接所用的配件，如气闸和连接端口。如果要将薄膜黏合在刚性零件上，则特别注意黏合区域的紧密性。

一次性隔离器的设置原理与硬质隔离器类似，其工作空间被包裹在可伸缩薄膜内。薄膜可以由低密度聚乙烯（LDPE）、聚氨酯（PU）或其他薄膜材料制成。LDPE膜的优点是它可以很容易地与其他聚乙烯部件焊接形成连接端口。一次性隔离器需要一个刚性框架或支架，用于固定薄膜和连接端口。

#### A.10.2 连续套袋技术

连续套袋技术一般应用于物料周转、局部密封、污染隔离等。由于连续套袋使用薄膜进行连接或转移，需要在使用过程中避免对其造成损伤。在多次反复使用套袋的环境中，需要预留充分长度的薄膜，避免中途更换引起泄漏风险。在操作连续套袋过程中，必须确保承接物料的袋子能够有足够的强度与密封，避免袋子破裂与掉落。连续套袋技术的工作原理，如图 A.9 所示。

在连续套袋初始状态，套袋折叠安装物料填充头外表，套袋的端部由一个密封夹封闭，与环境隔离，如图A.9a)所示。物料转移时，套袋被拉入受料容器中，打开物料填充头出料阀，物料下落装入受料容器，如图A.9b)所示。物料转移完成后，在物料上方的套袋上连续扎紧两次，在其中间剪断，并用胶带密封剪口。转移出来的物料就和物料填充头分离，此刻物料和填充头都处在被套袋隔离的状态，如图A.9c)所示。

#### A.10.3 薄膜封闭技术

薄膜封闭技术主要分为机械封闭和热封闭。

a) 机械封闭：通过两个机械封闭结构，如卡钳、扎带，在尽量短的距离对薄膜两端进行封闭，并通过切割其中间的薄膜实现分离。在这个过程中，两个封闭结构之间距离越短，泄漏量越少；用胶带密封剪口，能进一步减少泄漏量。具有双压接和分离切割功能的专门机械封闭工具可以更方便地完成机械密封。

b) 热封闭：热膜封闭系统主要通过加热薄膜使其两边热熔贴合，最终形成密闭效果。热膜封闭系统包括两个薄膜焊接爪，通过两个焊爪将产生三道焊缝。中间焊缝可较宽，并在焊缝中间设有自动分离带，这可以降低污染风险。薄膜焊接装置的设计与温度或焊接工艺有关，取决于要焊接的材料。

注：如果薄膜张紧不正确，并且薄膜中的褶皱相互重叠，则焊缝可能发生问题。焊缝上和焊缝中的物质残留也可能导致焊缝问题。

### A.11 顶锥阀

### A.12 安全柜

安全柜通过外壳的密封性、高效过滤器（HEPA）和无泄漏操作对人员和产品有保护功能，并且可防止交叉污染。

### A.13 薄膜连接系统

薄膜连接系统多用于密闭投料以及产品的安全更换中，由于薄膜连接系统成本较低，而且满足OEL小于或等于 $ 10\ \mu g/m^{3} $ 的物料传递要求，所以在很多场合中都有应用。

薄膜连接是一次性使用系统，采用双层套袋的方式，实现密封系统条件下对物品进行投放。图 A.12 显示了应用场景之一的工作原理。

a）首先需要在设备上做两个连接法兰口，一个用于进料，另一个用于操作。每个法兰上有两个凹槽，各用一个O形圈来实现薄膜与法兰之间密封。在投料时段以外的常态下，两个法兰的靠近口部的凹槽上各套有一个薄膜口袋，用O形圈扎住密封以实现与环境的隔离，如图A.12a）所示。

b) 当需要传递物品时，将传递的物品放在一个新的薄膜袋中，并在适当处扎紧，以保持物品与环境的隔离。然后将此薄膜袋套在进料端口的第二个凹槽上，用第二个凹槽上的 O 形圈密封。再将另一个新的薄膜袋套在操作口的第二个凹槽上，如图 A.12 b) 所示。

c）然后借助于操作口上新套上的薄膜袋的隔离，拽掉两个法兰口原来的薄膜袋和O形圈，如图A.12 c)、图A.12 d)所示。

d）把拽下来的薄膜袋和O形圈收容到操作口的新薄膜袋的底部，双重扎紧后在其中间剪下废弃的薄膜袋，并用胶带密封剪口，即完成废弃薄膜袋的分离和与环境的隔离，如图A.12e)所示。

e）解开进料口上薄膜袋中部的扎紧，物品即可传递到设备内部，如图 A.12 f) 所示。

f) 为方便下次进料，可以把进料口上的薄膜袋在靠近法兰口处双重扎紧，在其中间剪下废弃的多余薄膜袋，并用胶带密封剪口，如图 A.12 g) 所示。

g）再在两个法兰口各补充一个O形圈后，形成常态下的法兰口双O形圈密封。整个进料过程完成，恢复到非进料时的常态，如图A.12a)所示。

根据上述原理，也有操作口与进料口合二为一的做法，过程是相似的。

### A.14 层流系统(低湍流气流系统)

层流系统不是严格意义上的密闭系统，但为密闭生产系统提供了一种较为安全的配料形式。

在工业生产现场，理想的层流状态几乎是不可能实现的，因此这里提到的是低端流气流。低端流气流是以垂直方向和水平方向的为常见。在低端流的情况下，空气在平行层中均匀有序地流动，每个层不混合。

通常采用垂直流动，由于重力与流动方向一致，故这是一种较稳定的流动类型。该系统通常布置于相对独立的隔间中，用来称重或取样，主要用于保护在其中工作的人员。这种隔间也被称为层流或下流式隔间。此外，层流系统一般也有透明的抗静电PVC帘布，作为工作区域的边界。要注意的是，实际上该系统存在由操作员、工具、机器和产品容器引起的干扰。

空气进入该系统后，会流经预过滤器和二次过滤器。在通风装置的作用下，过滤后的清洁的空气反送回操作区。少部分空气送出系统，以利于补充新鲜空气，并可调节之间的比例。气流以垂直、低紊流位移的形式均匀地分布在整个横截面上。对于活性较强的物质，建议增加使用新鲜空气的比例乃至全部采用新鲜空气。

# 固体制剂设备颗粒密封性能评估测试

### C.1 采样

#### C.1.1 供试品

##### C.1.1.1 替代材料

在供应商现场和用户现场安装调试后验收的场合，从安全和成本的角度出发，评估测试建议使用“替代”材料加入供试品，而不是用实际的药物成分来进行设备颗粒密封性能评估测试。替代测试数据可用于预测在类似操作条件下使用其他材料的颗粒密封性能。

替代材料的化学和物理性质尽可能接近实际药物的特性。由于二者间总有或多或少的差别，故替代数据不被用作所有条件下的性能保证。在可能的情况下，使用者在该设备使用该药物后，进行后续测试，以验证使用替代物获得的结果。

替代材料建议不含可能造成健康危害的污染物，可以仅用手套且不使用其他个人防护装备的情况下方便地处理。

如果使用不常见的替代材料，则建议在该测试开始前评估其是否能安全地在被测试设备中处理和清洁。

##### C.1.1.2 常用替代材料

符合《中华人民共和国药典》的乳糖可作为常用的替代材料，建议颗粒度选择200目。其优点是：分析检测限（LOD）能达到3 ng每滤膜（相当于OEB5级），能适用于广泛的粒径分布范围，经济、无毒、高水溶性、非药品。其缺点是：易结块，少数对乳糖不耐受的试验人员要慎用。

其他较常用的替代材料还有萘普生钠、核黄素等，各有其优缺点，适合不同情况时选用。

##### C.1.1.3 供试品的组成

供试品由替代材料和一定比例的辅料组成，替代材料和辅料的比例建议模拟所替代的 API 在药品中的含量比例，以期获得相近的替代结果。也可与设备使用方协商一个双方认为合适的比例，来适应未来的不确定性。对于测试转移 API 用的容器，建议采取较高的比例，以满足该容器可以适应各种比例下的 API 转移要求，如带分离式蝶阀的容器。

##### C.1.1.4 供试品的形态

供试品的形态符合含被替代的 API 药物在正常生产中的形态，以期试验结果接近生产状况。如有孔包衣机测试时，采取含相应比例替代材料的素片进行。

##### C.1.1.5 供试品的储存

供试品按照其供应商的说明，储存在光照、湿度、温度都合适的环境中，以防止其关键参数的重大变化，如粒径或粉尘指数。测试材料不宜储存在试验围场内或测试区域内，避免增加污染风险。

##### C.1.1.6 供试品的处理

在处理供试品时，宜规范处理过程，以防止转移到测试区域发生污染。

建议使用层流系统等隔离设备进行供试品的分配处理，分配人员需合理着装和遵循卫生程序。建议分配人员不兼任进行密闭评估的人员。

供试品在转移到测试围场之前建议被密封，其容器表面擦拭干净。

#### C.1.2 采样环境

采样环境如下：

a）有被测试的设备正常运行的足够面积，并允许放置采样设备或其他测试设备；

b）有相对独立的空间，建议用围栏与相邻空间隔开，以避免相邻空间里的活动对测试的干扰和影响；

c) 气流方向稳定，以合理布置采样点位置；

d）清洁的测试环境，防止地面、被测设备和相关辅助设备上黏附的物质扰乱采样结果；

e）有合理的现场管控，避免无关人员随意走动带来空气扰动和污染；

f) 推荐的环境条件为: 温度范围:  $ 20^{\circ}C \pm 5^{\circ}C $ ; 相对湿度(RH):  $ 50\% \pm 10\% $ ; 室内正压:  $ +10Pa $  (最小) 相对于相邻空间(测试区域为封闭的情况下); 空气换气率(ACH): 每小时 3 次至 5 次;

g）对于用户现场的环境，尤其是在已经使用过的设备和场地，在测试前建议彻底清洁，并进行空气和表面采样，以确认残留水平符合检测要求；

h）如用户现场的环境中无法建立物理围场，建议对采样区附近的通道进行管制，防止交叉污染。

#### C.1.3 采样器材

采样器材如下：

a）采样器材的参考技术指标参照 GBZ/T 192.1—2007 中 A.1～A.5 执行；

b）滤膜和采样头的材质和规格的选择与所要测定的物质相关。在用乳糖作为替代材料时，滤膜材料建议用聚四氟乙烯，采样头可用聚苯乙烯或聚丙烯；

c) 采样泵流量与采样方式有关，常用 2  L/min，精度  $ \pm 0.1  L/min $ ，建议事先经过校正。

#### C.1.4 采样人员

采样人员如下：

a）进行采样的人员接受使用设备的培训，并了解测试程序；

b) 在采样期间，采样的人员采用适当的衣物/净化服技术，推荐的服装包括工作服、鞋套、帽子或头罩、手套；

c) 建议事先制定采样人员的项目分工清单，人员数量及其移动区域限于进行采样或操作设备所需的范围。

#### C.1.5 采样步骤

##### C.1.5.1 基本要求

参照 GBZ 159—2004 中第 4 章。

##### C.1.5.2 采样前的准备

在被测设备运行前对采样区域环境进行空气采样，以确认环境符合采样要求，且取得采样前区域内的空气中颗粒物的基准资料。

为正确选择采样点，建议在采样前对工作场所进行现场调查。调查内容主要包括：

a）采样中使用的原辅料的种类、数量、纯度、杂质及理化性质；

b) 采样流程中原材料的投入方式、所模拟的生产工艺；

c) 被测设备的基本工作原理和设备的完好性；

d） 制定采样计划。

##### C.1.5.3 采样点选择

C.1.5.3.1 空气中颗粒物的采样通常使用经过校准的采样泵引导空气作定向规则流动。

C.1.5.3.2 个人空气样本通常是将采样头放置在采样员的呼吸区（即离口腔 30 cm 内），采样泵建议固定在腰带、背带或口袋内，并通过一段柔性管连接到采样头。

C.1.5.3.3 固定采样位置建议选择在所测设备最容易发生泄漏的地方，建议事先进行分析。常见的最有可能发生泄漏的系统组件包括（且不限于）手套口、快速传递端口（RTP）、袋进袋出端口、可打开的门窗、气闸、分离式蝶阀接口、空气过滤器进出口、设备之间接口等。

C.1.5.3.4 在选定的固定采样点采样时，采样头放置在该点的下风位置，距离根据可能发生泄漏的系统组件进行设置。

C.1.5.3.5 样品中提交一定数量和类型的现场空白样本。空白样本通常以与样品相同的方式处理，但没有空气通过滤膜，作为比对使用。

##### C.1.5.4 采样数量

在选定的位置各采样一次为一轮，共进行三轮。

##### C.1.5.5 采样持续时间

被测设备在预设投料量运转过程中，采样过程持续进行。

#### C.1.6 采样记录

所有采样信息都在采样时收集，并记录在采样数据表中。采样信息至少包括日期、采样位置（采样点/采样人员）、样品编号、采样时间、环境温湿度等。

### C.2 检测

采样取得的滤膜小心从采样头中取出后，放入带聚四氟乙烯螺旋盖的适当大小的玻璃样品瓶中，期间避免干扰滤膜表面的颗粒物。按照《中华人民共和国药典》2020版（第四部）（通则0512）高效液相色谱法检测目标物质的含量。

如替代材料为符合《中华人民共和国药典》2020版（第四部）的乳糖，检测中的色谱条件与系统适用性试验按照《中华人民共和国药典》2020版（第四部）乳糖的有关章节进行。

### C.3 评估报告

评估报告的内容含颗粒密封性能评估测试过程记录、分析测试的条件和其他参数、结果的解释、遇到的任何问题的描述、以及可能的改进建议。

报告建议至少包括以下部分：

a）简介：说明测试意图，从事采样任务的所有人员的姓名和职责；被测设备、试验或替代材料选择、采样计划、被测设备的密封性能目标 CPT；

b）设备说明：测试设备的照片、型号、编号、设备的年限和使用状况；

c) 试验环境：试验环境照片，试验环境区域运行条件；例如，温度、湿度、空气流速或变化率、空气进排气位置（如果有）；

d）关键辅助系统的细节：包括进出料方式、容器、环境控制系统（温度、湿度控制、通风、室内真空

系统（如果有）、集尘等）、综合清洁系统（如 CIP/WIP）；

e）采样设备：采样设备说明、型号、照片；
