# 智能电网调度控制系统技术规范 第 2 部分: 术语

## 1 范围

本部分规定了智能电网调度控制系统中使用的术语、定义和缩略语。

本部分适用于智能电网调度控制系统标准的编制和系统设计、开发、建设，也可供专业技术交流参考。

## 2 基本概念

### 2.1 

## 应用 application

智能电网调度控制系统中由一组互相紧密关联的功能组成，用于完成某一方面的业务工作。

## 应用类 application categories

智能电网调度控制系统中由一组业务需求性质相似或者相近的应用构成，用于完成某一类的业务工作。

### 2.3 

## 安全校核类应用 security checking application

调度计划和电网运行操作（操作任务、操作票）安全校核的技术支撑，主要完成调度计划和电网运行操作的安全校核、稳定裕度评估，并提出调整建议。

### 2.4 

## 电网运行智能系统 operation smart system; OS2

支撑大电网运行，提供电网运行策划、监视、分析、决策、指挥、操作、控制和管理等功能，整合所有二次系统，发挥综合效益，具有互动性、智能化特点的一体化系统。

### 2.5 

## 电网运行驾驶舱类应用 power system operational cockpit application

电网运行 KPI 监视预警、信息挖掘、辅助决策和控制的技术支撑，主要包括运行 KPI 监视预警、运行全景视图、集中操控台等。

### 2.6 

## 调度管理类应用 operation management application

调度机构日常调度生产管理的技术支撑，实现电网调度规范化、流程化和一体化管理的技术保障。2.7

## 调度计划类应用 dispatching schedule application

调度计划编制业务的技术支撑，主要完成多目标、多约束、多时段调度计划的自动编制、优化和分析评估。

### 2.8 

## 基础平台 foundational platform

智能电网调度控制系统的基础，为各类应用的数据管理、信息传输与交换、公共服务、安全管理、资

源管理、模型管理、人机交互等提供通用的技术支撑，为整个系统的集成和高效可靠运行提供保障。

### 2.9 

## 基础资源平台 basic resource platform; BRP

电网智能运行系统的基础，基础软件、硬件、数据库、平台服务、数据服务、安全防护的集合，为各类应用的数据管理、信息传输与交换、公共服务、安全管理、资源管理、模型管理、人机交互等提供通用的技术支撑。

### 2.10 

## 实时监控与预警类应用 real-time supervisory control and early warning application

电网实时调度业务的技术支撑，主要实现电网运行监视全景化，安全分析、调整控制前瞻化和智能化，运行评价动态化。

### 2.11 

## 实时模式 real-time mode

在电网当前模型基础上，基于实时运行方式进行分析计算。

### 2.12 

## 研究模式 study mode

在选定的电网模型基础上，基于电网历史、预测方式进行分析计算。

### 2.13 

## 态 context

智能电网调度控制系统中为达成某些确定目标的一组应用的集合。态定义一个与时间相关的运行环境，根据运行环境的不同应分为实时态、研究态、规划态、测试态、反演态、培训态。一个应用可以在不同的态中被复用。

### 2.14 

## 域 domain

智能电网调度控制系统中基于空间、位置或责任区进行划分的区域，是每一套独立的调度控制系统所管辖的地区的名称。

注：改写 DL/T 1233-2013，定义 3.4。

### 2.15 

## 运行管理系统 operation management system; OMS

主站/厂站系统中实现电网运行分析、策划和管理相关功能的应用系统。

### 2.16 

## 运行监控系统 operation control system; OCS

主站/厂站系统中实现电网实时运行监视和控制相关功能的应用系统。

### 2.17 

## 智能变电站 smart substation

采用先进、可靠、集成和环保的智能设备，以全站信息数字化、通信平台网络化、信息共享标准化为基本要求，自动完成信息采集、测量、控制、保护、计量和检测等基本功能，同时，具备支持电网实时自动控制、智能调节、在线分析决策和协同互动等高级功能的变电站。

### 2.18 

## 智能电网调度控制系统 smart grid dispatching and control system

面向智能电网调度生产业务的自动化系统，对电网运行监视、操作控制、安全分析、计划编制、辅助决策、调度管理等业务提供技术支持。

## 3 基础平台

### 3.1 

## 安全标签 security label

具有数字签名的权限授权文件，内容包括设置的权限和证书序列号。安全标签必须与对应的证书配合使用。

### [DL/T 550—2014, 定义 3.8]

### 3.2 

## 安全操作系统 secure operating system

在自主访问控制、强制访问控制、标记、身份鉴别、客体重用、审计、数据完整性、隐蔽信道分析、可信路径、可信恢复等十个方面满足相应安全技术要求的操作系统。

### 3.3 

## 并行计算平台 parallel computing platform

用于进行各类电力系统分析计算的平台，由多个计算节点、管理节点构成，实现电网数据广播、计算结果汇总、计算任务管理、出错处理、数据备份等功能，可快速完成电力系统的计算和分析。

### 3.4 

## 场景数据 CASE

对应于某一时间断面下的应用实例内的电网运行数据的集合，包括方式数据、模型数据、图形数据等。

### 3.5 

## 调度数字证书 dispatching digital certificate

为电力监控系统及电力调度数据网络等关键应用、关键用户和关键设备提供身份安全认证的基础设施，是认证信任体系的核心内容。系统签发的证书和安全标签可应用于智能电网调度控制系统基础平台远程调阅、远方控制，调度计划制定、修订和下发，继电保护定值修改，纵向加密认证装置、单向隔离设备等安全设备的互通认证。

### 3.6 

## 电力系统通用服务协议 general service protocol for electric power system

基于 OSI 参考模型，采用面向简单服务接口规范的二进制实例化方法，建立统一的基于数据网络的通用服务机制，并兼容已有的常规通信协议，可支撑调度控制中心、变电站、发电厂内部及相互间的数据交互。

### 3.7 

## 电网模型 power grid model

满足电网运行监视、控制、分析计算等应用需求，表达电网设备属性及连接关系的数据集合。

[DL/T 550—2014, 定义 3.5]

### 3.8 

## 电网运行数据 powergrid operating data

用于电网运行监视和分析计算的数据，包括：电网主要设备的参数、设备运行时的拓扑关系和量测数据等。

### 3.9 

## 发布/订阅 publish/subscribe

一种消息的发送和接收模式。在这种模式中，发送者将具有某种分类特征的消息提交给消息总线

等消息设施；消息接收者按已订阅的消息分类从消息设施接收属于该分类的消息。

### 3.10 

## 服务 service

智能电网调度控制系统中组成功能的可被重用的最小单元，最小的功能可以没有服务。

### 3.11 

## 服务代理 service proxy

运行于某台计算机上的软件。使用服务代理的计算机先将信息发给代理服务器，由其转发，并且将收到的应答回送给该计算机。

### 3.12 

## 服务总线 service bus

屏蔽实现数据交换所需的底层通信技术，实现应用请求信息和响应结果信息的传输工具。

[DL/T 550—2014, 定义 3.4]

### 3.13 

## 服务总线管理中心 service bus management center

服务总线中完成服务注册和服务定位、查询功能的部件，屏蔽服务的实现和部署细节。对服务提供者提供服务注册和状态更新功能，对服务请求者提供服务定位和查询功能。

### 3.14 

## 公共服务 common service

基于 SOA 架构实现的，用于支持应用开发和集成的公用性服务，包括数据服务、电网模型服务、文件服务、画面服务、告警服务、权限服务、日志服务、消息邮件服务和工作流服务等。

### 3.15 

## 可信计算 trusted computing

在计算和通信系统中广泛使用基于硬件安全模块支持下的可信计算平台，以提高系统整体的安全性，包括认证密钥、安全输入输出、内存屏蔽/受保护执行、封装存储和远程证明5个核心技术概念。

### 3.16 

## 权限管理 authority management

智能电网调度控制系统中，为各类应用提供使用和维护权限的控制手段，是实现应用和数据安全访问管理的重要工具。

### 3.17 

## 身份认证 authentication

采用数字证书等方式实现访问主客体间身份强认证过程，实现数据安全传输的验证功能。

### 3.18 

## 时间序列数据 time series data

带有时间顺序标记或按照时间顺序排列的数据，能够反映事物随时间变化的情况。

### 3.19 

## 时间序列数据库 time series database

专门设计用于存储带时标的电网运行动态数据和实时稳态数据的数据库。

### 3.20 

## 实时数据库 real-time database

以内存作为存储介质的数据库及其管理系统。

### 3.21 

## 网络安全防护设备 network security device

部署在调度内部网与外部网之间、专用网与公共网之间的一组软件和硬件设备，用于构成内部网与外部网之间、专用网与公共网之间界面上的保护屏障。包括横向隔离装置、纵向加密认证装置、防火墙、防病毒系统、入侵检测/保护装置(IDS/IPS)等。

### 3.22 

## 系统安全防护 security protection framework

从操作系统安全、数据库安全、安全监视、身份认证、安全授权、网络和安全设备、性能指标等方面建立系统纵深防御体系。

### 3.23 

## 消息 message

具有规定格式，用于应用程序间传递信息的载体，由消息头和消息体两部分组成，也称消息报文。3.24

## 消息邮件 message mail

电网调度系统内部的专用简单邮件，简单实用，安全性高，不同于外部因特网及企业网上的通用电子邮件系统。

### 3.25 

## 消息总线 message bus

用于应用程序彼此间传递消息的公共通信传输工具。

[DL/T 550—2014, 定义 3.3]

### 3.26 

## 远程调阅 remote graphics browsing

一个调度主站系统通过服务代理远程访问另一个调度主站系统或变电站系统的画面和数据的一种方法。

### 3.27 

## 运行服务总线 operation service bus; OSB

由高速消息总线和通用服务总线共同组成的双总线，对外提供统一的总线服务接口。

### 3.28 

## 纵向加密认证装置 vertical encryption and authentication device

采用认证、加密、访问控制等技术措施实现电力二次系统数据的安全传输以及纵向边界安全防护的装置。

## 4 应用

### 4.1 

## AGC 控制区域 AGC control area

通过联络线与其他电网相连，可自动控制发电出力保持与其他控制区的交换计划平衡，协助整个互联电网进行系统频率调节，由发电机、负荷、输电系统及表计等构成的电气系统。

### 4.2 

## AVC 控制分区 AVC control area

根据无功电压的局域性划分出来的相互解耦的电压控制区域。

### 4.3 

## 安全约束机组组合 security constrained unit commitment: SCUC

在满足电力系统安全性约束的条件下，以系统购电成本最低等为优化目标，制定多时段的机组开停机计划。

### 4.4 

## 安全约束经济调度 security constrained economic dispatch:SCED

在满足电力系统安全性约束的条件下，以系统购电成本最低等为优化目标，制定多时段的机组发电计划。

### 4.5 

## 变电站运行驾驶舱 plants/substations operation cockpit

通过 KPI 综合、直观反映变电站(群)及其设备、环境运行状态，支持故障诊断和智能处理的应用。

### 4.6 

## 标准操作程序 standard operation procedure: SOP

将某一事件的标准操作步骤和要求以统一的格式描述出来，用来指导和规范业务流程。

## 不良数据 bad data

数据在采集、模/数转换、传输过程中会产生量测误差，误差的标准差为  $ \sigma $ 。误差大于  $ \pm3\sigma $  的量测数据为不良数据。

### 4.8 

## 不良数据辨识 bad data identification

通过状态估计的残差分析扫描量测中是否存在不良数据后，确定这些不良数据的具体量测点位置的功能。

### 4.9 

## 等值网 equivalent network

外网中与内网电气距离较远，对内网影响较小，可以进行简化等值的部分外网区域。

### 4.10 

## 电厂本地控制器 power plant local controller

调度端 AGC 控制对象，与电厂端的物理控制装置相对应，用于执行 AGC 控制命令。

### 4.11 

## 电力态势感知 power system situation awareness

对电力系统当前和未来状态的感知及理解，以及关键事件（故障、操作等）后电力系统状态的预测。

### 4.12 

## 调度计划 dispatching schedule

检修计划、交换计划和发电计划的总称。

### 4.13 

## 调度计划后分析评估 post evaluation analysis of dispatching schedule

对调度计划与实际执行效果之间的量化对比分析和跟踪评估，包含对负荷预测、水库来水预测、检修计划、发电计划、水电方案和考核结算等业务的分析评估。

### 4.14 

## 调度培训教案 dispatching training scenario

培训过程中初始运行方式和所有事件的总称。其中初始运行方式指培训初始的电网运行状态，包

括电网一次、二次设备等各种仿真对象的状态。

### 4.15 

## 调度员潮流 dispatcher power flow: DPF

实现实时方式和各种假想方式下电网运行状态的分析功能，在网络拓扑模型基础上，根据给定的注入功率及母线电压计算出各母线的状态量（电压的相角及幅值），计算出网络各支路的有功和无功功率。4.16

## 调度员培训模拟 dispatcher training simulator: DTS

一套数字仿真系统，它通过建立实际电力系统的数学模型，仿真再现各种调度操作和故障后的系统工况，并将仿真计算结果送到电力系统控制中心的模型内，为调度员提供一个逼真的培训环境。

### 4.17 

## 动态潮流 dynamic power flow

假定同一电气岛中的发电机组具有相同的频率，在考虑了发电机转动惯量、发电机调速器特性以及负荷频率特性的基础上，仿真故障和扰动下电力系统的频率变化和潮流分布。

### 4.18 

## 短路电流计算 short-circuit current calculation

针对规定的故障条件(包括各种短路故障和断线故障)，计算故障后各支路电流和各母线电压，用来校核开关遮断容量。

注：改写 DL/T 1033.2—2006，定义 2.2.42。

### 4.19 

## 断面限额管理 section limit management

用于稳定限额信息的电子化维护和管理，包括正常方式、检修方式及临时稳定限额调整的管理。4.20

## 短期交易 short term transaction

月内多日、日内多时段，区域之间、省（自治区、直辖市）之间达成的电力电量交易。

### 4.21 

## 发电计划 generation schedule

机组组合计划和出力计划。分为日前发电计划(未来一日或多日)、日内发电计划(未来1 h至数小时)和实时发电计划(未来5 min或15 min到1 h)。

### 4.22 

## 发电计划预分析评估 generation schedule pre-evaluation analysis

对同一周期不同发电计划方案间的安全裕度和经济性进行对比分析，定量评估各相关因素对不同调度目标、不同主体的影响。

### 4.23 

## 风电功率预测 wind power forecasting

风电场未来有功功率的预测，根据预测范围分为短期风电功率预测（预测未来3天，时间分辨率15 min）和超短期风电功率预测（预测未来15 min～4 h，时间分辨率15 min）。

### 4.24 

## 辅助服务 auxiliary service

为保障电力系统的安全稳定运行，保证电能质量，除正常电能生产、输送、使用外，由并网发电厂提供的用于电网调峰、调频、旋转备用等辅助性质的服务。辅助服务分为基本辅助服务与有偿辅助服务。

### 4.25 

## 基本辅助服务 basic auxiliary services

为保障电力系统安全稳定运行，保证电能质量，发电机组必须提供的辅助服务，包括一次调频、基本调峰、基本无功调节等。

### 4.26 

## 有偿辅助服务 paid auxiliary services

并网发电厂在基本辅助服务之外所提供的辅助服务。包括自动发电控制（AGC）、自动电压控制（AVC）、有偿调峰、有偿无功调节、旋转备用、黑启动等。

### 4.27 

## 辅助决策 auxiliary decision

基于静态安全校核和稳定计算校核，在满足静态安全、静态稳定、暂态稳定、动态稳定、电压稳定等安全稳定约束的条件下，计算调度计划和调度操作的校正措施，以消除或缓解各类越限、失稳等情况，为电网调度计划和调度操作提供决策支持。

### 4.28 

## 告警推送 alarm sharing

当智能电网调度控制系统综合智能告警功能诊断出设备故障后，将故障时间、故障设备、故障相别、重合情况以及故障测距等故障概要信息，通过基础平台的服务总线，自动传输给上级调度控制系统并告警。

### 4.29 

## 告警信息分流 alarm information diversion

告警信息根据在当前工作站登录的调度员或监控员所属的责任区进行过滤，工作站只显示当前登录用户所属责任区的电网或设备告警信息。

### 4.30 

## 告警直传 alarm direct transmittal

厂站监控系统向调度端进行告警信息传输。告警网关将本地告警信息转换为带厂站名和设备名的标准告警信息，传输给调度端。

### 4.31 

## 工作流 workflow

由多个参与者协调工作的流程，如调度计划流程、检修计划流程等，每类工作流程包括多个工作任务。

### 4.32 

## 工作流服务 workflow service

通过与消息邮件服务、图形浏览服务等模块协同工作，为用户提供模型设计、运行控制、管理监控等功能。

### 4.33 

## 故障集 fault set

智能电网调度控制系统中由调度人员和运行分析人员给出的包括各种可能的多重预想故障的集合。

### 4.34 

## 故障组 fault group

故障集中具有某种物理特征的故障子集。这些物理特征可以是：按开断元件类型划分，如线路、变

压器等；按地区划分，如 A 地区故障、B 地区故障等；按故障电压等级划分，如 500 kV、220 kV、110 kV 等。

### 4.35 

## 关键性能指标 key performance indicator; KPI

衡量达成运行目标水平和当前运行状态的指标。

### 4.36 

## 光伏发电功率预测 photovoltaic power forecasting

根据气象条件、统计规律等技术和手段，对光伏发电站有功功率进行的预报。根据预测范围分为短期光伏发电功率预测（预测未来3天，时间分辨率15 min）和超短期光伏发电功率预测（预测未来15 min～4 h，时间分辨率15 min）。

注：改写 NB/T 32011—2013，定义 3.3。

### 4.37 

## 合同管理 contract management

对双边交易电子化合同的集中管理，包括与其他市场成员签订的合同和组织市场成员之间签订的合同，包括电子签名（印章）管理、合同模版管理、电子证书管理等。

### 4.38 

## 恒定交换功率控制 flat tie-line control: FTC

不考虑系统频率, 只控制发电功率以维持联络线实际交换功率与计划交换功率一致的自动发电控制模式。

### 4.39 

## 恒定频率控制 flat frequency control; FFC

不考虑与外部控制区的功率交换，只控制系统频率在基准频率范围内的自动发电控制模式。

### 4.40 

## 缓冲网 buffer grid

外网中与内网电气距离较近，对内网影响较大的部分外网区域称为缓冲网。

### 4.41 

## 计算节点 calculation node

并行计算平台实现稳定分析功能的执行者，由多台计算服务器组成，提供潮流数据的接收、稳定分析计算、计算结果上传等功能，受管理节点的调配。

### 4.42 

## 机组群 unit group

若干机组的组合，用于计划对多个机组进行集合控制。

### 4.43 

## 交换计划 exchange schedule

调度机构编制的用于跨省、跨区联络线双边交换或交易的计划。

### 4.44 

## 交易管理 transaction management

获取双边交易意愿，将预交易计划送安全校核类应用进行校核，根据校核结果和交易调整建议，组织交易双方签订交易合同。

### 4.45 

## 交易合同 business contract

根据电力电量交易结果签订的合同。

### 4.46 

## 教员台 trainer worktable

教员使用的硬件和软件环境的总称，包括教员对培训过程进行监视、控制和评价功能的软件模块。4.47

## 检修计划 maintenance schedule

对发电、输电、变电等设备按照年度、月度、周、日前和临时检修做出的计划。

### 4.48 

## 检修计划预分析评估 maintenance schedule pre-evaluation analysis

对同一周期不同检修方案的安全裕度进行对比分析，定量分析各相关因素对检修计划的影响。

### 4.49 

## 经济机组 bid unit

用于计划编制的基本对象，可以为实际发电机组，也可以为等值机组或者负荷，与实际物理设备具有明确的对应关系，机组是经济模型与物理模型联接的桥梁，通过机组将经济模型中产生的各类交易、计划分解到物理设备上，进行电网的安全校核和运行分析。

### 4.50 

## 静态安全分析 static security analysis

逐个无故障断开线路、变压器、发电机、母线等单一元件或其预定义组合，分析其他元件是否因此过负荷和电网电压水平是否符合要求，用以检验校核断面结构强度和运行方式是否满足安全运行的要求。4.51

## 静态安全校核 static security check

对检修计划、发电计划、短期交易计划和电网运行操作（操作任务、操作票）等调度计划和操作，分析其基态潮流情况，实现待校核断面的灵敏度计算、静态安全校核和短路电流分析。

### 4.52 

## 考核结算 settlement

根据各类合同、计划和采集到的实际执行信息，实现各类结算主体的结算电量统计、运行情况考核、有偿辅助服务补偿，以及各类结算主体的电量结算。

### 4.53 

## 可观测性 observability

电网中是否配置了足够的量测，状态估计可以使用这些量测求解出电网全部节点的电压幅值和角度。

### 4.54 

## 可用输电能力 available transfer capability; ATC

在现有的输电合同基础上，实际物理输电网络中剩余的、可用于商业使用的传输容量。

注：改写 DL/T 861—2004，定义 11.1.3。

### 4.55 

## 快照 snapshot

在 DTS 中，系统周期自动或接受人工命令进行仿真状态的保存，用来支持培训过程的回退和重演。4.56

## 联合安全校核 joint security check

基于全网模型对全网日前发电计划、检修计划等调度计划和操作进行基态潮流、静态安全分析和短路电流计算，并实现基态越限告警设备的灵敏度计算。

### 4.57 

## 联络线偏差控制 tie-line bias control; TBC

既维持联络线实际交换功率与计划交换功率一致，又进行互联电网频率偏差调节的自动发电控制模式。

### 4.58 

## 灵敏度分析 sensitivity analysis

基于网络方程的线性化，研究电力系统变量之间相互影响的定量关系。

### 4.59 

## 流程节点 workflow node

流程节点是流程中的一系列环节步骤，一个流程任务由若干个节点组成。节点类型包括任务节点、活动节点、分支节点、聚合节点、决策节点和边缘节点，通过不同类型节点的组合实现流程逻辑。

[DL/T 1170—2012, 定义 3.4]

### 4.60 

## 流程任务 workflow task

工作流中的一个实例。

[DL/T 1170—2012, 定义 3.2]

### 4.61 

## 流域平均降水量 average precipitation of basin

一定时段内降落到某一流域面积上的加权平均降水量。

### 4.62 

## 流转过程 workflow process

工作流中一个流程实例经过各个节点的流转记录。

[DL/T 1170—2012, 定义 3.3]

### 4.63 

## 母线负荷预测 bus load forecasting

对母线有功负荷和无功负荷做出的预测，根据预测范围又分为短期母线负荷预测、超短期母线负荷预测。短期母线负荷预测可预测次日至未来多日的母线负荷，时间分辨率15 min。超短期母线负荷预测可预测未来5 min～1 h的母线负荷，时间分辨率5 min。

### 4.64 

## 区域控制偏差 area control error; ACE

由控制区当前的负荷、发电功率和频率等因素形成的偏差值，反映区域内发电与负荷的平衡情况。由联络线交换功率与计划值的偏差、系统频率与目标频率的偏差两部分组成，有时也包括时差和无意交换电量。

### 4.65 

## 扰动识别 disturbance recognition

根据扰动发生前后电力系统电气量变化的特征来判别扰动类型。

### 4.66 

## 数据申报 bidding process

电厂、变电站、地区供电公司按照各级电网调度机构的要求，在指定的时间范围内申报各类静态属性注册数据、运行技术参数和经济性参数等。

### 4.67 

## 水电调度 dispatching for hydropower

根据水库承担的水利任务与调度规则，在确保大坝安全的前提下，运用水库的调蓄能力，寻求最优运行策略及相应决策，从而实现有计划地对天然入库流量进行蓄泄，制定出水电站及其水库优化运行调度计划，以达到充分利用水能，增加发电量和保证系统安全运行的目的。

### 4.68 

## 水库群调度 dispatching for multi-reservoir

为满足防洪、水利要求，寻求水库群总效益最佳而对多个有水力和电力关联的水库所进行的统一联合调度。

### 4.69 

## 水务计算 hydro-calculation

进行与水量平衡相关的水库入库流量、出库流量、发电流量、弃水流量、耗水率和节水增发电量等常见水库统计量及统计指标的计算。

### 4.70 

## 条件故障 conditional fault

电网分析中设置的一类按条件自动触发设备开断的故障，包括条件监视元件与条件开断元件，两者配合使用，当条件监视元件满足预定义的触发逻辑时，自动触发条件开断元件执行。

## 同步相量 synchrophasor

以标准时间信号作为采样过程的基准，通过对采样数据计算而得到的相量。

注：改写 DL/T 280—2012，定义 3.1。

### 4.72 

## 拓扑逻辑防误 logic topology based anti-misoperation

采用电网拓扑分析方法，分析设备运行状态及设备间的连接关系，识别电气设备的当前综合工作状态以及可能的目标运行方式，通过防误闭锁规则逻辑判断，阻止可能的误操作并提供相应的告警说明信息。

### 4.73 

## 网络分析 network analysis

利用电网运行数据和其他应用软件提供的结果数据来分析和评估电网运行情况，确定母线模型，为运行分析软件提供实时运行方式数据，研究分析实时方式和各种预想方式下电网的运行情况；分析在电力系统中的某些元件或元件组合发生故障时，对电力系统安全运行可能产生的影响。

### 4.74 

## 网络拓扑分析 network topology analysis

根据逻辑设备状态，确定网络接线模型，建立网络母线模型和电气岛模型。

### 4.75 

## 稳定断面 stability section

为描述电网安全稳定性而定义的设备集合，一般为电网的割集。

### 4.76 

## 稳定限额 transmission limit

输电断面在满足不同的约束条件时，所允许的正向和反向最大传输功率。

### 4.77 

## 系统负荷预测 system load forecasting

未来系统负荷的预测，根据预测范围分为短期负荷预测（预测次日至未来多日，时间分辨率15 min）和超短期负荷预测（预测未来5 min～1 h，分辨率为5 min）。

### 4.78 

## 相量测量装置 phasor measurement unit: PMU

用于同步相量的测量和输出以及进行动态记录的终端装置，核心特征包括基于标准时钟信号的同步相量测量、失去标准时钟信号的守时能力、相量测量装置与主站之间能够实时通信并遵循有关通信协议。

### 4.79 

## 信息发布 information publication

电网调度机构按照规定的内容和格式向电厂和地区供电公司公开相关的注册参数和运行数据。

### 4.80 

## 学员台 trainee worktable

DTS 中学员使用的硬件和软件环境的总称，应尽可能与实际的控制中心环境一致。

### 4.81 

## 预想故障 predefined contingency

预想故障是由调度人员和运行分析人员设定的，它包括各种可能的故障及其组合，可以规定监视元件及条件故障，以自动产生多重故障。

### 4.82 

## 源端维护 source end model maintenance

在数据源端完成模型、图形、数据等的维护并发布来实现受端免维护或少维护的共享方法。

### 4.83 

## 在线安全稳定分析 online security and stability analysis

综合利用稳态、动态数据，通过稳态、动态、暂态多角度在线安全分析评估，以及电力系统运行全过程的稳定裕度评估，实现大电网运行的全面安全预警和多维多层协调的主动安全防御。

### 4.84 

## 在线外网等值 online external-network equivalence

上级调度为下级调度计算生成准实时的外网等值模型。

### 4.85 

## 责任区 area of response

调度员或监控员根据职责负责调度或监控的电网设备管辖范围，可以根据厂站、电压等级和间隔进行划分。

### 4.86 

## 智能运动网关 smart remote communication and control gateway

位于变电站站控层的一种远动装置，集成了变电站和主站系统之间的通信，实现变电站远动数据的直采直送，并提供变电站内数据、模型和图形的采集、存储、处理和传输服务，支持远程浏览、源端维护、顺序控制、电压无功控制等功能，宜具备防误闭锁、监控终端功能。

### 4.87 

## 中枢母线 pilot bus

AVC 控制分区中的枢纽母线，中枢母线电压代表了 AVC 控制分区的电压水平。

### 4.88 

## 状态估计 state estimation: SE

利用电网模型和参数、电网实时遥信遥测数据，求解电网实时运行状态，并检测和辨识量测中的不良数据。

注：改写 GB/T 2900.58—2008，定义 603-02-09。

### 4.89 

## 自动电压控制 automatic voltage control; AVC

通过自动控制程序的运行，根据电网实时运行工况在线计算无功电压控制策略，自动闭环控制无功和电压调节设备，以实现合理的无功电压分布。它是由主站自动电压控制软件、信息传输通道、信息接收装置、子站AVC控制系统及执行机构等环节组成的整体。

### 4.90 

## 自动发电控制 automatic generation control; AGC

通过自动控制软件，实现对控制区内各发电机组有功出力的自动调节分配，来维持系统频率和/或联络线交换功率在计划目标范围内的控制过程。它是由主站自动控制软件、信息传输通道、信息接收装置（远方终端）、机组协调控制系统（电厂监控系统）、执行装置、发电机组自动化装置等环节组成的整体。

### 4.91 

## 自动直流控制 automatic DC control; ADC

通过自动控制软件，实现对控制区内各高压直流、柔性直流输电系统有功出力的自动调节分配和解闭锁状态的自动控制。它是由主站自动控制软件、信息传输通道、信息接收装置（远方终端）、换流站控制系统等环节组成的整体。

### 4.92 

## 自动装置仿真 auto mechanism simulation

针对电力系统安全自动装置的建模仿真方法。用户无需编程即可定义所需要的装置仿真模型，模型中的量测、逻辑和动作环节可由用户根据装置的动作原理来指定。

### 4.93 

## 最大输电能力 total transfer capability

在满足系统各种安全可靠性要求的情况下，互联系统联络线上总的输电能力。
