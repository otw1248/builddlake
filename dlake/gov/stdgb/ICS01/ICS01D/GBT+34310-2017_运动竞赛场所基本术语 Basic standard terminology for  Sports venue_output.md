

# 中华人民共和国国家标准

GB/T 34310—2017

# Basic standard terminology for sports venue

# 运动竞赛场所基本术语

2017-09-07 发布

2018-04-01 实施

## 前言

本标准按照 GB/T 1.1—2009 给出的规则起草。

本标准由国家体育总局提出并归口。

本标准起草单位:北京体育大学、国家体育总局排球运动管理中心、国家体育总局自行车击剑运动管理中心。

本标准主要起草人：徐刚、刘冰、张凯。

# 运动竞赛场所基本术语

## 1 范围

本标准界定了相关运动竞赛场所的基本术语。

本标准适用于符合国际通用运动竞赛标准的，用于国家级及以上专业体育竞赛的各类体育运动竞赛场所。
