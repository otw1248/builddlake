## 前言

GB/T 4122《包装术语》分为七个部分：

——第1部分:基础;

——第2部分:机械；

——第3部分：防护；

——第4部分：材料与容器；

——第5部分：检验与试验；

——第6部分：印刷；

——第7部分:包装与环境。

本部分为 GB/T 4122 的第 2 部分。

本部分代替 GB/T 4122.2—1996《包装术语 机械》。

本部分与 GB/T 4122.2—1996 相比主要变化如下：

——补充了“相关包装机械”章节；

——增加了“毛重式充填机”等术语；

——删除了“自动包装线”、“自动包装机”等术语。

本部分由全国包装机械标准化技术委员会(SAC/TC 436)提出并归口。

本部分起草单位：中国包装联合会、深圳市美盈森环保包装技术有限公司、中国出口商品包装研究所、合肥通用机械研究院。

本部分主要起草人：王利、陈润洁、蔡少龄、朱婧、李建华。

本部分所代替标准的历次版本发布情况为：

GB 5035—1985、GB/T 4122.2—1996。

## 包装术语 第 2 部分: 机械

## 1 范围

GB/T 4122 的本部分规定了包装机械的通用术语及其定义。

本部分适用于包装机械的设计、生产、流通、使用及监督检验。

## 2 通用

### 2.1 

## 包装机械 packaging machinery

完成全部或部分包装过程的机器。包装过程包括成型、充填、封口、裹包等主要包装工序，以及清洗、干燥、杀菌、贴标、捆扎、集装、拆卸等前后包装工序转送、选别等其他辅助包装工序。

### 2.2 

## 充填机械 filling machine

将产品按预定量充填到包装容器内的机器。

### 2.3 

## 灌装机械 filling machine

将液体产品按预定量灌注到包装容器内的机器。

### 2.4 

## 封口机械 sealing machine, closing machine

在包装容器内盛装产品后，对容器进行封口的机器。

### 2.5 

## 裹包机械 wrapping machine

用挠性包装材料全部或局部裹包产品的机器。

### 2.6 

## 无菌包装机械 aseptic packaging machine

在无菌条件下，将经过灭菌的产品密封于经过灭菌的容器中，使产品在常温下能够满足保质要求的机器。

### 2.7 

## 标签机械 labelling machine

将标签贴在包装件或产品上的机器。

### 2.8 

## 清洗机械 cleaning machine

对包装容器、包装材料、包装辅助物以及包装件进行清洗以达到预期清洁度的机器。

### 2.9 

## 干燥机械 drying machine

对包装容器、包装材料、包装辅助物以及包装件上的水分进行去除以达到预期干燥程度的机器。

### 2.10 

## 杀菌机械 sterilization machine

对产品、包装容器、包装材料、包装辅助物以及包装件等上的微生物进行杀灭，使其降低到允许范围内的机器。

### 2.11 

## 捆扎机械 strapping machine

使用捆扎带缠绕产品或包装件，然后收紧并将两端通过热效应熔融或使用卡扣等材料连接的机器。

### 2.12 

## 集装机械 machine for the assembly of unit load

将若干个包装件或产品包装在一起，形成一个合适的搬运单元的机器。按集装方式分为托盘集装机、无托盘集装机。

### 2.13 

## 多功能包装机械 multi-function packaging machine

在一台整机上可以完成两个或两个以上包装工序的机器。

### 2.14 

## 辅助包装设备 auxiliary packaging equipment

对包装材料、包装容器、包装辅助物或包装件执行非主要包装工序的设备。

## 3 充填机械

### 3.1 

## 容积式充填机 volumetric filling machine

将产品按预定容量充填到包装容器内的机器。

### 3.2 

## 量杯式充填机 measuring cup filling machine

采用定量的量杯量取产品并将其充填到包装容器内的机器。

### 3.3 

## 可调容量式充填机 adjustable volume filling machine

采用可随产品容量变化而自动调节容积的量杯量取产品，并将其充填到包装容器内的机器。

### 3.4 

## 气流式充填机 stream type filling machine

利用真空吸粉原理量取定量容积产品，并采用净化压缩空气将产品充填到包装容器内的机器。

### 3.5 

## 柱塞式充填机 piston type filling machine

采用可调节柱塞行程而改变产品容量的柱塞筒量取产品，并将其充填到包装容器内的机器。

### 3.6 

## 螺杆式充填机 auger type filling machine

通过控制螺杆旋转的转数或时间量取产品，并将其充填到包装容器内的机器。

### 3.7 

## 计量泵式充填机 dosing pump type filling machine

利用计量泵中齿轮的一定转数量取产品，并将其充填到包装容器内的机器。

### 3.8 

## 插管式充填机 insertion pipe type filling machine

将内径较小的插管插入储粉斗中，利用粉末之间的附着力上粉，到卸粉工位由顶杆将插管中的粉末充填到包装容器内的机器。

### 3.9 

## 推入式充填机 push type filling machine

用外力将产品推入包装容器内的机器。

### 3.10 

## 拾 放式充填机 pick-off filling machine

将产品拾起并从包装容器开口处上方放入容器内的机器。可用机械手、真空吸力、电磁吸力等方法拾放产品。

### 3.11 

## 重力式充填机 gravity filling machine

靠产品自身重力落入或流入包装容器内的机器。

### 3.12 

## 定时式充填机 timed filling machine

通过控制产品流动的时间或调节进料管流量而量取产品，并将其充填到包装容器内的机器。

### 3.13 

## 称重式充填机 gravimetrie filling machine

将产品按预定质量充填到包装容器内的机器。按称重方式分为净重式充填机和毛重式充填机。

### 3.14 

## 计数式充填机 counting filling machine

将产品按预定数目充填到包装容器内的机器。

#### 3.14.1 

## 单件计数式充填机 counting filling machine with unit register

采用机械、光学、电感应、电子扫描方法或其他辅助方法逐件计算产品件数，并将其充填到包装容器内的机器。

#### 3.14.2 

## 多件计数式充填机 counting filling machine with multiple register

利用辅助量，如长度、面积等进行比较以确定产品件数，并将其充填到包装容器内的机器。

### 3.15 

## 转盘计数式充填机 round table counting filling machine

利用转盘上的计数板对产品进行计数，并将其充填到包装容器内的机器。

### 3.16 

## 履带式计数充填机 strip counting filling machine

利用履带上的计数板对产品进行计数，并将其充填到包装容器内的机器。

### 3.17 

## 料位式充填机 level filling machine

通过控制充填到包装容器内的产品料面高度的方法进行计量和充填的机器。

### 3.18 

## 净重式充填机 net weighing filling machine

称出预定质量的产品，并将其充填到包装容器内的机器。

### 3.19 

## 毛重式充填机 gross weighing filling machine

在充填过程中，产品连同包装容器一起称重的机器。

## 4 灌装机械

### 4.1 

## 负压灌装机 low vacuum filling machine

先对包装容器抽气形成负压，然后将液体充填到包装容器内的机器。

### 4.2 

常压灌装机 atmospherical pressure filling machine

在常压下将液体充填到包装容器内的机器。

### 4.3 

## 等压灌装机 isobar filling machine

先向包装容器充气，使其内部的气体压力和储液缸内的气体压力相等，然后将液体充填到包装容器内的机器。

### 4.4 

## 定量灌装机 dosing filling machine

通过控制充填到包装容器内的液体容积进行计量和灌装的机器。

### 4.5 

液位灌装机 level filling machine

通过控制充填到包装容器内的液体液面高度进行计量和灌装的机器。

## 5 封口机械

5.1

热压封口机 heat sealing machine  

用热封合的方法封闭包装容器的机器。

5.2

脉冲封口机 impulse sealer; impulse sealing machine  

通过电脉冲元件的瞬时加热以封闭包装容器的机器。

### 5.3 

超声波封口机 closing ultrasonic machine

利用超声波聚能器作用在包装容器封口部位，以封闭包装容器的机器。

### 5.4 

熔焊式封口机 fusion weld sealing machine 通过加热使包装容器封口处熔融封闭的机器。

### 5.5 

压纹式封口机 embossing closing machine

利用滚轮在包装容器封口处滚出螺纹或压痕以封闭包装容器的机器。

5.6

折叠式封口机 fold closing machine

在包装容器封口处，进行一次或多次折叠以封闭包装容器的机器。

### 5.7 

插合式封口机 tucking-in closing machine

通过插入或翻转包装容器上的规定部分以封闭包装容器的机器。

### 5.8 

滚压式封口机 roll-on capping machine  

通过滚压使金属盖变形以封闭包装容器的机器。

### 5.9 

卷边式封口机 double seaming machine

用滚轮将金属盖与包装容器开口处互相卷曲勾合以封闭包装容器的机器。

### 5.10 

压力式封口机 pressure closing machine

通过在封口器材的垂直方向上施加预定的压力以封闭包装容器的机器。

### 5.11 

旋合式封口机 screw-closure closing machine

通过旋转封口器材以封闭包装容器的机器。

### 5.12 

缝合式封口机 sewing machine

使用缝线缝合包装容器的机器。

### 5.13 

钉合式封口机 nailing machine; stitching machine

使用金属钉封闭包装容器的机器。

### 5.14 

胶带式封口机 adhesive tape sealing machine

使用胶带封闭包装容器的机器。

### 5.15 

粘结式封口机 adhesive sealing machine

使用粘合剂封闭包装容器的机器。

### 5.16 

结扎式封口机 binding sealing machine

使用线、绳等结扎材料封闭包装容器的机器。

## 6 裹包机械

### 6.1 

半裹式裹包机 part wrapping machine

由挠性包装材料裹包产品局部表面的机器。

### 6.2 

全裹式裹包机 full wrapping machine

用挠性包装材料裹包产品的所有表面的机器。

### 6.3 

折叠式裹包机 fold wrapping machine

用挠性包装材料裹包产品，将末端伸出的裹包材料折叠封闭的机器。

### 6.4 

扭结式裹包机 twist wrapping machine

用挠性包装材料裹包产品，将末端伸出的裹包材料扭结封闭的机器。

### 6.5 

接缝式裹包机 seam wrapping machine

用挠性包装材料裹包产品，将末端伸出的裹包材料热压封闭的机器。

### 6.6 

覆盖式裹包机 cover wrapping machine

用两张挠性包装材料覆盖在产品的两个相对面上，借热封或粘合封口以裹包产品的机器。

### 6.7 

## 缠绕式裹包机 spiral wrapping machine; convolute wrapping machine

用成卷挠性包装材料多圈缠绕产品的机器。

### 6.8 

## 现场发泡机 foam-in-place machine

能将不同组分发泡材料的混合物注入模具或产品与容器之间，使其发泡并固化的包装设备。

### 6.9 

## 拉伸包装机 stretch wrapping machine

使用拉伸薄膜在一定张力下裹包产品的机器。常用于把集积在托盘上或浅盘上的产品连同托盘或浅盘一起裹包。

### 6.10 

## 贴体包装机 skin packaging machine

将产品置于底板上，使覆盖产品的塑料薄片（膜）在加热和抽真空作用下紧贴产品，并与底板封合的机器。

### 6.11 

## 收缩包装机 shrink packaging machine

将产品用收缩薄膜裹包后，使薄膜收缩后裹紧产品的机器。

## 7 标签机械

### 7.1 

## 贴标签机 labelling machine

采用黏合剂将标签贴在包装件或产品上的机器。

#### 7.1.1 

## 直线式贴标机 straight line labelling machine

通过加标机构将标签贴在作直线运动的包装件或产品上的机器。

#### 7.1.2 

## 回转式贴标机 rotary labelling machine

通过加标机构将标签贴在作回转运动的包装件或产品上的机器。

### 7.2 

## 不干胶标签机 non-drying labelling machine

通过加标机构将不干胶标签贴在包装件或产品上的机器。

### 7.3 

## 收缩标签机 shrink labelling machine

用热收缩或弹性收缩的方法将筒状标签紧套在包装件或产品上的机器。

### 7.4 

## 订标签机 tag labelling machine

用钉、针、线等材料将标签固定在包装件或产品上的机器。

### 7.5 

## 挂标签机 tie-on labelling machine

用线、带、针、钉等材料将标签或吊牌悬挂在包装件或产品上的机器。

### 7.6 

## 套标机 sleeve machine

将筒标自动切断，准确套入容器的包装机械。

## 8 清洗机械

8.1  
  
干式清洗机 dry-cleaning machine  
  
使用气体清洗剂，以压力或抽吸方法清除不良物质的机器。  
  
8.2  
  
湿式清洗机 wet-cleaning machine  
  
使用液体清洗剂、蒸汽清除不良物质的机器。  
  
8.3  
  
机械式清洗机 mechanical cleaning machine  
  
借助工具擦刷以清除不良物质的机器。  
  
8.4  
  
电解清洗机 electrolytic cleaning machine  
  
通过电解分离清除不良物质的机器。  
  
8.5  
  
电离清洗机 ionization cleaning machine  
  
通过电离清除不良物质的机器。  
  
8.6  
  
超声波清洗机 ultrasonic cleaning machine  
  
通过超声波产生的机械振荡清除不良物质的机器。  
  
9 干燥机械  
  
9.1  
  
热式干燥机 heat drying machine  
  
通过热交换去除水分的机器。  
  
9.2  
  
机械式干燥机 mechanical drying machine  
  
通过离心、甩干等方法去除水分的机器。  
  
9.3  
  
化学式干燥机 chemical drying machine  
  
通过化学作用去除水分的机器。  
  
9.4  
  
真空干燥机 vacuum drying machine  
  
通过抽去包装容器内部空气达到预定真空度的方法去除水分的机器。  
  
10 杀菌机械  
  
10.1  
  
热式杀菌机 heat sterilization machine  
  
通过加热进行杀菌消毒的机器。  
  
10.2  
  
超声波杀菌机 ultrasonic sterilization machine  
  
通过超声波的直接作用进行杀菌消毒的机器。

### 10.3 

## 电离杀菌机 ionization sterilization machine

通过放射性同位素或高能电子束辐照，破坏细胞生理结构而使微生物致死的杀菌机器。

### 10.4 

## 化学杀菌机 chemical sterilization machine

通过施加灭菌剂进行杀菌消毒的机器。

### 10.5 

## 辐射杀菌机 radicidation sterilization machine

通过电磁波或放射线辐照进行物理杀菌的机器。

### 10.6 

## 微波杀菌机 microwave sterilization machine

通过频率在 300 兆赫兹到 300 千兆赫兹的电磁波进行物理杀菌的机器。

### 10.7 

## 杀菌-干燥机 sterilization and drying machine

对产品包装容器、包装材料、包装辅助器材以及包装进行杀菌并去除其水分使其达到预期干燥程度的机器。

## 11 捆扎机械

### 11.1 

## 机械式捆扎机 mechanical strapping machine

采用机械传动进行捆扎的机器。

### 11.2 

## 液压式捆扎机 hydraulic strapping machine

采用液压传动进行捆扎的机器。

### 11.3 

## 气动式捆扎机 pueumatic strapping machine

采用空气压力传动进行捆扎的机器。

### 11.4 

## 捆结机 tying machine

使用线、绳等结扎材料，使之在一定张力下缠绕产品或包装件一圈或多圈，并将两端打结连接的机器。按结扎方法可分为纵向、横向、环形捆结机。

### 11.5 

## 压缩打包机 baling press; baler

将松泡产品压缩打包，成为有规则形状包装件的机器。

## 12 集装机械

### 12.1 

## 单元包装机 machine for the assembly of unit load

将若干个包装件或产品包装在一起，形成一个合适的搬运单元的机器。

### 12.2 

## 堆码机 stacking machine

将预定数量的包装件或产品按一定规则进行堆积的机器。

### 12.3 

## 拆卸机 machines for the unloading of unit load

将集合包装件拆开、卸下、分离等的机器。

## 13 多功能包装机械

### 13.1 

## 充填-封口机 filling and sealing machine

将产品装入包装容器，并完成封口工序的机器。

### 13.2 

## 真空包装机 vacuum packaging machine

将产品装入包装容器后，抽去容器内部的空气，达到预定真空度，并完成封口工序的机器。
