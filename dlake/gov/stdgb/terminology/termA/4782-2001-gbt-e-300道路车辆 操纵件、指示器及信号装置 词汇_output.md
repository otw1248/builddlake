## 前言

由于计算机技术的迅速发展和在汽车上的广泛应用，出现了许多新的汽车电子控制装置，如主动悬架、巡行系统等。对本标准的修订就是为了适应这一新技术的发展。修订的内容较原标准增加了新的汽车电子控制装置的词汇，主要收录了国内外较为成熟的技术的术语，对于尚未成熟的技术，本标准未予以考虑。同时，对原标准中不适当的词汇进行了修改。

附录 A、附录 B 均为提示的附录。

本标准自生效之日起，代替 GB/T 4782—1984。

本标准由国家机械工业局提出。

本标准由全国汽车标准化技术委员会归口。

本标准起草单位：中国汽车技术研究中心。

本标准主要起草人：刘彦戎、孙惠、李岩和。

本标准 1984 年首次制定，本次为第一次修订。

## 1 范围

本标准规定了道路车辆用的操纵件、指示器及信号装置的词汇。

本标准适用于四个车轮以上(含四个车轮)的各种道路车辆。

2 操纵件 controls

2.1 操纵件 control

驾驶员用来改变车辆某个装置的工作状况或功能的构件。

2.2 方向盘 steering control 控制车辆行驶方向的操纵件。

2.3 变速杆 gear level/transmission control

2.4 加速踏板 accelerator pedal

用来改变发动机输出功率或转速的脚操纵件。

2.5 手油门 manual accelerator control

用来改变发动机输出功率或转速的手操纵件。

2.6 离合器踏板 clutch pedal

控制离合器接合或分离的脚操纵件。

2.7 行车制动器制动踏板 service brake control

控制行驶车辆减速或停车时所使用的制动系统的脚操纵件。

2.8 应急制动器操纵件 emergency (secondary) brake control 控制车辆紧急减速或停车的装置的操纵件。

2.9 驻车制动器操纵件 parking brake control

停车后使车辆保持静止状态的装置的操纵件。

2.10 发动机冷起动操纵件 engine cold start control

使发动机易于冷起动的辅助装置的操纵件(一般为手操纵件)。

2.12 后窗玻璃刮水器操纵件 rear window wiper control

控制后窗玻璃刮水器工况的操纵件。

2.13 侧窗玻璃刮水器操纵件 side window wiper control

中华人民共和国国家质量检验检疫总局 2001-07-03 批准

控制侧窗玻璃刮水器工况的操纵件。

2.14 前风窗玻璃洗涤器操纵件 windscreen washer control

控制前风窗玻璃洗涤器工况的操纵件。

2.15 后窗玻璃洗涤器操纵件 rear window washer control

控制后窗玻璃洗涤器工况的操纵件。

2.16 门锁操纵件 door latch control

控制车门锁开闭的手操纵件。

2.17 车门锁止操纵件 door lock control

用以阻止门锁操纵件动作的手操纵件。

2.18 车门玻璃升降器操纵件 window lift control

用来升降车门玻璃的手操纵件。

2.19 活动车顶操纵件 sliding roof control

用来开闭车身活动顶盖的手操纵件。

2.20 暖风操纵件 heater control

控制暖风装置工况的手操纵件。

2.21 空气调节装置操纵件 air conditioning controls

控制空调装置工况的手操纵件。

2.22 音响操纵件 radio receiver controls

音响开关、音量、调谐、波段选择等的旋(按)钮。

2.23 自动天线操纵件 automatic antenna control

使收音机杆式天线自动伸长或缩短的装置的手操纵件。

2.24 启动开关 starting control

接通或切断启动电路的手操纵件。

2.25 柴油机停机操纵件 diesel engine cut-off control

使运转的柴油机停机的装置的手操纵件。

2.26 起动机操纵件 starter control

发动机起动装置的手或脚操纵件。

2.27 灯光总开关 master lighting control

所有前照灯、位置灯、侧标志灯、牌照灯及示廓灯的手操纵件。

2.28 前照灯变光开关 headlights beam switching control

变换前照灯远光或近光的操纵件。

2.29 转向信号灯操纵件 direction indicator control

转向指示信号的操纵件。

2.30 危险警告信号操纵件 hazard warning signal control

由于车辆本身的紧急状况，对其他车辆将造成暂时的危险而发出警告的装置的操纵件。

2.31 前照灯光束对光操纵件 headlight beam aiming control

补偿车辆在不同载荷时因光束倾斜度的影响而对前照灯光束作垂直方向对光的操纵件。

2.32 前照灯刮水器操纵件 headlight wiper control

控制前照灯刮水器工作的操纵件。

2.33 前照灯清洗器操纵件 headlight cleaner control

控制前照灯清洗器工作的操纵件。

2.34 客厢顶灯开关 interior light control

用于控制客厢内照明顶灯的开关。

2.35 蓄电池断路器操纵件 battery isolating switch control

使蓄电池和电器线路间断路的手操纵件。

2.36 声响警告(喇叭)操纵件 audible warning (horn) control

向行人及其他车辆发出声响警告(鸣喇叭)的操纵件(不包括有特殊优先权的车辆，如消防车的响警告)。

2.37 灯光警告操纵件 optical warning control

驾驶员控制前照灯发出光警告的操纵件。

2.38 方向盘调节装置操纵件 steering wheels adjustment control

调节方向盘相对于驾驶员位置的装置的操纵件。

2.39 座椅调节操纵件 seat adjustment control

调节座椅相对于车体作线位移或角位移机构的手操纵件。

2.40 动力输出操纵件 power take-off control

用于控制车载动力输出装置的手操纵件。

2.41 倾卸货物操纵件 tilt control

使货厢倾斜，以便卸货的装置的手操纵件。

2.42 下坡缓速器操纵件 retarder control

在车辆下坡不用行车制动器及应急制动器时，使车辆缓速的装置的操纵件。

2.43 附加驱动车轮的操纵件 additional wheel drive control

传递发动机动力给附加驱动车轮的装置的操纵件。

2.44 差速锁操纵件 differential lock control

控制差速器锁止装置的操纵件。

2.45 水箱面罩操纵件 radiator grill control

打开前水箱面罩锁止机构的手操纵件。

2.46 发动机罩操纵件 front hood (bonnet) control

打开发动机罩锁扣的手操纵件。

2.47 行李箱盖操纵件 rear hood (boot) control

打开行李箱盖锁止机构的手操纵件。

2.48 散热器百叶窗操纵件 radiator shutter control

用以控制百叶窗开度的操纵件。

2.49 高低挡操纵件 range shift control

控制分动箱高档或低挡的操纵件。

2.50 外后视镜调节操纵件 outside rear-view mirror adjustment control

控制外后视镜调节装置的操纵件。

2.51 外后视镜电加热器操纵件 outside rear-view mirror electric heater control

控制外后视镜电加热器工作的操纵件。

2.52 巡行控制操纵件 cruise control system control

用来设定巡行车速的手操纵件。

2.53 自动变速器控制操纵件 transmission control switch control

用于锁死超速挡的手操纵件。

2.54 座椅加热器操纵件 seat heater control

用来控制座椅加热器的手操纵件。

2.55 雾灯开关 fog light control

用来开闭雾灯的手操纵件。

## 3 指示器 indicators

3.1 指示器 indicator

用刻度值或数字显示车辆及其部件的工作状况的仪器(表)。

3.2 电流表 battery charging condition indicator

显示蓄电池处于充电还是放电状态的指示器。

3.3 燃油油量表 fuel level indicator

显示燃油箱内油量的指示器。

3.4 发动机冷却液温度表 engine coolant temperature indicator/engine coolant thermometer 显示发动机冷却液温度的指示器。

3.5 发动机机油温度表 engine oil temperature indicator/engine oil thermometer

显示发动机机油温度的指示器。

3.6 发动机机油压力表 engine oil pressure indicator

显示发动机润滑系统中油压的指示器。

3.7 贮气筒压力表 air brake reservoir pressure indicator

显示制动系贮气筒内有效压力的指示器。

3.8 车速表 speedmeter

显示车辆行驶速度的指示器。

3.9 里程表 odometer

显示车辆行驶累计里程的指示器。

3.10 旅程表 trip odometer

易于归零的并能显示出车辆区段行驶里程的装置。

3.11 速度记录器 tachograph

能自动记录车速、行驶距离、时间间隔等的装置。

3.12 转速表 tachometer/revolution counter

显示发动机工作转速的指示器。

3.13 转数累积表 total revolution counter

显示发动机累积转数的指示器。

3.14 集中润滑系统油压表 centralized lubrication pressure indicator

显示集中润滑系统中润滑油工作压力的指示器。

3.15 自动变速器指示器 automatic gearbox indicator

显示自动变速器的工作挡位的指示器。

## 4 信号装置 tell-tale

4.1 信号装置 tell-tale

用光信号或音响信号显示车辆及其部件的功能状况的指示装置。

4.2 远光信号装置 high(upper)(driving)beam tell-tale

表示前照灯远光接通的信号装置。

4.3 近光信号装置 low(dipped)(passing)beam tell-tale

表示前照灯近光接通的信号装置。

4.4 前和(或)后雾灯信号装置 front and/or rear fog light tell-tale

表示前和(或)后雾灯接通的信号装置。

4.5 转向信号装置 direction indicator tell-tale

表示转向灯(左或右)已接通的信号装置。

4.6 危险警报信号装置 hazard warning system tell-tale

表示危险警报信号已接通的信号装置。该信号装置与转向信号灯可以是公用的，也可是独立的。

4.7 机油压力警报装置 oil pressure warning tell-tale 表示发动机润滑系统油压不足的信号装置。

4.8 机油油面警报装置 oil level warning tell-tale 表示机油油面低于或高于规定范围的信号装置。

4.9 温度警报装置 temperature warning tell-tale

表示发动机的冷却液的温度低于或高于规定范围的信号装置。

4.10 蓄电池充电状况警报装置 battery changing condition tell-tale 表示蓄电池充电状况或蓄电池本身的状态不良的信号装置。

4.11 燃油警报装置 fuel level tell-tale

表示燃油箱内存油即将用尽的信号装置。

4.12 前风窗电加热器信号装置 windscreen electric heater tell-tale(de-misting and de-frosting) 表示前风窗电加热器已接通的信号装置。

4.13 后窗电加热器信号装置 rear window electric heater tell-tale (de-misting and de-frosting) 表示后窗电加热器已接通的信号装置。

4.14 行车制动系警报装置 service brake system warning tell-tale 表示行车制动系统存在故障的信号装置。

警报装置亮时，表示可能有以下一种或几种情况：

——在制动系统中某处压力不足；

——制动衬片过量磨损；

——液压制动液量不足；

——制动能量储存不足。

4.15 ABS 警报装置 ABS tell-tale

表示防抱制动系统失效的信号装置。

4.16 驻车制动系警报装置 parking brake system warning tell-tale 表示驻车制动器还未松开的信号装置。

4.17 轮胎充气压力警报装置 tyre inflation pressure warning tell-tale 表示某一轮胎的充气压力已低于规定的最小工作压力的信号装置。

4.18 安全带警报装置 seat belt warning tell-tale

警告某个乘员的座椅安全带没有系住的信号装置。

4.19 被动约束气囊警报装置 passive restraint tell-tale

表示“撞车-扩张”的乘员的约束气囊机构不能充分工作的信号装置。

4.20 发动机自动停机警报装置 automatic enging stop tell-tale 表示发动机由于故障已自动地停机的信号装置。

4.21 门开警报装置 automatic door tell-tale 表示某个车门未关闭好的信号装置。

4.22 行李箱开启警报装置 rear hood (boot) tell-tale

表示行李箱未关闭好的信号装置。

4.23 乘员与驾驶员联系信号装置 passenger-driver communication tell-tale 表示乘员要求和驾驶员通话的信号装置。

4.24 无线电话信号装置 radio communication system tell-tale

表示管理中心要求和驾驶员通话的信号装置。

4.25 空气滤清器警报装置 air filter tell-tale 表示空气滤清器堵塞的信号装置。

4.26 机油滤清器警报装置 oil filter tell-tale 表示机油滤清器堵塞的信号装置。

4.27 驾驶室锁止警报装置 cab lock tell-tale

表示翻转驾驶室的锁止机构没有锁住的信号装置。

4.28 自动变速器警报装置 automatic transmission tell-tale 表示自动变速器系统出现故障的信号装置。

4.29 电控空气悬架警报装置 electronic air-suspension tell-tale 表示电控空气悬架系统出现故障的信号装置。

4.30 主动悬架警报装置 active suspension tell-tale 表示主动悬架系统出现故障的信号装置。

4.31 防撞警告系统警报装置 collision-avoidance warning system tell-tale 表示存在撞车危险或防撞系统存在故障的信号装置。

警告装置亮时，表示可能有以下一种或几种情况：

——与先行车车距过近；

——系统发生故障；

——激光雷达被污染。

4.32 自动变速器控制信号装置 automatic transmission control indicator 表示自动变速器超速挡被锁死的信号灯。

4.33 巡行控制信号装置 cruise control system indicator 表示车辆稳速行驶控制状态的指示灯。

4.34 防盗系统信号装置 anti-theft system indicator 表示车辆防盗系统处于启动状态的指示灯。
