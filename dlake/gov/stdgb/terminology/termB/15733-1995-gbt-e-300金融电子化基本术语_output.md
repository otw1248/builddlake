# 中华人民共和国国家标准

GB/T 15733—1995

# 金融电子化基本术语

# The basic terminology of financial computerization

## 1 主题内容与适用范围

本标准规定了金融电子化常用基本术语。

本标准适用于金融行业电子化应用领域。

## 2 术语

2.1 金融电子化 financial computerization

使用计算机、通信、网络技术与设备等手段以实现各类金融业务处理自动化。

2.2 电子资金转帐 electronic fund transfer(EFT)

在计算机系统中，资金以电子信息形式在帐户间转移。

2.3 大额支付系统 high value payment system(HVPS)

以电子方式逐笔实时处理每笔金额在限额以上和应客户要求每笔金额在限额以下的贷记交易，并以全额方式清算资金的应用系统。

2.4 批量电子支付系统 bulk electronic payment system(BEPS)

以电子方式批量处理借记交易及每笔金额在限额以下的贷记交易，并以净额方式清算资金的应用系统。

2.5 客户信息文件 customer information file(CIF)

与金融机构发生业务关系的有关客户的各种信息的文件，如包括客户的资信、企业财务信息等。

2.6 (帐号)校验数字 check digit

帐号组成部分之一，是利用帐号其他部分的数字，通过某种算法产生的数字标识，用于校验帐号的正确性。

2.7 货币代码 currency code 表示货币种类的代码。

2.8 自助银行(业务) self-service banking

银行利用专用电子设备为客户提供自我操作服务，以完成存取现金、查询帐户余额或资金转帐等业务。

从手工操作到计算机操作或从一种计算机系统到另一种计算机系统的业务数据的转换过程。

2.10 人机并行 parallel run

手工操作与计算机操作分别同时进行业务处理的方法。

2.11 集中式处理 centralized processing

一种计算机系统处理模式，其拓扑结构为星形网络，在中央结点和终端结点之间有一条链路连接，中央结点集中控制各终端结点的工作，并集中进行数据处理。

1996-06-01 实施



2.12 分散式处理 decentralized processing

一种计算机系统处理模式，指所有的数据处理工作分别在各个数据处理中心独立进行。

2.13 分布式数据处理 distributed data processing

除输入、输出功能外，将部分或全部处理、储存和控制功能分散在各数据处理结点、数据处理设备及其软件上协调进行的一种数据处理方式。

2.14 脱机业务处理 off-line operation

金融行业利用计算机进行业务数据处理的一种方式，指外围设备与中央处理机不直接相连或与实时控制系统、主程序不直接相连的数据处理。

2.15 联机业务处理 on-line operation

金融行业利用计算机进行业务数据处理的一种方式，指外围设备与中央处理机直接相连，并在中央处理机直接控制之下或与实时控制系统直接相连的数据处理。

2.16 减载业务处理 off-load operation

金融行业利用计算机进行业务数据处理的一种方式，指在业务处理高峰期间，为了减轻通信线路的“拥挤”，提高中央处理机处理速度，根据业务处理缓急要求，把部分允许暂缓处理的交易，不直接向中央处理机发送，而先记录到终端处理机的磁盘内，当业务高峰时间过去后，系统自动或再启动交易将磁盘交易传送到中央处理机处理。

2.17 (银行)电子日志 electronic journal

计算机对所处理的事项按时间顺序无遗漏地自动存储的记录。

2.18 个人识别号 personal identification number(PIN)

客户拥有的用来检验其身份的代码或口令。又称客户密码。

2.19 电子密押 electronic test key

根据银行之间的约定由计算机产生的支付业务专用的密码。

2.20 电子印鉴鉴别 electronic seal authentication

指银行利用计算机识别技术对印鉴真伪进行自动鉴别的一种技术。

2.21 (金融)报文 financial message

在发出金融机构(或其代理)和接收金融机构(或其代理)之间传输的信息，不包括用于传输目的的头和尾的信息。

2.22 报文鉴别 message authentication

用于报文发送方和接收方之间，保证报文来源、报文部分内容或全部内容有效性的技术。

2.23 报文鉴别代码 message authentication code(MAC)

报文中由约定计算产生的用于报文鉴别的代码。也称电子签名。

2.24 系统跟踪审计号 system trace audit number

由报文发送方定义的唯一标识交易的号码，该号码在整个交易期的全部报文中都保持不变。

2.25 密钥 cryptographic key

用于将明文转换成密文或者密文转换成明文的算法中的一个数值。

2.26 加密设备 cryptographic equipment

执行加密功能(如加密、鉴别、密钥生成)的设备。

2.27 数据完整性 data integrity

数据不被以非授权方式更改或破坏的性能。

2.28 加密校验数字 crypto check digit(CCD)

利用加密算法计算出的数字，用于提供数据完整性的一种校验方式。

2.29 银行专用机具 special banking machines and tools

根据银行或其他金融机构业务处理的特殊需要而设计和生产的专用机器设备和工具的总称。

2.30 自动柜员机 automatic teller machine(ATM)

代替银行出纳柜员基本职能、查询客户帐户信息和办理客户转帐的自动服务设备。

2.31 (银行)销售点终端 point of sales(POS)

电子资金转帐的一种终端设备。将其安装在消费现场，通过银行卡的使用，可实现消费活动的非现金货币支付。

2.32 金融交易卡 financial transaction card(FTC)

用于标识发卡者和持卡人，便于金融交易并为金融交易提供输入数据的一种卡片。

2.33 银行卡 bank card 同金融交易卡。
