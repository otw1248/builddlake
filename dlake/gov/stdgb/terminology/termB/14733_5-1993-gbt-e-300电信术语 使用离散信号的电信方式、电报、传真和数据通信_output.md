# 使用离散信号的电信方式、电报、传真和 数据通信

GB/T 14733.5-93

# Terminology for telecommunications Telecommunications using discrete signals, telegraphy, facsimile and data communication

本标准参照采用国际电工词汇 IEV 第 721《使用离散信号的电信方式、电报、传真和数据通信》(1985 年版)，并增补 CCITT 红皮书(1984)和蓝皮书(1988)及 GB 1417《常用电信设备名词术语》的相关内容。

## 1 主题内容与适用范围

本标准规定了有关电报、传真和数据通信方面的术语及其定义，所收集的术语主要涉及使用离散信号的电信方式，离散信号和使用离散信号的传输，电报和数据传输，传真以及电报和数据网、交换、操作和业务等五方面的术语。

本标准适用于电信技术领域。

## 3 术语

术语编号参照采用国际电工词汇 IEV 721 的编号(从 IEV 721 的第 721.11 条开始)。

3.1 使用离散信号的电信方式

721.11.01 远程处理 teleprocessing

电信与数据处理技术的结合，以实现在远端处理信息。

721.11.02 数据 data

用适合于自动处理的方式来表示的信息。

721.11.03 数据通信 data communication

数据处理设备之间传递信息的电信方式。

721.11.04 数据传输 data transmission

利用电信手段将数据从一地输送到另一地。

721.11.05 远程信息处理(业务) telematics(service)

通常利用远程处理技术使用户接收和发送公用信息或专用信息，其中包括文件查询、预定、银行往来或商业经营等的电信业务。例如：传真、可视图文、智能用户电报、电写……等。它补充了传统的电报、电话业务。

721.11.07 （字符）电报 $ ^{[1]} $  （alphabetic）telegraphy；character telegraphy

一种使用编码信号的电信方式。该方式中的每个信号或信号码元组都对应一个字符，例如：对应一个字母、一个数字或一个标点符号。

721.11.08 电写 telewriting

一种电信方式，在发送端将人工正在书写或绘制的文字、图形信息发送出去；在接收端记录笔受电信号控制实时地作相应移动，或同时在屏幕上显示出收到的内容。

注：接收端的收件为图形文件形式时，则可使用英文术语“teleautography”。

721.11:09 传真 facsimile(fax)

对文件(页面)进行扫描和光电变换的处理方式。在发送端将被扫描的固定图像变换成电信号传送给远端的接收机；在接收端将接收信号反变换，以得到被扫描原稿的副本。

721.11.10 文件传真 document facsimile 以两种光密度级复制原稿的传真。

721.11.11 图片传真 picture facsimile

其副本可呈现有灰度的多色调密度的传真。

721.11.12 相片传真 phototelegraphy

用照相处理工序复制副本的图片传真。

721.11.13 电报(业务) telegraph(service)

用电报技术传输信息的任何一种业务。

注：电报业务包括公用电报、用户电报等业务。

721.11.14 公用电报(业务) telegram(service), public telegram service 向公众提供电报发送并投送给收报人的电报业务。

721.11.15 用户电报(业务) telex (service)

利用起止电报机和公用电报网电路，用户可以直接和临时地相互进行通报的电报业务。

721.11.16 高速用户电报(业务) supertelex(service), high speed telex 以高于 50Bd 的调制速率工作的用户电报业务。

721.11.17 智能用户电报(业务) teletex(service)

一种以电报形式传送电文的电信业务，它向用户电报业务提供若干附加性能，特别是更多的打字机功能和某些远程电文处理功能。

注：术语“teletex(智能用户电报)”和“teletext(图文电视)”属于两种不同的概念。

### 3.2 离散信号和使用离散信号的传输

3.2.1 时间离散信号和数字信号；信号码元

721.21.01 时间离散信号 discretely-timed signal

在时间上顺序排列的若干码元所组成的信号，其中的每个码元都具有可以传递信息的一种

或多种特征，例如，它在时间上的位置、持续时间、波形和幅度等。

721.21.02 数字信号 digital signal

其信息用若干明确定义的离散值来表示，且离散值的某一特征量可以按时提取的时间离散信号。

721.21.03 模拟信号 analogue signal

表示信息的特征量可以在任何瞬时发生变化，而且可以在一连续区间内呈现为任意值的信号。

注：例如：一个模拟信号可以连续地随另一个表示信息的物理量的变化而变化。

721.21.04 （信号的）抽样 sampling(of a signal)

对信号抽取样值的过程。通常以相等的时间间隔来抽取样值。

721.21.05 （量的）量化 quantizing(of a quantity)

一种过程，将某一量的连续可能的取值范围分成若干个预定的相邻间隔，在一给定间隔内的任何值均用该间隔内单一的预定值来表示。

721.21.06 信号码元 signal element

构成离散信号的各个部分,每个组成部分以一种或多种特征量(如它的极性、幅度、持续时间或相对位置)区别于其他各部分。

721.21.07 数字(位) digit

从整数有限集中选择出来的一个组成元素。

注：① 在数字传输中，可以用一个信号码元来表示一个数字。

② 在数字传输使用的设备中，一个数字可以用一个特定的物理状态（例如，磁芯的两个磁化状态）来表示。

721.21.08 二进制数字(位) binary digit, bit

通常用于表示信息的二进制集中的一位数字，简称比特。

注：在电报传输中不能用“比特(bit)”来表示“单位码元(unit element)”和“单位(unit)”。

721.21.09 二进制符号 binary figure

在二进制记数法中，表示数字所用的两个符号（“0”或“1”）中任何一个。

721.21.10 n 进制数字(位) n-ary digit n 进制集中的一位数字。

721.21.11 字节 byte

作为一个整体来处理的一组规定数目的二进制数字。

721.21.12 二位字节 doublet byte

由两个二进制码元组成的字节。

721.21.13 三位字节 triplet byte

由三个二进制码元组成的字节。

721.21.14 四位字节 quartet byte

由四个二进制码元组成的字节。

721.21.15 五位字节 quintet byte

由五个二进制码元组成的字节。

721.21.16 六位字节 sextet byte

由六个二进制码元组成的字节。

721.21.17 七位字节 septet byte

由七个二进制码元组成的字节。

721.21.18 八位字节 octet byte

由八个二进制码元组成的字节。

注：没有任何限定条件下，“byte（字节）”这个词不应作为“octet（八位组）”的同义词。

721.21.19 n 位字节 n-bit byte

由 n 个二进制码元组成的字节。

721.21.20 代码 $$ 电码 $$  $ ^{[1]} $  code

一组用字符、符号或信号码元以离散形式表示信息的明确的规则体系。

721.21.21 代码变换 code conversion

将同一信息从一种代码表示形式变换成另一种代码表示形式。如：将一种电报码的字符信号或字符信号组变换成另一种电码的相应信号或信号组。

721.21.22 特征状态 significant condition

依照某种电码确定其意义的信号码元的状态。

721.21.23 特征状态数 number of significant condition

信号码元依照某种电码所能呈现的不同特征状态的数目。

721.21.24 二态 two condition

表明所使用的特征状态数是二的限定性术语。

721.21.25 三态 three condition

表明所使用的特征状态数是三的限定性术语。

721.21.26 四态 four condition

表明所使用的特征状态数是四的限定性术语。

721.21.27 转换 change over

离散信号由一种特征状态到另一种特征状态的变换。

721.21.28 过渡 transition

在离散信号中，把两个具有不同特征状态的相邻信号码元分隔开来的瞬变现象。

721.21.29 特征瞬时 significant instant

发生转换的那一瞬时。

注：从一种特征状态改变成另一种特征状态的瞬时。

721.21.30 特征间隔 significant interval

两个相邻的特征瞬时之间的时间间隔。

721.21.31 特征间隔的理论持续时间 theoretical duration of a significant interval 由电码所规定的特征间隔的精确持续时间。

注：在确定该持续时间时，必须考虑标准化的调制速率，如有必要，还要考虑平均调制速率。

721.21.32 单位间隔 unit interval

采用等长电码或等时调制系统中的最大时间间隔。诸如电报调制（或解调）中信号的特征间隔的理论持续时间均为这种间隔的整数倍。

721.21.33 单位码元 unit element

持续时间等于单位间隔的信号码元。

3.2.2 使用时间离散信号或数字信号的传输

721.22.01 等时 isochronous

在用特征瞬时表征信号或随时间变化的现象时，把特征瞬时分开的时间间隔，其持续时间在理论上等于单位间隔的持续时间或它的整数倍。

721.22.02 不等时 anisochronous

在用特征瞬时表征信号或随时间变化的现象时，把特征瞬时分开的时间间隔，其持续时间并不一定要全部等于单位间隔的持续时间或它的整数倍。

721.22.03 起止信号 start-stop signal

由一连串等时信号码元组构成的信号。其中的每个等时信号码元组都表示一个字符或一个码组，而且，其持续时间等于单位间隔持续时间的倍数，并由不固定的持续时间隔开。

721.22.04 同步的 synchronous

a. 在时间上与某一特定的现象或量值反复同时发生；

b. 每一个成对信号或时变现象都反复地用成对的、以恒定时间位移分隔的相应特征瞬时来表征。

721.22.05 同步传输 synchronous transmission

采用等时信号的传输。其中，发送机和接收机连续地在对应的特征瞬时之间以恒定的时间差进行工作。

721.22.06 不等时传输 anisochronous transmission

一种传输方式，在同一码组或字符内的任意两个特征瞬时之间的时间间隔总是单位间隔的整数倍，而在不同码组或字符的两个特征瞬时之间的时间间隔并不总是单位间隔的整数倍。

721.22.07 起止传输 start-stop transmission

采用起止信号的传输。其中，在每个表示字符或码组的等长信号码元组前部有一个起信号，在其后部有一个止信号。

721.22.08 （时间离散信号的）数字传输 digital transmission(of a discretely-timed signal)

用数字信号表示时间离散信号的传输。对于该时间离散信号是以大于其最小间隔倒数的速率来进行抽样和量化。

721.22.09 字符 character

经商定用来组织、表示或控制信息的元素集中的一个组成元素。

注：字符可以是字母、数字、标点符号或其他符号。它亦可用于扩充功能控制，例如：包含在报文中的间隔、回车或换行。

721.22.10 字符信号 character signal 表示字符的一组信号码元。

721.22.11 码组[(数据)块] block

由于技术上或逻辑上的原因所构成的、可作为一个整体处理的数字或字符序列。

721.22.12 码组起始信号 start-of-block signal 标识码组开始的信号。

721.22.13 码组结束信号 end-of-block signal 标识码组结束的信号。

721.22.14 码组信号 block signal 表示码组的一组信号码元。

721.22.15 起(始)信号 start signal

在起止传输中，位于每组信号码元前部的信号。这个信号使接收机件做好接收该组信号的各个码元的准备。

721.22.16 起(始)码元 start element

只有一个信号码元的起信号，其持续时间通常为一个单位间隔。

721.22.17（停）止信号 stop signal

在起止传输中，位于每组信号码元后部的信号。这个信号使接收机件做好接收下一个起信号的准备，或者使接收机件静止。

721.22.18 (停)止码元 stop element

只有一个信号码元的(停)止信号。其持续时间等于或大于规定的最小值。

721.22.19 A 态 A condition

在起止传输中，起码元的特征状态。

721.22.20 Z 态 Z condition

在起止传输中，止码元的特征状态。

721.22.21 等长电码 equal length code

字符信号均由相同数目的具有相同持续时间的单位码元所组成的电码。

721.22.22 n 单位代码 n-unit code

其字符信号均由 n 个单位码元所组成的等长代码。

721.22.23 代码组合 code combination

按照 n 单位代码构成的 n 单位码元的组合，这种 n 单位代码为每个单位码元指定一个特征状态。

721.22.24 A 码元 A element

在代码组合中被指定为 A 态的单位码元。

721.22.25 Z 码元 Z element

在代码组合中被指定为 Z 态的单位码元。

721.22.26 调制速率 modulation rate

单位间隔的持续时间的倒数或信号码元最短理论持续时间的倒数。

721.22.27 波特 Baud (Bd)

在时间离散信号(或数字信号)中，调制速率的单位或持续时间恒定的信号码元传送速率的单位；波特数等于最短信号码元或单位间隔的持续时间(以秒为单位)的倒数。

721.22.28 速率变换 speed conversion

将收到信号的调制速率变换成适合于后继设备的不同调制速率。

721.22.29 实际传送速率 actual transfer rate; transfer rate  

在单位时间内两点之间传送的二进制数字或字符或码组的平均数。

721.22.30 二进制速率 binary rate

信号码元在传输链路上的集合传送速率，它以若干比特/秒的归一化形式来表示。

注：①信号速率由下式给出：

 $$ \sum_{i=1}^{i=m}\frac{1}{T_{i}}\log_{2}N_{i} $$ 

式中：m——并行传输信道数；

 $ T_{i} $ ——第i条信道中信号码元的最短理论持续时间，s；

 $ N_{i} $ ——第i条信道上调制的特征状态数。

②二进制速率的单位为比特/秒。

721.22.31 有效传送速率 effective transfer rate

在单位时间内两点之间传送并被接收端确认为有效的二进制数字(位)或字符或码组的平均数。

#### 3.2.3 传输信道[通路] $ ^{[1]} $ 和传输方法

721.23.01 电话型信道 telephone-type channel

其特性适合于传输话音但用于传输其他信号的传输信道。

721.23.02 电话型电路 telephone-type circuit

两点之间允许双向传输的一对相互关联的电话型信道。

721.23.03 宽带信道 wideband channel

带宽大于电话信道的传输信道。

721.23.04 复用(功能) multiplex

将一公共信道划分成若干单独的分信道，而每一分信道都能在同一方向上独立传送信号的一种设备或功能。

721.23.05 复用 multiplexing

将来自若干单独的分信道的独立信号复合起来在一公共信道的同一方向上进行传输的过程。

721.23.06 分用 demultiplexing

恢复复用信号中的合成信号，并将这些信号在各自独立的信道中还原的过程。

721.23.07 复用器 multiplexer 实现复用的设备。

721.23.08 分用器 demultiplexer 实现分用的设备。

721.23.09 复用分用器 muldex

在同一设备中，由工作于相反传输方向的复用器和分用器构成的组合体。

721.23.10 复用信号: multiplex signal, multiplexed signal 将若干信号进行复用后所形成的合成信号。

721.23.11 时分复用 time division multiplexing (TDM)

为在一公共信道上传输若干个独立的信号，将他们分别配置在一分立的、周期性的时间间隔上复用。

721.23.12 频分复用 frequency division multiplexing (FDM)

为在一公共信道上传输若干个独立的信号，将他们分别配置在一分立的频带上的复用。

721.23.13 同类复用 homogeneous multiplex

对所有工作于相同二进制速率下的分信道的复用。

注：对二进制速率可以增加某些附加条件。

721.23.14 异类复用 heterogeneous multiplex

对所有工作于不同二进制速率下的分信道的复用。

注：对二进制速率可以增加某些附加条件。

721.23.15 单工 simplex

信息在两点之间只能单方向发送而不能双向同时发送的工作方式。

721.23.16 双工 duplex

信息在两点之间能够在两个方向上同时发送的工作方式。

721.23.17 差动式双工 differential duplex

一种双工系统，在发送端电流向相反的方向分流，抵消它们对本端收报机件的影响；而在接收端，电流则以相同的方向流过，并使收报机件动作。

721.23.18 桥式双工 bridge duplex

一种双工系统，每一端的收报机件对本端所发电流是按惠斯登电桥电位平衡的原理取得中和。

721.23.19 增压式双工 incremental duplex

一种差动式双工或桥式双工系统。在这种系统中，当两个站同时发送一个同样的信号码元时，加到线路上的电位是相加的。

721.23.20 反压式双工 opposition'duplex

一种差动式双工或桥式双工系统。在这种系统中，当两个站同时发送一个同样的信号码元时，加到线路上的电位是相反的。

721.23.21 单向(传输) unidirectional

在链路上，只能在一个预先指定的方向传送用户信息。

721.23.22 双向(传输)bidirectional

在链路两点之间能够双向同时传送用户信息。

注：在两点之间，相关联的传输信道上的容量和二进制速率在两个方向上无需相同。

721.23.23 单向(建立呼叫) one way

呼叫建立总是在单向上发生的工作方式。

721.23.24 双向(建立呼叫) both way

呼叫建立在双向上发生的工作方式。

注：两个方向上的业务流量无需相同。

721.23.25 频道（frequency）channel

频谱中用于传送信号的部分，可以用两个规定的限值或中心频率及相关联的带宽或任何等效的表示法来确定。

注：为了用单工方式进行双向通信，可以对频道进行时间分割。

3.2.4 直流传输

721.24.01 直流传输 direct current transmission

对导线施加由直流电源供给的电压，以实现具有不同特征状态的离散信号传输的方式。

721.24.02 单流传输 single current transmission

靠施加同一极性的电压而产生单向电流的直流传输方式。

721.24.03 双流传输 double current transmission

靠施加极性相反的两个电压而产生方向相反的二态电流的直流传输方式。

721.24.05 话下传输 sub-telephone transmission

在电话线上使用一个或多个比电话信号占用频带低的频带所进行的传输。

721.24.06 幻线电路 phantom circuit

从两条实线电路的导线引导出来的一条额外电路，每一条实线电路的两根导线实际上并联使用。例如：在两条电话电路上叠加的一条电报电路。

721.24.07 地回幻线电路 earth phantom circuit

从一条实线电路的导线引导出来的一条额外电路，这两根导线实际上并联使用，并通过两个端点之间的大地或水域构成回程。例如：通过大地回程在一条电话电路上叠加的一条电报电路。

721.24.08 地回双幻线电路 earth-return double phantom circuit 从两对并联使用的金属导线引导出来的一条额外的地回电路。

721.24.09 双幻线电路 double phantom circuit

从两条幻线电路的导线引导出来的一条额外电路，每一条幻线电路的四根导线实际上并联使用。

721.24.10 德西特电报转换器 Desieter telegraph translation

一种装置。这种装置在两条导线的不同方向上提供两条独立工作的直流传输通路。

注：这种装置是为了避免市区用户电报网使用地回制而设计的。

3.2.5 载波传输——复用

721.25:01 载波传输 carrier transmission

利用发送机发出的离散信号去调制一个或多个载频的传输方式。

721.25.02 时分复用传输 time division multiplex transmission 采用时分复用技术的传输。

721.25.03 差分调制 differential modulation

任何一个信号码元的特征状态与其前面那个信号码元的特征状态相关的调制方式。

721.25.04 固定基准调制 modulator with a fixed reference

使每个信号码元的特征状态通过与预先选作基准的信号码元进行比较来确定的调制方式。



721.25.05 幅移键控 amplitude shift keying, amplitude shift modulation

一种调制方式，用于调制的离散信号的每一种特征状态，都用周期性正弦振荡的一个规定幅度值来代表。

721.25.06 频移键控 frequency shift keying (FSK)

一种角调制方式。其周期性正弦振荡的频率在一组离散值之间变化，而每一个离散值都表示用于调制的离散信号的一种特征状态。

721.25.07 相移键控 phase shift keying (PSK)

一种角调制方式，用于调制的离散信号的每一种特征状态，都用周期性正弦振荡的一个规定相位来代表。

721.25.08 差分相移键控 differential phase shift keying (DPSK)

一种角调制方式。这种调制方式用已调波相位的不连续变化来表示调制的离散信号的每个特征状态，该已调波相位的不连续变化是相对于前一信号码元的相位而言的。

721.25.09 倒相调制 phase inversion modulation 每个相移的取值都相差半个周期的相移键控。

721.25.10 四相相移键控 quadrature phase shift keying (QPSK)

一种相移键控方式，其相移取四种不同的值，各值间的差均为 $ 90^{\circ} $ 的倍数。

721.25.11 （频移调制的）调制指数 modulation index (in frequency shift modulation) 在二态频移键控方式中，频移值(赫)对调制速率(波特)的比值。

721.25.12 相干调制 coherent modulation

在表征载波相位的瞬时与已调波的特征瞬时之间存在着某种预定关系的调制方式。例如：载波过零点时即发生相位反转的倒相调制。

721.25.13 话上传输 super-telephone transmission

使用频率高于话音频率(通常是高于3400Hz)的单频交流或多频交流的传输方式。

721.25.14 带间传输 interband transmission

用话路中两条较宽频带之间的窄频带构成传输信道的载波传输方式。

721.25.15 带内传输 intraband transmission

在话路频带内占用一段适当的窄频带，使电话和时间离散信号在其中同时进行传输的载波传输方式。

721.25.16 语音+单工(设备) speech+simplex(equipment)

利用时间离散信号加在单载频上来提供单工传输的话路带内传输设备。

721.25.17 语音+双工(设备) speech+duplex(equipment)

利用时间离散信号加在双载频上来提供双工传输的带内传输设备。

721.25.18 双分(复用)器 bisector

一种同步时分复用设备。利用这种设备，可在一条工作于标称调制速率的传输信道上传输相当于两个与这种标称调制速率相同的起止字符信号，从而使字符的实际传输速率增加一倍。

721.25.19 第一级数据复用器 first data multiplexer

一种设备，以相同的或不同的标准接收若干个二进制数字序列，并按时分制以相当于脉码复用信道时隙的速率把它们组合成单一的二进制数字流。

721.25.20 第二级数据复用器 second data multiplexer

一种设备，以相同的调制速率接收来自几个第一级数据复用器的若干个二进制数字序列或接收若干信号，并按时分制以脉码复用基群的数字率把它们组合成单一的二进制数字流。

721.25.21 帧 frame

按顺序排列并重复出现的一组时隙。这些时隙构成信号的一个完整周期，在该周期中每个时隙的相对位置都是确定的。

721.25.22 子帧 sub-frame

每帧中固定数目的非邻接信道时隙汇合在一起，提供一个数字信道，其规定速率高于单个信道时隙速率。

721.25.23 帧定位 frame alignment

由接收设备产生的帧与接收到信号的帧具有预定的固定相位关系，从而使每一帧中的单个时隙能唯一地识别出来。

721.25.24 帧定位恢复时间 frame alignment recovery time

从接收终端设备得到有效帧定位信号到帧定位建立之间所经过的时间。

注：帧定位恢复时间包括重复检验帧定位信号的有效性所需要的时间。

3.2.6 传输性能——时间畸变

721.26.01 完好(离散)信号 perfect(discrete) signal

是一种时间离散信号，该信号的所有特征间隔都与正确的特征状态相吻合，并且精确地与代码所确定的理论持续时间保持一致。

721.26.02 理想瞬时 ideal instant

特征瞬时与每个系统所规定的某种条件相一致的瞬时。

721.26.03 时间畸变[电报畸变] time distortion [telegraph distortion]

当特征瞬时与相应的理想瞬时不一致时，对离散信号所产生的不良影响。

721.26.04 不正确(离散)信号 incorrect (discrete) signal

其中一个或多个码元的特征状态与代码所预定的特征状态不一致的时间离散信号。

721.26.05 （特定特征瞬时的）单个畸变度 degree of individual distortion (of a particular significant instant)

给定的特征瞬时偏离相应理想瞬时的代数值相对于规定的单位间隔之比。

注：① 按照惯例，当特征瞬时在理想瞬时之后发生时，把偏移视为正值。反之，当特征瞬时在理想瞬时之前发生时，则把偏移视为负值。

②单个畸变度通常用百分比来表示。

721.26.06 等时畸变度 degree of isochronous distortion

在一段规定的时间间隔内，就等时信号的各特征瞬时而言，单个畸变的最大值与最小值的代数差对单位间隔的平均持续时间之比。

721.26.07 起止畸变度 degree of start-stop distortion

在一段规定的时间间隔内，到达的起止信号其特征瞬时的各单个畸变度的最大绝对值。

721.26.08 总起止畸变度 degree of gross start-stop distortion

在假设的单位间隔完全适合于标称调制速率时所确定的起止畸变度。

注：按照惯例，当特征瞬时在理想瞬时之后发生时，把总起止畸变度视为正值。反之，当特征瞬时在理想瞬时之前发生时，则把总起止畸变度视为负值。

721.26.09 同步起止畸变度 degrree of synchronous start-stop distortion

在假设的单位间隔适合于实际的平均调制速率时所确定的起止畸变度。

注：实际上，同步起止畸变度是靠调节畸变测试仪的扫描速率来进行测量的。

721.26.10 标准化测试畸变度 degree of standardized test distortion

当发报端的信号是完好的并符合规定的电文时，在规定时间内所测得的接收信号的单个畸变度。

721.26.11 （传输通路的）固有畸变 inherent distortion (of a transmission channel)

当传输通路输入端的信号完好时，在其输出端接收信号的时间畸变。

注：①固有畸变包括传输通路中产生的所有畸变，例如：偏畸变、特性畸变和不规则畸变等。

②固有畸变的概念可以扩展到传输通路的组成部分，例如：电报继电器、电报中继器或交换机。

721.26.12 偏畸变 bias distortion

因特征瞬时的两个转换方向的平均单个畸变度不同，对二态离散信号所引起的时间畸变。

721.26.13 偏畸变度 degree of bias distortion

对不同值(A 态或 Z 态)的各信号码元进行测量,其平均持续时间的差值与单位间隔之比。

721.26.14 特性畸变 characteristic distortion

在特定特性的传输通路中，因信号传输而产生瞬态效应所引起的时间畸变。

721.26.15 不规则畸变 fortuitous distortion

对通路或设备有影响的偶发因素所引起的时间畸变。这种时间畸变使得对任何特征瞬时的单个畸变度都不能预测。

721.26.16 周期性畸变 cyclic distortion

由带有周期性特征因素而引起的时间畸变，这种时间畸变使特征瞬时序列中的各单个畸变度显出其周期性特征。

721.26.17 常规畸变度 conventional degree of distortion

一种单个畸变度，它在一段延续的时间内，很少超过规定概率。

721.26.18 远端畸变 teledistortion

根据在传输通路输出端所做的观察，对这条通路输入端的某个特定离散信号估计出来的时间畸变。

721.26.19 发报机畸变 transmitter distortion

在规定的标准条件下，在发报机输出端测得的时间畸变。

721.26.20 超前不等时并行畸变度 degree of early anisochronous parallel distortion



721.26.20 超前不等时并行畸变度 degree of early anisochronous parallel distortion

一个比值，其分母是特征间隔的理论持续时间，其分子是这个理论持续时间与另一持续时间之间的最大差值。另一持续时间是指在离散信号的所有相干特征瞬时当中最前一个与下一组相干特征瞬时当中最前一个相隔的持续时间。

721.26.21 滞后不等时并行畸变度 degree of late anisochronous parallel distortion

一个比值，其分母是特征间隔的理论持续时间，其分子是所有并行通路内离散信号所出现的最后一个与最前一个相干特征瞬时之间的最大时间差值。

721.26.22 （收报机或终端设备的）改正力 margin(of a receiver or terminal)

在规定条件下当信号到达输入端时，打印收报机或终端能正确译码所允许的单个畸变度的

最大值。

721.26.23 净改正力 net margin

调制速率为标称值时，设备输入端的改正力。

721.26.24 （给定设备的）有效改正力 effective margin (of a given apparatus) 在实际工作条件下，在指定的收报机上测得的改正力。

721.26.25 （某型设备的）标称改正力 nominal margin (of a type of apparatus)

在标准的工作和调整条件下，对某种给定类型设备所规定的有效改正力的最小值。

721.26.26 理论改正力 theoretical margin

假定设备在完好条件下进行工作，从这个设备的制造数据估算出来的改正力。

721.26.27 （起止设备的）改正力 margin (of a start-stop apparatus)

利用起止设备能对所有字符信号正确译码所允许的总起止畸变度的最大值。这些字符信号可能逐一出现或按照标准调制速率快速出现。

721.26.28 同步(起止)改正力 synchronous (start-stop) margin

将输入信号的调制速率调整到收报机时基特性的最佳值时，所测得的起止设备改正力的最大值。

721.26.29 同步收报机的改正力 margin of a synchronous receiver 用等时畸变度确定的同步收报机的改正力。

3.2.7 差错和差错控制

721.27.01 损伤 mutilation

使信号码元不正常地从一种特征状态变成另一种特征状态。

721.27.02 差错数字 erroneous digit 由于受损伤而被变义的数字。

721.27:03 差错二进制数字 erroneous binary digit 由于受损伤而被变义的二进制数字。

721.27.04 差错字符 erroneous character

其中含有一个或多个差错数字的字符。

721.27.05 差错码组 erroneous block

其中含有一个或多个差错数字的码组。

721.27.06 单个差错 single error

前后至少各有一个正确数字的差错数字。

721.27.07 双位差错 double error

在同一字符或码组中具有两个差错数字组成的群。

721.27.08 三位差错 triple error

在同一字符或码组中具有三个差错数字组成的群。

721.27.09 差错突发 error burst

在含有差错的一组数字中，两个相邻差错数字之间的正确数字的个数小于规定的数目，这些差错数字可以作为一个单元来处理。这样的差错数字称为差错突发。

721.27.10 差错突发长度 error burst length

从差错突发的第一个差错数字开始到最后一个差错数字结束的这一段序列中的数字个数。

721.27.11 数字差错率 digit error ratio

接收到的差错数字的个数与在相应期间发送出的正确数字的总数之比。

721.27.12 字符差错率 character error ratio

接收到的差错字符的个数与在相应期间发送出的正确字符的总数之比。

721.27.13 码组差错率 block error ratio

接收到的差错码组的个数与在相应期间发送出的正确码组的总数之比。

721.27.14 差错控制 error control

任何一种用来减少信息在记录、处理及传送中发生差错的技术措施。

721.27.15 码组检验 block check

按照一定的规则对数据码组的结构进行检验所遵循的差错控制过程。

721.27.16 差错检测 error detection

使用适当的代码在接收端发现差错的差错控制方法。

721.27.17 不重发的差错检测 error detection without repetition

这种差错检测将接收端发现的每个差错字符或每个差错码组从数据中删除并在删除的地方留下适当的标志，或者将差错保留并标明差错的存在。

721.27.18 环路检验 loop checking

将接收到的数据从接收端自动地重发给发送端，并与存储在发送端的原始数据进行比较的差错控制方法。

721.27.19 回显 echoplex

让每个发送出的字符从远端站返回后在本机得到显示的环路检验方法。

721.27.20 纠错 error correction

使用纠错码、检错码或环路检验来达到纠正绝大部分差错的差错控制。在使用检错码和环路检验的情况下，对识别出来有差错的信号进行自动重发。

721.27.21 检错及反馈重发系统 error detecting and feedback system (ARQ)

一种使用检错码纠错方式的系统。在这种系统中，接收端发现的每一差错都能导致在反向信道上向发送端发出功能信号，命令发送端重发最后发出的固定信号序列。

721.27.22 信息反馈重发纠错 error correction by information feedback and repetition

采用环路检验并随后自动重发识别出有差错的字符信号或码组信号的纠错方式。

721.27.23 冗余码 redundant code

符号数或信号码元数比表示信息所必需的数目多的代码。

例如：1）使用国际二号电码，所有字符的五单位码不是冗余码。

2）诸如仅使用由四个 A 态码元和三个 Z 态码元构成的国际三号电码的七单位码是冗余码。



721.27.24 码字 code word

在冗余码中可以有效地记录下来的代码组合。

721.27.25 汉明间距 Hamming distance

两个等长码字的对应位中不相同的数字位的个数。

721.27.26 防错码 error protection code

为进行检错或者同时进行检错和纠错而设计的代码。

721.27.27 检错码 error detecting code

其结构规则可以把引起这种结构偏离的任何差错自动检测出来的冗余码。

721.27.28 纠错码 error correcting code

能同时把检测出的部分差错自动纠正的检错码。

721.27.29 定比码 constant ratio code

在每个字符或功能信号中使用的各类码元均具有指定数目的检错码。

注：在 n 个码元中包含有 P 个 A 态码元的 n 单位二态定比码用“n 中取 P 码”来表示。

721.27.30 范杜伦码 Van Duuren code

同步无线电报中所使用的七单位二态定比码。这种定比码在每个字符信号中使用4个A态码。

注：这种代码是由 CCIR 建议 342 规定的，其字母表为国际三号电报码。

721.27.31 循环码 cyclic code

码字在每次循环移位以后仍是码字的代码。

721.27.32 争用 contention

当两个站或多个站在同一条公用传输信道上同时试图进行发送时，或者当彼此通常以单工方式操作的两个站同时试图进行发送时所引起的状态。

721.27.33 恢复规程 recovery procedure

为解决站间争用或纠正在传输中产生的某些差错而设计的一整套规则。

721.27.34 （自动重发纠错传输的）时间效率系数 efficiency factor in time (of a transmission with automatic repetition for the correction of errors)

在规定的调制速率下无重发控制时，自动发送一份电文所需的时间与按给定的差错率采取重发差错控制接收同一份电文实际所需要的时间之比。

721.27.35 重发信号 signal repetition

在检错和反馈重发纠错系统中，为要求重发或再次传输而使用的功能信号。

注：在国际三号电码中，这种信号对应于代码组合 AZZAZAA。

721.27.36 重发周期 repetition cycle

是一个字符序列，其最小字符数取决于检错和反馈重发系统的环路时延，该时延是提供自动重发信息所必需的。

721.27.37 请求周期 request cycle

当检测到差错时，在检错和反馈重发系统中所要求的重发周期。

注：见 CCIR 的建议 342。

721.27.38 响应周期 response cycle

当收到重发信号时，在检错和反馈重发系统中发送的重发周期。

注：见 CCIR 的建议 342。

721.27.39 非打印周期 non-print cycle

在自动检错和反馈重发系统中，接收机从检测到差错或从开始重发信号所用的操作时间，它与重发周期的持续时间相同，在这段时间内收到的所有信号均不打印。

注：见 CCIR 建议 342。

721.27.40 重发选通 gated RQ

在非打印周期内检测是否出现重发信号的过程。

注：见 CCIR 建议 342。

721.27.41 重发测试 tested RQ

在非打印周期内，检查信号是否重发以及接收到的所有字符中 A 码元数与 Z 码元数之比值是否正确的过程。

721.27.42 检测重发周期 testing repetition cycle

检查信号是否已经重发以及接收到的所有字符中 A 码元数与 Z 码元数之比值是否正确的非打印周期。

721.27.43 预校正 precorrection

在信道的发送端,人为地给信号加上一定的时间畸变,以便全部地或部分地补偿因信号传输所引起的特性畸变的影响。

721.27.44 信息比特 information bit

由数据源产生但不用于传输系统差错控制的比特。

721.27.45 业务位 service digit

与信息比特结合起来用于数据交换的二进制码元。

721.27.46 检验比特 check bit

与字符信号或码组信号结合起来用于检验该字符或该码组当中有无差错的二进制码元。

721.27.47 字符检验 character check

对每个字符的构成是否符合防错码规则而进行检验的差错控制过程。

721.27.49 检验位 check digit

按照一定规则，用某种非冗余码定义的与若干组码元相关联的数字位。其目的在于将这种非冗余码变换成一种防错码。

721.27.50 偶检验 even parity check

在表示一个字符或一个码组的二进制数字中，检验同一类码元（“1”或“0”）的个数是否为偶数的差错控制过程。

721.27.51 奇检验 odd parity check

在表示一个字符或一个码组的二进制数字中，检验同一类码元（“1”或“0”）的个数是否为奇数的差错控制过程。

721.27.52 奇偶检验码 parity code

每一个字符信号或码组信号所包含的同一类码元(“1”或“0”)的数目总是偶数或总是奇数的防错码。

721.27.53 奇偶检验位 parity digit

附加到某个 n 进制数字组中的一个 n 进制数字，它使所有数字的模 n 和总是等于 0 或等于任何预定的数字。

注：在二进制的情况下，奇偶检验位即为奇偶检验比特。

721.27.54 弗丹自动重发系统 automatic repetition Verdan system

一种电报系统，其中的每个信号都不止一次地自动发送，被重复的信号以恒定的延时与原发信号隔开。

721.27.55 （信号）质量检测 （signal）quality detection

根据诸如信号幅度、信噪比或时间畸变等指标来确定所接收到的信号是否已经偏离了可接受的形式，其中不包括对被复原的数字信号的含义或值所进行的检验。

721.27.56 剩余差错率 residual error-ratio, undetected error-ratio

不正确地接收下来而未被差错控制设备检测到或未被纠正过来的比特、单位码元、字符或码组的数目与发出相应的总数之比。

721.27.57 信息加密 information security

防止信息未经认可被泄露、转移、修改或破坏(不管是有意的还是无意的)所做的保护。

721.31.16 等时调制 isochronous modulation

每个特征间隔的持续时间均等于一个单位间隔或等于单位间隔的整数倍的电报调制。

721.31.17 同步电报 synchronous telegraphy 使用同步传输的电报系统。

3.3.2 功能控制和功能信号

721.32.01 功能控制 function control

设备执行某项基本操作的控制，其中不包括记录或打印报文或数据中的字母、数字、标点符号或图形符号。

721.32.02 功能信号 function signal

传送某一功能控制所用的一组信号码元。

721.32.03 控制字符 control character

在特定的上下文中出现的字符，这种字符启动、修改或停止某项操作。

注：①可以将控制字符记录下来，在随后的动作中加以使用。

② 控制字符并不是图形字符，但在某种环境中可以有一个图形表示。

721.32.04 （电报和数据传输的）译码 translation (in telegraph and data transmission) 收报机根据接收信号重建报文中的正文（其中包括把它们记录下来）的功能。

721.32.05 （收报机的）选择 selection (in a telegraph receiver)

译码的一项基本操作。通过这项操作，从接收的信号中选出所要打印或译码的控制功能或

符号。

721.32.06 本机记录 local record

所发送的报文在与发送机件相关的接收机上做出的显示。

721.32.07 格式控制符 format effector

保证字符在打印或显示时的最终格式所用的功能控制。

721.32.08 换行 line feed

使打印或显示的位置进到下一行的同一字符位置的格式控制字符。

721.32.09 回车 carriage return

使打印或显示位置回到该行起始点的格式控制字符。

721.32.10 键位转移 case shift

收报机的译码机构从一种字符键位向另一种字符键位的转移。

721.32.11 字母键位 letters case

一种键位，通过键位转移将电报电码的各个字符（主要是字母字符）和各种控制功能归在这种键位中。

721.32.12 字母转移 letters shift

一种键位转移，在此键位，信号被译成下段字符（主要是字母字符）或被译成字母键位的控制功能。

721.32.13 字母转移信号 letter-shift signal

一种信号，它限定收报机一定要将所收到的全部信号译成下段字符或字母键位的控制功能。

721.32.14 数字键位 figures case

一种键位，通过键位转换将电报电码的各个字符（主要是数字和符号）和各种控制功能归在这种键位中。

721.32.15 数字转移 figures shift

一种键位转移，在此键位，信号被译成上段字符（主要是数字字符）或被译成数字健位的控制功能。

721.32.16 数字转移信号 figure-shift signal

限定收报机一定要将所收到的全部信号译成上段字符或数字键位控制功能的信号。

721.32.17 输格信号 space signal

对应于一个电码组合，它使打印位置前进一个字符间距但不打印的信号。

721.32.18 输格字母转移 letters shift-on space

一种措施。采用这种措施，收报机能在收到输格信号后，在输格的同时，使数字键位向字母键位转移。

721.32.19 删除信号 erasure signal

用来使前一个信号无效的信号。

721.32.20 “你是谁”信号(控制功能) “who are you”signal (function), WRU signal

对应于一个电码组合的信号，当电报终端或数据站收到这个电码组合时，使应答码机件发出应答码。

721.32.21 应答码 answer-back code

识别一个特定的电报终端或数据站所用的唯一的字符序列。

721.32.22 同步控制 synchronization control

同步传输所用的一种功能控制。这种功能控制所产生的信号可以使终端之间，实现同步或实现同步校正，特别是在没有其它信号正在传输期间。

721.32.23 正文开始信号 start of text signal

表示报文的正文开始或报文报头结束的一个字符信号或一串字符信号。

721.32.24 正文结束信号 end of text signal

表示报文的正文结束的一个字符信号或一串字符信号。

721.32.25 报头开始信号 start of heading signal

表示报文的报头开始的一个字符信号或一串字符信号。

用于表示一份或多份正文传输结束的一个字符信号或一串字符信号。

721.32.27 传输控制(功能) transmission control (function)

仅为了适应在电信网上进行传输而使用的控制功能。

721.32.28 标识请求 identification request

请求远程终端给以响应的传输控制，这种响应可以包括终端标识或终端状态。

终端在作出肯定回答时所用的传输控制。

终端在作出否定回答时所用的传输控制。

让打印或显示位置设定到下一页的第一行或设定到其对应行所使用的格式控制字符。

721.32.32 横向制表 horizontal tabulation

让打印或显示的位置进到同一行中下一个预定字符位置所使用的格式控制字符。

721.32.33 纵向制表 vertical tabulation

让打印或显示的位置进到下一个预定行中相同字符位置所使用的格式控制字符。

让打印或显示的位置沿着同一行退回一个字符位置所使用的格式控制字符。

721.32.35 设备控制 device control

控制与终端相关的某些辅助设备所使用的功能控制，特别是对这些辅助设备“开”、“关”倒换所使用的功能控制。

根据某项协定，指出前面的数据有差错必须作废的功能控制。

721.32.37 取代字符 substitute character

一种控制字符。用这种控制字符来代替被认为是无效的或有错误的字符或在所使用的特定设备上不能表示的字符。

用于填充一部分数据媒体或时间间隔的控制字符。在字符串中插入或去掉这种控制字符不影响信息内容。

操纵视听呼叫装置以便引起注意的功能信号。

改变紧跟其后的有限个代码组合的意义，以便提供附加的功能控制。

721.32.43 移出 shift out, code shift

一种功能控制，这种功能控制，通过正常图形字符集用另一种被认可的字符集代替的办法来改变某种代码，以便提供一种新的代码或代码扩充。

721.32.44 移入 shift in, code reversion

一种功能控制，这种功能控制终止使用由代码移出字符所启动的字符集，并回复到正常的字符集。

根据逻辑的或层次的顺序对数据进行分隔或限定所使用的功能控制，其精确的含义必须根据各种情况具体规定。

电码表中用于呼叫的一个字符或字符的一部分。它可以与某些规定的信号状态一起在其它的交换电路上使用。

721.33.01 电报通路 telegraph channel

在两点之间单向传输电报信号的途径。

在两点之间单向传输数据信号的途径。

两个终端机之间的电报通路。

允许在两点之间进行双向传输的两条数据信道的组合。

传输方向与正在传送用户信息的方向完全一致的传输信道。

721.33.07 反向信道 backward channel; return channel

与前向信道相关联的传输信道，允许在与前向信道相反方向上传递确认信号和其他的功能控制信号。

注：在双向同时传送信息的情况下，反向信道的方向在任何时刻总是与用户信息的方向相反。

721.33.08 状态信道 status channel

指明某个比特组是用于传送数据还是用于控制的信道。

721.33.09 发送信道 transmit channel

相对于电路的某一给定端而言，用于发送用户信息的信道。

721.33.10 接收信道 receive channel

相对于电路的某一给定端而言，用于接收用户信息的信道。

721.33.11 电报中继器 telegraph repeater

接收电报信号并在下一个线路段上实时转发出相同含义的信号所使用的器件。

721.33.12 电报再生中继器 telegraph regenerative repeater 转发信号而无时间畸变的电报中继器。

721.33.13 调制变换器 modulation converter

用同一种电码表示输入信号和输出信号,但使用不同调制方式的电报中继器。

721.33.14 （电报的）假设参考连接 hypothetical reference connection (in telegraphy)

在全球用户电报网或其他电报网的两个终端之间建立的假设连接，这种假设连接大体上符合最复杂的情况，以便研究为保证满意运行所必需的传输性能和交换性能。

721.33.15 指定备用电路 nominated reserved circuit

当主用电路(或原用电路)发生故障时,分配给多路电报系统使用的电路(通常为电话业务所用)。

721.33.16 串行传输 serial transmission

以连续的(邻接的或不邻接的)时间间隔传输字符信号或码组信号的各个信号码元。

721.33.17 并行传输 parallel transmission

在多条分开的信道上同时传输字符信号或码组信号的各个信号码元。

721.33.18 相干特征瞬时 coherent significant instant

在并行传输中，同一字符信号的各个码元在所有信道上同时启动的特征瞬时。

721.33.19 字符串行传输 character-serial transmission 将连续的字符一个接一个地按顺序进行传输。每个字符的各个码元可以串行或并行传送。

721.33.20 字节串行传输 byte-serial transmission

将连续的字节一个接一个地按顺序进行传输。其中每个字节的各个码元可以串行或并行传送。

721.33:21 多信道(系统) multi-channel

在两点之间，使用两条或多条信道，按相同方向传播同一个信号的传输系统。

721.33.22 音频复用电报 voice-frequency multiplex telegraphy (VFT) 在电话型通路中使用频分复用的电报传输。

721.33.23 音频复用(电报)电路群 set of voice-frequency multiplex (telegraph) circuits 同时集结在一条电话型通路中的各条音频复用电报电路的组合体。

721.33.24 间歇式等时传输 burst isochronous

当传输通路速率高于输入的二进制速率时所采用的传输过程。被传送的二进制数字以该通路的数字率发送，而且按一定间隔中断传输，以便产生所需要的平均二进制速率。中断时间总是取二进制数字周期的整数倍。

721.33.25 四工系统 quadruplex system

在一条单独的电报电路上，每一方向都能同时独立地传输两份报文的电报系统。

721.33.26 电报复用 telegraph multiplex

将一公共传输通路分成若干条单独的电报通路，其中的每一电报通路都能在同一方向独立地传送电报信号。

721.33.27 字符交错电报 character interleaved telegraphy

一种时分复用电报。在这种电报中，从每一条独立通路轮流送来的各个字符，在一条公共通路上顺序地发送，而不把每个字符的各单位码元分隔开。

721.33.28 比特交错电报 bit interleaved telegraphy

一种时分复用电报。在这种电报中，一条公共通路上发送的每个字符的各信号码元被来自不同通路并属于其他字符的各信号码元所隔开。

721.33.29 博多电报 Baudot telegraphy

一种同步电报。通常是使用国际一号电码的字符交错电报。

721.33.30 范杜伦无线电报系统 Van Duuren radiotelegraph system, TOR

一种自动检错和重发的无线电报系统。它一般是由二条或四条使用范杜伦电码的通路组合而成的时分复用电报系统。

注：①这种系统的主要特征在国际无线电咨询委员会(CCIR)的建议342中有所规定。

② TOR 是“无线电路电传(Teleprinting Over Radiocircuits)”的缩写。

721.33.31 频移电报 frequency shift telegraphy (FSK)

采用调频的电报，其中的每个特征状态在稳态条件下均用一个规定频率的正弦信号来表示。

721.33.32 (四频)双路频移电报 four-frequency duplex telegraphy

一种频移电报，在这种电报中，对应于两条电报通路的四种可能的信号码元组合，每一种组合均由一个单独的频率表示。

注：双路频移电报用于无线电报。

721.33.33 信号记录电报 signal recording telegraphy

自动记录所收到的信号并随后由报务员译码的电报。

721.33.34 本机操作 in-local

不向线路发送信号的终端操作。例如：事先准备一份报文以便随后自动发报。

721.33.35 自动转发 automatic retransmission

通常将事先收到并已记录好的信号经过短暂的延迟之后再自动发送出去。

721.33.36 印字电报 printing telegraphy

将所收到的字符信号自动记录成打印字符的电报。

721.33.37 镶嵌电报 mosaic telegraphy

以镶嵌图形构成字符的电报，镶嵌图形的各个单位均作为单独的信号码元发送。

721.33.38 海尔系统 Hell system

镶嵌电报的一种方式。在所用的电报电码中，每个字符都用固定数目的单位码元表示。靠一个连续加墨水的旋转螺旋杆，在信号码元的控制下接触移动着的纸带来形成所接收的字符。

721.33.39 集合(等时)信号 aggregate (isochronous) signal

由来自时分复用器各分信道的信号组合而成的等时信号。

721.33.40 平衡集合信号 balanced aggregate signal

每个特征状态的码元数均相等的集合同步信号。

721.33.41 复用集合比特率 multiplex aggregate bit rate

时分复用器的比特率，它等于用户可用的分信道二进制速率的总和加上这个时分复用器各帧所需要的管理比特的速率。

721.33.42 字符周期 character cycle

时分复用器的每个分信道在公共信道上发完一个字符所用的时间。

721.33.43 码元同步 element synchronism

在同步传输中，本地时标的码元与接收到的信号码元完全一致的状态。

721.33.44 码元同步(过程) element synchronizing 调整到码元同步状态的过程。

721.33.45 同步比特 synchronization bit

用于字符或码组同步的二进制数字。

721.33.46 同步校正 synchronous correction

在同步传输系统中，使接收机和发送机保持规定相位关系的方式。

721.33.47 校正信号 correction signal

在同步传输系统中用于同步校正的专用信号。

721.33.48 自同步校正 correction from signals

在没有校正信号的情况下，通过恢复报文信号特征瞬时的位置来完成同步的校正方式。

721.33.49 相位关系 phase relationship

在同步传输中，接收设备的时钟信号与输入信号之间或者接收设备与发送设备的时钟信号之间的相对相位。

721.33.50 字符同相 character in phase

当本地时标的字符周期与接收到信号的字符周期完全一致时的状态。在这种状态下，便于在相应的接收分信道上收到某一特定信道传输的集合等时信号的各个字符信号。

721.33.51 子信道 sub-channel

通过时间分割的方法得到传输信道。该信道上所分配的速率为标准信道实际字符传输速率的分量。

721.33.52 子信道同相传输 sub-channel phase transmission

一条特定子信道上发送的字符在对应的子信道上接收到的状态。

721.33.53 定位 orientation

a. 在同步传输中，考虑到信号的传播时间或接收设备的响应时延，在接收分配器的位置和发送分配器的位置之间有意安排的相位差；

b. 为了改善收报机的改正力使收报机凸轮的静止位置进行角度调整。

721.33.54 定位范围 orientation range

在两个移位方向上，能够正确译码的移位限值。

721.33.55 分信道器 sector

总是与时分复用器提供的一条分信道相关联的设备部件。

721.33.56 空占电路状态 idle circuit condition

在已建立的连接中，电路在既不传送字符信号也不传送监控信号时电路的特征状态。

721.33.57 字符间休止状态 inter-character rest condition

在没有时钟的数据传输中，两个连续传送的数据字符之间的一段时间所存在的特征状态。

3.3.4 电报和数据传输设备

721.34.01 用户装备 subscriber's installation

设在用户住处的线路、终端及其任何辅助设备、专用交换机、控制单元和任何其他设备。

721.34.02 （电报）终端（telegraph）terminal

接到或准备接到电报用户线末端或电报电路末端的设备。这些设备既可以建立或接收呼叫，也可以存储和转发信号，并且均可被唯一地识别出来。

721.34.03 数据终端设备 data terminal equipment (DTE)

数据站的功能单元。它可以作为数据源、数据宿或者两者兼备并按照某一链路协议来完成数据传送控制。

721.34.04 分组型终端 packet mode terminal

能够对分组进行控制、格式化、发送和接收的数据终端设备。

721.34.05 数据电路终接设备 data circuit terminating equipment (DCE)

用于数据终端设备与线路之间的功能单元。其功能是建立、保持和释放连接，并且提供任何一种代码变换或信号变换所需要的功能。

721.34.06 网络终接单元 network termination unit

当数据电路由专门数据网提供时所使用的一种数据电路终接设备。

721.34.07 数据站 data station

包含有数据终端、数据电路终接设备及其公共接口的组合体。

721.34.08 数据源 data source

为数据链路提供数据的设备。

721.34.09 数据宿 data sink

从数据链路接收数据的设备。

721.34.10 调制解调器 modem

将调制器和解调器组装在同一台设备内的组合体。

721.34.11 接口 interface

两个功能单元之间，由交换电路的公共物理互连特性、信号特性和功能特性所规定的共同界面。

721.34.12 接口(规范) interface (specification)

对某一接口的公共物理互连特性、信号特性和功能特性的技术要求。

721.34.14 起止设备 start-stop apparatus 按起止系统设计的电报设备。

721.34.15 电传机 teleprinter, teletypewriter

由字母数字键盘发报机与打印字符收报机组合而成的起止设备。

721.34.16 纸页式电传机 page teleprinter 以纸页形式打印字符的电传机。

721.34.17 纸带式电传机 tape teleprinter  

在连续的纸带上按单行打印字符的电传机。

721.34.18 字母数字键盘 alphanumeric keyboard

由字母数字键与功能键组合成一体的装置，该组合装置对电报设备的发报机或数据源的操作进行控制。

721.34.19 电动式键盘 motorized keyboard

在这种字母数字键盘中，把组合杆移动到按键选择位置所需要的能量是由驱动设备（键盘属于它的一部分）的电机供给。

721.34.20 银步式键盘

721.34.20 锯齿式键盘 saw-tooth keyboard

在这种字母数字键盘中，通过操作人员的操作把组合杆移动到按键选择位置。

721.34.21 存储式键盘 storage keyboard

在这种字母数字键盘中，通过按键所产生的组合并不直接控制发报机，而是传送给一组或多组存储部件，存储部件随后对发报机进行控制。

721.34.22 纸带阅读器 tape-reader

读取记录纸带并产生与纸带上记录数据相对应的信号装置。

721.34.23（电传机）控制单元 control unit (teleprinter)

与电传机相配合的单元。其中包括在交换网上操作这种装置所必需的辅助设备。

721.34.24 直接打印机 direct printer

在采用不等长电码(例如莫尔斯电码、二态电报电码)的系统中所用的电报打印机。这种打印机直接用输入信号完成打印。

721.34.25 (纸带)凿孔机 (tape) perforator

在纸带上按照预定的电码凿出不同的洞孔组合来记录电报信号的设备。

721.34.26 键盘凿孔机 keyboard perforator 由字母数字键盘控制凿孔的凿孔机。

721.34.28 复凿机 reperforator, receiving perforator

主要是由受接收到的电报信号或数据信号控制的纸带凿孔机组成的收报机。

721.34.29 印字复凿机 printing-reperforator

在凿孔的同时在凿孔纸带上打印出相应的字符或代表相应功能控制符号的复凿机。

721.34.30 中导孔 centre holes, feed holes

在电报纸带上所凿出的小孔。当纸带放入适当的输送机构时，这些小孔能使纸带前进或后退。

721.34.31 无屑凿孔 chadless perforation

在纸带上凿孔时，洞孔只是部分地凿穿，纸屑或切片仍浮留在纸带上。

721.34.32 道 track

记录媒体上的一条位置线，它依次受同一写入器件的影响，或者依次影响同一读出器件。例如，在磁鼓表面上与某一磁头相关的环形部分。在纸带上能被给定的凿孔器凿孔或被给定的读出器读出的位置线。

721.34.33 发报机 telegraph transmitter 在电报通路上发送电报信号的设备。

721.34.34 键盘发报机 keyboard transmitter 受字母数字键盘控制的发报机。

721.34.35 自动发报机 automatic transmitter

信号的形成不受报务员控制而受信号记录媒体驱动的发报机。

721.34.36 自动编号发报机 automatic numbering transmitter

能在发送每份报文之前自动发送一个流水号的自动发报机。

721.34.37 自动转报机 automatic retransmitter

自动按照所记录的输入信号转发电报信号的设备。

721.34.38 凿孔纸带转报机 perforated-tape retransmitter

一种自动转报机。它包括一个能将纸带直接送入自动发报机的复凿机。

721.34.39 复凿孔和纸带阅读两用机 coupled reperforator and tape reader

能将所有用凿孔记录下来的信号(包括最后一个信号)转发出去的凿孔纸带转报机。

721.34.40 带可控输带装置的自动发报机 automatic transmitter with controlled tape-feed mechanism

利用外部同步装置发来的脉冲(例如,时分复用系统情况下)控制凿孔纸带移动的自动发报机。

721.34.41 应答器 answer-back unit

电报终端或数据终端在收到“你是谁？”信号之后立即自动发出其应答码的部件。

721.34.42 应答模拟器 answer-back unit simulator

其本身并不属于电传机的一部分，但在收到特定的“你是谁？”信号之后能完成与应答器相同功能的设备或程序安排。

721.34.43 纸带打印机 tape printer

读出所记录的信号(例如，在凿孔纸带上记录的信号)，并在纸带上或纸页上打印出相应字符而不受发报干扰的设备。例如：莫尔斯打印机或五单位打印机。

721.34.44 莫尔斯电键 Morse key

靠人工操作形成莫尔斯电报信号的器件。

721.34.45 音响器 sounder

将莫尔斯信号转换成音响信号的收报机，该音响信号由两种连续的声音之间的不同间隔来表征。

721.34.46 莫尔斯波纹收报机 Morse undulator

莫尔斯系统所用的一种接收机，其中有一个双位置的记录针，能依照输入信号的两个状态将墨水馈送给一条连续移动的纸带，从而形成可读的记录。

721.34.47 波纹收报机 siphon recorder

通过细虹吸管将墨水馈送给记录针以实现收报的莫尔斯波纹收报机。

721.34.48. 双工设备 duplex apparatus

包含发送和接收部分，且其配置容许双工操作的设备。

721.34.49 半双工设备 half-duplex apparatus

包含发送和接收部分，且其配置容许在两个方向上进行传输，但不能同时传输的设备。

721.34.50 电报继电器 telegraphy relay

主要用于中继电报信号，例如，为了构成电报中继器而设计的继电器。

721.34.51（电报）振动继电器 vibrating relay (telegraph)

具有附加绕组的电报继电器。当正常绕组中没有线路电流时，附加绕组被激励，使衔铁在其停止点之间产生某种有规律性的振动，从而达到提高继电器灵敏度的目的。

721.34.52 代码变换器 code converter 完成代码变换的设备。

721.34.53 电报调制器 telegraph modulator 受电报信号控制的调制器。

721.34.54 电报解调器 telegraph demodulator 受电报信号控制的解调器。

721.34.55 电报鉴频器 telegraph discriminator

将调频电报信号变换成直流电流传输信号的鉴频器。

721.34.56 无线电报变换器 radio telegraph converter

以音频或中频接收电报信号，并将它们变换成能让电报记录器工作的信号码元所使用的装置。

721.34.57 移位寄存器 shift register

由若干存储单元组成的寄存器。当这个寄存器的所有各级加上一个移位脉冲时，每个存储单元的状态就会按预定的方向转移给相邻的存储单元。

721.34.58 （接到电话线路的）声耦合 acoustic coupling (to telephone line)

利用换能器将非电话设备耦合到电话线的方法。换能器将声波发送给电话送话器，同时，从电话受话器接收到声波。

721.34.59 (数据传输的)声耦合器 acoustic coupler (for data transmission) 利用手持式送受话器将数据终端设备耦合到电话线路的电声换能器。

### 3.4 传真

#### 3.4.1 设备和业务

721.41.01 文件传真一类机 document facsimile equipment, Group 1

使用未采取任何传输信号频带压缩措施的双边带调制，并且适于在电话型电路上按标称扫描密度4.33线/mm、以大约6min时间传送一份GB148《印刷、书写及绘图纸幅面尺寸》B4幅面文件的文件传真机。

721.41.02 文件传真二类机 document facsimile equipment, Group 2

采用信号频带压缩技术，在电话型电路上按标称扫描密度3.85线/mm（或选用4.33线/mm）、以大约3min时间传送一份ISO A4（或选用GB 148 B4）幅面文件的文件传真机。

注：这里，信号频带压缩可用编码和（或）残余边带调制方式来实现，但不包括减少文件信息冗余度的处理。

721.41.03 文件传真三类机 document facsimile equipment, Group 3

调制处理之前，减少文件信息的冗余度。这样，在电话型电路上若以4800bit/s速率、大约1min的时间传送一份ISO A4幅面典型打字文件的文件传真机。文件传真三类机还可以对线路信号采用频带压缩措施。

721.41.04 滚筒式发送机 drum transmitter

原稿固定在旋转的滚筒上，并用读出头进行螺旋式扫描的传真发送机。

721.41.05 平面式发送机 flat bed transmitter 原稿平放并进行逐线扫描的传真发送机。

721.41.07 连续式接收机 continuous receiver

在相邻两条扫描线之间以固定间距移动着的记录媒体上逐线进行记录，并可以连续地记录多页报文，无需操作员在相邻报文间更换记录媒体的传真接收机。

721.41.08 电写机 telewriter

发送端人工控制书写笔在书写板上移动的同时，接收端的笔受电信号控制作相应移动或同时在屏幕上显示收到内容的一种电信设备。

721.41.09 用户传真 telefax

在电话交换网上用户终端之间的公用传真传输业务。

721.41.10 （数据网）用户传真 datafax

在公用数据网上用户终端之间的公用传真传输业务。

721.41.11 （局设）公用传真 bureaufax

在电话交换网或专线上利用电信局(或所)设置的公用终端的公用传真传输业务。

#### 3.4.2 传真工作特性

721.42.01 （传真发送机的）扫描 scanning (in facsimile transmitter)

将一份要用传真方式复制的文件分解成众多像素，并把这些像素连续地用电信号表示的过程。

721.42.02 （传真接收机的）扫描 scanning (in facsimile receiver)

将接收信号中依次得到的像素通过传真的复制合成为图像的过程。

721.42.03（发送）扫描点 scanning spot (at transmission)

读出头在扫描一份传真发送文件的过程中，在某个给定瞬间对这份文件进行探照的那一部分面积。

721.42.04 （接收）扫描点 scanning spot (at reception)

记录媒体在扫描过程中能保证传真副本合成的最小记录面积。

注：① 在相片传真的情况下，扫描点是指光敏记录媒体在给定的瞬间受到曝光的那一部分。

② 在直接记录的情况下，扫描点是记录媒体与扫描部件接触的那一部分。

721.42.05 扫描场 scanning field

在传真接收或发送的图像传输期间，被扫描点实际探照过的总面积。

721.42.06 扫描线 scanning line

传真扫描点从扫描场的一侧扫到另一侧，每进行一次扫描所探照过的区域。

721.42.07 扫描线周期 scanning line period

传真发送或接收时，扫描点经过相邻两条扫描线上的两个对应点间的一段时间间隔。

721.42.08 扫描线频 scanning line frequency

单位时间的扫描周期数。

注：对滚筒式传真机来说，扫描线频等于单位时间内滚筒的转速。

721.42.09 宽高比 aspect ratio

在传真中，一幅图面或扫描场的宽度与高度之比。

721.42.10 像素 picture element, pixel, pel

传真扫描的任一瞬间，在发送端分解原稿的最小面积（与发送扫描点一致）；在接收端，记录媒体上能有效复制合成的最小记录面积（$与接收扫描点一致）。

721.42.11 （传真发送机的）纵向分辨力 longitudinal resolution (of a facsimile transmitter)

在规定条件下，沿扫描线方向由传真发送机发出的最短信号形成可辨认的最小图像细节的尺寸。

注：① 相片传真中，发送分辨力取决于上述最小细节的尺寸及其有效亮度。

② 文件传真中，如果最小细节的反差适当，则所发送的信号应相当于标称黑信号或标称白信号。

721.42.12 （传真发送机的）横向分辨力 transverse resolution (of a facsimile transmitter)

在规定条件下，垂直于扫描线方向由传真发送机发出的最短信号形成可辨认的最小图像细节的尺寸。

注：①相片传真中，发送分辨力取决于上述最小细节的尺寸及其有效亮度。

② 文件传真中，如果最小细节的反差适当，则所发送的信号应相当于标称黑信号或标称白信号。

721.42.13 （传真接收机的）纵向分辨力 longitudinal resolution (of a facsimile receiver)

在规定条件下，沿扫描线方向由能使传真接收机记录的最短信号形成可辨认的最小图像细节的尺寸。

注：①相片传真中，最小细节的纵向和横向尺寸相应于像素的纵向和横向尺寸。

② 文件传真中，纵向分辨力是由能驱动接收机的最短标称黑信号或标称白信号所产生的线长来确定。

721.42.14 （传真接收机的）横向分辨力 transverse resolution (of a facsimile receiver)

在规定条件下，沿垂直于扫描线方向由能使传真接收机记录的最短信号形成可辨认的最小图像细节的尺寸。

注：① 相片传真中，最小细节的纵向和横向尺寸相应于像素的纵向和横向尺寸。② 文件传真中，横向分辨力等于扫描行距。

721.42.15 点频率 dot frequency

a. 传真发送机在一秒钟内发送连续像素数的一半；

b. 在传真中，表示沿扫描线方向交替出现的黑与白像素的信号基频。

721.42.16 （传真）对相 phasing (in facsimile)

使接收机扫描场上的一个参考点与发送机上的相应点准确保持一致，以保证记录媒体上图像有正确的定位。

721.42.17 对相信号 phasing signal

为保证对相，由传真发送机发往接收机的一种特定信号。

721.42.18 白对相 phasing on white

利用相应于标称白的对相信号对相。该对相信号是在损失时间内发送相应于标称黑的短时中断信号。

721.42.19 黑对相 phasing on black

利用相应于标称黑的对相信号对相。该对相信号是在损失时间内发送相应于标称白的短时中断信号。

721.42.20 （传真）同步 synchronization (in facsimile) 使发送机和接收机的扫描线频相同的建立过程。

721.42.21 （传真发送机的）光阑孔径 aperture (of a facsimile transmitter) 传真发送期间，能通过光线并有效控制扫描点尺寸的小孔。

721.42.22 （传真接收机的）光阑孔径 aperture (of a facsimile receiver)

受接收图像传真信号调节的光线所通过的最后一个小孔。通过小孔的影像在光敏记录媒体上被聚焦成一个点。

721.42.23 空段 dead sector

滚筒式发送机的滚筒表面上，不能用来传输图像信号的部分扫描时间。

721.42.24 损失时间 lost time

扫描线周期中不能用于传输图像信号的那部分时间。

注：对于滚筒式传真机来说，损失时间就是空段的扫描时间。

721.42.25 记录螺旋轮 writing helix

在某些连续式的传真接收机中，使用带有螺旋式凸缘线的旋转部件。它与扫描线的相交断面确定了给定瞬时像素在该线上的位置。

721.42.26 记录杆 writing bar, writing edge

在某些连续式传真接收机中，与记录螺旋轮联合使用的一种有凸缘直杆的部件。以便确定给定瞬时记录媒体上扫描的位置。

721.42.27 滚筒因数 drum factor

在滚筒式发送机或滚筒式接收机中，滚筒的扫描线有效长度与滚筒直径之比。

721.42.28 扫描行距 scanning pitch

相邻两条传真扫描线对应边间的距离。

721.42.29 扫描密度 scanning density

在垂直于扫描线方向，单位长度内的扫描线线数。

注：扫描密度是扫描行距的倒数。

721.42.30 扫描速率 scanning speed

传真扫描点在原稿或记录媒体上移动的线速度。

721.42.31 扫描线有效长度 usable scanning line length 传真设备所能达到的扫描线最大长度。

721.42.32 扫描线全长 total scanning line length

扫描速率与扫描周期的乘积。

注：扫描线全长等于扫描线有效长度加上扫描速率与损失时间乘积之总和。

721.42.33 合作因数 factor of cooperation 扫描线全长与扫描密度的乘积。

721.42.34 合作指数 index of cooperation

合作因数被  $ \pi $  值除所得的商。

对滚筒式发送机和滚筒式接收机，合作指数也等于滚筒直径与扫描密度的乘积。

721.42.35 压缩扫描 limited scanning

为缩短传真传输时间，以正常使用扫描行距的两倍或其他整数倍所进行的扫描。

721.42.36 复制比 reproduction ratio

传真副本的线性尺寸与原稿对应尺寸之比。

721.42.37 正向扫描位移 positive scanning shift

传真扫描部件对于原稿的相对移动。即从左向右逐行扫描原稿表面时，从原稿的顶部到底部形成若干条扫描线。

注：滚筒式传真机中，向左螺旋式扫描为正向扫描位移。

721.42.38 负向扫描位移 negative scanning shift

传真扫描部件对于原稿的相对移动。即从右向左逐行扫描原稿表面时，从原稿的顶部到底部形成若干条扫描线。

注：滚筒式传真机中，向右螺旋式扫描为负向扫描位移。

721.42.39 最大反差 maximum contrast

图像中最亮部分的亮度与最暗部分的亮度之比。

注：这个参数值通常用以10为底的对数表示，即等于相应光密度的差值。

721.42.40 亮度范围压缩 compression of the luminance range 有意地或偶然地改变传真信号，使接收图像的亮度范围与原稿相比缩窄了。

721.42.41 亮度范围扩展 expansion of the luminance range 有意地或偶然地改变传真信号，使接收图像的亮度范围与原稿相比扩展了。

721.42.42 亮度补偿 luminance compensation

相片传真中,为了正确复现图像的亮度,按照记录媒体感光特性的变化接收信号。

721.42.43 标称黑 nominal black

相应于纯黑部分信号的电平或频率值。

721.42.44 标称白 nominal white

相应于纯白部分信号的电平或频率值。

721.42.45 加重反差 accentuated contrast

对原稿中所有像素信号值进行的处理，凡亮度低于规定中间值的均按标称黑发送；亮度高于规定值的均按标称白发送。

721.42.46 图片黑 picture black

对应于传真发送原稿纯黑部分的信号(如电平或频率)值。

721.42.47 图片白 picture white

对应于传真发送原稿纯白部分的信号(如电平或频率)值。

721.42.48 色调制孔 tone control aperture

为了能按照收到传真信号的幅度和光敏记录媒体的特性来改变照射到记录材料上的光通量而设计的一个光孔。

721.42.49 半色调图像 half-tone picture

色调范围在传真系统的图片黑和图片白两界限之内的图像，有时也称之为中间色调图像。

注：在报纸传真系统中系指网点图像。

721.42.50 色调阶 tone wedge

在黑和白之间包含有若干个光密度等级的光密度阶。

#### 3.4.3 传真副本中的损伤和畸变

721.43.01 孔径畸变 aperture distorsion

由于传真发送和接收的扫描点有一定尺寸而产生的一种效应，它使(图像或细节的)轮廓变得模糊，并使小于扫描点的细节丢失。

721.43.02 拖尾 failing

原稿文件上亮度发生突变(如由黑到白)时，在接收文件上沿扫描方向表现出来无规律变化的一种传真损伤。

注：对于文件传真，拖尾这个术语不适合于复制比像素窄的细节。而在某些情况下，还可以在发送机中有意地扩展这种窄的细节信号。

721.43.03 回波效应 echo effect

传真副本中，由于传播等现象而引起在扫描方向出现偏离正常图形轮廓的第二条或多条轮廓的一种损伤。

721.43.04 （传真中的）干扰图型 interference pattern (in facsimile), moire

传真副本中出现的一种损伤，它在图片上表现为又叠加了多少有些规则的图型或密度变化的线条。

注：干扰图型一般是由周期性干扰所引起。

721.43.05 纵向抖动 longitudinal judder

由于传真滚筒或记录螺旋轮的不规则旋转使相应于原稿中有规则的线条，在传真副本中出现轻微波动或间断。

721.43.06 横向抖动 transverse judder

由于传真扫描行距不规则性而使传真副本中同时出现重叠和脱节。

721.43.07 噪声黑区 noisy blacks

由于噪声引起信号电平变化，使传真副本黑色区出现不均匀的现象。

721.43.08 （传真中的）重叠 overlap (in facsimile)

由于扫描线宽度大于扫描行距或扫描行距不规则而在传真副本中出现的一种损伤。

721.43.09 脱节 underlap

由于扫描线宽度小于扫描行距或扫描行距不规则而在传真副本中出现的一种损伤。

721.43.10 歪斜 skew

由于发送和接端扫描速率不同，使传真副本中应垂直于扫描线方向的线条变成倾斜于扫描线方向的一种损伤。

721.43.11 部分色调反转 partial tone reversal

传真副本中出现的一种损伤，它表现为不是由白平滑地变到黑，而是由白变到黑，再返回到白，或与此相反。

注：例如，这种损伤可能是由于副载波调频时黑和白信号的范围调整不正确而引起的。

### 3.5 电报和数据网、交换、操作和业务

#### 3.5.1 电报和数据网

721.51.01 公用电报网 public telegram network

为实现公用电报业务而建立的电信网。

注：公用电报网由主管部门或经认可的私营机构提供、经营和管理。

721.51.02 国际公用电报网 gentex network

欧洲等国的主管部门或经认可的私营机构之间使用的电报交换网，以提供国际公用电报业务。

721.51.03 用户环路 subscriber loop, subscriber's line

用户住宅内的设备与提供所需业务的本地电信局之间的链路。

721.51.04 用户线 local line

指用户设备与一次复用器或集中器之间的用户线部分。如果不使用集中器或复用器，则指用户设备和交换局之间的用户线部分。

721.51.05 （电报和数据传输中的）专用电路 dedicated circuit (leased line) (in telegraphy and data transmission)

不使用交换设备并且专门为一组终端提供固定连接的电报电路或数据电路。

721.51.06 国际电路 international circuit

一种直接将位于不同国家的两个交换机连接起来的电路。

721.51.07 洲际电路 intercontinental circuit

一种将位于不同洲不同国家的两个交换机连接起来的电路。

721.51.08 电路组 set of circuits

在两个交换机之间建立的一组电路。当建立连接时，不考虑优先级，可以选择其中的任何一条电路。

721.51.09 （电报和数据传输中的）专用网 private network (in.telegraphy and data transmission) 不经过公用网的交换设备便可以彼此通信的一组终端所组成的网。

721.51.10 溢出座席 overflow position

在专用网中的一个指定终端，当被叫终端的标识被遗漏或无法与所选择的终端建立连接时，由网将呼入改接到该指定终端。

721.51.11 封闭专用网 closed private network

只允许在网内的终端之间相互通信的专用网。

721.51.12 网间互通 interworking between networks

可以使连接在某种电信网上的终端与其他网上的终端进行通信的手段。

721.51.13 数据网 data network

为一组数据终端设备提供相互连接的手段。

721.51.14 同步数据网 synchronous data network

一种在数据电路终接设备(DCE)和数据交换设备(DSE)之间，以及各数据交换机之间采用某种同步方式的数据网，其中的信令、速率由网中的定时设备来控制。

721.51.15 时基 time base

一种标准振荡，根据该振荡周期中相应瞬间来确定它的时间间隔。

721.51.16 时标 time scale

相继邻接的、预先确定的时间间隔。

注：时标中的时间间隔不一定都具有同样的持续时间。

721.51.17 周期性时标 cyclic time scale

由周期性重复的一串时间间隔构成的时标。

注：组成一个周期的单个时间间隔不一定都具有相同的持续时间。但在理想情况下，所有的周期都是相

同的，任何偏差都必须限制在指定的限值以内。

721.51.18 公用数据网 public data network

为了向公众提供数据通信业务而建立的一种电信网。

721.51.19 节点 node

在数据网中，一个或多个功能单元与数据信道或数据电路的相连点。

721.51.20 支路 branch

数据网中任意两个直接相连的节点之间的路由。

721.51.21 通道 path

数据网中任意两个节点之间的路由。

721.51.22 （终端）端口 （terminal）port

节点的一个功能单元,数据能通过它进出数据网。

721.51.23 数据链路 data link

连接两个数据站的电信设施。

721.51.24 电路交换型连接 circuit switched connection

根据要求，在两个或两个以上数据站之间建立的连接。这些数据站可以独占该连接，一直到拆线为止。

721.51.25 多点连接 multipoint connection

三个或者三个以上的数据站之间建立的连接。

721.51.26 点对点连接 point to point connection 只在两个数据站之间建立的连接。

721.51.27 虚电路 . virtual circuit

在按分组交换方式工作的数据网中，由网提供的用于数据站之间传送数据的设施。这些设施模拟了电路交换连接所提供的功能。

721.51.28 永久性虚电路 permanent virtual circuit

在分组交换网中，两个终端设备之间建立的永久性连接。

721.51.29 虚呼叫 virtual call

通过虚电路建立的呼叫。

721.51.30 （电报和数据传输的）透明度 transparency (in telegraph and data transmission)

在设备调制速率不超过设计要求的唯一条件下，电信网以任何形式的二进制码元序列传输信号的能力。

721.51.31 代码不相关 code independent

对数据链路、数据网或操作方式的一种限制。其特征是允许任何二进制码元序列以指定的二进制速率传输。

3.5.2 电报与数据交换；信令

721.52.01 连接 connection

在电信网中的两点或多点间为提供信息传送而在信道或电路、交换设备，以及其他功能部件之间建立的暂时联系。

721.52.02（完整）连接（complete）connection

在电信网中各种终端间，为提供信息传送而在信道或电路、交换设备以及其他功能部件之间建立的暂时联系。

721.52.03 （用户电报）呼叫（telex）call

由所连接的用户电报终端设备要求建立和可能使用的一次完整连接的过程。

721.52.04 用户业务交换机 subscriber service exchange

在同一地区用户之间或本地区用户与其他交换机之间建立相互连接的公用交换机。

721.52.05 中转交换机 transit exchange

能在其他电报交换机之间建立连接的电报交换机。

721.52.06 电报交换机 telegraph switching exchange 为交换电报业务而安装在某地的一套设备。

721.52.07 数据交换机 data switching exchange

为交换数据业务而安装在某地的一套设备。

注：一个数据交换机可能只提供电路交换，或只提供分组交换、或二者兼而有之。

721.52.08 子局 sub-centre

为一组终端提供服务并将它们的业务量汇集到一个较大母局的交换局，其全部业务量的路由分配均由母局来决定。

721.52.09 线路集中器 line concentrator

设置在本地网远方的一种交换设备，它使用户业务交换机和许多用户之间的业务量通过少量的线路进行传输。

注：用户业务交换机一般应提供与线路集中器相配合的设备。

721.52.10 电报中继电路 telegraph trunk circuit

设置在两个相互通信的电报交换机之间的固定电报电路。

721.52.11 电报汇接电路 telegraph junction circuit 连接子局与母局的电报电路。

721.52.12 跨线业务 overline service

多条用户线电路组合起来合用同一个地址。这样，呼叫该地址便可以接入其中任何一条空闲线路。

721.52.13（电报的）溢出 overflow (in telegraphy)

当不能同被叫站建立连接时，由网将呼叫或报文改发到一个指定的座席，以便稍后再发。

721.52.14 电路交换 circuit switching

根据请求提供一个专用的完整连接，将两个或多个终端暂时连接起来，直至释放为止。

721.52.15 报文交换 message switching, store and forward switching

电信网中分发报文的处理过程。它包括在网的某些节点中接收报文，存储并转发报文。

721.52.16 分组交换 packet switching

电信网中分发报文的处理过程。这一过程首先将报文分成带有地址的分组；然后在一定的网节点中接收、存储，并把它们转发到适当的传输信道上；在接收端，再将接收到的分组重新组成报文。

注：一个分组只有在传输时才占用一条传输信道。因此，该信道可以用于传送属于同一报文的其他分组或属于其他报文的分组。

721.52.17 复凿机交换 reperforator switching

一种纸带中转系统。来自复凿机的纸带直接送入固定连接的自动发报机，该发报机能通过交换连接到一条输出通道。

注：这种交换可以是人工的、自动的或半自动的。

721.52.18 字符交换 character switching

根据要求在两个或多个终端之间建立暂时的连接，并采用存储和转发的处理方式把字符逐个地从一条线路转发到另一条线路。

721.52.19 电路交换机 circuit switching exchange, switch (circuit)

与一组电路相关联的一套设备。这套设备可根据要求将这些电路暂时互连以构成连接。

721.52.20 报文交换机 message switching exchange, switch (message)

与一组电路相关联的一套设备。这套设备可以接收、存储并转发报文，而无需在这些电路之间提供固定的连接。

721.52.21 国内用户电报号码 national telex number

通常，供主叫用户进行选择以连接到本国被叫用户的一个数字序列。

721.52.22 本地用户电报号码 local telex number

比国内用户电报号码短的数字序列，用来连接有限地区内的被叫用户。

721.52.23 缩位码选择 short code selection

利用比完整地址短的代码来建立连接。

注：该代码还可以应用于一组地址。

721.52.24 缩位地址 short address .

一种比正常地址少若干字符的地址，例如，用于特种业务或特殊用户的地址。

721.52.25 直接呼叫 call without selection signals, direct call

在不使用选择信号的情况下，与用户预先指定的一个或一组目的终端建立连接。

注：网将主叫信号转换为一条建立连接的指令。

721.52.27 选择纠错 selective error correction

一种使用户不必重复全部选择过程便可以纠正选择序列中差错的功能。

721.52.28 可选用的选择信号 alternative selection signals

电信网能接受选择信号时可用的几种不同电码（例如：国际二号电码和国际五号电码）。对某个指定用户来说，每次呼叫所选用的电码可固定为一种，也可由用户灵活选用。

721.52.29 长途用户电报网前缀 prefix giving access to the long-distance telex network

用户电报网中本地号码区的用户选择网内其他区的用户时，所附加的数字序列。

721.52.30 国际网前缀 prefix giving access to the international network

为使用户能连到国际网，必须选择的一个数字或数字序列。它放在长途用户电报网前缀之前。

721.52.31 洲际网前缀 prefix giving access to the intercontinental network

为使用户能连到洲际网，必须选择的一个数字或数字序列。它放在长途用户电报网前缀之前。

721.52.32 目的地代码 destination code

识别被叫用户所在国或该国一个特定网的数字序列。

注：用户电报目的地代码在国际电报电话咨询委员会的建议 F69 中已有规定。

721.52.33 国际选择序列 international selection sequence 国际两级选择信号中的第一个数字序列。

721.52.34 国际两级选择 international two-stages selection

采用两个数字序列建立国际呼叫的过程。第一个序列表示被叫的国家或网；而第二个序列表示该国家或网中的被叫用户。

721.52.35 （信令中的）标号 label (in signalling)

一组数据内或附加在该组数据上的一个或多个字符。它包含与该组数据有关的信息(包括其标识)。

721.52.36 （电路交换中的）业务量路由选择 traffic routing (in circuit switching)

根据一定的规则指定一组电路。对于某一给定的试呼叫，可以利用这组电路与指定的交换

机建立连接。

721.52.37 正常(业务量)路由选择 normal(traffic) routing

根据一定的规则指定一组电路。对于某一给定的试呼叫，如果该组电路中有空闲电路，则将按照第一优先级的原则选择其中一条作为路由。

721.52.38 迂回业务量路由选择 alternative traffic routing

对于一个给定的试呼叫，如在正常业务量路由的电路组中没有可用的电路，则按照给定的规则指定另一组电路作为路由。

721.52.39 首选电路组 first choice set of circuits

一组具有第一优先级的电路，但该电路组中必须有可用的空闲电路。

721.52.40 应急路由 emergency routing

在正常业务量路由和任何迂回业务量路由的电路组均不能使用的特殊情况下所选用的路由。

721.52.41 重选路由 re-routing

当某个转接机出现拥塞时，改变呼叫的方向使其返回到前一个与用户已建立连接的交换机，以便从该机寻找一条迂回路由。

721.52.42 交换信号 switching signal

为建立和拆除一次呼叫而在两个交换机之间或者交换机和终端之间传送的一种信号。

721.52.43 前向交换信号 forward switching signal 由主叫向被叫传送的交换信号。

721.52.44 返回交换信号 return switching signal 从被叫向主叫回送的交换信号。

721.52.45 电路空闲状态 free circuit condition

电路在可以用来建立呼叫时的特征状态。

721.52.46 准备好 ready

表示数据电路终接设备准备接受呼叫请求信号或者数据终端设备准备接受呼入信号时，DTE 和 DCE 之间接口的状态。

721.52.47 未准备好 not ready

表示数据电路终接设备不准备接受呼叫请求信号或者数据终端设备不准备接受呼入信号时，DTE 和 DCE 之间接口的状态。

721.52.48 数据准备好信号 ready for data signal

由数据电路终接设备发送到数据终端设备的一种呼叫控制信号，用它来表示该数据连接可以进行数据传递。

721.52.49 数据信道的信令状态 data channel signalling conditions

在呼叫建立和拆线过程中，交换机间数据信道的各种状态。例如，中继线空闲，中继线已占用，准备交换数据，拆线请求等。

721.52.50 主叫信号 calling signal

在电路或用户线上传送表示请求建立一次呼叫的前向交换信号。

721.52.51 呼叫控制规程 call control procedure

建立、保持和拆除一次数据呼叫所需要的一整套交互信号。

721.52.52 呼叫证实信号 call-confirmation signal

在响应主叫信号时，为证实已收到主叫信号而发送的一种返回交换信号。

721.52.53 呼叫接受信号 call accepted signal

通过反向信道发送的表示终端可以接受本次呼叫的信号。

721.52.54 选择信号 selection signals

向交换机发送的一种前向信号序列，这是建立呼叫所需的信号。

721.52.55 （电路交换中的）地址 address (in circuit switching)

用于指明呼叫目的地的那部分选择信号。

721.52.56 （信息处理中的）地址 address (in information processing)

在不使用任何中间参数的情况下，标识存储器或装置的字符或字符组。

721.52.57 着手选择信号 proceed-to-select signal

为响应主叫信号或在呼叫证实信号之后，由交换机回送的一种返回交换信号，用它来表明该交换机已准备好接收选择信号。

721.52.58 （用户）业务类别信号 （user）class of service signal 在选择信号中，标明主叫用户业务类别的字符或字符组。

721.52.59 前置信号 presignal

一种在选择信号的开头发送的业务类别信号。

721.52.60 后置信号 postsignal

一种在表示被叫终端的数字序列之后发送的业务类别信号。

721.52.61 选择结束信号 end of selection signal

一种与选择信号一起发送的交换信号，它位于被叫用户号码的数字之后，用来表明属于该号码的数字已结束。

721.52.62 键盘选择 keyboard selection

在电报电路交换中，由电传机键盘或自动设备发出电报字符信号所形成的选择序列。

721.52.63 （电报交换中的）拨号选择 dial selection (in telegraph)

在电报电路交换中，利用拨号盘或自动设备发出的拨号脉冲串所形成的选择序列。

721.52.64 呼叫连接信号 call-connect signal

通过反向信令信道返回的交换信号，用它来表明该呼叫已到被叫站。

721.52.66 拆线证实信号 confirmation of clearing signal 表示拆线信号已被执行的返回交换信号。

721.52.67 业务信号 service signal

由电信网自动发向主叫终端的信号，它表明一次呼叫的进程或试呼叫失败的原因。

721.52.68 占线信号 engaged signal, busy signal

一种表明被叫站占线或者不可占用的业务信号。

721.52.69 阻止信号 barred signal

一种表明由于接入受阻，呼叫不能建立的业务信号。

721.52.70 呼叫建立时间 call set-up time

从主叫用户发出呼叫信号到收到呼叫连接信号之间的时间间隔。

721.52.71 预选时延 preselection delay

从主叫用户发出呼叫信号到收到着手选择信号之间的时间间隔。

721.52.72 选择时间 selection time

从主叫用户收到着手选择信号到交换机收完选择序列之间的时间间隔。

721.52.73 选择后时间 post-selection time

从主叫用户发完选择序列到收到呼叫连接信号之间的时间间隔。

721.52.74 呼叫有效时长 effective duration of a call

从主叫用户收到呼叫连接信号到发出拆线信号之间的时间间隔。

721.52.75 存储程序控制 store program control (SPC) 利用一套存储的并可修改的指令对交换机进行控制。

721.52.76 共路信令 common channel signalling

将许多电路的信令信息以标号报文的形式，在一条单独的信道上传送的信令方式。

721.52.77 随路信令 channel associated signalling

将单个信道上传输业务所需的信令，通过信道本身或通过与其相关的固定的一条信令信道进行传送的信令方式。

注：当在该信道上传送某条电路信令时，这条术语也适用。

721.52.78 拆线时延 call clearing delay

从终端开始发送拆线信号到反向线路上呈现电路空闲状态之间的时间间隔。

721.52.79 不成功呼叫 unsuccessful call

未能建立完整连接的试呼叫。

721.52.80 呼叫拒收信号 call not accepted signal 被叫终端为表明其不接受呼入而发送的一种呼叫控制信号。

721.52.81 未接通呼叫 lost call

由于网拥塞而被拒绝的连接请求。

721.52.82 碰撞 head on collision

在一条可以建立双向呼叫的传输通道上，当两端同时或几乎同时争用它时存在的状态。由于传播时延，不会出现远端抢先占用该通道的情况。

721.52.83 （用户电报的）终端连接方法 terminal connection method (of telex)

用户电报的用户线与为用户服务的交换机之间的接口特性。

721.52.84 速率转换集中器 speed converter concentrator

一种将来自低速信道的数据暂时存储起来，并在高速信道上转发出去（或进行相反操作）的装置。

721.52.85 数据集中器 data concentrator

一种配置在通道两端的装置，使用这种装置可使一条通道为多个终端提供服务。

721.52.86 系统控制站 system control station 传输系统中负责维护和排除故障的站。

3.5.3 电报操作和业务

721.53.01 （公用）电报 telegram

委托电报局所传送并投递到收报人的报文。

注：电报包括委托给电报局、所或投送给收报人的文件。

721.53.04 无线电报 radiotelegram

发自或发往移动电台或移动地球站的电报，传送这种电报的全部或部分路由经过移动业务或移动卫星业务的无线通信信道。

721.53.05 用户电报对话方式 telex conversation mode

使用用户电报终端设备的两用户间，通过用户电报连接进行对话的一种通信方式。

721.53.06 无线用户电报呼叫 radiotelex call

发自或发往移动电台或移动地球站的用户电报呼叫。传送这种呼叫的全部或部分路由经过移动业务或移动卫星业务的无线通信信道。

721.53.07 公用电报网接入口 access to the public telegraph network

供用户电报终端设备向公用电报局所发送电报以及从公用电报局所接收电报的端口。

721.53.08 用户业务类别 user class of service

规定公用电信业务用户可用的呼叫特征的类别。

注：用户业务类别的特征，诸如二进制速率、终端工作方式、电码结构、禁止出入。

721.53.09 公用用户电报亭 public telex booth 向公众提供用户电报终端的场所。

721.53:10 只发终端(设备) outgoing only terminal 能向电信网发出呼叫但拒收输入呼叫的终端设备。

721.53.11 只收终端(设备) incoming only terminal 能接收电信网发来的输入呼叫但不能发出呼叫的终端设备。

721.53.12 阻止接入 access barred

电信网的一种功能，它禁止某些用户对某些业务、路由或终端设备的来去呼叫。

721.53.13 限制服务 restricted service

禁止某个用户终端设备接入到在正常情况下所有用户都可接入的某些业务、路由或终端设备的一种服务。

721.53.14 优先级 priority

在每一选择级，分配给专用网或闭合用户群中某个指定终端设备所发出的呼叫比正在建立过程中的所有其他较低优先级的呼叫享有优先建立的可能性。这种可能性适用于这种特权终端设备所发出的每个呼叫或仅仅是指定的某些呼叫。

721.53.15 绝对优先级 absolute priority

在每一选择级或某些选择级，分配给专用网或闭合用户群中某个指定终端设备所发出的呼叫比已建立的所有其他较低优先级的呼叫享有优先建立的可能性。这种可能性适用于这种特权终端设备所发出的每个呼叫或仅仅是指定的某些呼叫。

721.53.16 被叫用户优先级 priority for called subscriber

对被叫用户终端设备的全部呼叫或某些呼叫具有优先级或绝对优先级。这种功能是通过主叫终端设备发送一个适当的信号而起作用。

注：优先级可有若干等级，每一级相对于较低级来说都是优先（等）级或绝对优先（等）级。

721.53.17 制止本机操作 in local override

一种网的性能，它可以制止正在进行本机操作的终端设备，以便将输入呼叫接到该终端设备。

721.53.18 直接出网选择 direct outgoing selection

专用网中的一种功能，无需人工介入便可使专用用户电报网的终端与其指定的网中的终端设备建立呼叫。

721.53.19 直接入网选择 direct incoming selection

专用网中的一种功能，无需人工介入便能使用户电报网中的终端设备与其指定的专用网中的终端设备建立呼叫。

721.53.20 综合编号的直接入网选择 direct incoming selection with integrated numbering

使用单个选择序列的直接入网选择，这种选择序列由识别专用网的一些数字和随后加入的一些识别该网中被叫终端设备的数字组成。这种完整的数字序列构成一个完整地址，纳入用户电报网的编号方案。

721.53.21 两级选择的直接入网选择 direct incoming selection with two-stage selection 使用两个选择序列以选择专用网中所需终端设备的一种直接入网选择。第一个序列识别专用网，第二个序列识别该专用网中的终端设备。只有第一个序列纳入用户电报网的编号方案。

721.53.22 闭合[封闭]用户群 closed user group

公用交换网中的一个用户群，该群中的各终端设备只能在本群内相互通信。

注：一台终端设备可以属于几个闭合用户群。

721.53.23 部分闭合用户群 partially closed user group

用户群中的某些终端可以呼叫公用交换网中通常都能接通的任何其他终端设备，也可以接收它们的呼叫。而其他一些终端设备只能与本用户群内的终端设备通信。这种用户群称为部分闭合用户群。

注：某些情况下，某些指定终端的对外连接只限于输出呼叫。

721.53.24 互锁码 interlock code

一种在通常情况下是前向发送的、而在某些情况下又是反向发送的信息，该信息用以表明本呼叫中有闭合用户群。

721.53.25 用户选用业务功能 user facility

除提供通常的业务外，还根据电信网的用户要求提供选用业务。

注：可按每个呼叫提供用户选用业务或在一段商定的时间内提供用户选用业务。

721.53.26 自动呼叫 automatic calling

为建立连接，主叫终端不需要人工介入，而由网的规程所要求的序列操作。

721.53.27 自动应答 automatic answering

呼叫数据终端设备自动响应呼叫信号，而且无论被叫终端设备是否有人值守都可以建立这种呼叫的应答。

721.53.28 人工应答 manual answering

只有当被叫用户的信号表示已准备好用人工操作接收呼叫时而建立的呼叫应答。

721.53.29 自动标识 automatic identification

不需要人工介入，传送到所连接的终端的主叫终端标识，反之亦然。或者当一个连接已建立时，相互传送终端标识。

注：这种标识可由网络提供，也可由终端设备提供。

721.53.30 （电信网进行的）线路标识 line identification by the network

电信网为响应两个已建立连接的用户当中任何一个的请求而发出适当的线路标识或地址标识。

721.53.31 日期时间自动指示 automatic date and time indication

由网将一次呼叫开始的日期和时间自动指示给主叫终端或同时指示给主叫和被叫终端设备。

721.53.32 时长指示 indication of duration

在付费终端拆线信号之前或在适当时间通过回叫由网发给付费终端的一次呼叫的计费时间。

注：可自动提供这种信息，也可按要求提供。

721.53.33 费用指示 indication of charge

在付费终端发出拆线信号之前或在适当时间通过回叫由网向付费终端指出一次呼叫的费用。

注：这种信息可自动提供，也可按要求提供。

721.53.34 呼叫账单通报 statement of call account

应用户、主管部门、闭合用户群或专用网的要求，由网发出他们的通报费用的详细账单，或某一指定时间内的账单。

721.53.35 共享终端 shared terminal

可以提供某些用户共同使用同一终端并分担相应成本和费用的设备。

721.53.36 共享终端计费 accounts for shared terminal 给共享终端的各用户分别计费。

721.53.37 呼叫内容存储 storage of call content

按用户要求，由网将该用户所发或所收呼叫的部分内容或全部内容在规定的时间内存储起来。

721.53.38 已存呼叫内容检索 retrieval of stored call content 向先前要求存储呼叫内容的各用户传送呼叫内容。

721.53.39 按请求统计 statistics on request

按照用户请求，由网将其呼叫细节，例如：国际呼叫、国内呼叫、到某些用户的呼叫或总呼叫次数，以规定的报头发给用户的一种功能。

721.53.40 记录报文 recorded message

由被叫用户控制的一种性能，借此可将该用户的输入呼叫连接到一部记录报文用的发报机。

721.53.41 预占(空闲时连接) camp-on(connect when free)

因被叫终端设备已占用，或网拥塞而未曾接通的呼叫，由网加以保持，一旦有空闲立即自动连接。

721.53.42 有回叫的预占 camp-on with recall

主叫终端设备实现拆线并尽可能立即回叫的预占。

721.53.43 （电报和数据通信中的）用户缺席服务 absent subscriber service (in telegraph and data communication)

能通过业务信号自动通知主叫终端设备，被叫用户终端设备因故不能用于呼叫的一种性能。

721.53.44 呼叫改向 call re-direction

根据被叫用户的要求，将呼叫改接到预先指定的另一目的地，并用业务信号通知主叫终端设备的一种性能。

721.53.45 改址拦截 changed address interception

由网发给主叫终端设备的自动通知，其内容是被叫终端设备的新地址，随后是呼叫转移或释放主叫终端设备。

注：也可简化为发出业务信号后就拆线。

721.53.46 存储转发 store and forward

将报文或部分报文先存储起来，然后把它们转发到指定的一个地址或多个地址的过程。

721.53.47 存储装置 storage installation

提供存储转发功能的装置。

注：此装置可配备在终端设备上，也可以配备在集中器里。

721.53.48 改向地址 redirection address

在反向信道发送的由若干个地址信号组成的信息，这些地址信号指明该呼叫将要转移或已经转移到的完整地址。

721.53.49 延时传递 delayed delivery

将所存储的报文延迟到预先规定的时间再发送的一种存储转发过程。

721.53.50 报文优先级 message priority

存储转发或报文交换系统中的一种功能，这种功能可使用户将其报文附加已提供的若干优先级当中的一个等级。

721.53.51 报文分隔(功能) message spacing

一种功能，这种功能使用户能够要求网在每次成功呼叫的终了、拆线之前，向其终端设备发送数个换行字符，以便在相邻两份打印报文之间留出空白间隔。

721.53.52 报头[信息头] header

一份报文或一个分组的开始部分，它含有业务信息。

721.53.53 预约呼叫 booked call

用户可以要求网在指定时间内呼叫其终端设备(有或没有可听信号)的一种过程。

721.53.54 对网再呼叫 network recall

在一次呼叫的报文传输阶段，为请求其他服务功能用户再次呼叫网。

721.53.55 多址呼叫 multi-address call

由网建立并包括一个以上被叫终端的呼叫。

721.53.56 会议呼叫 conference call

由任一终端设备所发出的信号能同时被所有其他终端设备接收的一种多址呼叫。

注：各终端可以发送的顺序，由相互协商决定。

721.53.57 广播呼叫 broadcast call

主叫终端将信号同时发送给所有被叫终端设备的一种多址呼叫。

721.53.58 有限制的会议呼叫 restricted conference call

某些被指定的终端设备只能向相关终端设备中的一个或几个发出呼叫、或者根本不能发出呼叫的一种会议呼叫。

721.53.59 广播会议呼叫 broadcast conference call

只有一部指定的终端设备能向其他终端设备发送并从其他终端设备接收的一种有限会议呼叫。

721.53.60 （海事移动业务的）群呼叫 group call (in a maritime mobile service)

一种广播呼叫。这种呼叫允许一个海岸站同时发送给所有被叫船站，而且没有收妥承认信号。

721.53.61 自动顺序连接 automatic sequential connection

由公用数据业务提供的一种功能，利用这种功能可将与指定的一组地址相对应的每个数据终端设备，按预定顺序自动地连接到某一指定地址所对应的数据终端设备。

721.53.62 传递证实 delivery confirmation

一种用户选用业务功能，此业务可以给发送数据终端设备提供有关给定的分组已传递到指定地址（一个或多个地址）的信息。

721.53.63 电信安全(措施) telecommunication security

使电信信号的信息内容只能由经过批准并具有相应功能的设备来提取的一种保护措施。

721.53.64 数据收集 data collection

一种业务功能，它将来自指定地址群的少量数据收集起来，并将它们在网内汇集成单份报文后，传送到另一个指定的地址。

#### 3.5.4 数据传输操作和业务

721.54.01 链路协议 link protocol

通过数据代码、传输方式、控制规程和恢复规程，提供数据链路操作的一套规则。

721.54.02 突发式传输 burst transmission

一种传输方式。在该方式中，来自特定信息源的信息通常以一连串短时间间隔高速传送。

721.54.03 （数据传输中的）呼叫 calling (in data transmission) 在数据站之间建立连接的一系列操作。

721.54.04 人工呼叫 manual calling

一种补充业务，它允许主叫终端设备的选择信号以任意字符速率输入。这些字符可以由数据终端设备或数据电路终接设备产生。

721.54.05 应答 answering

为了完成在数据站之间建立连接，响应主叫站的过程。

721.54.06 询问 interrogation

一个终端装置或数据站请求另一终端装置或数据站标识其身份或给出其状态的过程。

注：数据传输中，由主站请求从站标识其身份。

721.54.07 探询 polling

每次只请求一个数据站发送的过程。

721.54.08 选择(过程) selecting

请求一个或多个数据站接收的过程。

721.54.09 分组排序·packet sequencing

确保分组传送到接收数据终端设备的顺序与从发送数据终端设备发出时的顺序一致的过程。

721.54.10 控制站 control station

在多点或点对点连接中，指定作为主站并且监控轮询选择、询问和恢复程序的数据站。

721.54.11 分支站 tributary station

在多点连接中，除控制站以外的任何数据站。

721.54.12 主(控)站 master station

接受请求并保证将数据传送到一个或多个从站的数据站。

注：在给定的时刻内，一条数据链路上只能有一个主站。

721.54.13 从属站 slave station

由主站选定进行数据接收的数据站。

721.54.14 被动站 passive station

在一个多点连接中，任何等待轮询或等待选择的分支站。

721.54.15 主站 primary station

根据链路协议，对数据链路提供全面控制的数据站。

721.54.16 次站 secondary station

根据链路协议,响应命令的数据站。

721.54.17 组合站 combined station

既能完成主站功能又能完成次站功能的数据站。

721.54.18 （分组交换数据传输中的）路由选择 routing (in packet switch data transmission) 由数据分组在网中选择通道。

721.54.19 适应性路由选择 adaptive routing

根据网络的状态,对相应于同一报文中各个数据分组的路由进行修改。

721.54.20 分组装配 packet assembly

一种由数据网提供的用户功能，它可以使非分组方式的终端以分组交换方式传送数据。

721.54.21 分组拆卸 packet disassembly

一种由数据网提供的用户功能，它将分组数据以适当的形式传送给非分组终端。

721.54.22 分组格式 packet format

在管理分组中控制数据结构的一套规则。它规定了构成分组的各个字段的长度和内容。

721.54.23 数据报业务 datagram service

一种由分组交换网提供的业务。它把能够装在一个分组中的短报文传送到由分组地址标明的目的地。

721.54.24 定位器 aligner

一种用于调整某种数据结构的码元，使之与另一种结构的特定码元相一致的装置。某些情况下是在两种结构之间进行变换。

721.54.25 包封 envelope

由一个字节加上若干数据网操作所需的附加比特所组成的二进制数字组。

721.54.26 永久锁定的包封 permanently locked envelope

总是被相应于整数个包封的若干个比特分隔开来的包封。

721.54.27 数据传送阶段 data transfer phase

在通过网互连的数据终端设备之间，可以传送用户数据信号的数据呼叫阶段。

721.54.28 网控制阶段 network control phase

为建立呼叫、拆除呼叫或控制数据传送阶段的信令，在数据终端设备和网之间交换网控制信号的数据呼叫阶段。

721.54.29 传送时间 transfer time

从发送数据终端设备向网提供用户数据单元开始，到该数据单元完全传送到接收数据终端设备为止，其间所经过的时间。

721.54.30 流量控制 flow control

对数据网中两个指定节点之间(例如,数据终端设备和数据交换机之间)的分组传送速率进行控制的过程。

721.54.31 发送流量控制 transmit flow control

一种流量控制，它使从一个终端发出的数据速率不高于远地终端可接收的速率。

注：①该控制过程适用于数据终端设备和邻近的数据交换局之间或两个终端设备之间。在后一种情况下，其传输速率可根据网或远地数据终端设备的请求来加以控制。

②该控制过程可以在数据传输的两个方向上独立地操作，因此允许在两个传输方向上使用不同的数据传输速率。

721.54.32 适应性信道分配 adaptive channel allocation

信道的传输容量不是预先确定，而是根据需求来指定的一种复用方法。

## A2 离散信号和使用离散信号的传输

A2.1 二进制串行信号(传送)速率 binary serial signalling rate

在串行二态传输的特定情况下，二进制串行信号速率等于单位间隔的持续时间的倒数，用比特/秒表示。

A2.2 最小间隔 minimum interval

指一个系统中最短特征间隔的持续时间。该系统所使用的代码，其特征间隔的理论持续时间不都是单位间隔的倍数。

A2.3 可接受的最小间隔 minimum acceptable interval 用单位间隔表示的可以接受的最小特征间隔。

A2.4 闭路工作 closed-circuit working

当发报机静止时，在电路中有电流流通的单流传输。

A2.5 开路工作 open-circuit working

当发报机静止时，电路中没有电流流通的单流传输。

A2.6 调频 frequency modulation

在电报中，用不同频率的交流电流表示特征状态的调制方法。

A2.7 换频调制 frequency-exchange modulation

从一个频率变到另一个频率并不需要相位连续的调频方法。

A2.8 帧再同步 frame resynchronization 重新建立已丢失的帧同步。

A2.9 成帧比特 framing bit

用于帧同步的二进制位。

A2.10 内务信息 housekeeping information

附加在数字信号上的一些信号。用这些信号使与该数字信号相关的设备正常工作，并在可能情况下提供一些辅助功能。

A2.11 复用链路 multiplex link

让 DTE 在一条单独的电路上获得多条通到数据网的输入输出信道的途径。

注：现已确定的三种可能的方法：

a. 分组交错；

b. 字节交错；

c. 比特交错。

A2.12 开销比特 overhead bits

用户信息比特以外的所有比特。

A2.13 调制的特征状态 significant condition of a modulation

在进行电报调制时，与所选特征量化值相关的器件所处的相应状态。

注：例如：在二态调制（或二元调制）中，有通常称为 A 态和 Z 态（或 0 和 1）的两种特征状态。

A2.14 工作范围 range

电传机接收正确电报信号的时间上限值到下限值之间的范围。

A2.15 电报连接的字符差错率 character error ratio of a telegraph connection

收到的不正确电报字符数与发出的电报字符总数之比。

注：① 在一个电报连接的两个传输方向上可以有不同的字符差错率。

②在表示字符差错率时，必须说明通常用来进行观察的有限时间。对于一个建立时间足够长的连接，应当考虑到超出指定差错率的概率。

A2.16 比特差错率 bit error ratio

接收到的错误比特数与发送出的比特总数之比。

A2.17 码元差错率 element error ratio

接收到的错误码元数与发送出的码元总数之比。

A2.18 同类结构 homogeneous structure

在复用系统中具有相同特性（例如调制速率、字符格式、固有电报畸变等）的一组单独通路。

## A3 电报和数据传输

A3.1 字符格式 character format

对字符信号的综合描述,例如:指出其中所包含的单位码元数。

A3.2 五单位数字保护电码 protected 5-unit numerical code

一种五单位电码，其中十个数字0-9的电码组合均由三个传号和两个空号组成。这个特点使数字电码在传输中产生的差错容易发现。

A3.3 电码孔 code holes

凿孔纸带上代表传号信息的孔。

A3.4 输格 space feed

电传机在打印和凿孔过程中，使印字机构或纸页（纸带）前进一格的动作。

A3.5 无效字符 inactive character

在数据发送阶段发出的不代表信息的填充字符。

A3.6 空闲字符 idle character

在无信息发送时发出的控制字符。

A3.7 话下电报 subtelephone telegraphy

所用频段低于电话传输常用音频频段的电报。

A3.8 话上电报 super-telephone telegraphy

所用频段高于电话传输常用音频频段的电报。

A3.9 电码相关通路 code dependent channel

只能以规定的 n 单位电码格式传输电报信号的电报通路。

A3.10 多路音频电报 multi-channel voice-frequency telegraphy (MCVFT) 在电话通路内使用频分复用传输的电报。

A3.11 单路音频电报 single channel voice frequency telegraphy (SCVFT) 在一条电话通路中提供一条电报通路的音频电报。

A3.12 比特序列独立性 bit sequence independence

数字数据能在不施加任何限制的情况下以二进制数字序列的形式从一个用户传送到另一个用

户的性能。

注：这意味着在数据信号中可以接受任意长度的、连续的二进制数字序列“1”或“0”。

A3.13 字符定位 character alignment

对构成字符的连续比特序列所做的标识。

A3.14 代码相关系统 code dependent system

一种系统，其功能的正确实施依赖于数据终端设备通信时所使用的字符集或代码。

A3.15 数据传送(速)率 data transfer rate

在数据传输系统中，两个相应设备之间在每一单位时间内通过的比特或字符或信息组的平均数。

注：应当指明的相应设备有：调制解调器或中间设备或数据源及数据宿。

A3.16 信息承载信道 information bearer channel

这种数据传输信道用以传送通信所需要的全部信息，其中包括用户的数据同步序列、控制信号等等。因此，它的传输速率可能比传输单纯的用户数据所需的信号速率要高。

A3.17 信息信道 information channel

在两个终端之间的确定方向上传递信息时所涉及的传输媒体和介入的设备。其中包括调制器和解调器、任何的差错控制设备（不论安置在何处）以及反向信道（如果有的话）。

A3.18 信息速率 information rate

信息比特的传送速率(相当于 V.24 接口 103 电路或 104 电路的比特率)。

A3.19 逻辑信道 logical channel

在分组交换操作方式中，通过由相互关联的发送信道和接收信道所组成的数据链路同时进行双向传输的途径。

注：①采用分组交错，可以从一条数据链路中得到一定数量的逻辑信道。

② 多条逻辑信道可以存在于同一条数据链路之中。

A3.20 八位组定位 octet alignment

对一个八位序列进行同步的过程。

A3.21 八位组定时信号 octet timing signal

在串行发送的连续八位组序列中，确定每一个八位组的第一个比特的信号。

A3.22 简单多点电路 simple multipoint circuit

串接的 DCE 不超过两个并提供集中式多点操作的多点电路。

A3.24 对称信道 symmetrical channel

在发送和接收传输方向上，数据信号速率相同的传输信道。

A3.25 超时 time out

一种与设计好的、在预定时间过去之后必定要发生的事件相关的参数。

A3.26 广播中继器 broadcast repeater

连接数条信道的中继器。其中，一条是输入信道，其余的是输出信道。

A3.27 会议中继器 conference repeater

连接数条电路的电报中继器。这种中继器可从这些电路中的任何一条接收信号，并在其他所有的各条电路上自动转发这些信号。

A3.28 电子电传机 electronic teleprinter

用电子技术来实现其功能的电传机。

A3.29 人工电报机 manual telegraph set

供人工收发电报电码用的设备，例如发报电键，收报振荡器等。

A3.30 汉字译码机 Chinese character code translation equipment

接收从纸带式发报机、电报交换机等输出的电码信号，自动译成汉字和字符，并打印出报文的设备。

A3.31 静电打印机 electrostatic printer

用多针静电记录方式按输入的字形信号印出汉字和字符的设备。

A3.32 字符打印机 alphanumeric printer

接收从纸带式发报机、电报交换机等输出的电码信号并打印出字符的设备。

A3.33 (电报)定位器 range finder

电传机接收机构用来寻找正确工作范围并选定工作点所用的部件。

A3.34 自动停止开关 automatic stop switch

在电传机停止工作超过规定时间之后，能使电传机的驱动电源自动切断、转入等待状态。而且，当任何一方需要重新通报时，只要按指定键就能使电传机开启的器件。

A3.35 电报呼叫器 telegraph calling set

在进行电报电路交换时，局所或用户向电报交换机发送信号以选择所需对方局所或用户，并在通报结束后发送拆线信号的部件。

A3.37 报头格式器 message heading device

在每份电报的开始，自动发出发报时间、前缀和流水号的部件。

A3.38 汉字字形发生器 Chinese character generation unit

汉字译码机或字符显示器的主要部件。这个部件根据接收到的指令，从存储器中取出所需要的字形信号供打印或显示之用。

A3.39 时分复用电报设备 time division multiplex telegraph equipment 利用时间分割原理，在同一条电路上同时传输若干路电报的设备。

A3.40 电报分发设备 telegraph distributing equipment

电报局所内从分发台把电报分送到所需的电报机桌上去的机械设备。例如行车、输报带。

A3.41 输报带 belt conveyor

电报局所内输送电报的传送带装置。

A3.42 气压输报管 pneumatic tube

电报局所内利用空气压力输送电报的设备。

A3.43 电报调度测量台 telegraph dispatching and testing desk

电报局所内用来连接和调度电路、监测电报信号和电路工作情形的设备。

A3.44 音频直流变换器 VF/DC converter

把音频信号变换成直流信号或者把直流信号变换成音频信号的设备。

A3.45 单双流转换器 single current/double current converter

把单流信号变换成双流信号或者把双流信号变换成单流信号的设备。

A3.46 起止信号发生器 start-stop signal generator 能产生正确或有规定畸变起止信号的装置。

A3.47 起止信号畸变测试器 start-stop signal distortion tester 测量起止信号畸变的仪器。

A3.48 等时信号畸变测试器 isochronous signal distortion tester

测量等时信号畸变的仪器。

A3.49 快速音叉 tuning fork

校验电传机的电动机转速的音叉。

A3.50 DTE/DCE 接口 DTE/DCE interface

在数据通信网上，数据终端设备(DTE)和数据电路终接设备(DCE)之间的界面上所应用的一组规则。

注：在涉及通过数据通信网提供用户业务的内容时，常常用到该术语。根据用户业务的特征，这种接口可以构成一层或多层功能。

A3.51 接口速率 interface rate

在物理的交换电路上比特流的传送速率。

A3.52 中间设备 intermediate equipment

插在数据终端设备和信号转换设备之间，在调制前或解调后完成某些附加功能的辅助设备。

注：中间设备所有的输入和输出电路及信号都应符合规定的接口标准。

A3.53 复用接口 multiplex interface

以时分复用的方式传送用户信道比特流的 DTE/DCE 接口。

A3.54 信号变换设备 signal-conversion equipment

属于数据信道终端设备的一部分，其中至少包括一个调制器或一个解调器，以实现对发送信号进行调制和(或)对接收信号进行解调。

注：信号变换设备可以包括时钟、信号再生器，但不包括差错控制设备。

A3.55 空白 blank

使机械式电传机的收报部分发生“空跳”（既不打印字符，也不出现其他动作）的信号。

## A4 传真

A4.1 文件传真四类机 document facsimile equipment, Group 4

主要经由公用数据网传输，并在传输之前，减少文件信息冗余度的传真机。这类传真机采用适合于公用数据网的规程，并保证基本上无差错地接收文件。经合适的调制解调处理后，也可以在公用电话网上使用。

A4.2 传真机 facsimile apparatus

应用扫描技术，把固定的图像（包括相片、文字、图表等）转换成电信号传送到接收端，再以记录形式（在硬拷贝上）复制出来的电信设备。

A4.3 传真收发两用机 facsimile transceiver 兼有发送和接收两种功能的传真设备。

A4.4 相片传真机 phototelegraph apparatus

主要用于传送有连续色调的图像，并用照相记录法记录出符合一定色调密度级要求的图像副本的传真机。

A4.5 信函传真机 postfax apparatus

具有自动拆、封装置的传送信函用的传真机。

A4.6 报纸传真机 newspaper page facsimile apparatus

其发送原稿为包括有网点图片的报纸清样，而接收端可复制出供印刷制版用的软片的传真机。

A4.7 彩色传真机 color facsimile apparatus

传送并复制彩色图像的传真机。

A4.8 气象图传真机 weather-chart facsimile apparatus 传送气象图用的传真机。

A4.9 线路连接设备 line connection equipment

在传真机或传输设备与中继线之间或两个传输设备之间装用，以满足系统的某些特定要求（如对传真系统的某些特性进行补偿和匹配等）的接口设备。

A4.10 幅频变换器 AM-FM converter 把调幅信号变换成调频信号的装置。

A4.11 频幅变换器 FM-AM converter 把调频信号变换成调幅信号的装置。

A4.12 扫描轨迹 scanning track

扫描点在原稿或记录媒体上相对运动的轨迹。

A4.13 主扫描 main scanning

扫描点沿扫描线方向的扫描。

A4.14 副扫描 subscanning

扫描点垂直于扫描线方向的扫描。

A4.15 图像频率 picture frequency

由扫描原稿所产生的传真信号频率，它不包括已调波中的其他各频率分量。

A4.16 最高调制频率 maximum modulating frequency

要求传真传输系统传输的最高图像频率。它与主扫描方向的分辨力(即纵向分辨力)有关。

A4.17 最高键控频率 maximum keying frequency

数值上等于扫描速率除以扫描点沿主扫描方向尺寸的两倍。

A4.18 游程长度编码 run-length coding (RLC)

为了减少传真信号的冗余度，对连续的相同光密度像素的扫描游程长度（以像素数来表示）进行编码的方法。

A4.19 副载波调幅 subcarrier amplitude modulation

为了传输的需要或其他原因，把扫描产生的传真信号对一个低频载波调幅，再用这个已调波去调制一个高频载波。上述对低频载波的调幅称之为副载波调幅。

A4.20 副载波调频 subcarrier frequency modulation

为了传输的需要或其他原因，把扫描产生的传真信号对一个低频载波调频，再用这个已调波去调制一个高频载波。上述对低频载波的调频称之为副载波调频。

A4.21 连续色调 continuous tone

图片中包括黑和白在内的全部光密度范围。

A4.22 半色调特性 half tone characteristics

记录副本光密度与发送原稿光密度之间的关系。

A4.23 光密度 optical density

一个面积的光透射或光反射特性的量度。它是用入射光通量与透射或反射光通量之比的常用对数来表示。对前者称之为透射光密度，后者称之为反射光密度。

A4.24 传真测试样张 facsimile test chart

印有专门设计的图样(包括文字和图像)，用来测试传真设备或系统的部分性能。

A4.25 分辨力 resolution

描述形成图像细节能力的一种尺度。对三类机和四类机来说，传真发送的分辨力以每毫米像素数和每毫米扫描线数来表示。

A4.26 抽样失真 sampling distortion

在传真中，一条扫描线上对同一光密度像素的游程长度与抽样后所得出的游程长度之差。

A4.27 驱动斑纹 drive pattern

由于接收机驱动系统周期性的瞬时速率变化,导致记录图像的密度也发生周期性变化所呈现出的斑纹。

## A5 电报和数据网

A5.1 分散式多点传输 decentralized multipoint

一种可选择的用户功能。它使用户能在 $ n+1 $ 个DTE之间建立连接。每个DTE可以向其他n个DTE发送信息，而且每个DTE可以接收来自其他所有DTE的信息。

A5.2 网参数 network parameter

电信网设计中网的固有特性。

A5.3 专用数据网 private data network

由专用部门建立并经营用于数据通信的网。

注：专用网可否连接到一个或多个公用数据网上，取决于国家的管理体制。

A5.4 （电报）音频通路 （telegraph） voice frequency channel 一对音频电报设备之间多路电报信号同时传输的公共通路。

A5.5 重新选择 reselection

一种处理过程。在这过程中，一个 DTE（称作 DTEA）可以请求 PAD 拆除它与某个起止式 DTE 之间已经建立的虚呼叫，并在该起止式 DTE 与另一个 DTE（称作 DTEB）之间建立新的虚呼叫。DTEB 是由 DTEA 选择并通知 PAD 的。

A5.6 串行自动呼叫 serial automatic calling

DTE 可以通过 100 系列交换电路指示 DCE 完成呼叫建立功能的一个程序。从 DTE 到 DCE，每位拨号数字在交换电路 103 上以串行方式传输。

A5.7 测试段 test section

在两个有测量设备的站之间，能进行电报传输测试的一段通路。

A5.8 人工电报交换机 manual telegraph switching equipment 由人工操作并用以完成电路交换的电报交换设备。

A5.9 机电式电报交换机 electro-mechanic switching equipment 采用机电器件完成电路交换的电报交换设备。

A5.10 存储程序控制电子电报交换机 stored program controlled electronic switching equipment 采用数字电子计算机控制方式完成电路交换或报文交换的电报交换设备。简称程控电报交换机。

A5.11 电报集中器 telegraph concentrator 可以使少量的电报终端设备和较多个局所或用户进行通报的设备。

A5.12 同文电报接线设备 multi-address telegraph connector

能使一个局所或用户向若干不同地点的局所或用户发送同样内容报文的设备。

A5.13 撕断纸带接转方式 torn tape relay system

将需要转发的电报凿孔纸带按每份电报撕断，然后利用人工或局内去报分发设备，将电报送到通向接收局所或用户的发报机桌上，以便将电报发送出去的一种纸带接转方式。

A5.14 地址分隔符 address separator

在选择信号中用于分隔不同地址的字符。

A5.15 应答单音 answering tone

由被叫端向主叫端发出的单音。

A5.16 呼叫控制信号 call control signals

建立、保持和释放一次呼叫所必需的一组信号。

A5.17 呼叫建立 call establishment

用于建立一次数据连接的一系列操作。

A5.18 呼叫标识符 call identifier

一种网的实用信息，是由始发网分配给每个已建立或已部分建立的虚呼叫的标识。在与主叫DTE地址一起使用时，这些标识在一定时间内能唯一地识别虚呼叫。

A5.19 呼叫进行信号 call progress signal

由 DCE 发给 DTE 的呼叫控制信号，用以通知该 DTE 某次呼叫的进行情况或未能建立连接的原因或任何其他网的情况。

A5.20 呼叫释放时间 call release time

从终端设备发出拆线信号开始，到始发数据终端设备处于空闲电路状态这一段时间。

A5.21 呼叫请求信号 call request signal

在呼叫建立阶段，用于向 DCE 表明 DTE 希望建立呼叫连接的信号。

A5.22 重选呼叫路由 call rerouting

在试图建立一次连接期间，改变原定的呼叫路由。

A5.23 呼叫路由选择 call routing

由交换机从若干业务路由中选出一条指定的呼叫路由。

A5.24 被叫线路标识功能 called line identification facility

一种由网提供的功能，它使主叫终端能从网得知其呼叫已接通的地址。

A5.25 被叫线路标识信号 called line identification signal

一组发送到主叫 DTE，并使之可以识别被叫线路的字符序列。

A5.26 主叫线路标识功能 calling line identification facility

一种由网提供的功能，它使被叫终端能从网得知该呼叫发自何地址。

A5.27 主叫线路标识信号 calling line identification signal

一组发送到被叫 DTE，并使之可以标识主叫线路的字符序列。

A5.28 主叫站响应 calling station response

主叫 DCE 为对其检测到的应答单音作出响应而发送的一种单音或信号。应答单音的检测已由 CCITT 建议 V.25 作出规定。

A5.29 呼叫单音 calling tone

由主叫端发送的单音，该单音可以是1300Hz单音或在DCE中对应于二进制“1”的任何单音。

A5.30 阻止呼叫 calls barred

一种允许终端设备只发出呼叫或只接收呼叫(但不能同时进行)的功能。

A5.31 集中控制信令 centralized control signalling

通过一条(或多条)专用的传输信道来交换数据传输电路中呼叫控制信号的系统。

A5.32 清除 clearing

用于断开一次呼叫，并且返回到准备状态的一系列操作。

A5.33 编码带内信令 coded inband signalling

在正向信道中，控制信号通过数据方式进行交换的带内信令方式。

A5.34 连接进行中(信号) connection in progress

在 DCE 和 DTE 接口处的一种呼叫控制信号，它向 DTE 表示建立数据连接正在进行之中，并且随后准备传输数据信号。

A5.35 数据连接 data connection

通过交换设备将一定数量的数据电路互相串联起来，使数据传输能在数据终端设备之间进行。

注：① 在相互连接的数据电路中有一条或多条电路为数据虚电路时，整个连接称为数据虚连接。

② 整个连接包括位于数据终端设备一方的数据电路终接设备。

A5.36 数据传送请求 data transfer requested

在专用电路业务中，由 DCE 对 DTE 发送的一种呼叫控制信号，用来表示远端 DTE 希望交换数据。

A5.37 DCE 拆线指示 DCE clear indication

由数据电路终接设备(DCE)发送的呼叫控制信号,用来表示正在拆线。

A5.38 DCE 等待 DCE waiting

在 DCE 和 DTE 接口处的一个呼叫控制信号，它表示 DCE 正在等待呼叫建立程序中的另一个事件。

A5.39 分散控制信令 decentralized control signalling

通过在数据传输电路上发送信号的方式，交换有关该电路呼叫控制信号的系统。

A5.40 (建议 X.25 中的)诊断码 diagnostic code (in recommendation X.25)

一种唯一的代码组合，它与国际五号电码和二进制代码所表示的十六进制数相同，用于在DTE 和 DCE 之间传送信息以指出差错、故障或某一个 DTE 与网或与另一个 DTE 之间固有的不兼容性。

A5.41 DTE 忙 DTE busy

数据终端设备(DTE)由于不能接收另外的呼叫而处于不可占用的状态。

A5.42 DTE 拆线请求 DTE clear request

由数据终端设备(DTE)发送的请求拆线的呼叫控制信号。

A5.43 受控 DTE 未准备好 DTE controlled not ready 表示虽然 DTE 可以工作，但暂时还不能接受呼入。

A5.44 非受控 DTE 未准备好 DTE uncontrolled not ready 表示 DTE 不能接受呼入(通常是因为工作状态不正常)。

A5.45 DTE 等待 DTE waiting

在 DTE/DCE 接口处的一种呼叫控制信号状态，它表明 DTE 正在等待来自 DCE 的呼叫控制信号。

A5.46 选择结束 end of selection 表示选择信号结束的字符。

A5.47 业务功能请求分隔符 facility request separator 在选择信号中分隔不同业务功能请求的字符。

A5.48 带内信令 in-band signalling

利用 DCE 线路信号频带(数据信号在该频带的前向信道中传送)，在与其相连的 DCE 之间交换的控制信号。如果此时有 DTE 的数据，则该数据将被扰乱。

A5.49 网选择信号 network selection signals

表示是呼叫控制信息而不是网建立呼叫时所要求的地址信息的选择信号。

A5.50 带外信令 out-of-band signalling

在相互连接的 DCE 之间，利用前向信道中数据信号之外的信号进行控制信号的交换。此时，DTE 数据的传输不受破坏。

A5.51 分组长度选择 packet length selection

一种用户功能，它使 DTE 可以从一组指定的长度中选出某个认定的最大的用户数据字段长度。

A5.52 并行自动呼叫 parallel automatic calling

DTE 可以通过 200 系列交换电路，指示 DCE 完成呼叫建立功能的一个程序。从 DTE 到 DCE，

每位拨号数字在电路 206 到 209 上以并行方式传输。

A5.53 转接网标识 transit network identification

一种为每个转接网命名的网服务，该转接网控制已建立的或部分建立的虚电路中的一部分。

A5.54 通报速率 telegraph rate

传送电报的速率，通常用“波特”、“字符数每分钟”或“字数每分钟”来表示。

A5.55 缩位呼叫 abbreviated address calling

用户可使用比完整地址字符数少的地址进行呼叫的功能。

A5.56 接入拒绝概率 access denial probability 从接入请求开始到接入成功所经历的时间。

A5.57 接入时延 access delay

在特定的抽查中，被拒绝接入的接入尝试次数与全部接入尝试次数的比值。

注：这个比值实际上是对真正概率值的一种估计。

A5.58 呼叫路由 call route

用于在两点间提供连接的一组电路。

A5.59 占线终端设备空闲时的回叫 call back when busy terminal installation becomes free 在呼叫占线终端设备时，始发端可以请求网在该占线终端设备空闲时建立呼叫的功能。

A5.60 DTE 和 DCE 接口处的呼叫碰撞 call collision at the DTE/DCE interface

数据终端设备发出的呼叫请求信号和来自数据电路终接设备的入呼叫信号同时出现，使得双方的设备都不能接收所等待的应答。

A5.61 接入类别 category of access

数据终端设备借以接入公用数据传输业务的方法。在这种数据传输业务中，采用标准化的数据信号速率和 DTE/DCE 接口。已经规定的方法包括直接连到公用数据网或综合业务数字网，以及通过其他公用网连到上述网。

A5.62 集中多点功能 centralized multipoint facility

集中多点功能使中央 DTE 能同时向两个或多个远端 DTE 发送数据，并且能每次接收一个远端 DTE 发来的数据。而一个远端 DTE 发来的数据不传递给其他远端的 DTE。

A5.63 电路交换数据传输业务 circuit-switched data transmission service 在数据终端设备之间传递数据之前，需要建立电路交换连接的业务。

A5.64 拆线证实时延 (CLCD) clear confirmation delay

从被拆线 DTE 发送 DTE 拆线证实信号至收到 DCE 准备好信号之间的时延。

A5.65 拆线请求时延 clear request delay (CLRD)

从要求拆线的 DTE 发出拆线请求信号至收到 DCE 准备好信号之间的时延。

A5.66 具有外接通道的闭合用户群 closed user group with outgoing access

分配给闭合用户群中某个用户的一种功能。这种功能使该用户能在适当的情况下，与公用数据网传输业务的其他用户进行通信。这种功能可以分配给某些具有与其他任何公用交换网相连的数据终端设备的特定用户，使他们也能利用网的互通功能。

A5.67 数据信号传送速率透明性 data signalling rate transparency

一种网的参数，它使一个用户与另一用户之间进行数据传输时，在一定范围内对所用的数据信号速率无需有任何限制。

A5: 68 数据传送 data transfer

数据信号从数据源到数据宿的传输过程。

A5.69 数据用户部分的处理时间 data user part handling time  

从电文最后一个比特进入数据用户部分开始到电文最后一个比特离开数据用户部分所经过的

那段时间。

A5.70 切断时延 disengagement delay

从对某一特定用户开始试切断到该用户被完全切断所经过的时间。

A5.71 切断拒绝概率 disengagement denial probability

在特定的抽查中，被拒绝的试切断次数与全部切断次数之比。

注：该比值实际上是对真正概率值的一种估计。

A5.72 额外用户信息传递概率 extra user information delivery probability

在特定的抽查中，所有（未要求的）额外用户信息单元与目的地用户收到的全部信息单元的比值。

注：该比值实际上是对真正概率值的一种估计。如果数据源用户向目的地用户提供的信息单元中没有应输入系统的比特，那么该特定目的地用户所收到的信息单元就称为额外信息单元。

A5.73 虚呼叫业务的流量控制参数选择/协商和指示 flow control parameter selection/negotiation and indication for virtual call service

虚呼叫业务的一种用户功能，它为分组型 DTE 提供了 8～11 个业务等级，以选择、协商分组和窗口的大小以及分组接收序号  $ [P(R)] $  的意义，同时可使 DCE 在呼叫建立阶段结束时指出各参数的适当值。

A5.74 错误接入概率 incorrect access probability  

在特定的抽查中，错误的试接入次数与所有试接入次数的比值。

A5.75 信息 information

从远程数据处理的特定意义上讲，信息意味着信号通过电信信道进行传输。这种信息可以用数据或报文的形式来表示。

A5.76 各类用户业务之间的互通 interworking between user classes of services

使某用户业务类别的数据终端设备可与属于不同用户业务类别的另一数据终端设备进行通信的一种手段。

A5.77 网故障 network failure

在网上由于运转不正常而拒绝提供某种业务的情况。

A5.78 网传送时延 network transfer delay

电信网从始发 DTE/DCE 接口向目的地 DTE/DCE 接口传递某个实体所需的时间。根据采用的操作方式，该实体可为一个比特、一个分组或一份报文。

A5.79 分组式操作 packet mode operation

通过带地址的分组进行数据传输，这些分组只在分组传输期间占用传输信道，随后，这个信道即可用于其他数据终端设备之间的分组传输。

注：在某些数据网上，数据可以拆装成（可以通过数据终端设备，也可以通过网中的设备来进行）一些分组以便传输或复用。

A5.80 分组交换数据传输业务 packet switched data transmission service 一种以分组方式进行传输的业务，必要时按分组的方式对数据进行拆装。

A5.81 奇偶检验功能 parity function

一种由 PAD 提供的功能，它允许起止式 DTE 和分组式 DTE 根据 PAD 发送和接收字符的奇偶检验位来选择 PAD 应完成的操作范围。

A5.82 断电状态 power off condition

设备部件中的电源不能使用时的状态。

A5.83 优先功能 priority facility

使某一用户比其他用户具有优先权的功能。

例如，可以优先处理呼叫、传输分组及由网提供的其他服务。

A5.84 公用数据传输业务 public data transmission service 由主管部门通过公用数据网建立并经营的数据传输业务。

A5.85 接收拥塞 reception congestion 在交换中心发生的一种网拥塞状态。

A5.86 业务可用率(性) service availability

能够提供令人满意或可以容忍的业务的时间之和与总的业务时间之比。

A5.87 终端操作方式 terminal operating mode

在公用数据传输业务中，为满足国际用户业务类别的参数由数据终端设备指定的方式。

A5.88 全部呼叫连接时延 total call connection delay (TCCD)

从主叫 DTE 发出呼叫请求信号至收到准备发送数据信号之间的时间间隔。

A5.89 非常规传输代码 transmission code violation

不包括在传输代码中的数字。当少量使用时，这些数字可以在不明显影响信号频谱的情况下提供一些补充信息。

A5.90 透明数据传送阶段 transparent data transfer phase 可以在 DTE 之间传输任意比特序列的数据传送阶段。
