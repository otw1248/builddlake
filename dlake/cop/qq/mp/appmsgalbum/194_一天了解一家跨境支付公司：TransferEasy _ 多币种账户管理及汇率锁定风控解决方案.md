本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、公司概况**

以下内容基于公司公开资料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

2014 年 3 月成立，注册全名为北京银通易汇科技有限公司，品牌名为 TransferEasy。

**🌍****全球布局**

总部位于中国北京。业务涉及美国、加拿大、欧洲、澳大利亚、新西兰、香港、菲律宾、印度尼西亚等国家/地区，部分地区通过合作伙伴开展业务。

**🛡️****合法资质**

支付牌照：在加拿大 FINTRAC(加拿大金融交易和报告分析中心)完成 MSB 注册；持有香港 MSO 牌照；完成美国联邦 FinCEN(美国金融犯罪执法网络)MSB 注册和新泽西州货币传输牌照。

国际安全认证：通过 PCI DSS 4.0 Level 1 认证、ISO 27001 认证。

官方合作：微信支付、支付宝官方合作伙伴；亚马逊 Marketplace 授权支付服务合作伙伴，可在 Amazon Selling Partner Appstore 验证其 PSP 资格。

**📊****业务范围**

聚焦跨境金融基础设施服务、多货币虚拟账户、跨境整合支付、全球汇款服务。服务场景包括跨境贸易、外贸企业资金管理、金融机构合规防控。

**🎯****核心服务概览**

核心产品：TransferEasy Business Solution(企业结算)、HipoPay(整合支付网关)、BasaltMatrix(反洗钱及风险管理引擎)。

支持币种：基于业务涉及地区，至少覆盖 USD/EUR/CAD/AUD/HKD/CNY 等 **6 种** 主要国际货币。

到账速度：主流币种支持 **T+0** 到账，部分币种或场景为 T+1 至 T+2，具体以合同约定为准。

主要服务特点：多货币虚拟账户、汇率锁定、自主开发并持续优化的 BasaltMatrix 风控系统、易汇卡(虚拟卡)发行。

## **二、收费方式与服务规则**

**📋****服务项目列表****💱****汇兑与结算服务**

按交易金额、币种、业务类型、企业规模等因素实行定制化报价，涵盖全球汇款、外汇兑换、多币种收付款等核心功能。

**🔗****跨境整合支付**

为企业提供微信支付、支付宝等本地支付的统一接入服务，收费模式需与商务团队定制化协商。

**💳****易汇卡(虚拟卡)服务**

涵盖虚拟卡的发行、充值、交易处理服务，公开信息强调消费返现优惠，易汇卡基于国际卡组织网络发行，费用结构需与企业协商。

**🏦****虚拟账户与资金管理**

涉及海外虚拟账户的设立、资金归集、分账管理等服务，收费模式为企业定制化方案。

**📝****费用说明**

汇率报价基于市场中间价加服务商加价，具体费用结构采用定制化报价模式，可能包含汇兑加价、交易手续费、账户维护费、提现费用等项目。

## **三、国际化服务能力**

### (一)支付网络与覆盖区域

**🔄****国际清算方式**

提供全球汇款服务，通过 SWIFT 报文网络(全球银行间金融通信协会)传输支付指令，资金清算则通过全球银行间代理行网络和各国清算系统(如美元 CHIPS(纽约清算所银行同业支付系统)、欧元 TARGET2(跨国自动实时全额清算系统)、人民币 CIPS(人民币跨境支付系统)等)完成。

**🔗****本地支付整合**

侧重于亚太地区本地支付网络的整合：

中国：作为微信支付、支付宝官方合作伙伴，覆盖全球商户的中国游客钱包收单。

北美/亚太：支持在美国、加拿大、澳大利亚、新西兰、香港、菲律宾、印度尼西亚等地区开展业务。

**💵****多货币虚拟账户**

提供多货币虚拟账户服务，支持企业在全球范围内开设境外收款账户，进行资金归集与管理。

**📍****区域总部**

公司总部位于中国北京，并在加拿大拥有注册实体办公室，在主要的出海国和资金结算地建有本地支持能力。

### (二)合规与安全体系

**🏛️****牌照与认证**

在加拿大 FINTRAC(加拿大金融交易和报告分析中心)完成 MSB 注册(汇款业务)，持有香港 MSO 牌照，完成美国联邦 FinCEN(美国金融犯罪执法网络)MSB 注册及新泽西州货币传输牌照。通过 PCI DSS 4.0 Level 1 认证和 ISO 27001 认证。

**🛡️****反洗钱能力**

自主开发并持续优化 BasaltMatrix 反洗钱及风险管理系统，系统符合美国《银行保密法》(Bank Secrecy Act)、OFAC(美国海外资产控制办公室)制裁名单筛查、中国《反洗钱法》、香港《打击洗钱及恐怖分子资金筹集指引》、欧盟《反洗钱指令》等反洗钱合规要求。

系统功能覆盖风险识别、合规报告生成、制裁名单筛查等，符合全球监管标准。

**💰****资金托管服务**

公司与国际银行合作进行资金托管，遵循合规要求。

## **四、汇率管理与资金流转**

**📊****汇率管理方式****💹****汇率基准与来源**

汇率报价基于市场中间价加上服务商加价，实际汇率和加价比例可能因交易额、币种、客户类型而异，并提供 24 小时实时报价。

**🔄****换汇与避险工具**

多货币虚拟账户支持资金的实时兑换，满足全球资金管理需求。

汇率锁定服务：支持客户提前锁定外汇汇率(锁定时长从几分钟至 24 小时)，以对冲汇率波动风险。汇率锁定功能适用于有大额外汇交易且担心短期内汇率波动的场景，锁定后汇率固定不变，可规避市场汇率朝不利方向波动的损失。

**💵****资金账户与管理****🏦****账户类型**

提供多货币虚拟账户服务，支持企业及金融机构开设境外收款账户，实现多币种收付与资金归集管理。

**💧****资金池管理**

支持资金分账、账单代收、资金代付、代发工资/佣金等企业级资金流转场景，税务代扣功能涉及多国法规，可通过 API 接口实现自动化管理。

## **五、平台对接与区域重点**

### (一)平台与合作伙伴接入

**🛒****电商平台回款**

作为 **亚马逊 Marketplace 授权支付服务合作伙伴** ，支持跨境电商卖家的支付结算和收益管理。

**🤝****支付网络合作**

作为本地支付网络官方合作伙伴，重点在跨境整合支付领域，助力全球商户实现中国游客钱包的本地化收单。

**🔌****企业系统(ERP/财税)集成**

通过 API/SDK 接口，支持企业将支付数据与自有系统对接，实现资金流转、分账、支付与税务代扣等功能的自动化。

### (二)区域支付注意事项

**🌍****区域核心业务覆盖关键合规/风险提示**

北美 (CA/US)：资金结算、全球汇款在加拿大 FINTRAC 完成 MSB 注册；完成美国联邦 FinCEN MSB 注册和新泽西州货币传输牌照，需注意各州货币传输牌照要求。

亚太 (HK/PH/ID)：资金归集、微信/支付宝收单持有香港 MSO 牌照，需注意东南亚地区各国的本地化监管和支付渠道差异。

欧洲 (EU)：多货币虚拟账户服务在欧洲地区，业务模式主要基于全球清算网络支持。

## **六、附加服务与安全体系**

### (一)核心风险控制与安全能力

**🛡️****风控引擎(BasaltMatrix)**

自主开发并持续优化 BasaltMatrix 反洗钱及风险管理系统，该系统具备多国监管合规算法、可疑交易识别、制裁名单筛查等能力，适用于金融机构的合规防控需求。

**🔒****安全认证与合规承诺**

通过 PCI DSS 4.0 Level 1 认证(已提前通过最新版认证标准)和 ISO 27001 认证，表明在支付数据安全和信息系统管理方面遵循国际高标准要求。遵循多国反洗钱法规，所有投诉、建议、隐私相关问题承诺在 **7 个工作日** 内回复。

### (二)虚拟卡服务(易汇卡)

**💳****服务内容**

提供企业虚拟卡(易汇卡)的发行与交易处理服务，支持余额即时开卡、即时充值，并提供消费返现优惠。适用于在 Facebook、Google 等平台投放广告，或支付海外物流、员工采购等费用的场景，可即时开通多币种虚拟卡，设置消费限额和有效期，用于灵活、安全的海外支出。虚拟卡资金归集于主账户，便于企业财务集中管理。使用前需确认目标平台是否接受虚拟卡支付。

### (三)技术能力与开发者支持

**🔧****API/SDK集成能力**

提供 API/SDK(已发现 PHP SDK)供开发者对接自有系统。API 接口支持资金分账、账单代收/代付、订单状态异步通知(Webhook)等功能，以及企业级安全功能(包括数据加密、重复交易防护等)。采用私有 API 模式，API 文档向签约客户开放。企业需与商务和技术团队联系获取对接文档。

## **七、适用场景分析**

### (一)适合使用 TransferEasy 的场景

  * **🌐****外贸/电商资金结算企业** ：主要服务从事跨境贸易或电商业务的企业，提供多币种收付款、外汇兑换交易和全球汇款的一站式资金结算平台。

  * **💼****多渠道资金管理需求** ：适用于需要统一管理来自 Amazon Marketplace、独立站或其他渠道资金的企业，利用 BasaltMatrix 系统进行反洗钱和合规筛查，可作为资金集中管理基础设施。

  * **📊****海外采购/广告投放频繁的企业** ：适用于有大量海外采购或广告投放支出的企业，易汇卡(虚拟卡)提供即时开卡和返现功能。

  * **📈****对汇率稳定性有高要求的企业** ：适用于对汇率波动较为敏感、需要规避风险的企业，提供汇率锁定服务(锁定时长从几分钟至 24 小时)，帮助企业提前固定成本。




### (二)选择建议

在选择跨境支付服务商时，建议从以下维度进行综合评估：

**🎯****业务场景匹配度**

根据企业的主要市场、客户支付习惯、交易规模等因素，对比多家服务商的适配程度。

**⚖️****合规透明度与牌照**

在签约前，务必通过监管机构官方网站直接验证牌照真实性，以确保业务的合规性。

**🔧****技术集成模式**

由于 API 文档不公开，评估企业团队是否能接受私有 API 对接模式，并预留充足时间与官方技术团队进行对接。

**💰****费用与合同条款**

明确计费规则、服务边界、数据安全、争议处理等关键条款，特别是定制化费率和汇率加价比例，选择最符合业务需求的方案。
