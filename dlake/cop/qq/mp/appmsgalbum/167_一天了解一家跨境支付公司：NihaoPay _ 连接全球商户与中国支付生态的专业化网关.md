本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业基本档案**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考

**📅****成立时间与主体：**

2013 年成立；全球总部的法律实体为 Aurfy Inc.（NihaoPay），在美国加利福尼亚州注册

**🌍****全球布局：**

全球总部位于美国加州圣克拉拉；在中国上海、深圳及日本东京设有办公室，以支持其在亚太区的核心业务

**🛡️****合规资质：**

核心资质：作为银联（UnionPay）、支付宝（Alipay）、微信支付（WeChat Pay）的官方全球收单行

监管注册：运营主体持有 NMLS 资质定位为货币传输商，在美国财政部金融犯罪执法网络（FinCEN）完成货币服务企业（MSB）注册

业务覆盖：支付网络聚焦于中国三大主流支付方式；支持全球多个国家的商户注册，并提供 USD、CAD、EUR、GBP、AUD、JPY、HKD、SGD 等主要货币的结算服务

安全认证：通过 PCI DSS Level 1 认证（支付行业安全认证等级）

**🎯****行业渗透与应用场景：**

深度服务于希望触达中国消费者的全球各类商户，覆盖跨境电商、SaaS 平台、旅游、数字娱乐等行业；提供线上支付网关、线下扫码支付、电商平台插件及支持利润分享的 API 解决方案

## **二、全球化能力图谱**

###  （一）支付网络与区域布局

**💳****收单网络：**

为全球商户提供一站式接入中国支付生态的解决方案，支付网络能力主要围绕银联、支付宝、微信支付三大核心通道构建，提供 Visa、Mastercard、JCB、American Express、Discover 等国际卡组织的收单服务

**🔗****本地支付聚合：**

东亚（中国）：深度整合中国的银联、支付宝与微信支付

东南亚及其他区域：通过 Alipay + 产品支持 AlipayHK、GCASH、DANA、KakaoPay、TNG、TrueMoney 等当地主流钱包

**🔄****跨境清算与区域枢纽：**

支持向全球商户进行多币种结算，资金通过国际银行电汇与本地清算网络划转至商户账户以美国总部为全球运营中心，以上海/深圳、东京办公室为亚太区关键枢纽，提供 24/7 多语种客户服务

### （二）合规与安全体系

**🏛️****核心监管资质：**

运营主体 Aurfy Inc.持有 NMLS ，并已在美国财政部金融犯罪执法网络（FinCEN）完成货币服务企业（MSB）注册

**📜****行业认证：**

获得 PCI DSS Level 1 认证，国际卡收单服务采用 RSA-SHA256 加密和 OAEP 填充模式确保卡数据安全

**👤****合规流程：**

作为持牌支付服务提供商，遵循标准的 KYC/AML 合规流程，并依据美国相关法规对客户资金进行隔离（Safeguarding）

## **三、技术能力清单**

###  （一）智能技术与架构

**⚙️****平台架构：**

采用 RESTful API 架构，为开发者提供独立的沙盒测试环境

**🛡️****风控引擎：**

主营的支付方式（支付宝、微信支付）具备无拒付特性，基于钱包余额或银行账户的推送式支付，不存在传统信用卡支付的未授权交易风险，商户不会面临拒付造成的资金损失和运营压力，为商户提供交易后风险保障

**🔧****开发者生态：**

API 与 SDK：提供 RESTful API 及覆盖 PHP、Java、C# 等主流编程语言及 Android 开发环境的开源 SDK 和代码示例，托管在 GitHub 平台

开发工具：提供公开的 API 参考文档与沙盒测试环境

Webhook 机制：支持 IPN（即时支付通知）异步回调，并设有完善的重试策略

预置对接方案：提供官方 WooCommerce 插件（持续兼容最新版本）、Magento 2.X 模块，并通过第三方支持与 Shopify 的集成

### （二）支付与结算效率

**⏱️****结算时效：**

T+2 结算

**💵****资金管理技术：**

多币种结算：支持 USD、CAD、EUR、GBP、AUD、JPY、HKD、SGD 等主要货币的结算

自动扣费功能：新增支持微信支付自动扣费（Auto Debit）功能，支持签约管理、自动扣费操作及合约查询等完整功能

## **四、计费结构与服务规则**

**📋****整体定价原则：**

采用按交易计费模式，无月度维护费根据商户的交易量和合作规模，提供阶梯式折扣，具体以双方签署的《商户服务协议》为准

**🏦****结算费用：**

因支付方式、收款国家与地区而异美元结算通常通过美国自动清算所（ACH）或国际电汇完成，相关费用由银行及汇款渠道实际成本决定

**⚖️****争议处理：**

商户的争议处理流程简化，相关订单争议通过平台内部申诉渠道解决，无需通过国际卡组织或银行的清算拒付流程

**📌****小结：**

本部分重点梳理 NihaoPay 的计费结构与服务规则，不涉及具体数值细节每一位商户实际费率与费用标准应以商户服务协议的最终签署版本为准

## **五、汇率管理与资金流转**

**📊****汇率机制：**

支持商户在前端以人民币（RMB）定价，系统在交易时自动、实时地将金额转换为商户的结算币种汇率采用全包模式，成本已包含在交易率中，简化商户的成本结构

**🔄****换汇能力：**

系统内置即期换汇功能，支持交易时的实时汇率转换

**💵****多币种账户体系：**

提供多币种结算账户，商户可通过交易管理系统（TMS）实时查看各币种账户余额、交易历史和结算状态

## **六、生态对接与服务能力**

###  （一）电商与平台集成

**🛒****电商平台集成：**

提供面向主流电商平台的官方插件（如 WooCommerce、Magento），并支持与 Shopify 等平台的快速集成，满足独立站商户的接入需求

**🏢****平台与代理商能力：**

子商户管理：支持代理商-子商户两级管理模式代理商可在代理商管理系统（Agent Management System）中统一管理下属子商户的账户、交易、资金及结算情况，设置不同权限与费率策略

多方分账（Profit Sharing）：提供基于订单或全局设置的利润分享功能，支持按百分比或固定金额将单笔交易资金分配至多个参与方，执行时间具体以服务协议为准

### （二）企业系统与服务交付

**🔌****ERP/财税系统集成：**

主要通过开放 REST API 实现数据读取与交易记录导出，商户可获取交易流水、结算报告等用于本地财务核算

**📞****服务与支持：**

提供标准化的商户上线流程，并通过邮件、热线及专属客户经理提供支持；同时提供完整的 API 文档、示例代码及常见问题（FAQ）等自助服务资源

## **七、增值工具与行业解决方案**

###  （一）风控解决方案

**🛡️****核心优势：**

支付解决方案内置风险规避功能；支付宝与微信支付的支付机制从源头避免传统信用卡支付的未授权交易风险，降低商户的资金损失与运营压力

**🔍****欺诈检测能力：**

平台内置欺诈检测功能，为商户提供多层次的风险防护

### （二）行业方案要点：跨境电商/独立站

**🎯****适配方案组合：**

银联（UnionPay）+ 支付宝（Alipay）+ 微信支付（WeChat Pay）三合一收单 + 国际卡收单服务 + 主流电商平台插件

**💪****能力亮点：**

专注服务中国消费者，提供以移动支付为核心的服务体验及人民币定价功能，国际卡收单服务支持全球信用卡用户

## **八、适配场景与注意事项**

###  （一）适配场景

  * **🌏****希望拓展中国市场的海外企业** ：跨境电商、数字娱乐、旅游及教育品牌

  * **🛍️****独立站中小零售商** ：使用 WooCommerce、Magento、Shopify 等建站系统搭建站点，需要快速集成中国支付方式的商户

  * **🤝****支付渠道代理商** ：需统一管理下游商户并开展利润分享的渠道合作伙伴

  * **🛡️****寻求低风险支付解决方案的商户** ：希望规避传统信用卡高拒付率风险的企业

  * **🌐****跨境及泛亚太地区的收款商户** ：希望同时覆盖中国、日韩及东南亚消费者的多元化商户




### （二）注意事项

  * 📊**业务边界** ：为服务商户通过中国三大主流支付方式（银联、支付宝、微信支付）及国际卡组织实现收款，聚合部分亚太地区的主流本地电子钱包

  * 🔧**技术边界** ：平台接口高度开放，采用模块化 RESTful API 架构，便于快速集成各类电商平台及企业系统

  * 💰**资金与合规** ：商户应自行核查目标市场的监管要求，并确认 NihaoPay 在美国等关键市场的金融资质




## **九、区域合规与适配方案**

###  （一）东亚

**🎯****市场定位：**

连接全球商户与中国消费者的核心支付网关，为全球商户提供银联（UnionPay）、支付宝（Alipay）、微信支付（WeChat Pay）收单服务

**⚖️****合规与结算：**

所有交易在合规框架下处理；结算通过国际电汇及本地清算网络，将款项划转至商户指定账户

**📱****本地化适配：**

支持扫码支付、H5 支付、小程序支付及应用内支付，提供人民币定价功能

### （二）东南亚

**🎯****市场定位：**

通过 Alipay + 产品接入当地主流电子钱包生态，实现本地电子钱包支付聚合能力

**⚖️****结算与合规：**

依托美国 MSB 牌照，通过国际电汇方式结算至当地银行账户；需商户按照当地法规完成运营层面的合规备案

### （三）北美及欧洲

**🎯****市场定位：**

为北美和欧洲商户提供中国支付通道及国际卡收单服务，覆盖全球客户

**⚖️****合规与结算：**

在美国支持 ACH 本地清算与国际电汇两种可选方式，降低结算成本；在欧洲通过环球同业银行金融电讯协会（SWIFT）/ 国际银行账户号码（IBAN）网络结算，商户需自行满足本地监管要求

**📱****本地化适配：**

支持多币种直接到账结算，并借助当地清算网络优化到账效率
