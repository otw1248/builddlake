## **一、企业基本档案**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体：**

2005 年 7 月（美国加州成立）；核心运营主体为 Bright Market, LLC，由私募股权公司 Accel-KKR 控股，是一家专注于服务软件与 SaaS 行业的公司。

**🌍****全球布局：**

全球总部位于美国加州圣巴巴拉；在荷兰阿姆斯特丹、英国贝尔法斯特、加拿大哈利法克斯设有区域办公室；2024 年新设新加坡办公室和美国奥斯汀办公室。

**💼****核心商业模式：**

采用全权交易托管方（Merchant of Record - MoR）模式；FastSpring 作为其客户的法律销售实体，全权负责处理全球支付受理、销售税/增值税计算与代缴、欺诈防护、拒付管理及客户账单支持等所有交易相关的后端运营，承担税务合规、欺诈与拒付的全部财务损失及处理义务。

**📊****业务覆盖：**

服务全球超过**200个** 国家和地区、服务全球数千家 SaaS 与数字产品公司；支付网络支持全球主流银行卡、数字钱包及关键区域的本地支付方式；税务合规网络覆盖全球主要经济体。

**🎯****行业渗透与应用场景：**

深度聚焦软件即服务（SaaS）、人工智能、游戏、移动应用及在线教育等数字经济领域；提供集支付处理、订阅管理、全球税务合规、风险管理于一体的全栈式收入交付平台。

## **二、全球化能力概览**

###  （一）支付网络与区域布局

**💳****国际卡收单：**

受理能力：支持 Visa、Mastercard、American Express、Discover、JCB、银联等所有主流国际卡。

收单模式：作为 MoR，通过与全球多个支付网关与收单行合作处理卡支付，而非以单一主收单资质直连。

智能路由与支付级联重试：提供智能支付路由功能，可根据地区、成本、成功率等因素自动选择最优网关，并在主要路由失败时自动切换至备用处理器重试，以最大化授权成功率。

安全认证：支持 3D Secure 技术以满足欧洲强客户认证（SCA）要求。

**🔗****本地支付聚合：**

欧洲：支持 SEPA 直接借记、iDEAL（荷兰）、SOFORT、Klarna

北美：支持 ACH（美国）

亚太地区：支持支付宝、微信支付（中国）、UPI（印度）、KakaoPay（韩国）、PayPay（日本）

拉丁美洲：支持 Pix（巴西）、Mercado Pago

数字钱包：支持 Apple Pay 与 Google Pay

**🔄****跨境清算路径：**

资金流转模式：作为 MoR，所有资金首先进入 FastSpring 的账户体系。

**📍****区域枢纽：**

美国总部（加州）为全球战略与核心运营中心；阿姆斯特丹/贝尔法斯特办公室为欧洲运营与税务合规中心；新加坡办公室为亚太地区运营枢纽；奥斯汀办公室为北美区域运营中心。

**🌐****服务与语言：**

提供超过**25种** 语言的本地化结账体验。

### （二）合规与安全体系

**⚖️****核心模式（MoR，全权交易托管模式）：**

税务合规（核心优势）：在全球主要经济体及数十个国家/地区自动化处理销售税、增值税（VAT）、商品及服务税（GST）等多种税费的计算、代收与代缴，具备大规模税务代缴能力，承担所有相关审计与合规风险。

金融牌照：合规策略侧重于在全球范围内获得作为数字商品经销商所需的商事主体资格与税务登记，与传统支付服务商（PSP）所需的电子货币（EMI）或资金传输（MTL）等金融牌照类型不同。

**📜****行业认证：**

持有并每年更新 PCI-DSS Level 1 合规认证。

**🛡️****欺诈与拒付管理：**

责任承担：作为 MoR，承担所有欺诈交易与拒付的全部财务损失与处理责任。

技术集成：与 Sift 合作，采用机器学习与 AI 技术进行实时欺诈检测。

## **三、核心技术能力解析**

###  （一）智能技术与架构

**☁️****平台架构：**

基于 Amazon Web Services (AWS) 构建，采用云原生架构，具备自动弹性扩展能力，支持高并发负载处理。

系统可靠性：维护公开的系统状态页面，提供各核心服务的实时状态与历史运行时间数据，确保服务透明度。

**🛡️****风控引擎：**

技术构成：采用机器学习（与 Sift 合作）与规则引擎结合的双栈模式。

数据驱动：风控模型每月基于数百万个全球欺诈决策数据进行分析与优化，利用 Sift 网络中超过 **34,000 个** 站点的数据进行风险识别。

**🔧****开发者生态：**

API 与 SDK：提供完整的 RESTful API 及多语言集成支持，包括详细的 API 参考文档。

开发工具：提供完整的开发者文档中心、API 参考及支持 HMAC SHA256 签名验证的 Webhook 机制。

预置对接工具：通过第三方 iPaaS 平台（如 Integrately）支持与 Shopify、WooCommerce、QuickBooks 等系统的连接。注：官方 WordPress 插件已不再提供主动维护，建议通过 API 或第三方集成工具实现 WordPress 集成。

### （二）支付与结算效率

**📈****授权成功率优化：**

通过智能支付路由与支付级联重试功能，动态选择最优收单行并自动重试失败交易，以最大化授权成功率。

**💵****资金结算时效：**

结算周期：采用双周（每月 15 日和月末）结算模式，支持商户自定义结算频率。

延迟期：结算流程中包含约两周的结算缓冲期以确保交易稳定性。

**🔄****资金管理技术：**

多币种处理：支持超过**20种** 主流货币的支付受理，包括 USD、EUR、GBP、JPY、AUD、CAD、CHF、CNY、HKD、SGD 等主流货币，所有交易自动转换为商户设定的单一基础货币。

分账能力：提供合作伙伴分账账户功能，支持向合作伙伴（如经销商、联盟营销伙伴）自动分配销售收入，可按百分比或固定金额进行分账，并提供独立的合作伙伴登录界面进行账户管理。

## **四、计费结构与服务规则**

**📋****整体定价原则：**

采用全包式（All-Inclusive）的混合费率模型，结合基于交易量的分级阶梯定价；费率已包含支付处理、订阅管理、全球税务、欺诈防护等所有核心服务；具体费率和费用标准因商户规模、交易量和业务类型而异，建议直接联系 FastSpring 获取准确的定制化报价。

**💱****货币兑换（FX）：**

汇率数据源：使用 OANDA 提供的汇率数据，每日更新多次确保准确性。

汇差：在基准汇率之上加收固定汇差，主流货币与其他货币采用不同汇差标准。

**🏦****结算（出款）服务：**

结算方式：通过 ACH（美国）、SEPA（欧洲）、BACS（英国）、EFT（加拿大）、银行电汇等多种方式向商户支付款项。

最低结算金额：通过本地银行系统出款需达到一定最低金额，国际电汇有更高的最低金额要求。

**📝****其他费用：**

账户闲置费：连续 6 个月无销售活动的账户将收取月度管理费用。

风险筛查费：年销售额低于特定标准的账户需支付年度风险验证费用，用于满足 KYC/KYB、反洗钱（AML）、反恐融资（CFT）等监管合规要求。

货币转换费：非基础货币交易将产生货币转换费用。

## **五、汇率管理与资金流转**

**📊****汇率基准与透明度：**

明确使用 OANDA 作为汇率数据源，平台每日多次更新汇率数据以确保准确性；在 OANDA 基准汇率之上加收固定汇差，主流货币（USD、EUR、GBP、JPY 等）与其他货币采用不同的汇差标准，定价结构清晰透明。

**🔄****货币转换规则：**

所有交易均自动转换为商户设定的单一基础货币进行结算，支持 **23 +** 种货币的自动转换；未提供远期锁汇、货币掉期等外汇对冲工具。

**💵****多币种资金管理模式：**

作为 MoR，所有资金由 FastSpring 统一管理，商户无需自行开设多币种账户；合作伙伴分账账户功能支持将收入按预设百分比或固定金额自动分配给多个合作伙伴，每个合作伙伴拥有独立的登录界面管理账户信息。

**⏱️****到账与清算时效：**

SEPA 直接借记通常需要 5-7 个工作日完成清算并到账；ACH 直接借记（美国）一般需要 3-5 个工作日处理；银行电汇可能需要 3-5 个工作日，具体取决于银行间处理效率；本地银行转账根据各国银行系统差异，处理时间有所不同。

## **六、生态对接与服务能力**

###  （一）SaaS 与订阅平台

**🔄****订阅管理引擎：**

提供功能完善的订阅管理引擎，支持完整的生命周期管理，包括订阅创建、暂停（可配置最大暂停期）、恢复、升级、降级、取消等操作，并支持按百分比或固定金额的即时按比例计费调整。

**📧****逾期付款催缴管理（Dunning 管理）：**

提供高度自动化的催缴管理功能，支持在支付失败后 2、5、7、14、21 天等可配置时间间隔发送 HTML 或纯文本提醒通知，系统会在每次重试前自动尝试重新处理支付，并提供智能重试逻辑。

**🆓****试用期管理：**

支持灵活的试用期设置，包括免费试用（可选择是否预留支付信息）、付费试用（低价体验完整功能）、按用量计费试用、可配置试用期长度和自动转换规则、支持试用到期提醒和自动续费选项。

### （二）电商与企业系统

**🔌****第三方集成：**

通过丰富的 API 和 Webhook 接口支持与主流平台集成；通过 iPaaS 平台（如 Integrately、Zapier），可将 FastSpring 连接至平台支持的数千种第三方应用；支持与 Shopify、WooCommerce、QuickBooks 等系统的数据同步；提供 RESTful API 和完整的开发者文档支持自定义集成。

### （三）服务交付与支持

**👨‍💼****商户支持：**

提供 24/7 全天候客户支持服务，工作日工作时段内提供即时响应，非工作时间提供在线支持。

**👥****买家支持：**

开设独立的终端消费者支持门户，作为 MoR 全权代商户处理所有与支付、账单、退款、订阅管理相关的客户咨询。

**📈****系统监控：**

维护公开的系统状态页面，实时显示所有核心服务状态、历史运行时间数据和维护通知。

## **七、增值工具与行业解决方案**

###  （一）核心内置功能模块

**⚖️****全球税务与合规：**

自动处理全球范围内的销售税、增值税（VAT）、商品及服务税（GST）等多种税费的计算、代收与代缴，承担所有相关审计与合规风险；通过税务引擎自动处理全球数十个国家和地区成千上万种复杂的税率规则与组合。

**🛡️****欺诈与拒付保障：**

承担 100% 的欺诈交易与拒付损失，由内部专家团队负责所有争议处理；通过与 Sift 合作的 AI 机器学习风控系统，结合设备级保护、平台级风险调整和处理器级欺诈警报。

### （二）行业解决方案

**🖥️****SaaS 和 AI 公司：**

核心功能组合：全权交易托管方（MoR）模式 + 订阅管理引擎 + 全球税务合规 + API/Webhooks + 逾期付款催缴管理。

能力亮点：提供从收款、合规到订阅生命周期管理的全栈式解决方案，支持暂停/恢复、升降级、按比例计费等高级订阅功能。

**🎮****游戏行业：**

核心功能组合：直面消费者（D2C）支付平台 + 全球本地化支付 + 游戏行业专属风控 + 先进的游戏行业专属风控技术 + 1ClickPay 快速结账。

能力亮点：通过 MoR 模式使游戏开发者一键面向全球 **200 +** 市场直销，支持游戏内购买、虚拟货币、战斗通行证等多样化变现模式；提供专业的游戏网店设计服务。

**📚****在线学习/数字课程：**

核心功能组合：数字课程销售平台 + 与 LearnDash 的集成功能 + 订阅计费 + API/Webhook 自动化对接。

能力亮点：通过第三方集成工具（如 Integrately）支持与 LearnDash、Teachable 等学习管理系统的深度集成，提供比主流课程市场更低的费率，支持灵活的课程定价和订阅模式。

## **八、适配场景与注意事项**

###  （一）适配场景

  * **🌐****全球化 SaaS 与订阅制企业** ：核心聚焦于希望将复杂的全球支付、税务和合规统一外包、专注产品和市场增长的企业。

  * **💻****销售数字产品的开发者与公司** ：支持软件、游戏、电子书、在线课程等单品或订阅模式数字产品，可快速低门槛面向全球 **200 +** 地区销售，无需在各国设立实体或处理本地税务问题。

  * **🛡️****需完全转移风险的商户** ：对欺诈和拒付风险高度敏感的企业可将全部财务损失和运营负担托付给 FastSpring。




### （二）注意事项

  * 📦**业务范围限制** ：专注于数字、SaaS、软件、移动应用及在线内容，不支持实体商品销售。

  * 💰**成本结构** ：采用全包费率模式，费率较传统 PSP 可能偏高，但可减少企业法律、财务和运营额外支出，实际总成本需结合自身业务总拥有成本（TCO）分析。

  * ⏱️**资金到账时效** ：双周结算和资金延迟周期适合绝大多数全球化数字业务，对极高现金流时效要求的企业需充分评估。

  * 🔧**技术灵活性** ：平台在支付路由、收单行选择等底层交易流程上采用统一管理模式，部分高级自定义需求（如定制结算逻辑、特殊账户结构等）需提前评估；多实体逻辑、ERP 级深度定制不是主打方向。




## **九、全球统一的合规方案（MoR 模式）**

**🛡️****准入与合规要点**

作为全球统一的全权交易托管方，FastSpring 自身负责完成在全球 **200 +** 市场的商事主体注册、税务合规、销售税/VAT/GST 申报与缴纳、风险筛查等全部义务。客户无需在各地单独申请金融或税务牌照，仅需完成企业验证审核（KYB 流程，包括企业资质、工商注册信息、受益人身份、合规风险筛查等），签署一份全球主服务协议并遵循可接受使用政策，所有合规责任、税费政策、法定申报均由 FastSpring 统一承担和管理。

**🏢****本地化运营/风控重点**

平台自动集成并处理各地支付路由、3DS 强客户认证（SCA/PSD2）、本地化结账语言与币种，以及实时税率计算与合规报表生成，同时符合 CCPA、GDPR、SCA、经济实质认定等全球主要经济体的法定要求。全球客户可获得标准化 API 接口对接，实现结账体验定制与法规合规的自动化交付。

**📋****上线清单**

技术：通过开发者中心获取 API 凭证，利用 Webhook 接口和 Store Builder JavaScript 工具库进行集成测试和定制化开发，一次 API/开发者集成即可面向全球所有支持市场上线。

财务：配置结算基础货币与银行账户信息，熟悉双周结算周期与外汇价差规则。

运营：在后台统一配置商品目录、定价策略与订阅规则，设置逾期付款催缴管理参数，熟悉终端用户通过消费者支持门户的支持流程。
