本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

2007 年成立，法定名称 PT Nusa Satu Inti Artha，后更名为 DOKU，为印尼首家电子支付及风险管理公司。

**🌍****全球布局**

总部位于印尼雅加达，在巴厘岛、马来西亚吉隆坡（通过子公司 senangPay）及新加坡设有办公团队，覆盖 3 个国家、4 个城市。

**🛡️****合规资质**

\- 金融牌照：印尼央行（Bank Indonesia）PJP Category 1 牌照，涵盖支付网关、电子货币、电子钱包、汇款、QRIS 二维码支付共 5 项子牌照

\- 马来西亚牌照：子公司 senangPay 持有马来西亚央行（Bank Negara Malaysia）商户收单服务牌照，同时为 Visa 及 MasterCard 亚太区授权支付促进者

\- 安全认证：PCI DSS Level 1、ISO 27001、AES 256 加密标准

**📊****业务覆盖**

聚焦印尼与马来西亚两个市场，提供约 50 种印尼本地支付渠道及约 25 种马来西亚支付渠道，支持 IDR（印尼盾）与 MYR（马来西亚林吉特）交易处理，结算货币覆盖 IDR、MYR、USD、SGD。

**🎯****主要应用场景**

聚焦印尼本地电商收款、数字娱乐与游戏、在线旅游、保险、零售、物流等行业，提供支付网关、付款（Payouts）、钱包即服务（Wallet as a Service，WaaS）、子账户管理、WhatsApp 商务支付（PayChat）等能力。

## **二、服务范围与计费规则**

**📋****服务项目清单**

**收款服务（Accept Payments）**  
覆盖卡收单、银行转账、二维码、数字钱包、先买后付、便利店现金、信用卡分期及直接借记等主要支付类型。

**付款服务（Payouts）**  
Domestic Payouts（印尼国内 100+ 银行及电子钱包实时转账）、Cross-Border Payouts（跨境入境汇款）、Cash Out（便利店现金提取）。

**增值服务**  
包含 WaaS、子账户管理、促销引擎、商户贷款等。

**💰****计费规则**

印尼市场采用按交易收费模式，无设置费、无月费，仅对成功交易收取手续费（MDR），费率因支付方式、交易类型等因素而异；所有交易费用另加增值税（PPN）。马来西亚市场（senangPay）采用"年费订阅 + 交易费率"的双层模式。

**📝****费用项目补充说明**

可能包含退款服务费、跨境结算转账费、货币转换费、Shopify 插件附加费等项目。商户退款时，原始交易手续费（MDR）不予退还。

**📌****小结：** 本章聚焦收费项目与适用规则，不含数值，最终以《商户协议》为准。

## **三、支付网络与合规体系**

###  （一）支付网络与区域布局

**🇮🇩****印尼本地支付**

整合约 50 种本地支付渠道：

  * 银行转账：15 家银行 Virtual Account（BCA、Mandiri、BRI、BNI、BSI 等）+ 5 家网上银行 + 4 家直接借记

  * 数字钱包：DOKU e-Wallet、OVO、DANA、ShopeePay、LinkAja、i.saku

  * 二维码：QRIS（兼容所有 QRIS 生态钱包）

  * 便利店：Alfamart、Alfamidi、Lawson 及 Indomaret

  * 先买后付：Akulaku、Kredivo、Indodana


**🇲🇾****马来西亚本地支付（通过 senangPay）**

覆盖约 25 种渠道：FPX 网银（Maybank、CIMB、Public Bank 等 17 家银行）、Touch 'n Go、GrabPay、Boost、ShopeePay、ShopBack 等钱包、SPayLater、PayLater by Grab、Atome 等 BNPL。

**💳****卡支付网络**

直连 Visa、Mastercard、JCB、American Express，支持印尼本地发行借记卡及 GPN 本地卡网络，提供信用卡分期（3、6、12 期）、令牌化支付（一键支付、两步支付）、3DS 验证。

**🌏****跨境付款（Cross-Border Payouts）**

聚焦资金汇入印尼方向的跨境付款服务：全球汇款平台可通过 API 或 Dashboard 将资金从海外汇至印尼 126+ 家银行和电子钱包。与 Deutsche Bank 合作实现外汇转换与跨境结算。

**🤝****战略合作拓展**

2025 年与 Ant International（蚂蚁国际）签署战略合作，通过 Antom 和 2C2P 为印尼商户提供多币种交易能力，已实现中国游客在印尼商户使用支付宝付款。

### （二）合规运营与数据保护

**👤****反洗钱能力**

对接印尼人口与民事登记总局（Dukcapil）进行身份验证，提供 KYC/KYB 审核流程，配备 24/7 欺诈防范团队，交易限额监控覆盖单笔、日累计、月累计。

**💰****客户资金管理**

遵循印尼央行 PJP 监管框架下的资金管理要求，灾备节点部署于印尼境内（阿里云印尼数据中心）。

**🔒****数据保护**

适用印尼《个人数据保护法》（UU PDP，2024 年 10 月全面实施）及马来西亚《个人数据保护法》（PDPA 2010）。

**⚖️****合规运营**

PCI DSS Level 1 年度审计（满足 PCI SSC 300+ 项安全措施）、Bank Indonesia 定期监管。

## **四、技术能力与支付结算**

###  （一）技术架构与风控能力

**⚙️****API 体系**

采用 RESTful API 架构，提供 Non-SNAP API 和 SNAP API（符合印尼央行 SNAP 国家开放 API 标准）两代接口，覆盖约 100～150 个端点，涵盖支付创建、退款、Virtual Account、电子钱包、直接借记、PayLater、便利店、付款等核心流程。

**👨‍💻****开发者生态**

提供多种服务端 SDK，支持 Webhook 事件通知（支付成功/失败），提供多种语言的签名验证示例代码。

**🧪****测试环境**

提供完整沙盒环境，包含独立测试端点、测试密钥、Demo Site、Payment Simulator、测试卡号，无交易费用，并提供公开的 API 文档与开发者指南。

**🛡️****风控引擎**

自研 Fraud Detection System（FDS），运行 10 年以上，覆盖 AI/ML 驱动的实时交易风险检测、人工审核团队（Early Detection Unit）、令牌化技术、3D Secure 验证、发卡行识别码（BIN）国别过滤、拒付（Chargeback）保险功能（可选）与拒付争议管理。商户可通过 Dashboard 配置风控策略。

**🤖****AI 支付集成**

2025 年推出 DOKU MCP Server（Model Context Protocol），与 Google Agent Payments Protocol（AP2）合作，支持 AI 智能体直接调用支付接口完成交易创建、Virtual Account 支付、状态查询等操作。DOKU 称其为印尼首家支持 Agentic Commerce 的支付网关。

### （二）接入方式与开发者工具

**🛒****电商插件**

提供 Shopify、WooCommerce、Magento 官方插件，支持一键安装与配置。Shopify 插件在支付方式手续费基础上额外收取附加费。

**📱****无代码收款工具**

Payment Link（支付链接）、Digital Catalog（电子目录）、QRIS（二维码收款）、PayChat（WhatsApp 商务支付）、Virtual Terminal，共 5 种无需开发的收款方式，适合没有开发团队的小微商户。

**📲****移动端应用**

Juragan DOKU（商户 App）：支持 Payment Link 创建、动态 QRIS 码、Digital Catalog、实时交易报告。DOKU e-Wallet（消费者钱包 App）：注册用户超过 400 万。

### （三）支付与结算效率

**⏱️****到账时效分级**

印尼市场结算周期因支付方式和收单行而异：Virtual Account 和 QRIS 较快（T+1），电子钱包 T+1 至 T+2，卡支付 T+2 至 T+3，便利店相对较慢（T+4）。

**💵****结算模式**

提供 Aggregator 结算（资金先结算至 DOKU 后转发至商户）和 Direct 结算（资金从银行直接结算至商户）两种模式。支持自定义结算计划（每日、每月两次、每月），支持设置最低结算金额。结算报告支持 CSV、JSON 格式导出及 API 拉取。

**⚠️****注意事项：** 仅适用于拥有印尼本地银行账户的企业商户可享受标准结算周期；海外银行账户商户可能采用每周、每两周或每月结算，且有最低结算门槛要求。

## **五、增值服务与商户支持**

###  （一）钱包即服务（WaaS）

**📦****服务内容**

提供企业级数字钱包发行与管理能力，支持 API 集成和白标模式。创建钱包免费，仅在资金流动时收费，定价根据交易量、活跃用户数、API 调用量等因素确定。

**💡****应用场景：** 适用于需要为用户提供内置钱包功能（充值、消费、转账）的电商平台或金融科技企业，可通过 WaaS 快速部署，无需自行申请电子钱包牌照。

### （二）子账户管理（Sub Account）

**👥****服务内容**

提供 Default、Standard、Partner 三种子账户类型，支持代收款、代付款、账户间转账、余额查询、交易历史等功能。

**💡****应用场景：** 适用于电商平台或代理分销网络，可为每个入驻商户或代理商创建独立子账户，实现资金自动分账与结算，无需手动处理多方资金流转。

### （三）分账与资金控制

**💰****Split Settlement（分账）**

支持按百分比或固定金额分账，单笔最多 10 个银行账户，可通过 API 或 Dashboard 配置。

**🔒****Hold and Release**

支持资金冻结与释放，适用于担保交易、预售等场景。

### （四）行业方案

**🇮🇩****印尼本地电商**

**适配组合**

  * 支付网关（Checkout、Direct API）+ 多渠道本地支付（VA + QRIS + 钱包 + 便利店）

  * Payment Link、QRIS（小微商户无代码收款）

  * Split Settlement + Sub Account（平台分账）

  * Juragan DOKU App（移动端收款管理）




**上手清单**

开通前准备（运营/财务负责）

  * 营业执照及法人身份证明（KYB 认证）

  * 业务网站/应用信息

  * 银行账户信息（印尼本地银行）




对接阶段（运营/技术负责）

  * 无代码方案：注册后直接使用 Payment Link、QRIS（无需开发）

  * 插件方案：安装 Shopify、WooCommerce、Magento 插件

  * API 方案：对接 SNAP API，单渠道约 3～7 天，多渠道可达数周

  * 测试验证：沙盒环境小额测试，确认 Webhook 通知与对账数据




上线后运营（财务/运营负责）

  * 每日核对：Dashboard 检查订单与资金流水匹配

  * 结算管理：根据资金需求配置结算计划（每日、每月两次、每月）

  * 风控监控：查看 FDS 拦截情况，按需调整策略




### （五）服务交付与支持

**📝****入驻与审核**

支持企业、国际企业、个人三种账户类型，线上注册提交 KYB 资料，账户验证最多 48 小时；免开发产品（如 Payment Link、QRIS 等）可在注册后快速激活收款。

**📞****客户支持与帮助**

提供邮件、电话、WhatsApp、Dashboard 工单等多渠道支持，WhatsApp 渠道 24/7 可用，承诺 24 小时内响应；可申请专属客户经理。帮助中心与开发者文档双语公开访问，配套 Interactive Demo 和 Sandbox 环境。

## **六、适用场景与选型参考**

**🇮🇩****印尼本地电商收款**

面向印尼消费者的电商业务，DOKU 覆盖约 50 种本地支付渠道，在印尼市场的支付方式覆盖较为全面。如业务同时涉及欧美、拉美等区域，建议同时了解 Stripe、Adyen、2C2P 等多区域覆盖方案。

**🏢****多商户平台与市场**

运营电商平台、代理分销网络或 O2O 市场的企业，需要为入驻商户自动分账与结算，DOKU 提供分账、子账户和资金冻结释放等能力。

**🛍️****小微商户与无开发团队**

个体商户或小微企业在没有开发资源的情况下，可使用 DOKU 的 5 种无代码收款工具，注册后即可快速启用。处于印尼市场起步阶段的企业，可利用 DOKU 的零门槛入门模式（无设置费、月费）快速验证业务模型，待规模扩大后再评估 API 集成与定制化方案。

**💼****嵌入式金融服务**

需要在应用或平台中嵌入数字钱包功能的企业，可通过 DOKU 的 WaaS 能力快速部署，借助其电子钱包牌照降低合规门槛。

**🇲🇾****马来西亚本地收款**

业务同时覆盖印尼与马来西亚的企业，可通过 DOKU + senangPay 在两个市场使用统一服务商，senangPay 覆盖 FPX 网银 17 家银行及本地钱包。

**🌏****跨境入境收款（B2B 贸易）**

海外企业需要收取印尼客户货款或向印尼进行付款与资金下发时，DOKU Cross-Border Payouts 提供资金汇入印尼方向的跨境付款服务，适合跨境 B2B 贸易的印尼端收付款需求。如业务涉及多市场双向跨境收付款、多币种账户管理或汇率锁定服务，建议同时评估具备全球多区域覆盖能力的服务商。

**📝****选型建议**

建议在签约前与多家服务商沟通，明确计费规则、结算周期、退款政策、资金扣留条款等关键事项，选择最符合业务需求的方案。DOKU 提供从无代码到全 API 集成的多层次接入方案，企业可根据自身技术资源灵活选择。如需财务系统自动对账，可通过 API 或 CSV、JSON 导出方式对接。
