本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

2020 年成立，初创于美国加州硅谷，公司主体 DEUNA LLC 注册于墨西哥。

**🌍****全球布局**

总部位于美国纽约市，在帕洛阿尔托、迈阿密、墨西哥城、波哥大、基多、乌拉圭等地设有分支机构，覆盖 **15 个以上** 国家和地区。

**🛡️****合规资质**

通过 PCI-DSS Level 1 认证，架构基于 ISO 27001 信息安全管理及 ISO 22301 业务连续性标准设计；Visa 全球服务提供商注册（DEUNA LLC MEXICO）；具备 GDPR、HIPAA 数据保护相关声明。

**⚠️****说明：** DEUNA 以支付编排平台（Payment Orchestration Platform）模式运营，通过整合已持牌的支付提供商（如 Adyen、Worldpay、PagBank 等）为商户提供服务。

**📊****业务覆盖**

集成 **400 余家** 支付及风控服务商、**200 余种** 支付方式，重点覆盖拉美（墨西哥、巴西、哥伦比亚、智利、阿根廷、厄瓜多尔、乌拉圭）及北美市场。

**🎯****主要应用场景**

聚焦跨境电商、快餐连锁、航空、消费电子、SaaS 订阅等行业，支持一键结账、支付编排、订阅管理、自动对账、AI 驱动的支付优化等功能。

**💰****融资情况**

完成 A 轮融资，融资总额约 **3,800 万美元** ，投资方包括 Activant Capital（领投）、Valor Capital、Upload Ventures（原 SoftBank Early Stage）、L Catterton 等。

  


## **二、计费结构与服务规则**

**📋****定价模式**

采用按需定价，需通过销售团队获取报价。商业模式为 SaaS 订阅与交易分成相结合。

**📋****服务项目清单**

  * 支付编排平台订阅

  * 一键结账服务

  * 订阅管理与周期性支付

  * 自动对账与数据管理

  * Athia AI 智能优化引擎

  * 风控与欺诈防护


**📊****影响计费的因素**

具体取决于交易规模、集成范围、功能模块、服务级别、地区等因素。

**📌****小结：** 本章聚焦收费项目与适用规则，最终以《服务协议》为准。

## **三、支付网络与合规体系**

###  （一）支付网络与区域布局

**💳****国际卡收单**

支持 Visa、Mastercard、American Express、Elo（巴西）、银联等国际与本地卡组织，通过集成的支付提供商实现多币种记账与清算。

**👛****数字钱包覆盖**

  * 🌐国际钱包：Apple Pay、Google Pay、PayPal、Mastercard Click to Pay（覆盖**50多个** 国家）

  * 🇱🇦拉美本地钱包：Mercado Pago、DaviPlata（哥伦比亚）、PagBank（巴西）


**🔗****本地支付聚合**

覆盖多种支付类型：

  * 🇲🇽墨西哥：OXXO（现金）、SPEI（即时转账）

  * 🇧🇷巴西：Boleto（票据）、PIX（即时转账）

  * 🇨🇴哥伦比亚：DaviPlata、Baloto

  * 💳先买后付（BNPL）：Klarna、Affirm、Afterpay、Aplazo


**🔄****清算网络**

支持 SEPA、ACH、SWIFT、PIX、SPEI 等主流清算通道。

### （二）合规与安全体系

**📜****认证与标准**

  * PCI-DSS Level 1：年度现场审核、季度网络扫描

  * 架构基于 ISO 27001 信息安全管理标准设计

  * 架构基于 ISO 22301 业务连续性管理标准设计

  * GDPR、HIPAA：数据保护相关声明


**🔒****技术安全**

  * 传输加密：TLS 1.3

  * 存储加密：AES-256

  * 身份管理：AWS IAM + Okta，强制启用多因素认证（MFA）

  * 网络安全：资源隔离、网络隔离、流量监控

  * 安全开发：遵循 OWASP Top 10（Web 应用安全风险清单）2023 标准

  * 定期渗透测试与漏洞评估


**🏗️****基础设施**

依托 AWS Well-Architected Framework 搭建，具备业务连续性与灾难恢复能力。

## **四、技术架构与支付优化**

###  （一）智能技术与架构

**⚙️****API 架构**

采用 RESTful API，支持标准 HTTP 方法，提供支付创建、确认、查询、取消、扣款、退款等完整接口。

**📱****SDK 支持**

  * Web SDK：TypeScript/React Native

  * 移动端：iOS（Swift）、Android（Kotlin）

  * 文档相对完整，包含快速入门、API 参考、代码示例


**🔔****Webhook 与事件通知**

  * 支持支付、退款、客户等关键业务动态的实时推送

  * 签名验证：HMAC-SHA256（X-Deuna-Signature Header）

  * 自动重试失败的 Webhook


**🔧****开发者工具**

  * 沙盒测试环境

  * 测试卡号与场景

  * 控制台调试功能

  * 开发者文档




### （二）支付与结算效率

**🔄****动态路由**

无需编程即可配置路由规则，支持基于客户位置、支付方式、交易金额、风险等级、银行识别码（BIN）、发卡行等多维度条件的智能路由，自动选择最优支付处理通道。

**📈****支付优化成效**

根据企业公开的客户案例，智能路由可帮助商户降低交易成本约 **12%** ，转化率提升约 **30%** ，授权率提升 **8%至10%** ，挽回约 **15%** 的潜在流失交易额（GMV）。

### （三）区域要点与风险提示

**🇲🇽****墨西哥**

  * 支付方式：OXXO（现金）、SPEI（即时转账）、本地卡


**🇧🇷****巴西**

  * 支付方式：PIX（即时转账）、Boleto（票据）、PagBank


**🇨🇴****哥伦比亚**

  * 支付方式：DaviPlata、Baloto、本地卡


**🇺🇸****北美**

  * 支付方式：ACH 转账、本地卡收单




**⚠️****区域支付关键注意事项**

巴西的 Boleto 是线下支付方式，客户下单后需去银行或便利店付款，3至7天到账，存在弃付风险，建议在订单确认中注明支付期限。墨西哥的 OXXO 同样为现金支付，需注意到账延迟和线下环节的对账流程完整性。拉美整体而言，各国钱包限额差异较大，需提前在测试环境验证规则，税号与身份字段也需满足本地监管与税务要求。北美市场若高客单价与订阅业务并存，建议采用差异化风控策略。

## **七、适用场景分析**

**🌐****适合使用 DEUNA 的场景****🇱🇦****拉美市场拓展企业**

如果您的业务覆盖墨西哥、巴西、哥伦比亚等拉美市场，需要接入大量本地支付方式（如 OXXO、Boleto、PIX 等），DEUNA 在拉美市场覆盖了主流本地支付方式。

**🔗****多支付提供商整合需求**

如果您已对接多个支付提供商（如 Adyen、Worldpay、PagBank），需要统一编排和智能路由，DEUNA 支持无需编程即可配置的支付编排功能，可通过单一集成连接多个提供商。

**📈****高转化率追求企业**

如果您希望提升结账转化率、降低购物车放弃率，DEUNA 提供一键结账功能。

**🔄****订阅型业务**

如果您运营 SaaS 或订阅制业务，需要复杂的周期性计费、智能重试、账款催缴管理，DEUNA 具备订阅管理功能。

**🤖****AI 驱动的支付优化需求**

如果您需要实时优化支付路由、降低交易成本、提升批准率，DEUNA 的 Athia AI 平台可提供智能决策支持。

**💡****决策建议**

在选择跨境支付服务商时，建议从以下维度进行综合评估：

**🌍****业务场景匹配度**

根据您的主要市场、客户支付习惯、交易规模等因素，对比多家服务商的匹配程度。如果您的业务聚焦拉美市场，可重点了解 DEUNA、Kushki、Mercado Pago 等方案；如果需要全球覆盖，可同时了解 Adyen、Stripe 等方案。

**🏢****商业模式匹配**

DEUNA 以支付编排平台模式运营，整合已持牌的支付提供商为商户提供服务。这一模式适合需要多提供商整合的企业。

**📊****业务规模阶段**

DEUNA 聚焦企业级客户，采用按需定价模式。如果您处于业务起步阶段，可优先考虑提供标准化定价的方案；待业务规模扩大后，再评估支付编排平台的价值。

**📦****业务类型匹配**

DEUNA 主要服务 B2C 电商、连锁餐饮、SaaS 订阅等场景。如果您从事 B2B 外贸业务，需要大额跨境汇款、贸易融资等功能，可同时了解 PingPong、万里汇、XTransfer 等专注 B2B 跨境贸易的方案。

**🔧****技术实施能力**

评估您的团队是否具备 API 对接能力。DEUNA 提供完整的多平台 SDK 和开发者文档，支持 VTEX 等主流拉美电商平台集成，对接周期约 2 个月，提供专属客户支持。

**💰****费用与合同条款**

在签约前，务必与销售团队沟通，明确计费规则、服务边界、数据安全、争议处理等关键条款，选择最符合您业务需求的方案。
