## **一、企业基本档案**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

业务始于 2014 年，Fondy Ltd 于 2016 年 3 月在英国注册成立；公司总部位于英国伦敦，是一家专注于欧洲市场的金融科技公司。

**🌍****全球布局**

全球总部位于英国伦敦；在波兰华沙设有核心办公地点。

**🛡️****合规资质**

核心牌照：持有英国金融行为监管局（FCA）颁发的电子货币机构（EMI）牌照。

其他资质：拥有独立的 SWIFT BIC 代码，并在英国开放银行目录中注册（Open Banking Regulated Provider）。

**📊****业务覆盖**

根据官方网站数据，Fondy 为全球超过 **8，000 家** 企业客户提供服务，支付网络覆盖 **200 多个** 国家、**150 多种** 货币和 **300 多个** 支付渠道；核心业务区域集中在英国及欧盟。

**🎯****行业渗透与应用场景**

深度聚焦跨境电商、SaaS、在线平台与市场等领域；提供支付网关、订阅管理、分账支付、即时结算及虚拟 IBAN 等一体化解决方案，与多家跨境电商平台及在线市场建立合作。

## **二、全球化能力图谱**

###  （一）支付网络与区域布局

**💳****国际卡收单**

受理能力：支持 Visa、Mastercard 等主流国际卡的在线支付。

安全认证：支持 3D Secure 2.0 认证以提升交易安全性。

智能路由：提供支付路由功能，可在单个收单行失败时自动切换至备用收单行。

**🔗****本地支付聚合**

欧洲：支持 SEPA 信用转账与 Bancontact（比利时）。

亚太地区：支持 WeChat Pay 和支付宝等主流支付方式。

数字钱包：支持 Apple Pay 和 Google Pay。

**🎯****战略定位**

为欧洲商户提供核心的银行卡及银行转账（SEPA）支付解决方案，并辅以对中国等特定市场的主流支付方式支持；在欧洲市场，能够与 Stripe 等全球支付巨头在中小企业电商领域展开差异化竞争，相较于传统银行收单或单纯的支付网关，提供更好的资金管理与集成便利性。

**🔄****跨境清算路径**

网络接入能力：拥有独立的 SWIFT BIC 代码，具备接入全球银行间清算网络的能力。

区域枢纽：伦敦总部作为 FCA 持牌的 EMI 主体，是全球合规、运营与战略中心；华沙办公室支持中东欧市场的业务运营。

### （二）合规与安全体系

**👤****KYC/AML**

作为受 FCA 监管的 EMI 机构，需遵守严格的 KYC/AML 流程。

**💰****客户资金管理**

严格遵循 FCA 关于客户资金保障（Safeguarding）的规定，将客户资金与公司自有资金分离，存放于独立的托管账户；基于其 FCA EMI 牌照的合规账户体系，结合与 ClearBank 在清算层面的深度集成，在英国市场提供兼具合规性与高效率的即时/同日结算服务，打造其核心竞争壁垒。

## **三、技术能力清单**

###  （一）智能技术与架构

**🛡️****风控引擎（Fondy Antifraud）**

提供自研的 Fondy Antifraud 反欺诈系统，采用 **300 多项** 规则和算法进行风险评估。

**🔧****开发者生态**

API 与 SDK：提供 RESTful API、官方 Node.js SDK、JavaScript API，同时支持 PHP、Python 等多语言 SDK。

开发工具：提供包含公开测试凭证的沙盒环境及开发者文档中心。

预置集成（核心优势）：官方插件覆盖 Shopify、WooCommerce、Magento、Wix、Drupal、InSales、EcWid、OpenCart、PrestaShop 等主流和长尾电商平台及内容管理系统，集成生态广泛。

### （二）支付与结算效率

**⏱️****结算时效**

即时结算：支持即时结算（Instant Settlements）服务，商户可在几分钟内实现资金到账，包括夜间和公休日。

同日结算：通过与 ClearBank 的合作，Fondy Flow 虚拟 IBAN 账户可接入英国本地支付网络（Faster Payments， CHAPS， Bacs），实现当天到账。

**💵****资金管理技术**

为企业提供 Fondy Flow 多币种虚拟 IBAN 账户，支持在一个账户中管理和收付多种货币；技术能力聚焦于支付处理与电商集成；对商户的价值在于，无需在多个国家开设实体银行账户，通过 Fondy 平台即可实现多币种资金的本地收款及统一管理，大幅简化跨境业务财资管理操作的复杂度。

## **四、计费结构与服务规则**

**📋****整体定价原则**

采用基于交易量的分级阶梯（Tiered Pricing）定价模型，为不同规模的商户提供差异化费率。

**💳****核心服务**

核心服务包括银行卡支付（含国内、欧洲经济区及国际卡支付）、大额交易优惠、出款服务（单笔、批量）及虚拟 IBAN 账户服务，具体费率以官方定价页及《商户协议》为准。

## **五、汇率管理与资金流转**

**📊****汇率机制**

采用银行间汇率进行货币兑换，具体汇率价差情况未公开披露。

**🔄****换汇与对冲工具**

换汇能力：内置多币种即时兑换功能，支持常见货币之间的自动转换。

对冲工具：不提供远期锁汇或衍生品对冲工具，侧重于即时支付与结算。

**💵****多币种账户体系（Fondy Flow）**

Fondy Flow 多币种虚拟 IBAN 账户通过与 ClearBank 合作集成，实现统一账户下多币种收付。

## **六、生态对接与服务能力**

###  （一）电商与平台

**🛒****独立站集成（核心优势）：**

官方插件覆盖主流和长尾电商平台及内容管理系统（CMS），兼容性极强。

**🔄****订阅管理：**

提供订阅生命周期管理，包括创建、暂停、恢复、升级、降级和取消等功能。

**🏢****平台能力：**

支持分账支付和批量支付，适合市场平台模式；提供 Flow Identity 用于 KYC（客户身份验证）、KYB（商户身份验证）审核及子商户管理。

### （二）企业系统集成

**🔌****通过开放 API 及 Webhook，商户可将交易数据导入 ERP 或财务系统（如 SAP、NetSuite、Xero），实现后端系统对接。**

###  （三）服务交付与支持

**📞****支持体系：**

提供在线工单和邮件支持，工作时间内平均响应时间为几小时。

**📚****自助服务：**

拥有开发者文档中心、帮助中心和 API 参考文档，这些文档中包含示例代码和集成指南。

## **七、增值工具与行业解决方案**

###  （一）风控套件（Fondy Antifraud）

**🛡️****将风控能力产品化，应用于支付全流程；系统基于********300 多项****风控规则和算法引擎进行风险评估和实时反欺诈预警。**

###  （二）财资与资金管理套件（Fondy Flow）

**💼****将多币种资金管理及虚拟 IBAN 账户服务产品化，通过与 ClearBank 深度合作集成实现高效结算。**

###  （三）行业方案

**🌐****跨境电商**

适配组合：官方电商插件、Fondy Flow 多币种收款服务、Antifraud 反欺诈系统。

能力亮点：插件助力快速集成上线，Fondy Flow 实现英国及欧盟地区资金的高效收款与清算。

**🔄****SaaS & 订阅制业务**

适配组合：完整的订阅生命周期管理服务、全球银行卡收单服务、循环计费功能。

能力亮点：支持订阅生命周期管理，满足 SaaS 等按周期付费业务需求。

**🏪****平台 / 市场**

适配组合：分账支付服务、Flow Identity 身份审核服务、批量支付工具。

能力亮点：满足收益分成和多子账户模式平台的支付与合规管理需求。

**📋****附：快速落地清单**

  * **🔧****技术** ：在开发者中心获取沙盒环境的测试账号凭证，利用官方电商插件进行快速集成。

  * **💰****财务** ：与商务团队确认适用的阶梯费率，配置虚拟 IBAN 账户及结算规则。

  * **🛡️****合规** ：准备商户（KYB）审核材料，通过 Flow Identity 完成 KYB 审核，签署符合 FCA 监管的服务协议。

  * **👥****运营** ：商户后台支持订阅计划配置和分账规则设定。




## **八、适配场景与注意事项**

###  （一）适配场景

  * **🏪****欧洲中小企业使用各类电商平台及内容管理系统（CMS）** ：可利用 Fondy 官方插件，覆盖 Shopify、WooCommerce 等，实现快速集成与低代码落地。

  * **🇬🇧****对英国本地结算时效要求较高的商户** ：可利用 Fondy Flow 解决方案，通过与 ClearBank 合作接入英国本地清算网络，实现即时或同日结算。

  * **🔄****需要基础订阅或分账功能的平台** ：适合业务模式相对简单的 SaaS 或市场平台，可利用其原生的订阅管理与分账支付功能。




### （二）注意事项

  * 🌍**区域限制** ：业务高度集中于欧洲市场，虽然声称全球覆盖，但在北美、亚太、拉美等地区的本地支付方式及合规布局上，公开信息有限。

  * 📊**财务数据** ：英国 Fondy Ltd 公开财报显示资产和营收规模有限，与全球合并报表口径的业务规模存在差异，需审慎核查实际经营状况。

  * 🔧**技术透明度** ：现阶段未公开完整的 Webhook开发文档，建议在项目合作环节重点验证其技术可用性和集成深度。

  * 🛡️**合规状态** ：PCI-DSS 证书需由 Fondy 方及时更新并公开提供。




## **九、区域合规与适配方案**

###  （一）欧洲（核心优势区域）

**🛡️****准入与合规要点**

以英国 FCA 授权的 EMI 机构主体开展运营，业务运营遵循英国及欧盟的支付服务法规，通过英国 FCA 授权的 EMI 主体和欧盟境内业务实体合规运营。

**🏢****本地化运营**

深度整合英国清算网络（Faster Payments、CHAPS、Bacs），支持 SEPA 信用转账和 Bancontact。

### （二）北美、亚太及其他区域

**🎯****战略定位与能力边界**

在北美、亚太等区域以国际支付网关身份，为本地商户提供全球卡支付通道，但未深度整合当地支付方式或获取当地金融牌照。

**📊****业务适配性评估**

本地支付：除 WeChat Pay 和部分主流电子钱包外，北美、亚太等区域的本地支付方式覆盖范围有限。

本地合规：商户需自行确保其在当地业务运营的合规性。
