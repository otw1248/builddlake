本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业基本档案**

以下内容基于企业公开材料、监管公示与行业媒体整理；仅供参考。

**📅****成立时间与主体：**

2020 年 4 月在新加坡注册成立（TAZAPAY PTE. LTD.），公司总部位于新加坡。

**🌍****全球布局：**

以新加坡为全球总部和亚太运营中心，在加拿大与立陶宛设有子公司，分别覆盖北美与欧洲的合规与运营。

**🛡️****合规资质：**

亚太：持有新加坡金融管理局（MAS）主要支付机构牌照（MPI）；  
北美：持有加拿大金融交易与报告分析中心（FINTRAC）货币服务业务牌照（MSB）；  
欧洲：通过立陶宛子公司 Tazapay Europe UAB 申请相关金融服务牌照。

**✅****行业认证：**

通过支付卡行业数据安全标准（PCI-DSS 4.0 Level 1）。

**📊****业务覆盖：**

卡支付业务覆盖 173+ 国家；本地支付方式与出款网络覆盖 70+ 市场；提供 35+ 种货币的虚拟账户；支持 60+ 种货币的资金支出；年化支付处理量超过 100 亿美元。

**🎯****行业覆盖与应用场景：**

服务跨境电商、教育科技、SaaS、在线旅游与游戏等中小企业，提供全球收款账户、本地支付聚合、多币种资金管理、全球批量出款及 B2B 交易托管（Escrow，托管）等能力。

## **二、全球化能力图谱**

###  （一）支付网络与区域布局

**💳****国际卡收单：**

受理范围：通过合作收单行对接，支持在 173+ 国家接收 Visa、Mastercard、American Express 等；  
安全认证：支持 3D Secure 2.0，并可按市场差异化配置动态触发策略；  
智能路由：基于成功率、成本等因素动态选择处理路径；  
高级功能：面向中小企业的一体化收付平台定位下，网络令牌化、账户更新器等高级功能按客户需求配置。

**🏦****本地支付聚合：**

覆盖范围：支持 70+ 市场本地方式；  
欧洲：支持 SEPA（单一欧元支付区，含 SEPA Instant）与 SOFORT（德语区即时转账）；  
亚太：支持印度 UPI（统一支付接口）、东南亚 GrabPay、韩国 KakaoPay/NaverPay/Samsung Pay 等；  
拉丁美洲：支持巴西 Pix；  
扩展策略：按区域需求优先级逐步拓展覆盖范围。

**🔄****跨境清算路径：**

网络对接：支持 SWIFT 全球收付款；直连 SEPA Instant、PayNow（新加坡实时转账）、FedWire（美国联邦电汇）等区域即时清算网络；  
出款能力：向 70+ 市场出款，提供本地转账与 SWIFT 电汇两种路径，可按需选择时效与成本方案。

**📍****区域枢纽：**

新加坡总部：MAS 持牌主体，承担全球及亚太地区的运营管理与合规工作；  
立陶宛子公司：面向欧洲市场推进持牌与运营；  
加拿大子公司：MSB 持牌主体，服务北美市场。

### （二）合规与安全体系

**🛡️****核心监管牌照：**

新加坡：MAS 主要支付机构（MPI）；  
加拿大：FINTRAC MSB；  
欧盟：立陶宛子公司申请欧盟加密资产市场法规（MiCA）框架下的加密资产服务提供商牌照（CASP）。  
扩展计划：在美国、阿联酋、澳大利亚、香港推进牌照申请。

**✅****行业认证：**

PCI-DSS 4.0 Level 1。

**🔍****KYC/AML（了解你的客户 / 反洗钱）：**

技术对接：与 ComplyAdvantage 对接，对制裁名单及政治公众人物（PEP）进行筛查与交易监控；  
流程能力：覆盖事前身份与主体验证（KYC/KYB）；事中交易监控；事后可疑活动报告（SAR）提交。

**💰****客户资金管理：**

资金存管（按监管要求对客户资金进行隔离存管）—— 按监管要求进行客户资金隔离（Safeguarding），资金存放于新加坡授权银行。

## **三、技术能力清单**

###  （一）智能技术与架构

**☁️****平台架构：**

以 Amazon Web Services（AWS，云服务）为主要基础设施；

**🛡️****风控引擎：**

规则引擎与机器学习双体系，整合 ComplyAdvantage 能力；

**📊****数据采集与分析：**

支持设备指纹与交易模式分析；

**⚙️****商户配置功能：**

可自定义交易监控规则与 3D Secure 动态触发策略。

### （二）开发者生态

API 与 SDK（开发工具包）：提供 RESTful API 及 Node.js、PHP、Python、Ruby 等语言的 SDK，含 JavaScript SDK 前端集成；  
  
开发工具：提供文档中心、公开沙盒与支持幂等机制及指数退避重试功能的 Webhook；  
  
现成集成方案：提供 Shopify/WooCommerce 官方插件；加入亚马逊支付服务提供商计划（PSPP）。

### （三）支付与结算效率

到账时效：本地支付到账时效为 T+0 至 T+1；SWIFT 电汇到账时效为 T+1 至 T+5；  
  
结算周期：新用户默认为周结；可申请日结或双周结；  
  
资金管理功能（支持 35+ 种货币的虚拟账户统一管理）：提供 35+ 种货币虚拟账户统一管理；提供 Payouts API 面向 70+ 市场分发资金。

## **四、计费结构与服务规则**

整体定价：采用透明的混合费率模式（Blended Rate）；大交易量可定制报价。  
  
收款服务：本地支付方式收款费率为 0.8% 至 2.5%；国际卡收单费率为 3.8%＋0.5 美元 / 笔（非高风险行业）；  
  
结算（出款）服务：本地货币结算手续费：1,000 美元以下收取 5 美元，超过该金额则免费；电汇结算手续费：100,000 美元以下收取 35 美元，超过该金额则免费；  
  
争议处理：争议处理费为 15 美元 / 笔争议，预争议警告（RDR，Rapid Dispute Resolution）费用为 15 美元 / 次，仲裁申请费 500 美元由败诉方承担；  
  
其他费用：退款手续费 —— 卡支付 2 美元；本地方式 1 美元；认证失败费每次 0.2 美元（成功免收）；FIRC（国外入账证明书，印度）每份 4 美元；  
  
小结：费用以《商户协议》为准。

## **五、汇率管理与资金流转**

**📊****汇率与透明度**

报价机制：通过与 Currencycloud 等合作伙伴对接，提供实时且具有竞争力的汇率报价；  
汇率优势：接近银行间汇率，费用透明展示。

**🔄****换汇与对冲**

即期换汇：支持在多币种账户内按需转换；  
汇率对冲工具：暂不提供远期锁汇等衍生品服务。

**🏦****多币种账户体系**

账户结构：提供 35+ 种货币的全球收款虚拟账户；  
资金管理（Tazapay Balances）：自 2024 年 9 月起提供多币种余额管理，统一查看、持有与管理；  
账户功能：支持实时余额查询、自动对账报表生成、向 100+ 种货币灵活出款。

**⏱️****到账与清算时效**

时效详情：本地支付结算 T+0 至 T+1；SWIFT 电汇结算 T+1 至 T+5；  
资金利用效率：通过持有多币种余额，减少不必要的货币转换，提高资金周转；

**📌****小结：**

以全球收款账户为核心，结合 Currencycloud 汇兑网络，形成 “持有 — 转换 — 分发” 的一体化资金管理能力。

## **六、生态对接与服务能力**

###  （一）电商平台与市场

平台卖家回款：作为亚马逊支付服务提供商计划（PSPP）合作伙伴，为卖家提供多币种虚拟账户提取销售收入的回款服务；  
  
独立站插件：提供 Shopify 与 WooCommerce 官方插件，4.0 系列插件支持基于块编辑器的结账功能与暗色模式；  
  
B2B 市场：与 Bluerickshaw/OneAgrix/Complete Farmer 等合作，提供含托管（Escrow）功能的支付与资金安全整体解决方案。

### （二）平台与创作者经济

向创作者 / 供应商批量出款：通过 Payouts API 面向 70+ 市场向创作者、供应商批量下发资金的服务；  
  
多方分账：分账（Split Payments）与子商户进件功能可根据客户需求定制开发。

### （三）企业系统（ERP / 财税）集成

集成策略：通过 API 接口与 CSV 报表对接企业后台系统，实现数据同步与资金管理；暂未提供 SAP/Oracle NetSuite 等标准化预置连接器。

### （四）服务交付与支持

支持模式：专属客户代表；邮件、联系表单与 WhatsApp等；  
  
自助服务：提供知识库与开发者文档中心；

**📌****小结：**

通过虚拟账户＋Payouts API＋电商生态插件，为平台与市场提供关键资金管理能力。

## **七、增值工具与行业解决方案**

###  （一）风控套件

能力集成：规则引擎＋机器学习；对接 ComplyAdvantage；  
  
特色功能：提供预争议警告与快速争议解决（RDR）以降低运营成本。

### （二）财资与资金管理套件（Tazapay Balances）

产品定位：多币种资金管理能力产品化；  
  
核心能力：35+ 种货币虚拟账户；统一持有、管理与按需转换；提供详细余额报告。

### （三）数据与对账自动化套件

能力集成：商户后台、CSV 导出与 REST API；  
  
核心能力：Webhook 事件可靠通知，支持幂等与指数退避重试。

### （四）行业方案

游戏与数字娱乐：本地支付覆盖 70+ 国家；全球卡收单；60+ 国家同日到账出款；选择性 3DS 与机器学习优化风控；  
  
B2B 贸易平台：托管（Escrow）＋全球支付网络＋买卖双方保护；通过托管（Escrow）与全球支付网络，满足高金额交易的信任与安全需求；  
  
SaaS 平台：全球收款＋汇率管理＋全球出款；适配复杂跨境资金通道（如美国至印度的跨境支付），降低交易误报率与失败率。

  * **🛡️****合规** ：准备 KYB 材料（企业身份验证）并完成验证，签署商户协议，确认目标市场的法规要求；

  * **👥****运营** ：配置 3DS 与风控规则；熟悉争议处理；与客户代表建立沟通机制；

以 Tazapay Balances 为核心，结合托管与本地化支付，形成面向游戏与 B2B 等行业的特色方案。

## **八、适配场景与注意事项**

###  （一）适配场景

  * **🌍****深度本地化出海企业** ：覆盖欧洲、东南亚、印度；依赖高覆盖率本地支付提升转化；

  * **🛒****多渠道零售商** ：亚马逊＋独立站并行；需统一归集与管理全球资金；

  * **🎮****游戏与 B2B 平台** ：面向全球用户大规模分发或高价值交易保障；

  * **💻****全球化 SaaS** ：优化跨境订阅支付成功率；处理复杂通道；

  * **🎨****创作者经济平台** ：向 70+ 市场的创作者与供应商批量分发资金。




### （二）注意事项

  * ⚖️**合规** ：业务上线前需完成 KYB 验证；关注各目标市场牌照申请进度（如立陶宛子公司 2025 年 4 月成立，欧盟牌照申请正在推进）及对业务落地的影响；

  * 🔧**技术** ：订阅管理（含催款管理，Dunning）与平台分账（Split Payments）的标准化功能尚未完善；分账功能处于 Beta 阶段，预计 2025 年 Q4 全面发布；复杂需求建议提前评估定制开发可行性；

  * 💰**资金** ：需注意区分本地转账与电汇路径的费用差异及到账时效；争议处理会产生额外费用，且需关注预争议警告（RDR）的自动退款政策；

  * 📞**运营** ：主要服务支持渠道为邮件、表单及 WhatsApp；重大事项可通过专属客户代表协调处理；

  * 🛡️**牌照监管** ：关注美国、阿联酋、澳大利亚、香港等在申市场的监管进展对业务落地的影响。




## **九、区域合规与适配方案**

###  亚太

持有 MAS MPI；推进香港与澳大利亚牌照；整合 GrabPay、UPI、KakaoPay/NaverPay 等；印度提供 FIRC 服务每份 4 美元；上线前完成 KYB 并测试本地方式；

### 欧洲

立陶宛子公司 2025 年 4 月成立；按欧盟 MiCA/CASP 推进；遵循 GDPR 并提供数据处理协议（DPA，Data Processing Agreement）；支持 SEPA/SEPA Instant 与 SOFORT；与 Volt 合作开放银行支付；

### 北美

加拿大 FINTRAC MSB；美国牌照在申；作为亚马逊 PSPP 合作伙伴服务卖家；按当地要求配置风控；

### 其他区域

中东推进阿联酋等牌照；亚太计划日本团队与澳大利亚牌照；拉美与非洲以全球网络服务并支持巴西 Pix；

**📌****小结：**

以新加坡 MAS 牌照为核心；加拿大 MSB 覆盖北美；立陶宛子公司支撑欧洲；区域服务适配能力与当地牌照申请进度紧密相关，建议持续关注目标市场的监管政策变化。
