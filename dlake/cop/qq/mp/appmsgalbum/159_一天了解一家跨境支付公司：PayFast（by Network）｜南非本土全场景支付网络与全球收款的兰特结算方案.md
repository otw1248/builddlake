本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业基本档案**

以下内容基于企业公开材料、监管公示与行业媒体整理；仅供参考。

**📅****成立时间与主体：**

2007 年5月（南非开普敦）；公司全称 PayFast（Pty）Ltd（私营有限公司）；2019 年7月被 DPO Group收购；2021 年 9 月随 DPO Group 并入 Network International PLC（上市公司）；2024 年5月品牌重塑为 Network International；PayFast 作为产品名称保留。

**🌍****全球布局：**

运营总部设在南非开普敦；在约翰内斯堡设有办公室；母公司 Network International 总部位于阿联酋迪拜；并在埃及、约旦、尼日利亚及沙特阿拉伯等地设有办公室。

**🛡️****合规资质：**

南非储备银行（SARB）指定的清算系统参与者；南非支付协会（PASA）注册的第三方支付提供商（TPPP）；PCI-DSS Level 1 v4.0（支付卡行业数据安全标准一级）服务提供商认证。

**📊****业务覆盖：**

核心服务南非市场；服务商户超 **80,000** 家；支持 **18** 种以上支付方式；可受理全球发行的 Visa 与 Mastercard；统一以南非兰特（ZAR）结算。

**🎯****行业渗透与应用场景：**

面向中小电商（MSME）、订阅服务、市场平台、在线教育与非营利组织；提供独立站插件接入；支持订阅与循环扣款；多方分账；即时 EFT（电子转账）本地支付；即时提现。

## **二、全球化能力图谱**

###  （一）支付网络与区域布局

**💳****国际卡收单：**

受理全球发行的 Visa 与 Mastercard 信用卡和借记卡；通过赞助银行 Absa Bank Limited 接入卡组织并参与 SARB 结算；3D Secure 2.0（3D 验证）覆盖 Visa Secure 与 Mastercard Identity Check；交易包含 CVV（卡片校验码）校验；提供 Payvault 令牌化并符合 PCI-DSS Level 1；整体配置以南非市场需求为准。

**🔗****本地支付聚合：**

整合 **18** 种以上本地方式；即时 EFT 覆盖 FNB、ABSA、Standard Bank、Nedbank、Capitec、Investec、TymeBank、African Bank 及 Bidvest；支持 Apple Pay 与 Samsung Pay；集成 SnapScan、Zapper 二维码；接入 Mobicred、MoreTyme 分期；对接 SCode（现金条码支付）线下现金支付。

**🔄****跨境清算路径：**

收款经由 Visa/Mastercard；在本地折算为 ZAR 结算；出款以南非银行转账为主；清算网络以本地银行通道为核心。

**📍****区域枢纽：**

  * 🇿🇦开普敦负责产品、技术与客服；

  * 🇿🇦约翰内斯堡为企业客户提供服务；

  * 🌍可依托母公司网络触达中东与非洲其他国家。


**📞****服务与语言：**

提供英语与南非本地语言支持；客服覆盖工作日（08:00-18:00，GMT+2）及周六（08:00-14:00，GMT+2）。

### （二）合规与安全体系

**🛡️****牌照与认证：**

SARB 清算系统参与者；PASA 注册 TPPP；PCI-DSS Level 1 v4.0。

**👤****KYC/AML：**

商户上线需完成身份与账户核验。

**💰****客户资金管理：**

以支付促进商模式，通过 Absa Bank Limited 独立托管客户资金；遵循 SARB 对清算系统参与者的要求。

**⚖️****合规运营：**

遵循南非《个人信息保护法案》（POPI Act）；接口采用 SSL（传输层安全协议）加密；银行侧执行 2FA（双因素认证）；定期发布 State of Pay 报告并公示 PCI-DSS 认证。

## **三、技术能力清单**

###  （一）智能技术与架构

**☁️****平台架构：**

面向商户提供稳定的接入与结算体验；公开信息以产品与合规模块为主。

**📈****系统可用性：**

设有系统状态页面，覆盖各支付方式与银行通道；可用性指标以 SLA（服务等级目标）约定为准。

**🛡️****风控引擎：**

搭载 PayProtector 系统；自动化规则与人工审核结合；逐笔筛查可疑行为并比对全球负面卡片库；支持商户自定义黑名单与风险阈值。

### （二）开发者生态

**🔌****API 与 SDK：**

提供 RESTful API 与沙盒；官方 PHP SDK；主流电商插件。

**📚****文档与 Webhook：**

提供独立开发者文档与集成指南；Webhook 事件与重试策略按开发者后台指引及技术文档要求执行。

### （三）支付与结算效率

**⏱️****到账时效：**

标准提现约 **48** 小时；即时提现数分钟；即时 EFT 覆盖 9 家银行实现近实时转账。

**💰****资金管理技术：**

支持多币种定价，受理多币种并实时折算为 ZAR；提供分账支付，支持多方自动分账。

## **四、计费结构与服务规则**

**🌐****跨境收款：**

采用混合费率（固定费用 + 比例费用）与阶梯定价（交易金额越高，费率越低）；无月费与设置费；按成功交易金额计费；量大可议价。

**💳****国际卡收单：**

受理 Visa/Mastercard 信用卡与借记卡；标准费率 **3.2%＋R2.00** （以 2025 年 8 月官方费率表为准）。

**🏦****本地支付：**

  * 即时 EFT **2.0%** （最低 R2.00）；

  * SnapScan/Zapper **3.5%＋R2.00** ；

  * MoreTyme **5.5%＋R2.00** ；

  * SCode **4.5%＋R5.00** ；

  * Mobicred **3.2%** ；

  * Apple Pay/Samsung Pay **3.2%** （最低 R2.00）；

  * Capitec Pay **2.0%** （最低 R2.00）。


**📤****提现与对公结算：**

标准提现 **R8.70** ；即时提现 **0.8%** （最低 R14.00）；以上均不含 VAT（增值税）。

**✨****增值服务：**

订阅管理、分账支付与 PayProtector 基础风控计入标准交易费；高级风控与争议处理按合同约定执行。

**📝****费用与条款补充：**

不活跃账户在有余额且 12 个月无交易时，收取 115 兰特（R115）直至账户余额清零；其他费用以《商户协议》为准。

**📌****小结：**

费率以 2025 年 8 月公开价为依据；最终定价以商务谈判与《商户协议》为准。

## **五、汇率管理与资金流转**

**📊****汇率与展示：**

多币种定价支持实时换算；前端展示全包汇率。

**🔄****换汇与对冲：**

外币交易自动、实时折算为 ZAR 结算。

**🏦****账户与调度：**

采用综合余额账户；所有交易以 ZAR 余额呈现；分账可按固定金额、百分比或组合规则进行实时分配。

**⏱️****到账与清算时效：**

即时 EFT 近实时确认；国际卡收单按卡组织结算周期处理，且需额外叠加约 48 小时的银行侧核查时间；提现提供 48 小时与即时两类；即时提现在清算期结束后发起。

**📌****小结：**

通过多币种定价与即时换汇统一为 ZAR 结算；结合分账能力形成可预测的本地化资金流转路径。

## **六、生态对接与服务能力**

###  （一）电商平台 / 独立站接入

**🛒****独立站插件：**

WooCommerce 官方插件（活跃安装超 **30,000** ）；兼容订阅与预订；提供 Shopify 官方应用；同时支持 Magento 及 WHMCS（主机与账单系统）。

**🛠️****能力细项：**

控制台可发起全额退款或部分退款；通过 Payvault 令牌化保存卡信息；提供完整订阅生命周期管理；重点服务本地商户与独立站。

### （二）企业系统（ERP / 财税）集成

**📊****Xero（财务软件）：**

官方深度集成；支持发票管理与交易数据实时同步；可在发票启用点击支付。

**📋****对账与凭证：**

支持 CSV（即逗号分隔文件）报表导出。

### （三）本地化网络（南非要点）

覆盖即时 EFT、二维码、分期与现金网络等 **18** 种以上本地方式；上线需提交 KYB（企业身份核验）材料并完成沙盒测试。

### （四）服务交付与支持

**🔄****实施流程：**

账户注册；账户验证；集成配置；沙盒测试；上线。

**📞****支持体系：**

电话、邮件与在线聊天；周一至周五 08:00-18:00（GMT+2，南非时间）；周六 08:00-14:00（GMT+2）；系统状态页发布运行与维护信息；不同优先级工单响应按合同约定执行。

**📌****小结：**

官方插件与 Xero 集成叠加本地化支付覆盖，构成服务南非中小企业的核心能力组合。

## **七、增值工具与行业方案**

###  （一）风控套件（PayProtector）

数据信号涵盖设备指纹、IP 与地理位置、BIN（卡 BIN 号）、交易行为与全球负面卡片库；自动化规则与人工审核并行；提供基础欺诈报告与黑名单管理；支持商户设置黑名单、风险阈值与二次验证。

### （二）汇率与资金（FX 与 Treasury，外汇与资金管理）

以 ZAR 为核心的综合余额账户；在多币种定价下即时换汇。

### （三）数据与对账

提供交易与余额报表；支持 CSV 导出；数据治理遵循 PCI-DSS Level 1 与 POPI Act。

### （四）行业方案

**🛍️****电商 / 在线零售：**

  * 使用 WooCommerce/Shopify 插件；

  * 启用本地支付方式；

  * 即时 EFT 到账；

  * 结合实时监控与负面库交叉验证；

  * 完成注册、配置与沙盒测试。


**🔄****订阅服务：**

  * 使用订阅管理引擎；

  * 配合令牌化与循环扣款；

  * 可设置月度、季度、半年度、年度；

  * 启用令牌化并验证全流程。


**🏢****市场平台：**

  * 通过分账系统按固定金额、百分比或组合分配；

  * 适配平台经济模式；

  * 对多方交易进行监控并明确平台方责任；

  * 设定分账规则并完成测试。


**📋****附：快速落地清单****🔧****技术：**

  * 获取 API Key；

  * 配置沙盒（sandbox.payfast.co.za）；

  * 完成 API 集成与核心用例测试；

  * 可使用 Postman（接口调试工具）集合。


**💰****财务：**

  * 完成商户注册；

  * 确认费率；

  * 绑定南非本地银行账户。


**🛡️****合规：**

  * 准备 KYB（企业身份核验）材料（企业注册证书、身份证明、地址证明、银行账户证明）；

  * 签署商户协议；

  * 完成合规审核；

  * 提交 FICA（金融情报中心法）文件。


**⚙️****运营：**

  * 配置 PayProtector 参数；

  * 启用所需支付方式；

  * 熟悉客服流程。




**📌****小结：**

PayProtector 风控、订阅与分账引擎以及电商插件构成常用组合；可按行业场景部署使用。

## **八、适配场景与注意事项**

###  （一）适配场景

  * **🇿🇦****南非本地市场深耕企业** ；

  * **🛍️****中小电商** ；

  * **🔄****本地订阅类服务** ；

  * **🏢****本地市场平台** 。




### （二）注意事项

  * ⚖️**合规要求** ：签约主体为 PayFast（Pty）Ltd；受 SARB 监管并被指定为清算系统参与者；遵循 POPI Act；通过 Absa Bank Limited 参与结算。

  * 🔧**技术集成** ：上线前在沙盒完成支付与退款测试；Webhook 事件与重试策略按技术文档实施；提供 RESTful API 与 PHP SDK。

  * 💰**资金结算** ：外币收款统一并实时折算为 ZAR；提现需绑定与签约主体一致的南非本地银行账户；需提交 FICA（金融情报中心法）文件。

  * ⚙️**运营管理** ：明确标准提现与即时提现的时效与成本；费率以第四章为准；按照本地客服时间（工作日 08:00-18:00、周六 08:00-14:00，均为 GMT+2）联络支持；完成账户 FICA 验证。




## **九、区域合规与适配方案**

###  （一）南非（核心服务区域）

**🛡️****准入与合规要点：**

通过 KYB（企业身份核验）审核；业务符合本地法规；系统与流程遵循 SARB、PASA 与 POPI Act。

**⚙️****运营与风控重点：**

结合即时 EFT 与二维码等本地方式；配合 PayProtector 管理本地欺诈风险。

**📋****上线清单：**

提交企业注册证书、FICA（金融情报中心法）文件与董事身份证明；提供南非本地银行账户证明；完成 API 集成与沙盒测试报告。

### （二）其他全球市场

**🌐****战略定位与能力范围：**

以南非为核心市场；通过多币种定价接受全球 Visa/Mastercard 付款并以 ZAR 结算；可借助母公司网络获取区域协同。

**💡****客户建议：**

如需多区域深度本地化，可评估具备相应能力的方案；以南非为主并服务国际客户的企业，可依托多币种定价实现统一结算。

**📌****小结：**

在南非具备合规与本地化优势；通过多币种定价形成全球收款与本地结算的组合。
