本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

前身 Travelers Express Co. Inc. 于 1940 年在美国明尼阿波利斯成立，2004 年正式更名为 MoneyGram International, Inc.，注册于特拉华州。

**🌍****全球布局**

总部位于美国德克萨斯州达拉斯，运营中心位于明尼苏达州。在全球 36 个国家设有办公室，覆盖美洲、欧洲、亚太、中东、非洲等区域。欧洲业务通过比利时注册实体（MoneyGram International SA）以支付机构牌照覆盖整个欧洲经济区（EEA）。

**🛡️****合规资质**

金融牌照：美国 FinCEN MSB、全部 50 个州及华盛顿特区 Money Transmitter 牌照、比利时 NBB 支付机构牌照（覆盖 EEA）、印度 RBI MTSS 牌照等

**📊****业务覆盖**

支持 200 多个国家和地区、135 种以上货币，拥有约 500,000 个零售代理网点和 50 亿数字端点。每年服务超过 5,000 万消费者，年处理资金规模超过 2,000 亿美元。

## **二、计费结构与费用说明**

**📋****计费模式说明**

MoneyGram 的计费模式主要包含两部分：固定转账费与汇率点差。

**💳****计费规则**

收费项目涵盖跨境汇款转账费、账单支付、汇票、移动充值、加密货币买卖以及企业 API 服务费等。转账费因发送国、接收国、转账金额、付款方式（银行转账、借记卡、信用卡、现金）、收款方式（现金提取、银行存款、钱包）等因素动态变化。官网提供费用估算工具（Fee Estimator），用户可在转账前查询具体费用。

企业 API 产品（C2C 白标、B2B/B2C 付款、MoneyGram Ramps）的定价为非公开的定制化模式，需联系商务团队获取。

**📝****费用项目补充说明**

发送方通过信用卡付款时，发卡行可能额外收取信用卡预借现金手续费和利息（非 MoneyGram 收取）。收款方银行可能收取入账手续费（非 MoneyGram 收取）。MoneyGram 提供 Plus Rewards 会员积分计划，通过积累转账次数可获得费用折扣。

**📌****小结：** 本章聚焦收费项目与适用规则，不含数值，最终以官方费率页面或《服务协议》为准。

## **三、全球网络与合规安全体系**

###  （一）汇款网络覆盖

零售代理网点覆盖 Walmart、CVS、7-Eleven、Walgreens 等主要零售连锁，支持 20,000 多条汇款走廊。

### （二）合规与安全体系

**🛡️****反洗钱能力**

提供全流程反洗钱能力：交易前核查身份、交易中实时拦截可疑交易、交易后保留证据并报送监管机构。对接制裁名单与不良名单库，设有董事会合规与伦理委员会。

**🔒****安全措施**

  * 端到端加密

  * 生物识别登录（指纹／Face ID）

  * 全天候实时监控与欺诈保护

  * 2025 年与 Oscilar 合作引入基于 AI 的风险决策平台




## **四、产品与服务体系**

###  （一）消费者产品

  * **国际汇款** ：跨境 P2P 转账至全球，支持现金提取、银行存款、卡存款、移动钱包等接收方式

  * **账单支付（Bill Pay）** ：向美国约 13,000 家公司支付账单

  * **移动充值（Mobile Top-up）** ：通过 Ding 合作向全球手机发送通话和数据

  * **汇票（Money Orders）** ：美国及波多黎各地区

  * **加密货币买卖** ：通过 MoneyGram App 买卖和持有加密货币




### （二）企业与合作伙伴产品

  * **在线汇款 API（C2C 白标方案）** ：合作伙伴将 MoneyGram 汇款嵌入自有 App 或网站，面向银行、金融科技公司、电信运营商，支持转账、退款、状态查询等功能。

  * **企业付款 API（B2B/B2C）** ：单笔和批量跨境付款，接收方式包含现金、银行、钱包、卡等。其中批量付款支持单次调用批量处理。

  * **MoneyGram Ramps API** ：加密货币出入金 API（即法币与加密货币之间的兑换服务），供数字钱包和交易所嵌入现金与 USDC 互转能力，覆盖 170 多个国家。




### （三）区块链与稳定币创新

目前 MoneyGram 超过 50% 的交易已通过数字渠道完成，公司在区块链领域有多项布局：

  * 2025 年 9 月推出下一代 App，以 USDC 为基础的数字美元存储账户，率先在哥伦比亚市场上线

  * 2025 年 12 月与 Fireblocks 合作，将稳定币结算纳入资金管理体系

  * 接入 Mastercard Move 跨境汇款网络




### （四）开发者工具与集成

提供公开的开发者门户，包含集成指南、API 参考文档、视频教程和代码示例，支持在线直接测试 API。采用 RESTful API 架构，支持事件通知与回调。MoneyGram Ramps 提供沙盒环境，传统 API 同样提供独立的测试环境。

## **五、合作生态**

###  战略合作伙伴

支付网络：Mastercard（Mastercard Move 集成，覆盖 38 个接收市场）、Visa（Visa Direct 实时到账）

区块链基础设施：Stellar Development Foundation、Circle、Fireblocks

金融科技：Plaid（美国及欧洲开放银行账户认证）、Oscilar（AI 风险决策平台）

零售网络：Walmart、CVS、7-Eleven、Walgreens 等

## **六、适配场景分析**

基于 MoneyGram 的产品能力和网络覆盖，以下场景可作为参考：

**🏦****银行或金融科技公司嵌入汇款服务**

如果您是银行或金融科技公司，需要在自有 App 或平台中为用户提供跨境汇款功能，MoneyGram 提供 C2C 白标 API，通过单一集成即可接入全球汇款网络，无需自建合规体系和代理网点。

**💼****企业跨境批量付款**

如果您需要向海外员工、供应商或合作伙伴进行跨境付款（如薪资发放、佣金结算），企业付款 API 支持单笔和批量跨境付款，接收方式包含现金、银行、钱包、卡等。

**🔄****数字钱包或加密平台法币互转**

如果您运营数字钱包或加密交易平台，需要为用户提供法币现金与 USDC 之间的兑换通道，MoneyGram Ramps API 覆盖 170 多个国家的现金出入金网络。

### 决策建议

**📊****业务场景匹配度**

MoneyGram 的服务集中在 P2P 汇款和企业付款场景，不适用于电商收单、SaaS 订阅、平台分账等场景。如果您的需求是跨境电商收单或独立站支付，MoneyGram 并非对应方案，可同时了解 Stripe、Adyen、ONERWAY 等。

**🔧****技术集成能力**

MoneyGram 以 REST API 直接对接为主，不提供传统多语言 SDK 包，评估您的团队是否具备相应的技术能力。

**💰****企业定价透明度**

消费者端定价可通过费用估算工具查询，但企业 API 产品定价为非公开模式，建议在签约前充分了解计费规则、汇率点差机制和服务边界。
