本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

1981 年在爱尔兰 Kerry 郡 Killorglin 成立，全称 Foreign Exchange Company of Ireland（缩写 FEXCO）。

**🌍****全球布局**

总部位于爱尔兰 Killorglin，在约 **50 个国家** 设有业务运营。主要办公地点包括：都柏林、伦敦、爱丁堡、阿联酋、新加坡、中国、日本、菲律宾、澳大利亚、新西兰、巴西等。

**🛡️****合规资质**

持有爱尔兰央行 PI 牌照，覆盖欧洲 **30 个市场** ；英国 FCA 授权；通过 PCI-DSS 和 ISO 27001 认证；遵循 GDPR、PSD2、SCA 规范。

**📋****业务模式说明**

Fexco 并非传统在线支付网关（如 Stripe、Adyen 模式），核心业务围绕外汇、货币转换和 B2B 支付服务构建，主要产品线包括：

  * 动态货币转换（DCC、MCP）

  * payUnite 支付调度平台

  * 国际支付（PayDirect、Receivables）

  * Treasury 外汇风控

  * 零售外汇（Fexco Currency、No1 Currency）




## **二、计费模式说明**

Fexco 各业务线均采用定制化定价，不公开标准费率，具体取决于交易量、业务类型、货币对、支付频率、集成需求等因素。

DCC/MCP 服务的计费模式较为特殊，与传统按笔收费不同：采用汇率加点（FX margin）分成模式，Fexco 与合作方（收单行、商户）协商确定加点幅度，加点收入按约定比例分成。对合作方而言，DCC 是一项增量收入来源，而非传统意义上的"支付费用"。

**📌****小结：** 本章列出收费项目与适用规则，不含数值，最终以《商户协议》为准。

## **三、货币覆盖与合规体系**

###  （一）货币覆盖与区域布局

**💱****货币转换能力**

Fexco 以外汇服务起家，货币转换为核心能力：

  * 国际支付：支持 **165+ 种货币** 发送，**130+ 种货币** 在线操作，**20+ 种货币** 接收

  * DCC 服务：覆盖主流卡网络，自动识别外币卡并提供本币支付选择

  * MCP（多币种定价）：电商场景下的多币种定价方案


**🔄****清算路径**

支持 SEPA（单一欧元支付区）、SWIFT 电汇、本地 ACH、Faster Payments（英国）、支票等渠道。

**📍****区域覆盖**

  * 欧洲：爱尔兰 PI 牌照 + EEA 29 国牌照互认机制（passporting）+ 英国 FCA，全线产品覆盖

  * 亚太：菲律宾 Maya 合作部署 **112,000** 商户终端（2024 年），在新加坡、中国、日本、澳大利亚、新西兰等地设有办公室

  * 拉美：巴西 DAI ATM 网络 DCC 合作

  * 中东/非洲：DCC 服务覆盖，阿联酋设有办公室




### （二）合规与安全体系

**🏛️****牌照服务类型**

爱尔兰央行 PI 牌照覆盖的支付服务类型包括：汇款（Money Remittance）、支付发起服务（PIS）和账户信息服务（AIS）。

**🔒****安全体系**

  * PCI-DSS：支付卡行业数据安全标准合规认证

  * ISO 27001：信息安全管理体系认证

  * SSL 256 位加密传输；全平台强制多因素认证（MFA）

  * 定期委托独立第三方安全公司进行渗透测试，全员定期安全和反洗钱培训


**👤****反洗钱能力**

API 内置制裁名单筛查功能（Sanctions API），专职合规团队持续监控监管变化，客户资金独立托管。

## **四、核心服务与行业方案**

###  （一）DCC（动态货币转换）

Fexco 自 **1996 年** 起商业化 DCC 服务，目前覆盖 **500 万+** 商户点位和 **60+ 家** 合作收单行。

服务渠道：POS 终端 DCC、电商在线 DCC（eDCC）、ATM 取现 DCC，支持 Visa、Mastercard、Maestro、V-Pay、Diners、Discover 共 **6 个** 卡网络。

**Fexco Protect** ：外汇风险托管方案，Fexco 全额承担 DCC 交易的外汇风险，合作方无需改变现有 POS 设备，接入后可获得每日结算和佣金收入，无需自行承担外汇风险。

**Exchange Rate Guarantee** ：面向持卡人的汇率保障承诺，如持卡人能证明 DCC 汇率不如发卡行汇率，可申请差额退款。

合作案例包括 Qantas/Jetstar（航空 DCC）、菲律宾 Maya **112,000** 商户终端、巴西 DAI ATM 网络等。

### （二）MCP（多币种定价）

面向电商平台的多币种定价方案，通过单一 API 实现全平台多币种管理。消费者以本币浏览价格并支付，汇率全天更新，支持按货币和国籍调整加价幅度，提供国际客户消费趋势分析报告。

### （三）payUnite（支付调度平台）

面向邮轮、航空、酒店、零售等行业，年处理交易量 **$170 亿+** 。

核心能力：智能路由（自动匹配成功率最高、成本最低的收单通道）；多收单行支持与风险分散；线上与 POS 统一平台；支付令牌化（Tokenisation）；3DS 与欺诈检测；拒付管理；BNPL 集成（Affirm）；本地支付方式接入；托管支付页面（HPP）；统一报告与对账（支持按船队、市场、货币等维度查看）。

payUnite 于 2025 年 4 月完成跨行业发布，从邮轮扩展至航空、酒店和零售领域。

邮轮行业方案：覆盖全球约 **35%** 邮轮支付市场，提供从预订到离船的全流程支付管理，集成 Fidelio PMS（邮轮物业管理系统），支持船上 POS 与线上统一平台。

酒店行业方案：POS DCC + payUnite + Oracle OPI 酒店 PMS 集成，支持集团统一报告。

### （四）国际支付与收款

**PayDirect** ：企业跨境付款，支持 **165+ 种货币** 发送至约 **200 个国家** ，批量支付最多 **10,000 笔** ，内置银行验证工具，支持 API 对接 ERP/会计系统，按角色分级的审批流程与审计追踪。

**Receivables** ：国际收款，支持 **20+ 种货币** 接收，到账当日可转换为本地货币。

### （五）零售外汇

Fexco Currency、No1 Currency 品牌，**600+ 门店** 覆盖爱尔兰、英国、澳大利亚、新西兰及太平洋岛国。2025 年收购 Sainsbury's Travel Money 后，英国零售网点扩至 **460+ 门店** 。支持线上下单门店自取（Click & Collect）、Home Delivery 和门店现场兑换。

## **五、技术接入与服务支持**

###  （一）技术平台与集成方式

Fexco 提供多个独立技术平台服务不同业务线：

  * Fexco Payables API：国际支付 RESTful API，约 **21 个** 公开端点，覆盖支付创建/查询/取消/审批、收款人管理、汇率查询、银行验证等

  * DCC Developer Portal：DCC 集成开发者门户（需注册访问）

  * payUnite 平台：支持实时 API、托管支付页面、APM Link API，可与商户预订系统、POS、ERP 对接




集成方式：RESTful API + Token 认证，提供 API 代码示例文档。Fexco 采用 B2B 企业级集成模式，通过销售和技术团队协助完成对接，不提供标准化 SDK（npm、PyPI 等）或电商平台插件（Shopify、WooCommerce 等）。

### （二）服务支持

入驻流程：在线注册 → 销售团队电话沟通 → KYC 文档提交 → 账户开通

服务模式：所有客户分配专属客户经理，提供电话和邮件支持

培训资源：DCC Academy 在线培训门户（面向 DCC 合作伙伴）、API 文档与代码示例、FAQ

服务认证：CCA 全球客户体验认证

## **六、适配场景与选型建议**

**✅****适合使用 Fexco 的场景**

**收单行与支付服务商** 如果您是收单行或支付机构，希望为商户提供外币卡交易的增量收入，Fexco DCC 服务提供 Fexco Protect 零外汇风险方案，合作方无需承担汇率风险即可获得佣金收入。

**邮轮、航空与酒店企业** 如果您运营邮轮、航空或酒店业务，需要跨渠道（线上与 POS）统一支付调度，payUnite 已在邮轮行业有实际运营经验，2025 年扩展至航空、酒店和零售领域。

**企业国际供应商付款** 如果您需要向多个国家的海外供应商付款，PayDirect 支持多币种批量支付和外汇对冲工具，服务详情见第四章。

**中小企业跨境支付** 如果您是中小企业，需要替代银行的跨境支付方案，PayDirect 支持多币种付款。

**ATM 运营商** 如果您运营 ATM 网络，接入 DCC@ATM 服务可在外币卡取现时提供本币转换选择，创造新收入来源。

**📋****决策建议**

在选择跨境支付与外汇服务商时，建议从以下维度进行综合评估：

**业务场景匹配度** Fexco 的服务能力集中在 DCC、MCP、支付调度和国际支付领域，适合收单行、旅游与酒店企业、有国际付款需求的企业。如果您需要在线电商独立站收单或标准化支付网关集成（如 Shopify 插件），Fexco 并非直接方案，可同时了解 Stripe、Adyen 等方案；如果业务集中在东南亚本地支付接入，可对比 2C2P、dLocal 等区域化方案。

**技术集成方式** 如技术资源有限且需要自助快速集成，Fexco 不提供现成插件或标准化 SDK，可优先考虑其他方案。

**地理覆盖重点** 欧洲（特别是英国和爱尔兰）监管覆盖最为完善，在亚太和拉美地区主要通过 DCC 业务和合作伙伴覆盖。如业务集中在欧洲以外区域且需要当地持牌服务商，建议在签约前确认具体市场的合规安排。

**费用与合同条款** 所有 B2B 业务线采用定制化定价，无公开标准费率，需通过销售团队获取报价。在签约前，务必与多家服务商沟通，明确计费规则、服务边界和争议处理等关键条款。
