本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业基本档案**

以下内容基于企业公开材料、监管公示与行业媒体整理；仅供参考。

**📅****成立时间与主体：**

2011 年 6 月 16 日；母公司移卡有限公司 Yeahka Limited 在开曼群岛注册；中国运营主体深圳市移卡科技有限公司于当日成立；香港联合交易所主板上市公司；股票代码 09923.HK。

**🌍****全球布局：**

全球总部位于中国深圳；在香港、新加坡、美国设有办公室；分别作为国际金融中心、亚太业务枢纽与北美市场据点。

**🛡️****合规资质：**

  * **亚太** ：中国人民银行支付业务许可证；香港 MSO（货币服务经营者）牌照；新加坡 MPI（主要支付机构）牌照；日本信用卡号码等合约签订业务经营者注册

  * **北美** ：美国 FinCEN MSB（货币服务业务）注册；亚利桑那州 MTL（资金传输许可证）

  * **卡组织** ：已取得 Visa 会员资格；为 Mastercard NUCC 外卡收单首批合作伙伴；2024 年 10 月加入银联国际（UPI）成员机构


**📊****业务覆盖：**

覆盖中国、香港、新加坡、日本、美国及部分东南亚国家。

**🎯****行业渗透与应用场景：**

聚焦跨境电商、平台与市场、SaaS（软件即服务）与数字服务、连锁零售等；支持全球多币种收付款与换汇；API（应用编程接口）驱动的多方分账与批量下发；AI 结合支付的场景化运营。

## **二、全球化能力图谱**

###  （一）支付网络与区域布局

**💳****国际卡收单：**

具备 Visa 直连/会员能力；与 Mastercard-NUCC 外卡收单合作；加入银联国际网络；具体接入形态与路由以签约与对接网络为准；提供基于 AI 的智能路由以提升交易成功率与效率；日本子公司按行业安全标准建设，并通过相应合规审查。

**🔗****本地支付聚合：**

  * **东亚市场** ：整合大中华区支付方式（支付宝、微信支付、银联）；在日本获得资质后可直接接入本地扫码支付体系

  * **东南亚市场** ：通过投资企业富匙科技，网络延伸至新加坡、印度尼西亚、越南、马来西亚等；服务星巴克、无印良品、必胜客、New Balance、Levi's、KOI、Mr.Coconut 等品牌

  * **其他区域** ：全球化策略现阶段聚焦亚太与北美


**🔄****跨境清算路径：**

清算能力基于多国牌照与银行合作网络构建；具体对接与路由策略在商务合作与协议中明确。

**📍****区域枢纽：**

  * 🇨🇳深圳总部负责研发与运营

  * 🇭🇰香港负责资本市场与本地业务及 MSO 合规

  * 🇸🇬新加坡负责亚太拓展与 MPI 牌照运营

  * 🇺🇸美国负责北美市场准入与 MSB 与 MTL 合规


**📞****服务与语言：**

SLA（服务等级协议）以双方签署的合同为准。

### （二）合规与安全体系

**🛡️****牌照矩阵（主体资质验证）：**

  * 🇨🇳中国大陆：支付业务许可证

  * 🇭🇰香港：MSO 牌照

  * 🇸🇬新加坡：MAS（金融管理局）MPI 牌照

  * 🇺🇸美国：FinCEN MSB 注册；亚利桑那州 MTL

  * 🇯🇵日本：经济产业省信用卡号码等合约缔结业务者注册


**🏆****行业认证：**

日本子公司 Yeahpay Japan Limited 已通过 PCI-DSS（支付卡行业数据安全标准）Level 1 并取得 AOC（合规证明函）。

**👤****KYC 与 AML：**

引入 AI 与生物识别技术优化商户验证与风险管理；流程与第三方数据库对接规则在商业合作中提供。

**💰****客户资金管理：**

客户资金隔离（Safeguarding）条款及托管安排在商户协议中明确；遵循各持牌地监管要求。

## **三、技术能力清单**

###  （一）智能技术与架构

**🏗️****平台架构：**

采用私有云进行集中化资源管理与调配；支持高并发交易处理。

**🧠****风控引擎：**

AI 模型与规则引擎双栈；基于交易与行为数据持续训练；提供按需的商户自定义规则能力。

**📊****数据与洞察：**

商户后台整合支付处理、营销与数据分析；专用报表 API 与对账 API 的技术规格在合作后提供。

**🔌****开发者生态：**

通过开放 API 平台与约 5000 家 SaaS 合作伙伴连接；API 与 SDK（软件开发工具包）及沙盒等资源在商务合作确定后授权提供。

### （二）支付与结算效率

**⚡****交易处理性能：**

2025 年上半年披露的应用层支付日交易笔数峰值约 6000 万笔。

**⏱️****资金结算时效：**

提供定制化提现服务以匹配周转需求；不同支付方式与区域的结算周期（T+N）以合同约定为准。

**💰****资金管理技术：**

支持多币种交易、外汇与跨境汇款；可为平台型客户创建虚拟子账户；不提供资金池归集与自动化调度等专业财资功能。

## **四、计费结构与服务规则**

**📝****整体定价：**

面向中大型与企业级客户的定制化非公开模式；费率与费用根据交易规模、行业、区域与方式组合通过商务谈判确定。

**💳****国际卡收单：**

覆盖 Visa、Mastercard、银联国际；计费综合考虑跨境交易费与网络费等成本。

**🏦****本地支付：**

按不同市场供给本地化定价；区域能力成熟度决定费率竞争力。

**📤****提现与结算：**

提供定制化提现；加急等费用在商户协议中约定。

**💱****货币兑换（FX，外汇）：**

提供外汇服务；汇率与价差结构以协议为准。

**✨****增值服务：**

高级风控、合规支持、争议处理等在企业方案整体报价中体现。

**📌****小结：**

本章不含具体数值；一切以《商户服务协议》为准。

## **五、汇率管理与资金流转**

**📈****汇率机制：**

  * **资质** ：持有香港 MSO、新加坡 MPI、美国 MTL 等多地牌照，具备在多市场提供合规外汇兑换服务的资质条件

  * **风险管理** ：监控汇率波动以管理自有货币风险

  * **报价** ：提供实时汇率展示与透明定价；面向商户的基准与价差结构以协议为准；不提供远期锁汇等衍生品对冲


**🏦****多币种账户体系：**

  * **账户结构** ：以企业/平台账户为主；个人端开通范围视持牌地监管与产品边界而定；支持多币种收付款；为平台客户提供虚拟子账户以管理复杂分配

  * **资金调度** ：提供 API 驱动的多方分账与批量下发；收款后可对境内外供应商或合作方付款


**⏱️****到账与清算时效：**

在 80 余国具备本地服务能力；支持当日可用并可按需换成当地币种；不同来源、币种与区域的时效（SLA）以与上游机构协同后的合同条款为准。

**📌****小结：**

依托全球牌照布局，形成多币种收付兑与复杂资金分配的资金服务能力。

## **六、生态对接与服务能力**

###  （一）电商平台与独立站接入

**📦****平台回款：**

支持主流电商平台资金回款。

**🛒****独立站插件：**

通过开放 API 与生态伙伴对接 WooCommerce / Shopify / Magento 等；香港市场常见方式包括 Alipay CN、AlipayHK、微信支付、银联、Visa、Mastercard 等（具体开通以签约与对接为准）；与 Shopify、Magento 的集成多通过 SaaS 合作伙伴或定制化项目实现。

**🛠️****能力细项：**

支持收单、退款与快速结算。

### （二）平台与创作者经济

**📊****多方分账：**

API 支持后台配置与前台自动化；适配于供应链与佣金分润场景。

**💸****大规模出款：**

API 支持批量下发至银行账户或钱包。

**🏢****账户体系：**

支持为平台客户创建与管理虚拟子账户。

### （三）企业系统（ERP 与财税）集成

**🔌****集成策略：**

借助与约 5000 家 SaaS 合作伙伴的生态连接对接 ERP 与财务系统；必要时提供定制化集成。

### （四）服务交付与支持

**👥****实施与支持：**

由专属业务团队、解决方案顾问与渠道伙伴提供；不依赖公开自助渠道。

**📚****文档与资源：**

技术文档、实施指南与服务协议在商业合作启动后定向提供。

**📡****系统状态：**

重大事件与运维通报通过指定渠道向签约客户发布。

**📌****小结：**

以 API 能力与 SaaS 生态为核心，为电商与平台类客户提供深度集成的支付方案。

## **七、增值工具与行业方案**

###  （一）风控套件

**🛡️****能力集成：**

作为整体解决方案核心组件提供；非独立产品。

**🧠****技术核心：**

基于交易数据与 AI 的模型加规则双栈进行动态风险识别。

**⚙️****商户配置：**

行业模板与可配置项由专家团队按项目落地。

### （二）财资与资金管理

**💰****能力集成：**

多币种账户、分账与批量付款能力集成在整体方案中；非独立功能模块。

**🔄****核心能力：**

多币种收款、付款与货币兑换；API 自动化分配。

### （三）数据与对账自动化

**📊****能力集成：**

数据与对账为后台与 API 的基础能力；支持 API 与报表导出；提供采集与核验服务。

### （四）行业方案

**🌐****跨境电商与外贸：**

多币种收款与本地化对公结算与风控和汇兑一体化；突出一站式收款、付款与货币兑换。

**🏢****平台经济（供应链与佣金分账）：**

API 分账与批量下发加 KYC（了解你的客户）协作与供应商管理；支持全流程自动化。

**🍽️****连锁零售与餐饮：**

一站式收款、付款与货币兑换加AI 智能代理加营销自动化；富匙科技在东南亚的 AI 智能代理面向餐饮，为用户提供语音或文本咨询、下单与支付；基于行为数据优化推荐；支持创意营销自动生成。

**📋****附：快速落地清单****🔧****技术：**

  * 对接解决方案顾问

  * 获取授权API 密钥与沙盒访问

  * 按私有文档集成


**💰****财务：**

  * 确认定制费率与结算周期

  * 完成财务账套与币种映射


**🛡️****合规：**

  * 按清单准备 KYB 材料

  * 签署含 DPA（数据处理协议）的服务协议


**⚙️****运营：**

  * 与服务团队配置分账规则与风控策略




**📌****小结：**

将风控、资金与数据能力融入一体化方案；结合富匙科技在东南亚的 AI 应用，为零售、文旅与休娱等行业提供智能升级。

## **八、适配场景与注意事项**

###  （一）适配场景

  * 亚太市场深耕者

  * 平台型企业

  * 中大型出海企业

  * 合规优先型企业

  * 餐饮、零售、美容、汽车、奢侈品等垂直行业客户（提供前中后台一体的 SaaS 服务）




### （二）注意事项

  * ⚖️**合规** ：确认法律主体与牌照覆盖范围；确保与目标市场匹配

  * 🔧**技术** ：以定制化集成为主；需投入相应研发资源

  * 💰**资金** ：关注结算周期、汇率机制与提现规则

  * ⚙️**运营** ：明确客户经理与技术支持的职责边界；公开自助资源有限

  * 📈**发展阶段** ：更贴合具规模与技术能力的中大型企业；初创与即插即用诉求商户需评估匹配度




## **九、区域合规与适配方案**

###  （一）亚太地区（核心优势区域）

**🛡️****准入与合规要点：**

  * 🇨🇳中国大陆：支付业务许可证

  * 🇭🇰香港：MSO 牌照

  * 🇸🇬新加坡：MPI 牌照，可提供账户开立、境内外转账、商户收单等

  * 🇯🇵日本：完成信用卡号码等合约缔结业务者注册；通过 PCI-DSS Level 1 并取得 AOC（合规证明函）


**⚙️****运营与风控重点：**

整合支付宝、微信支付、银联等；具备按市场接入本地网络的能力。

**📋****上线清单：**

按目标市场提交 KYB 材料；签署本地数据与隐私协议。

### （二）北美地区（战略布局区域）

**🛡️****准入与合规要点：**

完成 MSB 注册与亚利桑那州 MTL；遵循联邦及州级监管要求。

**⚙️****运营与风控重点：**

适配以信用卡为主的交易与争议规则。

**📋****上线清单：**

提供满足 BSA（美国银行保密法，规范金融机构信息申报的法律）与 AML（反洗钱）要求的 UBO（最终受益人，实际控制企业的自然人）与业务材料。

### （三）欧洲、拉美及其他区域

**📍****定位与能力：**

现阶段聚焦亚太与北美；其他区域多依赖合作生态网络开展。

**🔍****适配评估：**

与 YeahPay 就合作伙伴网络的服务范围与合规保障进行评估。

**📌****小结：**

合规与牌照为全球化基石；亚太与北美具备准入优势。
