## **一、企业基本档案**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体：**

2017 年 9 月（VertoFX Ltd 在英国注册成立）；公司总部位于英国伦敦，是一家专注于服务新兴市场中小企业的金融科技公司。

**🌍****全球布局：**

全球总部位于英国伦敦；在尼日利亚拉各斯、肯尼亚内罗毕、美国纽约等地设有核心区域办公室，团队分布于全球多个市场。

**🛡️****合规资质：**

全球牌照体系：在英国持有 FCA 电子货币机构（EMI）牌照；在美国持有 FinCEN MSB 注册及多州货币传输许可证；在尼日利亚持有央行 IMTO 许可；在南非持有 FSP 许可证；在阿联酋获得 DFSA 授权。

**📊****业务覆盖：**

服务网络覆盖 **190+** 国家，支持 **49 种** 货币；核心优势为发达市场与新兴市场（尤其是非洲）之间的 B2B 贸易提供支付与流动性解决方案。

**🎯****行业覆盖与应用场景：**

深度聚焦跨境电商、金融科技、B2B 贸易等在新兴市场有业务的企业；提供多币种账户、全球支付、外汇交易（特别是流动性不足货币）及 Verto Atlas 以 API 为核心的嵌入式金融解决方案。

## **二、全球化能力布局**

### (一）支付网络与区域布局

**💳****国际卡收单与发行：**

战略定位：B2B 账户间支付与财资管理，而非传统的在线卡支付收单。

发卡能力：通过与 Monavate Limited 合作，发行依托 Mastercard 网络的企业卡，支持在线与线下消费。

**🔗****本地支付整合：**

非洲：推出直接向肯尼亚 M-Pesa 移动支付钱包付款的功能，深度适配东非市场的支付生态。

欧洲：通过与 Banking Circle 的合作，提供德国 IBAN，支持 SEPA 区域的收付款。

北美：支持美国的 ACH 与 Fedwire 支付网络。

亚太地区：支持向中国的企业账户进行人民币（CNY）付款。

**🔄****跨境清算路径：**

网络对接：支持 SWIFT 国际电汇，并直接对接 SEPA、ACH、Fedwire 等区域性清算网络。

### (二）合规与安全体系

**👤****KYC/AML：**

技术合作：与 Resistant AI 合作，通过自动化文档验证技术，显著提升了商户入驻审核效率。

流程机制：实施严格的反财务犯罪合规与 KYC 流程，对所有入驻企业进行官方文件核查与风险评级。

**💰****客户资金管理：**

资金隔离：作为受 FCA 监管的 EMI 机构，严格遵循客户资金隔离存管（Safeguarding）规定，将客户资金存放于授权银行（如 Banking Circle）的指定托管账户中，与公司自有资金完全分离。

## **三、技术能力清单**

### (一）智能技术与架构

**☁️****平台架构：**

平台基于 AWS 云基础设施及高弹性微服务架构搭建，技术策略侧重于通过 API 为企业客户提供底层金融服务能力。

**🛡️****风控引擎：**

采用 AI 与机器学习技术进行实时交易监控，以识别异常活动和潜在欺诈风险。

**🔧****开发者生态（Verto Atlas）：**

API 与 SDK：提供 Verto Atlas 以 API 为核心的嵌入式金融解决方案套件，核心为 RESTful 标准 API 接口。

开发工具：开放沙盒测试环境、API 文档及 Webhook 事件通知系统。

预置集成：公司战略聚焦 API 驱动嵌入式金融生态，与主流电商平台或 ERP 系统的预置插件集成并非其核心。

### (二）支付与结算效率

**⏱️****结算时效：**

时效分级：依照不同币种、支付渠道提供结算时效表，美国 ACH 通常 1–3 个工作日到账，Fedwire 当日完成，均设定有每日截止时间。

交易截止时间：对时效要求较高的交易设定有明确的每日截止时间，逾时则顺延隔日处理。

**💵****资金管理技术：**

多币种账户：多币种钱包支持 **49 种** 货币，并可通过 Banking Circle 等合作方提供虚拟 IBAN 及本地账户（覆盖 **12 个以上** 主流市场）。

自动化付款功能：提供全面的 Payouts API，支持批量与单笔国际付款、外汇转换与受益人的自动化管理。

## **四、计费结构与服务规则**

**📋****整体定价策略：**

采用基于功能模块的分级订阅模式（Tiered Pricing），包含免费版（Verto Free）及多个付费版本，并为大型企业提供定制化报价。

**💸****核心服务定价：**

订阅计划：免费版、轻量版、新兴市场版、企业版（定制报价）等分级订阅制。

转账服务：Verto 客户间钱包转账免费；本地付款服务；国际 SWIFT 转账服务；超出套餐额度的付款收取额外费用。

卡片服务：虚拟卡与实体卡管理费、ATM 取现费、拒付处理费、卡片闲置管理费等。

**💱****货币兑换（FX）：**

点差标准：根据客户计划等级收取阶梯式外汇点差，主流货币与稀有货币点差标准不同。

**✨****增值服务：**

其他服务：多币种账户管理；虚拟 IBAN 服务；API 集成服务等。

## **五、汇率管理与资金流转**

**📊****汇率基准与透明度：**

汇率来源：可提供接近银行间中间价（Mid-Market Rate）的汇率，并通过 WhatsApp 机器人等工具提供实时汇率查询。

透明度：采用 "基准汇率 + 固定百分比价差" 的透明定价结构。

**🔄****换汇与对冲工具：**

即期换汇：提供 Exchange Now 服务，支持实时按需货币兑换。

自动化换汇：提供 Auto Exchange 服务，允许用户设置目标汇率，系统自动监控并执行交易。

批量换汇：支持通过 API 进行 **49 种** 货币的批量外汇转换。

远期锁汇：提供汇率锁定服务以对冲风险。

**💵****多币种账户体系：**

账户功能：提供支持 **49 种** 货币的多币种账户。

虚拟账户（vIBAN）：通过与 Banking Circle 合作，为客户提供以客户名义开立的多币种虚拟 IBAN，支持在 SEPA 区域像本地企业一样收付款。

## **六、生态对接与服务能力**

### (一）核心业务平台集成

**⚙️****嵌入式金融（Verto Atlas）：**

API 平台：Verto Atlas 是一套以 API 为核心的嵌入式金融解决方案套件，提供统一 API，集成多币种账户、全球支付及外汇流动性，服务于金融科技公司、银行、电商与 SaaS 平台等，支持将 Verto 核心金融基础设施直接嵌入到第三方系统之中。

**📊****ERP / 财务集成：**

集成策略：公司以 API 为核心战略，通过开放接口为企业后台系统和第三方开发者提供基础金融服务能力，实现定制化集成。

**👥****供应商入驻：**

技术集成：与 Resistant AI 合作，采用 AI 驱动文档真伪识别和风险检测，自动化文档验证显著提升了供应商或客户的入驻审核效率，支持国际合规扩展。

### (二）服务交付与支持

**👨‍💼****专人支持：**

提供专门的技术对接支持，可预约会议与专家一对一沟通。

**📚****自助服务：**

提供 API 文档、技术指南与视频演示，提供帮助中心与开发者沙盒自助环境。

## **七、增值工具与行业解决方案**

### (一) 风控与合规套件

**🛡️****能力说明：**

风控与合规能力由平台基础服务统一提供，自动化流程贯穿商户入驻和日常交易风控。

**🧠****核心能力：**

整合 Resistant AI 的自动化文档造假检测技术，结合自身 AI 与机器学习模型进行实时交易监控，自动识别多格式文件真伪和异常，大幅提升国际合规速度与精度。

### (二) 财资与资金管理套件

**💼****核心功能：**

Verto 专门提供财资管理解决方案，包括多币种账户、虚拟 IBAN、汇率对冲与自动化换汇策略。

**💪****核心能力：**

提供独特的 FX 市场平台，支持 **49 种** 主流及流动性较弱的货币交易，有效解决新兴市场的汇率与流动性难题。

### (三) 行业方案

**🏪****B2B 市场平台适配方案组合：**

Verto Atlas API + 大规模付款 API + 多币种虚拟账户。

能力亮点：支持向全球 **100 个以上** 国家和地区的卖家和服务商批量付款，无需在每个市场建立本地法律实体。

**🏭****制造业 / B2B 贸易适配方案组合：**

FX 市场平台 + 汇率锁定 + 全球支付。

能力亮点：帮助企业有效管理在非洲等新兴市场的汇率敞口，通过本地化支付降低交易成本。

**✈️****旅游与酒店业适配方案组合：**

多币种商业账户 + **49 种** 货币管理 + 全球支付。

能力亮点：简化国际交易，为客户提供安全无缝的支付体验，实现快速、低成本的本地和国际资金收发。

## **八、适配场景与注意事项**

### (一) 适配场景

  * **🌍****深耕新兴市场、尤其非洲、中东北非与亚太地区的企业** ：尤为适合需频繁处理区域内高波动、流动性较弱货币的 B2B 贸易和制造业公司。

  * **🏢****需要嵌入式金融能力的平台型企业** ：如电商平台、物流 SaaS 平台、旅行平台、HR / 财务软件与金融科技公司，通过 API 可集成多币种账户、全球支付、实时汇率转化等财资能力于自身产品体系中。

  * **💼****寻求高级财资管理的跨国中小企业** ：适用于需要虚拟 IBAN、多币种资金池管理、远期锁汇、自动化汇率策略等工具优化全球资金效率及外汇风险对冲的运营企业。




### (二) 注意事项

  * 📊**业务范围** ：Verto 高度专注于 B2B 跨境支付与财资管理，不涵盖电商 C 端支付插件或订阅管理等零售及面向消费者（C 端）交易。

  * 💰**成本结构** ：平台免费版仅含基础功能，核心增值能力（如更优汇率、批量国际付款、企业专属支持等）集中于付费版本，企业需据实际交易量和功能需求综合评估使用成本。

  * 🔧**技术门槛** ：平台核心依托 API 嵌入式服务能力，充分利用其服务需企业具备一定的技术开发及系统集成资源。




## **九、区域合规与适配方案**

### (一) 欧洲（核心市场）

**🛡️****准入与合规要点：**

VertoFX Ltd 在英国持有 FCA 颁发的电子货币机构（EMI）牌照，严格执行客户资金隔离，业务运营符合 GDPR 等欧盟数据保护条例。

**🏢****本地化运营：**

与 Banking Circle 合作，深度整合 SEPA 支付体系，为客户提供德国本地 IBAN 账户服务，支持 SEPA 区收付款，账户以企业名义开立，实现本地合规与全球资金归集。

### (二) 北美（核心市场）

**🛡️****准入与合规要点：**

VertoFX Inc. 在 FinCEN 注册为货币服务企业（MSB），并持有多州货币传输许可证（MTL），业务严格遵循 BSA/AML 合规及相关反洗钱法规。

**🏢****本地化运营：**

深度整合美国本土 ACH 与 Fedwire 清算网络，支持企业便捷收付款与定制化资金管理。

### (三) 新兴市场（非洲 / 中东，核心优势区域）

**🛡️****准入与合规要点：**

分别持有尼日利亚央行（CBN）颁发的 IMTO 许可、南非 FSCA 颁发的 FSP 牌照、阿联酋 DFSA 授权，并在肯尼亚、坦桑尼亚等地通过与本地银行合作完成合规落地。

**🏢****本地化运营：**

深度整合 M-Pesa 等移动支付钱包生态和本地银行体系，聚焦解决非洲等高壁垒新兴市场的货币流动性与定制支付场景，专为流动性不足货币打造专业化 FX 市场平台。
