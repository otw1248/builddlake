本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

1994 年成立，注册名称 Servicio UniTeller, Inc.，注册于美国新泽西州，运营总部位于美国得克萨斯州奥斯汀。

**🏛️****股权结构与母公司背景**

2006-2007 年被 Grupo Financiero Banorte（墨西哥大型金融集团之一，在墨西哥证券交易所上市，BMV: GFNORTEO）全资收购，为 Banorte USA Corporation 旗下子公司。

**🌍****全球布局**

在马尼拉（菲律宾）设有办公室。2024 年 8 月完成 MORE Payment Evolution 收购，新增欧洲、南美洲、非洲等 **15 个** 国家的办事处与业务网点，直接持牌国家从 5 个增加到 **15 个以上** 。

**🛡️****合规资质**

  * **金融牌照** ：美国 FinCEN MSB 注册，**50 个州及华盛顿特区** ，共 **51 个** 司法管辖区（即各州和特区）持有 Money Transmitter License

  * **国际牌照** ：覆盖加拿大、墨西哥、菲律宾、危地马拉及拉美 11 国（阿根廷、巴西、智利、哥伦比亚、秘鲁等，通过 MORE 并购获得）、非洲（塞内加尔、尼日利亚），欧洲牌照申请中

  * **安全认证** ：PCI DSS 认证（支付卡行业数据安全标准）


**📊****业务覆盖**

支持 **120 多个** 国家和地区、**80 多种** 货币，接入 **5000 多家** 银行和移动钱包渠道，年交易处理规模（TPV）约 **170 亿美元** ，其中 120 多个国家的覆盖包含直接牌照运营和支付代理网络两种模式，具体覆盖范围以签约时点及合同清单为准。

**🎯****主要应用场景**

主要业务包括跨境汇款处理、B2B 企业跨境支付、白标汇款解决方案、本地收款（Payins）等，服务汇款公司（MTOs）、银行、零售商、金融科技公司等 **100 多家** B2B 合作伙伴。消费者端通过 uLink 品牌提供移动汇款服务。

## **二、服务项目与计费模式**

**📋****服务项目清单****🏢****B2B 支付网络服务**

  * 跨境收款处理

  * 跨境付款处理

  * 企业跨境支付

  * 白标平台定制

  * 零售代理网络

  * 本地收款/代收


**👤****B2C 消费者汇款服务**

  * 国际汇款（uLink/Remit2home）

  * 电子钱包服务

  * 预付卡服务（uLinkcard）


**📊****计费规则**

**B2B 产品** ：采用非公开的定制化定价，具体取决于交易量、目的地国家、支付方式、服务层级等因素。提供按笔收费（per-transaction）、月度订阅、交易量折扣等多种计费模式。

**B2C 消费者汇款** ：根据发送金额、目的地国家、支付方式、收款方式等因素采用差异化收费。电子钱包账户无设置费、月费、闲置费。

**具体费率以《商户协议》为准。**

## **三、支付网络与合规能力**

###  （一）支付网络与区域布局

**💳****支付方式覆盖**

整合多种支付形态，覆盖卡支付、银行转账、数字钱包、现金支付等：

  * **卡组织** ：Visa、Mastercard、JCB、UnionPay（银联）

  * **拉美本地支付** ：PIX（巴西实时支付系统）、SPEI（墨西哥银行间转账）、Boleto（巴西线下票据）、银行转账

  * **非洲本地支付** ：银行转账、移动钱包（尼日利亚、肯尼亚、乌干达、坦桑尼亚等）

  * **亚洲** ：银行转账（越南等），通过合作网络覆盖更多亚太市场

  * **国际清算通道** ：SWIFT（环球银行金融电讯协会）、SEPA（欧洲银行间即时转账系统）、ACH（美国自动清算中心）

  * **现金支付** ：智利、厄瓜多尔、秘鲁、萨尔瓦多、乌拉圭、加纳等国提供现金取款渠道


**🔄****跨境汇款路线**

MORE 并购后，汇款路线从北美-拉美、北美-亚洲主要汇款路线，扩展至欧洲-拉美、欧洲-非洲、北美-非洲、拉美内部等新路线。

### （二）合规与安全体系

**🛡️****反洗钱能力**

涵盖 eKYC（电子身份验证）、OFAC（美国海外资产控制办公室）筛查、实时交易监控等反洗钱环节。首席法务与合规官持有 CAMS（注册反洗钱专家）资质。

## **四、技术能力与风控体系**

**⚙️****核心技术特性****🔌****单一 API 架构：**

提供单一 API 接入全球支付网络，支持 P2P、B2C、C2B、B2B 多种交易类型，无需对接多个 API。

**⚡****实时交易能力：**

支持实时交易状态跟踪与报告、收款验证（确认收款人信息及支付通道状态）、外汇交易与实时汇率报价、自动换汇。

**🛡️****风控与合规技术**

  * **风控引擎** ：采用 AI 与机器学习模型，实现实时交易筛查、账户盗用防范、异常交易行为识别，并定期根据合作伙伴规模和业务复杂度进行风险评估

  * **合规自动化** ：eKYC 自动验证、OFAC 与制裁名单自动对接、可疑交易自动筛查

  * **欺诈零责任保障（Zero Fraud Guarantee）** ：UniTeller 按合作协议承担相应欺诈损失


**🔧****对接方式**

UniTeller 采用企业级 API 模式，技术文档通过商务合作渠道获取，主要面向金融机构和大型企业。提供代理商门户（Agent Portal）和合作伙伴门户（LRC Portal）。

## **五、合作生态与解决方案**

###  （一）合作伙伴网络

**🏦****金融机构合作：**

为 Western Union、MoneyGram、Wells Fargo 等提供处理服务，与 Visa 建立战略合作关系，与 Walmart、Remitly 等建立业务合作。

**🤖****技术合作：**

2021 年与 Thunes 合作，拓展在亚太和非洲市场的支付网络，通过 Thunes 网络接入 **552 家** 银行和 **28 个** 电子钱包。

**💳****卡组织合作：**

与 Mastercard 合作推出 uLinkcard 可充值预付卡，支持国际汇款、家庭预算管理等功能，为移民群体提供日常金融服务。

### （二）白标解决方案（Digital Link）

提供白标汇款平台，支持品牌定制和合规对接。

**💡****应用场景：** 如果您是银行或汇款公司，希望推出自有品牌的跨境汇款 APP，Digital Link 白标方案覆盖品牌定制、合规支持、技术对接等环节，无需自建支付网络。多米尼加 Banreservas 银行通过该方案推出 Remesas Reservas 品牌 APP，该案例从立项到上线约 2-3 个月，实际周期因项目复杂度而异。此外，Elektra（墨西哥零售商）、Remesas Bantrab（危地马拉银行）等也通过白标方案上线运营。

### （三）本地收款服务（Payins）

提供本地支付方式收款 API，覆盖拉美 **7 个** 国家（阿根廷、巴西、智利、哥伦比亚、墨西哥、秘鲁、厄瓜多尔）、非洲 **10 个** 国家（尼日利亚、肯尼亚、加纳等）、亚洲（越南），支持银行转账、现金、钱包等多种支付方式。

**💡****应用场景：** 如果您需要在拉美或非洲市场收取本地货币付款（如巴西 PIX、墨西哥 SPEI），Payins 服务提供单一 API 接入多国本地支付方式，无需逐国对接本地支付网络。

### （四）服务交付与支持

提供英语和西班牙语客户支持，覆盖美国、墨西哥、菲律宾地区，支持电话与邮件咨询。

## **六、区域合规与运营要点**

###  （一）北美

**🛡️****准入与合规要点**

  * 全美 Money Transmitter License 全覆盖

  * 遵循 BSA（银行保密法）/PATRIOT Act 反洗钱合规要求

  * FinCEN MSB 注册

  * OFAC 制裁交易筛查义务

  * 受 CFPB Regulation E（汇款转账规则）约束


**🏢****运营要点**

  * 各州金融监管机构年度报告义务

  * 消费者汇款需提供准确费用披露和差错处理流程

  * 大额交易建议设置分级风控规则




### （二）拉美

**🛡️****准入与合规要点**

  * **墨西哥** ：依托母公司 Banorte 联合运营，支持 SPEI 本地转账

  * **南美** ：通过 MORE 并购覆盖阿根廷、巴西、智利、哥伦比亚、秘鲁、乌拉圭等 11 国

  * **危地马拉** ：直接运营牌照


**🏢****运营要点**

  * 票据支付（Boleto 等）存在到账较慢、买家弃付等风险。建议配置到期提醒与催缴机制。

  * 各国税费规则差异大，对账时需区分税费和手续费的计算口径




**⚠️****拉美各国对汇款公司的牌照要求和外汇管制政策差异较大，跨国运营需逐国确认合规要求。**

###  （三）欧洲

**🛡️****准入与合规要点**

  * 欧洲牌照申请处理中，通过 MORE 并购覆盖 SEPA 支付网络

  * 需遵循 PSD2、SCA（强客户认证）、GDPR（数据隐私保护）等欧盟监管要求




**⚠️****强制 SCA 验证可能影响支付转化率 5-10%，需提前评估豁免策略（如小额交易免验证等）。**

###  （四）亚太与非洲

**🛡️****准入与合规要点**

  * **菲律宾** ：直接运营

  * **非洲** ：塞内加尔、尼日利亚持有 Money Transmitter 牌照

  * **亚太更多市场** ：通过合作网络覆盖




**⚠️****通过合作网络覆盖的市场，支付方式和到账时效可能受合作方网络状态影响，建议提前测试目标汇款路线的可用性和时效。**

## **七、适用场景与选型参考**

**🎯****UniTeller 的主要适用场景****🏦****汇款公司、银行与金融机构**

UniTeller 的核心客户群体是汇款公司（MTOs）、银行和金融科技公司等机构客户。其单一 API 接入 **120 多个** 国家的支付网络，支持多种交易类型。对于希望推出自有品牌跨境汇款服务的银行或 Neo-Bank，Digital Link 白标方案已有多家银行和零售商通过该方案上线运营。

**🌎****拉美汇款路线业务**

UniTeller 依托母公司 Banorte 背景和 MORE 并购后的拉美多国牌照，在北美-拉美汇款路线具有相对深入的覆盖，支持拉美主流本地支付方式及银行转账和现金取款。

**🏢****企业跨境付款与新兴市场本地收款**

Business Payments 产品支持多币种结算与本地支付方式对接，主要面向有拉美、非洲等新兴市场付款需求的企业（如供应商付款、工资发放等）。Payins 本地收款服务则提供 API 接入多国本地支付方式，覆盖拉美 **7 个** 国家和非洲 **10 个** 国家。

**📊****选型参考**

在选择跨境支付网络基础设施时，建议从以下维度进行综合评估：

**🎯****业务场景匹配度**

UniTeller 的定位是 B2B 跨境支付网络基础设施，服务 MTOs、银行、金融科技公司等机构客户。如果您是跨境电商商户，通常需要的是支付网关（Payment Gateway），而不是支付网络，可同时了解 Stripe、Adyen、ONERWAY 等方案。如果您是汇款公司或金融机构，需要接入全球支付网络，可同时了解 Thunes、TerraPay 等方案。

**📍****目标汇款路线覆盖**

UniTeller 在北美-拉美汇款路线具有相对深入的覆盖。如果您的业务同时涉及亚太或欧洲市场，建议同时了解对应区域有直接牌照的服务商，对比覆盖深度和到账时效。

**🔧****技术对接模式**

UniTeller 采用企业级 API 模式，需通过商务沟通获取技术文档和对接支持。如果您的团队偏好自助注册、文档公开的开发者平台，需提前了解接入流程和所需时间。

**💰****费用与合同条款**

签约前建议与多家服务商沟通，明确计费规则（按笔、订阅、交易量折扣）、外汇成本、合规覆盖、数据安全等关键条款，选择最符合业务需求的方案。
