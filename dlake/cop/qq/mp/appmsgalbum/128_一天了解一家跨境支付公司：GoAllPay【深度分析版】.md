## 一、费率体系全景解析

### 🔍 公开费率明细（特定场景）

跨境收款核心费率表（基于公开信息）

### 📌 费率定制化关键点

**非标准化定价模式：** 官网未统一公开 "标准版" 或分级套餐的各项费率明细，多币种、多国家、多支付通道等自助接入产品需商户联系销售获取定制化报价。

**影响费率的核心因素：** 商户所属行业与业务风险等级、月度 / 年度交易量与单笔金额、支付方式与通道、交易地域与币种、结算周期与币种、系统集成复杂度等。

**议价空间：** 大客户 / 企业级商户（交易量大、额度高）可议价享受较低费率，高风险行业因清算难度、风控成本高，通常费率较高。

## 二、汇率管理与资金流转

### 🔄 汇率机制特点

**基准汇率体系：**

  * 数据源：采用合作银行当日实时汇率（如人民币结算主要参照平安银行实时汇率）

  * 透明化优势：明确承诺 "无汇率加成、汇率完全公开透明"，无额外外汇手续费，区别于部分平台的隐藏点差模式

  * 特殊场景处理：特殊兑换规模或币种的点差方案需与销售团队个别磋商确认




### ⏱️ 到账时效分级

### 资金流转优化措施

  * **本地化清算布局：** 与本地银行和清算网络深度合作，在主流国家 / 地区通过直连、本地收单通道缩短资金流转链路

  * **智能路由系统：** 支付网关支持自动选择最快、最优的结算路径，减少中间环节

  * **实时追踪能力：** 商户可通过 API 和后台管理系统实时查询结算进度和每笔资金流向，启用后可自动接收到账提醒




## 三、合规架构与安全体系

### 🏛️ 资质矩阵

### 🛡️ 资金安全保障

  * **客户资金隔离机制：** 所有客户资金与公司自有资金独立隔离，存放于受监管银行的独立账户体系（如平安银行等），确保运营风险不影响用户资金

  * **合规流程管理：** 商户须通过 KYC（身份认证）、AML（反洗钱审查）流程，系统对触发风控规则的大额、频繁或异常交易进行拦截并发起追加身份核查

  * **跨境资金合规：** 遵循各国地区外汇管理与结售汇合规要求（如中国国家外汇管理局、香港金融管理局等），支持监管对账监控

  * **数据安全措施：** 支付全程采用 "银行级加密" 保护，通过 HTTPS（SSL/TLS）加密协议进行所有数据和交易信息的安全传输，实施分级权限管理




## 四、生态对接与服务能力

### 🧩 电商全链路整合

**平台直连能力：**

  * 已支持平台：包括 Shopify、Magento、WooCommerce 等电商独立站场景，HostBill 等主流 SaaS 购物车系统，以及 Wish 平台（官方支付合作伙伴）




**集成方式：**

  * **标准化 API/SDK 支持：** 提供在线支付 API、APP 支付 SDK，支持 Java、PHP、.NET、Python 等主流编程语言

  * **一键式插件：** 针对 HostBill、独立站商城类系统已发布 "一键激活" 插件，便于快速部署

  * **预构建支付网关模块：** 涵盖线上、App、小程序、线下扫码等多渠道场景，可直接嵌入网站或移动应用




**对接优势：** 实现 "一次对接，多渠道全覆盖"，兼容 30 + 支付方式，显著降低技术集成难度，部分渠道可一天内完成部署

### 🌍 本地化服务网络

**全球服务布局：**

  * **总部枢纽：** 中国上海，以技术研发、管理与结算清算为核心

  * **客户支持：** 官网及服务中心支持中英文服务，提供 7×24 小时客服邮箱、工单系统与在线留言窗口，注册、运营、开发等环节可通过专属客户经理获取支持




**本地化优势：** 设有本地运营团队和项目对接专员，为重点区域大客户提供本地合规、开发集成、上线辅导等专业技术服务

## 五、增值服务与行业方案

### 🛠️ 效率工具矩阵

  * **统一支付后台 / 管理面板：** 提供一站式支付管理，包含订单管理、资金结算、对账报表、退款操作等功能，支持多用户账号管理与权限分配

  * **智能路由系统：** 依据实时交易状态、支付通道健康度、成本和成功率等参数，自动选择最优链路

  * **基础风险控制工具：** 内置多层风控，包含黑白名单管理、风险交易拦截、异常交易实时告警、IP 地理位置识别与监控，支持自定义风险规则配置

  * **自动化对账与报表系统：** 全自动对账功能减少人工核算负担，可生成多维度财务与交易报表，支持定时生成和一键下载

  * **资金管理工具：** 支持自动结算配置、分周期 / 分批次结算、自动归集资金，支持一站式批量付款和灵活分账

## 六、适用场景与企业类型

### ✅ 高契合度客户画像

  * **跨境电商平台与独立站：** 出海中小企业、大型品牌电商、长尾独立站、区域跨境平台，需覆盖多国多渠道、多币种、合规快速结算

  * **跨国企业与大型 B2B/B2C 商户：** 跨区域 B2B、贸易服务、SaaS 平台，有分账、分币种归集需求

  * **数字娱乐与游戏行业：** 需要多国支付、高并发交易的游戏、数字内容、流媒体企业

  * **旅游、酒店、票务行业：** 票务预订、OTA、连锁酒店，需解决一站式结算、退改签资金管理问题

  * **重点拓展新兴市场的企业：** 需本地钱包接口和东南亚、拉美、非洲多币种适配的企业

  * **强需求风控与合规企业：** 法律敏感、高退款风险、需强 KYC/AML 的大型机构




### ⚠️ 落地操作建议

  * **资质准备：** 高风险行业或需加强合规监控的品类，需提前准备完善的资质材料，因清算难度、风控成本高，审核周期可能延长

  * **技术对接：** API 联调建议参考官方完善的文档，利用沙盒环境和代码示例进行调试，官方提供交互式 API 探索器便于自测

  * **费率协商：** 大客户 / 企业级商户可与销售团队协商定制化费率，建议提前整理交易量、业务类型等关键信息，以获取更优报价

  * **风险注意：** 明确禁止虚拟货币、赌博、烟草等行业，高风险行业需接受额外 KYC 及合规审查，需关注服务协议中的拒付管理、不活跃账户管理等潜在费用条款
