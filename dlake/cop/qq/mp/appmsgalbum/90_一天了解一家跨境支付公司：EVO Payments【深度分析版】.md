核心数据速览

## 一、费率体系全景解析

### 🔍 全流程费率矩阵

### 📌 成本优化关键点

**隐性费用豁免：** 无外汇手续费（FX Fee）、开户费，官网公示全费用项（对比传统银行 3% 的 "junk fee"）。

**小额交易友好：** 收款最低 1 美元 / 笔，支持 PayPal 余额免费提现（PCI DSS 年费 $150 / 商户含审计服务）。

**动态费率调整：** 月交易量≥1 万笔（北美）或单日 SEPA 清算≥500 笔（欧洲）可触发费率减免。

**技术接入成本：** 基础 API 接口免费开放，但需每年支付 $150 PCI DSS 合规年费，含 Sysnet 专业审计服务。

## 二、汇率管理与资金流转

### 🔄 三层汇率策略

**动态货币转换（DCC）系统**

  * 数据源：直连路透社、彭博终端，每 5 秒更新汇率

  * 技术原理：通过智能合约自动执行汇率对冲

  * 风险控制：当汇率波动超过 1.2% 时，系统自动触发二级审核机制，保障大额交易安全




**多币种资金池机制**

  * 功能：在 50 个市场开设本地账户，自动归集零散余额至主结算账户

  * 操作流程：支持手动设置归集阈值（最低 $100），或启用智能动态归集模式

## 三、合规架构与安全体系

### 🏛️ 资质矩阵

### 🛡️ 安全技术标准

**PCI DSS Level 1 认证：** 全链路 SSL/TLS 1.3 加密，卡号采用 Tokenization 替代存储，符合 PCI PTS 6.0 终端安全标准

**AI 反欺诈引擎：** 整合 5000 + 风险规则，实时分析 IP 属地 / 设备行为

**资金隔离机制：** 商户资金存放于 FDIC 承保银行（摩根大通 / 美国银行），与自有资金 100% 隔离，每日流动性覆盖率维持 110%

## 四、生态对接与服务能力

### 🧩 电商全链路整合

**API/SDK 接入：** 提供 Java/Python/PHP 开发工具包，支持订单创建、支付状态查询、退款处理等 23 项核心功能，通过 HTTPS 协议传输 JSON/XML 格式数据，联调周期平均 7 个工作日附沙盒测试环境，支持模拟 100 + 交易场景验证接口稳定性

**BRC 平台能力：** 自动匹配银行清算文件与商户账单，差异项高亮提示支持按海关 HS 编码导出结算报告，适配外管局申报格式

**ERP 系统集成：** 深度对接 SAP、Oracle 等主流 ERP 系统，实现订单 - 支付 - 库存数据实时同步

### 🌍 本地化服务网络

**欧洲节点：** 直连 SEPA 即时清算，单笔≤€50 万收费€0.02，单日≥500 笔费率降至€0.015与 KAL 合作推出 ATM 集成方案，欧洲经济区银行可 48 小时内部署 Visa/Mastercard 受理终端

**亚太创新：** 子公司 EVONET 与日本 KDDI 达成 MPM 模式合作，开通微信支付扫码受理功能，覆盖全日本 au PAY 商户，实现实时日元结算2025 年计划扩展至韩国、泰国市场，接入更多本地电子钱包

**北美优化：** ACH 网络 T+1 到账零附加费，月交易量≥1 万笔豁免延迟清算费与美国银行合作开设本地清算账户

## 五、增值服务与行业方案

### 🛠️ 效率工具矩阵

**业务资源中心（BRC）**

  * 实时交易监控：支持按日期 / 参考号 / 金额区间筛选交易，导出格式含 Excel/CSV/PDF提供自定义仪表盘，商户可拖拽配置核心指标（如交易成功率、拒付率）

  * 多币种资金池：支持 150 + 币种余额归集，零散资金自动整合至主账户支持设置资金池预警阈值

  * 电子发票管理：自动生成含交易明细的月度账单，保存期 36 个月，支持按欧盟 VAT 格式开具发票




**自动化合规工具**

  * PCI DSS 管理：在线提交三要素（PCI profile/SAQ/合规证明），系统自动校验文件完整性，年度到期前分阶段提醒

  * 牌照到期预警：监控全球 50 个市场牌照有效期，支持自定义邮件接收人

## 六、适用场景与企业类型

### ✅ 高契合度客户画像

#### 跨境零售企业：

  * **核心需求：** 年交易额≥300 万美元，需全渠道支付集成（如 3C、服饰类目卖家）

  * **典型案例：** 跨国服务平台：餐饮、酒店等需多币种动态定价的企业

  * **适配条件：** 技术型商户：支持 ERP 系统对接，B2B 应收账款自动化缩短回款周期




#### 非营利组织：

  * **核心需求：** 需定期捐赠管理的教育机构或公益组织，循环账单功能支持月/季/年周期捐赠




### ⚠️ 落地操作建议

#### 合规准备：

  * 欧盟业务需提前 2 个月完成 EMI 牌照备案，提交商户信息、资金托管方案等材料；日本市场需通过子公司 EVONET 合作，完成本地代理行 KYC 流程




#### 技术对接：

  * API 联调预留 3 周周期，重点测试异常交易场景（如退款、拒付）




#### 汇率策略：

  * 月流水超 100 万美元建议启用 DCC 系统建议设置汇率波动预警阈值（如 1.5%）




#### 资金管理：

  * 商户资金需存放于 FDIC 承保银行，每日关注流动性覆盖率（LCR）报告
