本文对跨境支付公司的界定：持支付牌照主营跨境收付款 / 换汇，或为该业务提供技术支撑的持牌服务商。

## **一、企业概况**

以下内容基于企业公开材料、监管公示与行业媒体整理，仅供参考。

**📅****成立时间与主体**

Paysend PLC 于 2010 年在苏格兰注册，2017 年推出全球卡对卡转账服务。总部位于英国伦敦，注册地址位于爱丁堡。

**🌍****全球布局**

伦敦为全球运营中心，设有多个区域办公室：贝尔格莱德（技术研发中心）、迈阿密（美洲总部）、都柏林（欧洲枢纽）；团队采用远程办公模式，分布于全球多个国家。

**🛡️****合规资质**

  * 金融牌照：英国 FCA EMI、爱尔兰 CBI EMI、美国 FinCEN MSB

  * 美国州牌照：覆盖 50 个司法管辖区（47 个州、华盛顿特区、波多黎各和美属维尔京群岛）

  * 卡组织资质：Visa/Mastercard/中国银联主要成员及认证处理商

  * 安全认证：PCI-DSS


**📊****业务覆盖**

支持从 **49** 个国家/地区发起转账，可向 **170+** 个国家/地区收款，覆盖银行卡、银行账户、电子钱包等多种收款渠道。

**🎯****主要应用场景**

聚焦个人跨境汇款，提供卡对卡转账、卡对银行账户、卡对电子钱包、银行对银行、手机号转账等多种方式；同时面向中小企业提供在线收单服务，面向银行与金融科技平台提供 API 支付解决方案。

**👥****用户规模与市场表现**

消费者用户 **1200 万+** ，2025 年入选 FXC Intelligence 全球前 100 跨境支付公司。

## **二、计费结构与收费规则**

**📋****定价模式**

采用固定费用模式（按笔收费），即无论转账金额大小，同一通道的转账费用固定。部分通道采用根据金额浮动的费率，具体通道以官方页面为准。

**📝****服务项目清单****💸****跨境汇款（C2C）**

按发送国家/地区收取固定转账费用，不同国家费用标准不同。

**💱****汇率加价**

在中间市场汇率基础上加收一定比例的汇率差价，加价幅度因通道而异。

**🏦****账户管理**

无账户月费。

**✨****附加服务**

  * ATM 取款：每月有一定免费额度，超出部分按比例收费

  * 跨币种支付：按比例收取货币转换费

  * 退款与拒付：按具体产品条款执行


**📌****费用项目补充说明**

固定费用模式在大额转账场景下单笔费用占比更低，小额频繁转账场景下需与按比例收费模式对比评估。

## **三、全球支付网络与合规体系**

###  （一）支付网络与区域布局

**💳****卡网络直连**

与 Visa、Mastercard、中国银联建立主要成员合作关系，具备认证处理商资质。通过 Visa Direct 和 Mastercard Send 实现向全球符合条件的银行卡实时转账。

**🔢****支持的卡类型**

  * 国际卡：Visa、Mastercard、Maestro

  * 本地卡：中国银联、Humo（乌兹别克斯坦）、Uzcard（乌兹别克斯坦）


**📱****支持的数字钱包**

  * 中国：支付宝、微信支付

  * 印度：UPI

  * 巴西：PIX（央行即时支付系统）

  * 全球：Apple Pay、Google Pay（用于充值转账）


**🏦****银行转账通道**

支持 SWIFT 国际电汇、SEPA 欧洲即时转账、PIX 巴西即时支付、UPI 印度统一支付接口等主流通道。

**📍****区域运营中心**

  * 🇪🇺欧洲：伦敦（全球总部）、都柏林（欧洲枢纽），受 FCA/CBI 监管，支持 SEPA 即时转账

  * 🇺🇸北美：迈阿密（美洲总部），覆盖 FinCEN MSB 合规与美国-拉美汇款通道

  * 💻技术：贝尔格莱德（技术研发中心）


**🗣️****服务与语言**

官网与 App 支持 **20+** 种语言，包括英语、中文、西班牙语、法语、德语、葡萄牙语、越南语、捷克语等。

### （二）合规与安全体系

**📜****牌照与认证**

  * 持有英国 FCA EMI、爱尔兰 CBI EMI、美国 FinCEN MSB 等牌照

  * 遵循 GDPR、DPA 2018 等数据与隐私规范


**🛡️****反洗钱能力**

通过第三方服务商提供 KYC/AML 能力，接入 OFAC/UN/EU/UK 制裁名单，提供实时交易监控、政治公众人物（PEP）筛查、可疑活动报告等功能。

**🔒****技术安全措施**

采用行业标准加密、多重身份验证与生物识别技术，符合 GDPR 规范。

## **四、支付时效与技术能力**

###  （一）到账时效

**⏱️****个人用户到账时效**

**90%** 的转账在 **15 秒内** 完成，**95%** 的转账在 **2 分钟内** 完成。实际到账时效取决于收款方式、收款国家、银行处理时间等因素：

  * 卡对卡转账：通常实时到账

  * 银行账户转账：取决于当地银行处理时间

  * 电子钱包：通常实时到账




**💡****应用场景：** 如果您需要向海外家人紧急汇款，卡对卡转账方式可实现实时到账，无需等待银行处理周期。

**🏢****企业用户到账时效**

根据 Paysend Enterprise 披露，**50%** 的支付在 **5 秒内** 完成，**99%** 在 **24 小时内** 完成。

### （二）技术架构

**📱****消费者端**

移动 App 支持 iOS 和 Android，提供转账前费用预览、重复转账、PDF 收据、实时通知等功能。

**🔌****企业级 API（Paysend Enterprise）**

面向银行、金融科技公司、支付服务商、零工平台等提供 RESTful API，支持 **170+** 通道、**120+** 币种，直通处理率（STP）达 **99%** 。

面向中小企业（Paysend Business）提供支付网关，支持交易结果实时通知。

### （三）多币种账户与资金管理

**💵****消费者多币种账户**

支持 **13** 种货币持有：GBP、EUR、USD、CNY、CZK、PLN、DKK、HUF、NOK、SEK、CHF、RON、BGN。用户可在 App 内查看各币种余额，进行货币兑换。

**🏢****企业级多币种能力**

Paysend Business 支持最多 **38** 种货币收款，Paysend Enterprise API 支持 **120+** 种货币。

**📊****汇率机制**

汇率参考中间市场汇率，实际加价幅度因通道而异，建议在转账前通过 App 确认。

**📈****交易限额**

采用分级限额机制，不同验证等级对应不同的日限额与月限额。完成更高级别身份验证可获得更高限额。

## **七、适配场景分析**

**✅****适合使用 Paysend 的场景****🌍****海外务工人员向家人汇款**

如果您在英国、美国、加拿大、澳大利亚等国家工作，需要定期向中国、印度、菲律宾、墨西哥等国家的家人汇款，Paysend 提供卡对卡即时转账，固定费用模式在大额单笔转账场景下单笔费用占比更低。

**💱****多国多币种资金管理**

如果您持有多个国家的银行卡或电子钱包，需要在不同货币之间灵活转换，消费者账户支持 13 种货币持有，可在 App 内统一管理。

**🌎****欧美向拉美汇款**

如果您从美国向墨西哥、危地马拉、多米尼加等拉美国家汇款，Paysend 与 Mastercard 在该通道有深度合作，支持即时转账。

**🏢****中小企业跨境收款**

如果您是中小型跨境电商或服务提供商，需要接受海外客户付款，Paysend Business 提供支付链接、在线收银台、WooCommerce 插件等集成方式，支持 38 种货币收款。

**💼****全球薪资发放与平台分账**

如果您是人力资源平台、零工平台或市场平台，需要向全球员工、自由职业者、卖家进行结算，Paysend Enterprise API 支持 170+ 通道、120+ 币种，直通处理率达 99%。

**🔌****跨境支付能力嵌入**

如果您是银行、金融科技公司或支付服务商，希望在自有产品中嵌入跨境支付能力，Paysend Enterprise 提供 API 和可定制品牌的嵌入式组件，支持快速集成。

**🤔****决策建议**

在选择跨境汇款或跨境支付服务时，建议从以下维度进行综合评估：

**🔄****转账通道匹配度**

根据您的发送国和接收国，对比不同服务商在该通道的覆盖情况、到账速度、费用水平。如果从美国向墨西哥汇款，Paysend 与 Mastercard 的合作通道具有到账速度优势；其他通道可同时了解 Wise、Remitly 等方案。

**💰****转账金额与频率**

如果您单笔转账金额较大（如 5000 美元以上），固定费用模式的优势明显；如果您小额频繁转账（如每次 100 美元以下），按比例收费模式可能更具性价比。

**📱****收款方式偏好**

如果收款人习惯使用银行卡收款，卡对卡转账是便捷选择；如果收款人使用支付宝/微信支付/UPI 等钱包，需确认目标通道是否支持该收款方式。

**🏢****企业级集成需求**

如果您需要将跨境支付能力集成到自有系统中，可评估 Paysend Enterprise API 的技术文档、支持的通道和币种、直通处理率和到账时效等指标，并与 Currencycloud、Nium、Thunes 等同类方案进行对比。
